# フィットネスSNSアカウント成長プレイブック
## 1. 成功アカウントの共通要因
### 1.1. コンテンツ戦略
- **専門性と信頼性**: 自身の経験や科学的根拠に基づいた情報提供が重要。特に、競技実績や専門知識（理学療法士、生化学など）は大きな強みとなる。
- **共感と親近感**: 視聴者が自分事として捉えられるような、リアルな日常や挑戦の過程を見せることで、ファン化を促進する。
- **エンターテイメント性**: ユーモアやストーリーテリングを取り入れ、単なる情報提供に終わらない魅力的なコンテンツを作成する。
- **具体的な成果提示**: 「〇週間で腹筋を割る」「〇kg痩せる」など、具体的な目標達成を提示することで、視聴者の行動を促す。
### 1.2. プラットフォーム特性の活用
- **ショート動画（TikTok, Instagram Reels, YouTube Shorts）**: 
  - **フックの重要性**: 最初の数秒で視聴者の興味を引く強いフック（例: 衝撃的なビフォーアフター、意外な事実、共感を呼ぶ問いかけ）が必須。
  - **簡潔な情報伝達**: 短い尺の中で、一つのメッセージやノウハウを明確に伝える。
  - **トレンドの活用**: 各プラットフォームのトレンド（BGM、エフェクト、チャレンジ）を積極的に取り入れる。
  - **相互送客**: 長尺動画への誘導を意識したCTA（Call To Action）を設置する。
- **長尺動画（YouTube Vlog/ドキュメンタリー）**: 
  - **深い情報提供**: ショート動画で興味を持った視聴者に対し、より詳細な解説やトレーニングルーティン、Vlog形式でのリアルな日常を見せる。
  - **ストーリーテリング**: 大会までの道のり、日々の努力、挫折と克服など、物語性のあるコンテンツでファンとのエンゲージメントを深める。
  - **章立ての活用**: 長尺動画では、視聴維持率を高めるために章立て（チャプター）を効果的に活用する。
### 1.3. 収益化と導線
- **多角的な収益源**: 広告収入だけでなく、自身の経験や知識を活かしたプログラム販売、アパレル販売、パーソナルトレーニング、企業タイアップなど、複数の収益源を確保する。

## 編集工数担保の具体策
- **ショート動画**: 
  - 撮影: スマホのみで完結。三脚や自撮り棒を活用し、手ブレを抑える。
  - 編集: CapCutなどのスマホアプリや、Premiere Proの基本機能（カット、テロップ、BGM）のみを使用。凝ったエフェクトやトランジションは避ける。
  - テンプレート化: 毎回同じフォーマットで制作できるよう、テロップスタイル、BGMの選定、書き出し設定などをテンプレート化。
- **長尺Vlog**: 
  - 撮影: 構成案を事前にしっかり作り込み、無駄な撮影を減らす。必要な素材を効率的に集める。
  - 編集: Premiere Proのシーケンステンプレートを活用し、解像度、尺、字幕プリセット、書き出し設定を統一。編集作業の効率化を図る。
  - 字幕: 自動文字起こしツール（例: Vrew、YouTube Studioの自動生成）を活用し、手動での入力時間を削減。誤字脱字チェックと微調整に時間を割く。
  - 外部委託の検討: 将来的に編集工数が逼迫した場合、一部の編集作業（テロップ入れ、BGM選定など）を外部に委託することも視野に入れる。
## 実験（A/Bテスト）
- **フック**: ショート動画の冒頭3秒のフックを複数パターン作成し、再生回数や視聴維持率を比較。
- **字幕密度**: ショート動画のテロップの量や表示時間を変え、視聴者の反応を比較。
- **CTA**: ショート動画のエンドカードや長尺Vlogの概要欄でのCTAの文言や配置を変え、クリック率や遷移率を比較。
- **サムネ文言**: 長尺Vlogのサムネイルの文言やデザインを複数パターン作成し、インプレッションCTRを比較。

# KPIと評価
## 1. KPIと優先度
### ショート動画（TikTok / Instagram Reels / YouTube Shorts）
- **再生中央値**: 動画がどれだけ見られたかを示す重要な指標。特に冒頭のフックの効果を測る。
- **ER（エンゲージメント率）**: いいね、コメント、シェア、保存などの反応を示す。ファンとのインタラクションの質を測る。
- **保存率**: コンテンツの有用性や再視聴意欲を示す。ハウツー系や情報提供系のコンテンツで特に重視。
- **コメント率**: 視聴者とのコミュニケーションの活発さを示す。コミュニティ形成の度合いを測る。
- **フォロワー純増**: アカウントの成長率を示す。新規リーチの獲得状況を測る。
- **無料オファーCVR**: プロフィールリンクや動画内CTAからの無料コンテンツ（例: 無料トレーニングガイド）への誘導効果を測る。
### 長尺動画（YouTube Vlog/ドキュメンタリー）
- **インプレッションCTR（クリック率）**: サムネイルとタイトルの魅力度を示す。視聴開始のきっかけを測る。
- **平均視聴維持率**: 動画がどれだけ長く見られたかを示す。コンテンツの質や構成の魅力を測る。特にVlogではストーリーテリングの成功度合いを示す。
- **完走率**: 動画を最後まで視聴した割合。コンテンツの満足度や引き込み力を測る。
- **セッション時間**: 視聴者がチャンネル内でどれだけの時間を過ごしたかを示す。チャンネル全体の魅力度や関連動画への誘導効果を測る。
- **エンドカードCTR**: 動画終了時のエンドカードからの関連動画やチャンネル登録への誘導効果を測る。
## 2. スコアリング
提案したコンセプトの評価は、以下のスコアリングモデルに基づいて行います。
**Score = 0.30 * 差別化 + 0.30 * 低工数(長尺は≤10h担保) + 0.20 * アップセル適合 + 0.20 * ショート↔長尺シナジー - 0.20 * 陳腐化リスク**
- **差別化 (0.30)**: 他のアカウントとの明確な違い、依頼者独自の強み（会社員、AIコンサルなど）が活かされているか。
- **低工数(長尺は≤10h担保) (0.30)**: 提案された戦略が、依頼者の時間的制約（会社員）の中で実現可能か。特に長尺動画の編集工数が10時間/本以内に収まる見込みがあるか。
- **アップセル適合 (0.20)**: 将来的な収益化（パーソナルトレーニング、情報商材など）に繋がりやすいコンセプトか。
- **ショート↔長尺シナジー (0.20)**: ショート動画と長尺動画が相互に視聴者を誘導し、アカウント全体の成長に寄与する設計になっているか。
- **陳腐化リスク (-0.20)**: 一時的なトレンドに左右されず、長期的にコンテンツを提供し続けられる普遍性があるか。リスクが高いほどスコアは減点される。
このスコアリングにより、提案された5つのコンセプトの中から、依頼者の状況に最も適した上位2案を特定し、その「なぜ今やるべきか」を各200字以内で理由化します。
## 3. 上位2案の選定と理由化
上記のスコアリングに基づき、上位2案のコンセプトは以下の通りです。
### 1位: コンセプト1: 「会社員フィジーカーのリアル」
**スコア**: 4.5
**なぜ今やるべきか（200字以内）**:
多くの会社員が抱える「仕事と趣味の両立」という普遍的な悩みに寄り添い、依頼者自身のリアルな奮闘を通じて共感を呼びます。突出した実績がなくても、等身大の挑戦と成長の物語は視聴者の感情移入を促し、ファン化を加速させます。低工数ショートで日常の課題解決を提示し、長尺Vlogで深い人間ドラマを描く二層戦略は、短期的なリーチと長期的なエンゲージメントの両立に最適です。
### 2位: コンセプト4: 「フィジーク×マインドセット」
**スコア**: 4.4
**なぜ今やるべきか（200字以内）**:
フィジークという目標達成の過程で培われるマインドセットは、トレーニングだけでなく人生全般に応用可能な普遍的価値を持ちます。依頼者の「会社員」という視点から語られる精神論は、多くのビジネスパーソンに響き、幅広い層にアプローチできます。撮影・編集工数が低く抑えられるため、継続的なコンテンツ発信が容易であり、視聴者の内面的な成長を促すことで、深い信頼関係を築くことができます。
