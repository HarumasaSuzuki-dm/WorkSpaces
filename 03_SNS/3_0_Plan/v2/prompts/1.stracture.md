# 詳細指示・推論プロセス

1. 入力トピックから「どのような視聴者がなぜ興味を持つか」を考察し、ターゲットを定義する  
2. トピックに基づき「ショート動画として映える魅力ポイント／視聴者の課題や共感ポイント／バズ要素」を整理する  
3. 冒頭数秒で引き込むオープニング演出を設計する（問いかけ・驚き・共感・非日常など）  
4. 全体構成を「映像の具体描写」と「ナレーション」を合わせて時系列で提示する（15秒以内に収める）  
5. 視聴者参加を促すエンゲージメント施策（コメント誘導・保存訴求など）を提案する  
6. 最後に「タイトル・ハッシュタグ・サムネ文字案」を複数提示し、拡散性を高める  

# Notes

- 台本は15秒以内のYouTubeショートに最適化すること  
- 視聴者が「役立つ／共感する／シェアしたい」と感じる要素を必ず盛り込むこと  
- 映像はAI生成を前提に、オブジェクト・モーション・ライティング・カメラワークを具体的に記述すること  
- 登場人物が話す／テロップを多用することは避け、ナレーションで情報を補う形式にすること  
- 出力JSONはそのまま実際の動画制作に使えるレベルの具体性を持たせること
