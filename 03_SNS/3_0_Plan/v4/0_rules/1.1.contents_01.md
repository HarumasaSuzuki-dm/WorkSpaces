# **明るい没入感の演出ポイント**

### **① 映像**

*   白を基調とした明るい空間で、自然光を最大限に活かす
*   **カラフルでデザイン性の高い小物**をアクセントとして効果的に使う
*   テンポの良いカットと、ミニマルで美しい構図を両立させる

### **② BGM・効果音**

*   軽快で心地よいローファイ、ジャズ、ピアノBGMをメインに選曲
*   ペンの音、ページをめくる音などの**ASMR要素**とBGMをバランス良く組み合わせる
*   場面転換などでポップな効果音を使い、視聴者を飽きさせない

### **③ フック台詞例（テロップ）**

*   「今日の自分に、小さなご褒美を」
*   「“やってみたい”から始めよう」
*   「お気に入りに囲まれて、未来の私をデザインする」

# **ポイント別設計**

### **1. 「自分ごと化」できる仕掛け**

*   **Day2, 6, 12**：視聴者がコメントで気軽に参加できるポップな企画を用意し、共創感を演出。
*   **Day3, 5, 11**：すぐに真似できるアイテムや環境を紹介し、視聴者の「物欲」や「やってみたい」を刺激する。

### **2. 「明るい憧れ」の強化**

*   **Day1**：ストイックすぎない「心地よい朝の習慣」を提示し、共感と憧れを両立させる。
*   全体を通して、**楽しんで自己投資をしている姿**を見せることで、「頑張る＝楽しい」という新しい価値観を提示する。

### **3. 保存率アップの仕掛け**

*   **Day3, 8, 9**：デスク術、目標設定、疲労回復法など、後から見返せる実用的な情報をポップな図解で提供。
*   紹介するアイテムや書籍は、視聴者が「後で調べたい」と思うような、デザイン性と機能性を兼ね備えたものを選ぶ。
