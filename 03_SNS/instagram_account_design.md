## 2. コンセプト

### コアメッセージ
> **「日常の中にある、まだ語られていない物語」**

### スローガン
> **「今日も、笑って、撮って、創っていく。」**

### コンセプト文
コンサルという「正解を出す仕事」から、写真という「感じたものを残す表現」へ。
大きな転換点に立っている今だからこそ撮れる視点がある。

東京の見慣れた風景、隣にいる人、何気ない一日。
それらを「映画のワンシーン」のように切り取ることで、自分自身の新しい物語を紡いでいく。

正解がない世界は、最初は怖かった。
でも正解がないから、自分で決めていい。
それって実は、めちゃくちゃ自由なことだった!

だから今日も、笑って、撮って、創っていく。

### キーワード
| カテゴリ | キーワード                       |
| -------- | -------------------------------- |
| トーン   | 明るい、軽やか、でも芯がある     |
| 視点     | 本質をつく、ハッとさせる、気づき |
| 場所     | 東京という舞台、近所、路地、窓辺 |
| 関係性   | 隣にいる人、二人の風景、共に歩く |
| 姿勢     | 挑戦、変化を楽しむ、自分で決める |

### ブランドパーソナリティ
- **トーン**: 明るい、ポジティブ、でも薄っぺらくない
- **話し方**: フランク、でも本質をつく、「それな!」と思わせる
- **姿勢**: 完璧を見せない、過程を共有する、怖いままやる

## 4. ポジショニング

### 競合との差別化

```
              作品志向・アート系
                    ↑
                    |
    アート系     　　　　　     
    フォトグラファー 　　　　
                    |
  ←─────────────────┼─────────────────→
  非日常・旅行      |           日常・東京
                    |
                ┌─────┐
                │ ここ │ ← 明るい × 本質 × 日常
                └─────┘
    インフルエンサー系   日記系アカウント
                    ↓
              カジュアル・記録志向
```

### 独自の強み（USP）
1. **明るいけど本質をつく** — 読んだ人が「確かに!」「それな!」となる
2. **キャリア転換中の視点** — 「怖いままやる」姿勢に共感
3. **二人の風景** — 顔を出さないことで、見る人が自分を投影しやすい

### ポジショニングステートメント
> 東京で暮らす、人生の転換期にいる一人のフォトグラファーが、
> 明るく軽やかに、でも本質をついた視点で、日常を切り取っていく。
> 見る人が「自分も何か始めてみようかな!」と思えるアカウント。

## 6. コンテンツ戦略

### コンテンツピラー（4本柱）

#### A. 東京の日常風景（メイン：50%）
| 項目         | 内容                                         |
| ------------ | -------------------------------------------- |
| 内容         | 近所の光、路地、窓からの景色、夕暮れ、雨の日 |
| 狙い         | フォロワー獲得、リーチ拡大、世界観の確立     |
| 頻度         | 週2投稿                                      |
| キャプション | 風景から得た「気づき」をシェア               |

**投稿例：**
- 朝の光が差し込む窓辺 → 「朝の光ってなんでこんなにやる気出るんだろう!」
- 雨上がりの路地 → 「雨の日、撮る側になると"待ってました!"になる」
- 夕暮れの電線と空 → 「毎日見てるのに、今日初めて気づいた」
- 駅のホームの何気ない瞬間 → 「みんな誰かの大切な人なんだよな〜」

#### B. 二人の風景（サブ：20%）
| 項目         | 内容                                         |
| ------------ | -------------------------------------------- |
| 内容         | 後ろ姿、シルエット、手元、並んで歩く足元     |
| 狙い         | エンゲージメント向上、ファン化、物語性の強化 |
| 頻度         | 週1投稿                                      |
| キャプション | 関係性から得た「気づき」、共感できるあるある |

**投稿例：**
- 並んで歩く後ろ姿 → 「"どこでもいい"が言える関係、一番贅沢!」
- カフェで向かい合う手元 → 「話さなくても気まずくない、これ大事」
- 夕日を見る二人のシルエット → 「同じもの見て"お!"ってなる瞬間」
- 二人分のコーヒーカップ → 「相手の好みを覚えてるって、それだけで愛」

#### C. 日常の気づき（サブ：20%）
| 項目         | 内容                                             |
| ------------ | ------------------------------------------------ |
| 内容         | 電車、カフェ、街中で感じたこと                   |
| 狙い         | 共感獲得、シェア促進、「この人面白い」と思わせる |
| 頻度         | 週1投稿                                          |
| キャプション | 本質をついた気づき、ハッとさせる視点             |

**投稿例：**
- 電車の車窓 → 「席譲って断られた。"譲る"は選択肢を渡すことなんだな」
- カフェの風景 → 「隣の人が愚痴ってた。その人も誰かの大切な人」
- 街中のスナップ → 「見てなかったんじゃなく、見ようとしてなかっただけ」

#### D. 自分自身の変化（アクセント：10%）
| 項目         | 内容                               |
| ------------ | ---------------------------------- |
| 内容         | 転職・新生活への心境、撮影への想い |
| 狙い         | 人間味、共感、長期的なファン獲得   |
| 頻度         | 月2-3投稿                          |
| キャプション | 正直な気持ち、でもポジティブに     |

**投稿例：**
- 新しい街の風景 → 「来月から新しい環境! 怖いよ普通に!笑 でもやる!」
- カメラを持つ自分の影 → 「正解がないって、最初は怖かった。今は楽しい!」
- 東京タワー → 「この街で過ごす日々、ちゃんと残しておきたい」

## 7. 投稿ガイドライン

### 投稿時間
| 曜日 | 推奨時間    | 理由                               |
| ---- | ----------- | ---------------------------------- |
| 平日 | 20:00-22:00 | 仕事終わりのリラックスタイム       |
| 土曜 | 11:00-13:00 | 昼のゆったり時間                   |
| 日曜 | 18:00-20:00 | 週末の終わり、新しい週への切り替え |

### 投稿形式の使い分け

| 形式                   | 用途                                   | 頻度  |
| ---------------------- | -------------------------------------- | ----- |
| フィード（単写真）     | メイン作品、世界観構築                 | 週3-4 |
| フィード（カルーセル） | シリーズもの、ストーリー性のある複数枚 | 週1   |
| リール                 | スライドショー、撮影風景、How-to       | 週1-2 |
| ストーリー             | 日常、リアルタイム、舞台裏             | 毎日  |

### トーン&マナー
| 要素           | 方向性                                             |
| -------------- | -------------------------------------------------- |
| 文体           | フランク、話し言葉、「〜だよね」「〜なんだよな〜」 |
| 句読点         | 少なめ、リズム重視                                 |
| びっくりマーク | 積極的に使う!                                      |
| 絵文字         | 控えめ（使っても1-2個）                            |
| 長さ           | 短め〜中程度（読みやすさ重視）                     |

### NGルール
- ❌ 彼女の顔が特定できる写真
- ❌ 勤務先・転職先の社名が特定できる情報
- ❌ ネガティブすぎる投稿（愚痴、批判）
- ❌ 他アカウントの批判・比較
- ❌ 過度な宣伝・セールス感
- ❌ 説教臭い言い方、上から目線

## 8. キャプション・文章テンプレート

### キャプションの基本構造

```
[一文目：キャッチーな入り or 情景描写]

[二文目以降：そこから得た「気づき」]

[最後：軽やかな締め or 問いかけ]

---
📍 [場所（任意）]

#ハッシュタグ
```

### カテゴリ別テンプレート

#### A. 東京の日常風景

**パターン1：気づき型**
```
いつもの駅のホーム、光の入り方えぐい!

毎日通ってるのに全然気づいてなかった。
見てなかったんじゃなくて、見ようとしてなかっただけなんだよな〜

カメラ持つと世界変わる!

---
📍 Tokyo

#東京 #日常風景 #気づき
```

**パターン2：対比型**
```
雨の日って憂鬱になりがちだけど、
撮る側になると「待ってました!」になる。

立場変わると、世界の見え方まで変わるの面白い!

#雨の日 #東京散歩 #写真好きな人と繋がりたい
```

**パターン3：シンプル感嘆型**
```
この光、やばくない?!

朝7時の窓辺、毎日こんなドラマチックだったのか...!

#朝活 #東京 #光
```

#### B. 二人の風景

**パターン1：会話想起型**
```
「今日どこ行く?」「んー、どこでもいい!」

これ言える関係、たぶん一番贅沢!
目的地より「誰と行くか」なんだよな結局。

#休日 #二人暮らし #東京デート
```

**パターン2：あるある共感型**
```
二人で歩いてるとき、同じもの見て「お!」ってなる瞬間ある。

言葉にしなくても伝わる何かがあるの、
なんかいいよね!

#カップル #何気ない日常
```

**パターン3：本質抽出型**
```
相手のコーヒーの好み、覚えてる。

それだけで十分「好き」の証明になると思う!

#カフェ #二人の時間
```

#### C. 日常の気づき

**パターン1：エピソード→気づき型**
```
電車で席譲ったら「大丈夫です!」って断られた。

最初ちょっとモヤったけど、よく考えたら
「譲る」って相手に選択肢を渡すことなんだよな。
受け取るも断るも、相手の自由!

押し付けたら親切じゃなくなる。写真も同じかも!

#日常 #気づき
```

**パターン2：観察→展開型**
```
カフェで隣の人が電話で怒ってた。

「なんでわかんないかな〜」って。
いや気持ちわかる!笑

でもさ、「わかってほしい」と「わかる」の間には
結構な距離があるんだよね。
自分も気をつけよ...!

#カフェ #考えごと
```

**パターン3：シンプル本質型**
```
写真の良いところは「言い訳がきかない」ところ!

撮ったものがすべて。
狙ったか偶然かも、見る人には関係ない。

結果で語る。仕事も人生も同じだな〜

#写真 #本質
```

#### D. 自分自身の変化

**パターン1：正直吐露型**
```
来月から新しい環境で働く!

「安定を手放すの怖くないの?」ってよく聞かれる。
怖いよ普通に!笑

でも「怖い」と「やらない理由」は別物だと思ってる。
怖いままやる、それでいい!

#転職 #新しい挑戦 #人生の転機
```

**パターン2：Before→After型**
```
前職、毎日「正解」を出す仕事だった。
写真には正解がない。だから最初めっちゃ怖かった!

でも今は「正解がない」が楽しい。
正解がないから、自分で決めていい。
これって実は、めちゃくちゃ自由なことだった!

#写真 #キャリアチェンジ
```

**パターン3：決意表明型**
```
今日も、笑って、撮って、創っていく。

大げさなことはできないけど、
毎日シャッター切ってたら、
気づいたら「物語」になってると思うんだよね!

#毎日投稿 #写真のある生活
```

## 10. 成長戦略

### フェーズ別戦略

#### Phase 1：基盤構築期（0-1,000フォロワー）
**期間**：1-3ヶ月

| 施策               | 詳細                            |
| ------------------ | ------------------------------- |
| 投稿の質と一貫性   | 週3-4投稿を死守、世界観を固める |
| プロフィール最適化 | 離脱されないプロフィールに      |
| ハッシュタグ研究   | 中小タグでの上位表示を狙う      |
| コミュニティ参加   | 同ジャンルへのコメント・いいね  |

#### Phase 2：成長加速期（1,000-5,000フォロワー）
**期間**：3-6ヶ月

| 施策             | 詳細                           |
| ---------------- | ------------------------------ |
| リール強化       | 写真スライドショー、How-to系   |
| コラボ           | 同規模アカウントとの相互紹介   |
| ストーリー活性化 | 投票、質問箱でエンゲージメント |
| 独自ハッシュタグ | #今日の東京 等を定着させる     |

#### Phase 3：確立期（5,000フォロワー〜）
**期間**：6ヶ月以降

| 施策                 | 詳細                         |
| -------------------- | ---------------------------- |
| 撮影依頼の受付本格化 | ポートフォリオ整備、料金設定 |
| オリジナルコンテンツ | プリセット販売、撮影講座等   |
| メディア露出         | Webメディア掲載、取材対応    |

### 日々のルーティン

**毎日（10-15分）：**
- ストーリー1本投稿
- 同ジャンル5アカウントにいいね＆コメント
- DMチェック・返信

**週1回（1時間）：**
- 投稿分析（インサイト確認）
- 翌週の投稿計画
- ハッシュタグ効果検証

**月1回（2時間）：**
- 月間振り返り
- 競合分析
- 戦略調整

## 補足：長期的なブランディング

### 新しい環境での継続

転職後も「個人の物語」として継続することで、新しい職場の「挑戦と物語を共に」という哲学の体現者になれる。
ただし、以下のルールを守る：

- ✅ あくまで個人アカウントとして運用
- ✅ 会社名・業務内容は直接言及しない
- ✅ 「新しい挑戦を始めた」程度の表現に留める
- ❌ 会社の宣伝・採用目的の投稿はしない

### 将来的な展開可能性

1. **撮影サービス**：カップルフォト、プロフィール撮影
2. **プリセット販売**：Lightroom用、自分の色味
3. **写真教室**：初心者向け、オンライン/オフライン
4. **書籍・ZINE**：東京の日常をまとめた写真集
5. **展示会**：ギャラリーでの個展

## 最後に

このアカウントは、ただの写真アカウントではない。

人生の転換期にいる一人の人間が、
明るく、軽やかに、でも本質を見失わずに、
「今日も、笑って、撮って、創っていく」姿を発信する場所。

見る人が「自分の日常も悪くないかも!」
「自分も何か始めてみようかな!」と思えること。

それがこのアカウントの存在意義。
