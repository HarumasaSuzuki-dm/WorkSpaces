## 2. コンセプト

### コアメッセージ
> **「日常の中にある、まだ言葉にされていない気づき」**

### スローガン
> **「今日も、笑って、撮って、創っていく。」**

### コンセプト文
コンサルという「正解を出す仕事」から、「感じたことを言葉にする表現」へ。
大きな転換点に立っている今だからこそ言える視点がある。

電車の中、カフェ、東京の路地裏。
何気ない瞬間に「それな！」と思える気づきを、短い言葉で切り取る。
時々、その気づきを写真と一緒に。

正解がない世界は、最初は怖かった。
でも正解がないから、自分で決めていい。
それって実は、めちゃくちゃ自由なことだった！

だから今日も、笑って、撮って、創っていく。

## 7. 投稿フォーマット

### 通常ポスト（メイン）

**パターン1：気づき型（最も多用）**
```
[具体的なエピソード/状況]

[そこから得た気づき・本質]

[軽やかな締め or 余韻]
```

例：
```
電車で席譲ったら「大丈夫です！」って断られた。

最初ちょっとモヤったけど、
"譲る"って相手に選択肢を渡すことなんだよな。
受け取るも断るも、相手の自由。

押し付けたら親切じゃなくなる。
```

**パターン2：対比型**
```
[一般的な見方/Before]

でも[視点を変えると/After]

[気づき]
```

例：
```
雨の日って憂鬱になりがちだけど、
撮る側になると「待ってました！」になる。

立場変わると、世界の見え方まで変わるの面白い。
```

**パターン3：シンプル本質型**
```
[一行で本質をつく言葉]

[補足 or 余韻]
```

例：
```
相手のコーヒーの好み、覚えてる。

それだけで十分「好き」の証明になると思う。
```

**パターン4：写真付き**
```
[写真]

[写真から得た気づき（短め）]
```

例：
```
[朝の窓辺の写真]

この光、やばくない？！
朝7時の窓辺、毎日こんなドラマチックだったのか。
```

### スレッド（週1-2本）

**用途：**
- 深い気づきの展開
- 自己紹介・価値観の共有
- ストーリー性のある内容

**構成例：**
```
1/5
転職を決めた時の話をします。

2/5
前職はコンサル。毎日「正解」を出す仕事だった。
でもある日気づいた。「正解を出す」ことに疲れてる自分がいた。

3/5
写真を撮り始めたのはその頃。
最初は「正解がない」ことが怖かった。
何を撮ればいいかわからない。これでいいのかわからない。

4/5
でもある日、「正解がない」=「自分で決めていい」って気づいた。
それって実は、めちゃくちゃ自由なことだった。

5/5
だから転職を決めた。
怖いよ普通に。でも「怖い」と「やらない理由」は別物。

怖いままやる。それでいい。
```

## 9. 拡散・エンゲージメント戦略

### リポストされやすい投稿の特徴
- **共感性が高い**：「わかる！」と思わせる
- **言語化が上手い**：モヤモヤしてたことを言葉にしてくれる
- **シェアしたくなる**：「これ、友達にも見せたい」
- **短くて伝わる**：一目で理解できる

### 日々のルーティン

**毎日（15-20分）：**
- 通常ポスト1-3本
- 同ジャンル5アカウントにいいね＆リプライ
- 自分の投稿へのリプライに返信
- タイムライン確認（ネタ収集）

**週1回（30分）：**
- スレッド1本投稿
- 投稿分析（インプレッション、リポスト数確認）
- 翌週の投稿ネタ整理

**月1回（1時間）：**
- 月間振り返り
- バズった投稿の分析
- 戦略調整

### エンゲージメント向上施策

| 施策               | 実施方法                                     |
| ------------------ | -------------------------------------------- |
| リプライで絡む     | 同ジャンルのアカウントに価値あるコメント     |
| 引用ポストを歓迎   | 引用されたら反応する、会話を広げる           |
| 質問を投げかける   | 「みんなはどう思う？」で双方向性を作る       |
| タイムリーな話題   | 季節、天気、時事ネタに軽く乗る               |
| 固定ポストを活用   | 自己紹介スレッドで新規フォロワーを獲得       |

### やらないこと
- ❌ フォロバ目的の大量フォロー
- ❌ 「いいねお願いします」等の直接的な要求
- ❌ 無関係なトレンドへの便乗
- ❌ 炎上への参戦

## 10. ハッシュタグ方針

### 基本ルール
- **使用数**：0-2個（多くても3個まで）
- **タイミング**：必要な時だけ（毎回つけない）
- **配置**：本文末尾に自然に

### 使用する場面
- 写真投稿時：`#ファインダー越しの私の世界` `#東京カメラ部`

### 使用しない
- 大量のハッシュタグ
- 内容と関係ないタグ
- `#フォロバ100` 等のスパム的タグ

## 13. リスク管理

### 炎上対策
- **発信前チェック**：「これ、悪意を持って読まれたらどう見える？」
- **政治・宗教・社会問題**：基本的に触れない
- **批判的な内容**：特定の人・企業を名指ししない
- **誤解されそうな表現**：補足を入れる or 投稿しない

### 炎上時の対応
1. 即座に反論しない（一呼吸置く）
2. 誤解であれば丁寧に説明
3. 自分に非があれば素直に謝罪
4. 必要に応じて投稿削除
5. 悪質な場合はミュート/ブロック

### プライバシー保護
- 彼女の顔が特定できる投稿はしない
- 会社名・業務内容は直接言及しない
- 位置情報はリアルタイムで出さない

このアカウントは、ただの「つぶやき」ではない。

人生の転換期にいる一人の人間が、
日常の中から「それな！」と思える気づきを発信する場所。

明るく、軽やかに、でも本質を見失わずに、
「今日も、笑って、撮って、創っていく」姿を見せていく。

見る人が「自分の日常も悪くないかも！」
「この人の視点、好きだな」と思えること。

それがこのアカウントの存在意義。
