### 前提条件
- 演者：斉藤様
- 編集者・撮影者：鈴木、辻、2名
- 作業時間：1〜3時間/日

### 対談・講義形式動画の制作ステップと工数の目安

- 作業時間：8時間〜18時間程度
- 日数：最短2日〜9日程度

| ステップ | 主な作業内容 | 工数の目安 | 備考 |
| :--- | :--- | :--- | :--- |
| **1. 企画・構成** | テーマ決め、話す内容の要点や流れをまとめる（台本作成） | **2時間～4時間** | 対談形式なので、一字一句の台本というよりは、話の構成やキーポイントをまとめた資料を作成するケースが多いです。専門的な内容のため、入念な情報収集やファクトチェックが必要になります。 |
| **2. 撮影準備** | カメラ2台（演者ごと）、マイク、照明のセッティング | **30分～1時間** | 演者が座る位置や画角を決めたら、撮影自体はスムーズに進むことが多いです。 |
| **3. 撮影** | 構成案に沿って対談を撮影 | **30分～1時間** | 動画の尺が12分〜18分なので、休憩や撮り直しを考慮しても、実際の撮影時間はそこまで長くなりません。一発撮りに近い形ならさらに短縮可能です。 |
| **4. 編集** | ①カット編集<br>②テロップ・図解挿入<br>③BGM・効果音挿入 | **合計 4時間～10時間** | **この形式の動画で最も時間がかかる工程です。**<br>① 話のテンポを良くするためのカット編集（2〜4時間）<br>② 要点をまとめたテロップや、簡単な図解の挿入（2〜5時間）<br>③ BGMや効果音は控えめなので、時間はかかりません（30分程度） |
| **5. サムネイル作成** | 動画のテーマが伝わるようなデザイン、テキスト配置 | **30分～1時間** | フォーマットがある程度決まっているため、比較的短時間で作成可能です。 |
| **6. 書き出し・公開設定** | 動画の最終チェック、YouTubeへのアップロード、概要欄作成 | **30分～1時間** | 専門的な内容を補足する情報を概要欄に記載する場合、少し時間がかかります。 |
| **合計リードタイム** | | **8時間～18時間** | 日数にして**最短2日〜9日** |

#### リードタイムの変動要因

  * **動画の尺と情報量:** 長尺の動画や、詳細な解説を多く含む場合は、編集にさらに時間がかかります。
  * **テロップの細かさ:** 視聴者が見やすいように、全ての会話にテロップをつけたり、図解を多く挿入したりすると、編集工数は増加します。

### シリーズ系（ドキュメンタリー）動画の制作ステップと工数の目安

  - 作業時間：**10時間〜30時間程度**
  - 日数：**最短2日〜15日程度**

| ステップ | 主な作業内容 | 工数の目安 | 備考 |
| :--- | :--- | :--- | :--- |
| **1. 企画・構成** | シリーズ全体のテーマ設定、各エピソードの進捗（トレーニング内容、食事、コンディションなど）、次回への引き作り | **2時間～4時間** | 日々の記録が中心となるため、大まかな流れを決める程度で、詳細な台本作成は不要な場合が多いです。 |
| **2. 撮影準備** | カメラ（複数台）、マイク、照明（必要な場合）、トレーニング器具の準備 | **30分～1時間** | ジムでの撮影が多いため、機材セッティングは比較的シンプルです。 |
| **3. 撮影** | トレーニング風景、食事風景、日常の様子、解説・コメント撮り | **1時間～3時間** | 実際にトレーニングしている時間に加え、様々なアングルからの撮影や、その場の思いつきでのコメント撮りなどが入ります。実際には1日での撮影を想定します |
| **4. 編集** | ①素材の取込・整理<br>②カット編集<br>③テロップ・字幕挿入<br>④BGM・効果音挿入<br>⑤インサート映像（回想シーンなど） | **合計 6時間～20時間** | **素材量が多いため、カット編集とテロップ挿入に時間がかかります。**<br>① 30分～1時間<br>② 3時間～8時間（複数日の素材を繋ぎ合わせる、テンポの良い編集）<br>③ 1時間～5時間（解説やコメントの聞き取り、文字起こし、テロップデザイン）<br>④ 30分～1時間<br>⑤ 30分～1時間（過去の映像や写真などを挿入する場合） |
| **5. サムネイル作成** | 動画の内容を象徴するインパクトのある画像、テキスト配置 | **1時間～2時間** | シリーズ内で統一感を持たせつつ、各エピソードの魅力が伝わるデザインが求められます。 |
| **6. 書き出し・公開設定** | 動画の最終チェック、YouTubeへのアップロード、タイトル・概要欄・タグの設定 | **30分～1時間** | シリーズ物として、過去の動画へのリンクなど、概要欄の充実も重要になります。 |
| **合計リードタイム** | | **10時間～30時間** | 日数にして**最短2日〜15日** |

-----

#### リードタイムの変動要因

  * **動画の尺と情報量:** 長尺の動画や、詳細な解説を多く含む場合は、編集にさらに時間がかかります。
  * **撮影素材の量:** 複数日分のトレーニングや食事、日常風景をまとめて編集する場合、素材の選定にも時間がかかります。
  * **テロップの細かさ:** 視聴者が見やすいように、全ての会話にテロップをつけたり、図解を多く挿入したりすると、編集工数は増加します。
  * **シリーズ構成の複雑さ:** ストーリー性を持たせるために、過去の映像をインサートしたり、特殊な演出を加えたりする場合は、企画・編集の工数が増えます。

### シリーズ系（トレーニング解説・コラボVlog形式）動画の制作ステップと工数の目安

  - 作業時間：**15時間〜40時間程度**
  - 日数：**最短3日〜20日程度**

| ステップ | 主な作業内容 | 工数の目安 | 備考 |
| :--- | :--- | :--- | :--- |
| **1. 企画・構成** | シリーズ全体の方向性、各エピソードのテーマ設定（トレーニング部位、解説ポイント）、雑談内容の準備、コラボ相手との打ち合わせ | **3時間～6時間** | トレーニングノウハウの解説、パーソナリティを出す雑談、コラボ相手との相乗効果を考慮した構成が必要です。 |
| **2. 撮影準備** | カメラ（複数台）、マイク、照明、トレーニング器具、コラボ相手とのスケジュール調整 | **1時間～2時間** | コラボ相手がいる場合、撮影場所や時間調整が複雑になることがあります。 |
| **3. 撮影** | トレーニング風景、解説、雑談、コラボ相手との対談、オフショット | **2時間～5時間** | 長時間のトレーニングを記録し、要所での解説や雑談を盛り込むため、撮影時間が長くなる傾向にあります。 |
| **4. 編集** | ①素材の取込・整理<br>②カット編集<br>③テロップ・字幕挿入<br>④BGM・効果音挿入<br>⑤インサート映像（解説図、回想シーンなど）<br>⑥コラボ相手のチャンネルとの連携 | **合計 8時間～25時間** | **動画の尺が長く、多角的な要素が盛り込まれるため、最も工数がかかります。**<br>① 1時間～2時間<br>② 4時間～10時間（トレーニングの動き、解説、雑談の流れを考慮した編集）<br>③ 2時間～8時間（専門用語の解説や、会話の聞き取り、テロップデザイン）<br>④ 30分～1時間<br>⑤ 30分～1.5時間<br>⑥ コラボ相手の動画への誘導や、共同制作の要素がある場合、その調整時間。 |
| **5. サムネイル作成** | 動画の魅力とパーソナリティが伝わる画像、テキスト配置（コラボ相手との写り込みも考慮） | **1時間～2時間** | 有名インフルエンサーとのコラボの場合、視認性とクリック率を高めるデザインが特に重要です。 |
| **6. 書き出し・公開設定** | 動画の最終チェック、YouTubeへのアップロード、タイトル・概要欄・タグの設定（コラボ相手のチャンネル紹介も含む） | **1時間～2時間** | 概要欄には、コラボ相手のチャンネルリンクや関連動画の紹介、使用機材のリンクなど、多くの情報を盛り込むことが推奨されます。 |
| **合計リードタイム** | | **15時間～40時間** | 日数にして**最短3日〜20日** |

-----

#### リードタイムの変動要因

  * **動画の尺と内容の密度:** 1時間近い動画であるため、編集作業は非常に膨大になります。解説の詳しさや雑談の量によっても大きく変動します。
  * **コラボ相手との連携:** スケジュール調整や動画の方向性、公開タイミングなど、コラボ相手との綿密な連携が必要となり、そのためのコミュニケーションコストが発生します。
  * **編集の凝り具合:** 雑談部分の面白さを引き出すためのエフェクトや、トレーニングフォームを分かりやすく解説するための図解など、細部にわたる編集作業は工数を大きく増加させます。
  * **パーソナリティの出し方:** 視聴者が親しみを感じるようなキャラクター設定や話し方を意識し、それを動画全体で一貫させるための工夫も、見えない工数として発生します。
