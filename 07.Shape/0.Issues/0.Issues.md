### **パーソナルジム経営戦略支援のための事前インプット情報：論点整理**

経営戦略を立案するにあたり、大きく以下の4つの視点から情報を整理・分析します。

1.  **マクロ・市場環境分析**：ジムを取り巻く外部環境の機会と脅威は何か？
2.  **顧客分析**：誰に、どのような価値を提供すべきか？
3.  **競合分析**：誰と、どのように戦うのか？
4.  **自社分析**：自社の現状の立ち位置と、強み・弱みは何か？

---

#### **第1章：マクロ・市場環境分析**

事業の成長機会や外部リスクを把握するための論点です。

| 大論点 | サブ論点 | 主な分析手法・調査項目 |
| :--- | :--- | :--- |
| **1. 市場規模とトレンド** | 1-1. パーソナルジム市場の規模、成長率、今後の予測 | ・業界レポート、調査会社のデータ分析 |
| | 1-2. 市場の最新トレンド（例：女性専用、食事指導特化、短期集中型 vs 継続型、オンライン化、AI活用、ウェルネス領域への拡大等） | ・業界ニュース、専門誌、競合の動向調査 |
| | 1-3. 異業種からの参入動向（例：フィットネスクラブ、IT企業等） | ・ニュース検索、プレスリリース分析 |
| **2. マクロ環境 (PEST分析)** | 2-1. **政治・法規制 (Politics)**：健康増進政策、広告規制（景品表示法等）、消費者保護関連法規 | ・官公庁の発表、法規制の確認 |
| | 2-2. **経済 (Economy)**：景気動向、可処分所得の変化、消費者の自己投資意欲 | ・経済指標、消費者動向調査 |
| | 2-3. **社会 (Society)**：健康・フィットネスへの意識変化、ライフスタイルの多様化（働き方、余暇の過ごし方）、人口動態（高齢化等） | ・各種統計データ、トレンド調査 |
| | 2-4. **技術 (Technology)**：トレーニング機器の進化、予約・顧客管理システムの普及、ウェアラブルデバイス、オンライン指導技術 | ・技術系ニュース、展示会情報 |

#### **第2章：顧客分析**

誰をターゲットとし、そのインサイトを深く理解するための論点です。

| 大論点 | サブ論点 | 主な分析手法・調査項目 |
| :--- | :--- | :--- |
| **1. 顧客セグメンテーション** | 1-1. どのような目的（ニーズ）を持った顧客層が存在するか？（例：ダイエット、ボディメイク、健康維持、機能改善） | ・既存顧客データ分析、アンケート調査 |
| | 1-2. どのような属性の顧客層が存在するか？（年齢、性別、職業、所得、居住エリア） | ・既存顧客データ分析、国勢調査データ |
| **2. ターゲット顧客の解像度向上** | 2-1. 現状のメインターゲット顧客層は誰か？ | ・既存顧客データ分析 |
| | 2-2. ターゲット顧客のペルソナ設定（ライフスタイル、価値観、情報収集方法、抱えている課題や悩み） | ・顧客インタビュー、アンケート深掘り |
| | 2-3. 顧客のKBF（Key Buying Factor：購入決定要因）は何か？（価格、立地、トレーナーの質、効果、口コミ、設備、施設の雰囲気） | ・顧客アンケート、失注顧客へのヒアリング |
| **3. 顧客単価とLTV** | 3-1. 平均顧客単価（コース料金）と入会金 | ・財務データ分析 |
| | 3-2. 平均継続期間と継続率／離反率 | ・顧客データ分析 |
| | 3-3. LTV（顧客生涯価値）の算出 | ・上記データからの算出 |

#### **第3章：競合分析**

競争環境を理解し、自社のポジショニングを明確にするための論点です。

| 大論点 | サブ論点 | 主な分析手法・調査項目 |
| :--- | :--- | :--- |
| **1. 競合の特定** | 1-1. **直接競合**：商圏内（例：駅、市区町村）の他のパーソナルジム（大手チェーン、個人経営） | ・マップ検索、Web検索、覆面調査 |
| | 1-2. **間接競合**：総合フィットネスクラブ、24時間ジム、ヨガ・ピラティススタジオ、オンラインフィットネスサービス等 | ・Web検索、業界分析 |
| **2. 競合の戦略分析** | 2-1. **提供価値**：各競合のコンセプト、ターゲット顧客、強み（例：価格、専門性、食事指導、立地） | ・競合ウェブサイト、パンフレット分析、口コミサイト |
| | 2-2. **サービス**：プログラム内容、料金体系、営業時間、設備 | ・同上 |
| | 2-3. **マーケティング**：Webサイト、SNS活用状況、Web広告出稿状況、チラシ、キャンペーン内容、口コミ・評判 | ・Webサイト分析、SNS分析、広告検索ツール |
| | 2-4. **推定される経営状況**：店舗数、トレーナー数、顧客の入り具合（覆面調査等から） | ・公開情報、現地調査 |

#### **第4章：自社（クライアントジム）分析**

戦略の実行可能性を判断し、活用すべき資産を洗い出すための論点です。

| 大論点 | サブ論点 | 主な分析手法・調査項目 |
| :--- | :--- | :--- |
| **1. 経営・財務状況** | 1-1. 経営理念、ビジョン、事業目標 | ・経営者ヒアリング |
| | 1-2. **主要KPI**：売上、利益、会員数、新規入会数、継続率、体験からの入会率、店舗稼働率、顧客単価 | ・財務諸表、顧客管理データ分析 |
| | 1-3. **コスト構造**：変動費・固定費の内訳（家賃、人件費、広告宣伝費、減価償却費等） | ・財務諸表分析 |
| **2. 提供価値 (バリューチェーン)** | 2-1. **サービス**：トレーニングプログラムの独自性・効果、食事指導の質、トレーナーの専門性・指導力・接客スキル | ・経営者・トレーナーヒアリング、顧客アンケート |
| | 2-2. **設備・施設**：トレーニングマシンのラインナップと質、施設の清潔感・デザイン性、アメニティの充実度 | ・現場確認、顧客アンケート |
| | 2-3. **立地**：商圏のポテンシャル、アクセスの良さ | ・地図分析、周辺環境調査 |
| **3. マーケティング・営業** | 3-1. **集客チャネル**：各チャネル（Web広告、SNS、紹介、チラシ等）の効果測定（問合せ数、CPA、入会率） | ・広告データ、アクセス解析データ分析 |
| | 3-2. **Webプレゼンス**：Webサイトの質、SEO順位、SNSアカウントの状況（フォロワー数、エンゲージメント） | ・Webサイト分析ツール、SNS分析 |
| | 3-3. **ブランド**：市場での認知度、顧客からの評判・口コミ | ・口コミサイト分析、顧客アンケート |
| **4. 組織・オペレーション** | 4-1. **人材**：トレーナーの採用基準、育成・研修制度、評価制度、定着率 | ・人事担当者・トレーナーヒアリング |
| | 4-2. **業務プロセス**：予約・決済システム、顧客管理方法、カウンセリングから入会までの流れ | ・業務フローの可視化、担当者ヒアリング |
