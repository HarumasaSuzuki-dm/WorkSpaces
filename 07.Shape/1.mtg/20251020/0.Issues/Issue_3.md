以下に「初回MTG」のための質問リストと議論の流れ（アジェンダ案）をまとめました。
第1・第2論点で洗い出した強み・課題・疑問点をもとに作成していますので、仮説検証と深掘りの双方を意識した構成になっています。

---

## 質問リスト（重要度の高いもの10問）

1. **強みの仮説検証**

   * HPからは「初心者が日本一来やすいジム」を掲げ、トレーナーとの距離が近いことやコミュニティ作りを重視していると感じました。御社が最も大事にしている価値観は何でしょうか？
   * SNSを活用した集客が印象的です。集客面で最も成功しているチャネルは何か、その理由は何でしょうか？
2. **サービス内容と方針の確認**

   * 現在はローンや分割払いに対応していないとのことですが、あえて対応しない理由や背景を教えてください。
   * 返金保証や途中退会時の対応について、今後導入予定はありますか？導入する場合はどのような条件を検討していますか？
   * 食事指導や健康サポートをどの範囲まで提供していますか？今後、管理栄養士や医師との連携を強化する予定はありますか？
3. **ターゲットと拡大戦略**

   * 現在主なターゲットは20〜40代の初心者と推察していますが、今後シニア層やリハビリ需要への対応を拡大する考えはありますか？
   * 現在の店舗数や拡大ペースはどうなっていますか？将来的な出店計画やフランチャイズ展開の構想をお聞かせください。
   * Shape Fit Festivalなどのイベントがブランド戦略や収益にどのように寄与しているか、今後の展望を教えてください。
4. **技術・サービスの展望**

   * AIやオンラインプログラムなど、テクノロジーを活用したトレーニングサービスの導入は検討中でしょうか？導入する場合、どのような体験を提供したいと考えていますか？
5. **本音・背景の深掘り**

   * 今回、外部コンサルタントに相談しようと考えたきっかけや背景は何でしょうか？達成したい目標や解決したい課題を率直にお聞かせください。

## 議論の地図（アジェンダ案）

| 時間配分    | 議題            | 目的                                                                        |
| ------- | ------------- | ------------------------------------------------------------------------- |
| **5分**  | イントロダクション     | 参加者の紹介、ミーティングの目的や流れを共有する。                                                 |
| **10分** | ビジネスモデルの理解確認  | こちらで整理した提供価値・顧客像・収益モデルの仮説を提示し、認識にギャップがないか確認する。                            |
| **15分** | 強み・独自性の深掘り    | SNS活用やコミュニティ作り、低価格で完全個室などの強みがどのように生まれたか、実感している効果や課題を伺う。                   |
| **15分** | 潜在的な課題と疑問点の検証 | 支払い方法、返金保証、医療連携の有無、設備面、テクノロジー導入、採用・出店戦略など情報が薄い点について意見を伺い、課題意識や計画を共有いただく。  |
| **10分** | 将来の展望と相談事項    | シニア層への拡大、オンラインサービス、Shape Fit Festivalの活用、他事業展開など将来ビジョンを伺い、支援できるポイントを洗い出す。 |
| **5分**  | まとめ・次回アクション設定 | 本日の議論を振り返り、優先的に深掘りすべきテーマや次回までの宿題を整理する。                                    |

### 補足

* 質問は仮説の検証（Yes/No）と深掘り（Why/What/How）を意識して設計しました。
* 議論の流れは60分程度を想定し、時間配分は目安です。社長の話が盛り上がりそうな部分には余裕を持たせてください。
* 当日の議論を可視化するため、簡単なホワイトボード図やメモを用意すると良いでしょう。
