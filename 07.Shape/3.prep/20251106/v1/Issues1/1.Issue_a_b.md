# 1. 収益性（どれだけ儲かったか）

## 1-1. 売上高の成長性と構成

* **上半期（～8月）売上高合計：68,650千円（実績）／84,713千円（予算）で予算比▲16,063千円。** 
* **事業別の売上構成（上半期）**

  * SFF：32,347千円、YouTube：18,780千円、フィットネス：11,836千円、FC収入：3,678千円、アパレル：2,009千円。合計の内訳として表記あり。 
    → **構成比ざっくり**：SFF 47%、YouTube 27%、フィットネス 17%、FC 5%、アパレル 3%（小数点四捨五入；68,650千円に対する比率）。

> 備考（前年比較）：表には**前年実績列**が設けられていますが、上半期累計の前年値の記載が確認できないため、**本アウトプットでは予算比中心**に評価しています。該当表の列見出しは確認可能です。

## 1-2. 利益率（粗利・営業・経常）

* **売上原価率（上半期）：27.8%**（＝粗利率72.2%）。 
* **売上総利益（上半期）：49,572千円（実績）／84,078千円（予算）**で予算比▲34,506千円。 
* **販管費合計（上半期）：45,036千円（実績）／80,210千円（予算）**で予算比▲35,174千円（＝想定より使っていない）。 
* **営業利益（償却後・上半期）：4,274千円（実績）／2,952千円（予算）** → **予算超過＋1,322千円**。**営業利益率：6.2%（実績）／3.5%（予算）**。 
* **経常利益（上半期）：5,141千円（実績）／3,870千円（予算）** → **予算超過＋1,270千円**。 

**要点**：
売上は予算未達（▲16,063千円）だが、**販管費の抑制（特に業務委託費の大幅未執行）**により粗利から販管費を差し引いた結果、**営業・経常ともに予算を上回る黒字**を確保。

## 1-3. 事業セグメント別の収益性（概観）

* 資料には**全社**P/Lの他、**フィットネス事業P/L／YouTube事業P/L**の個別様式があり、月次～四半期の**勘定科目別明細**までブレイクダウンされます（該当様式の見出し）。  
* ただし、当該抜粋範囲では**“事業別の営業利益額（累計）”の確定数字**は読み取れず（見出し・項目は確認できるが数値セルが未抽出箇所あり）。**全社では営業黒字**（上記の通り4,274千円）。

# 2. 費用構造（何にお金を使っているか）

## 2-1. コスト構造の把握（損益分岐点の目安）

* **上半期 粗利率＝72.2%**（= 1 − 売上原価率27.8%）。 
* **販管費（固定費に近い費目の集合と仮定）：45,036千円**。 
* この前提での**損益分岐点売上** ≒ **45,036 ÷ 0.722 = 約62,377千円（上半期）**。実績売上68,650千円はこれを**上回っているため、営業黒字**と整合。
  （計算根拠：上記粗利率・販管費。※厳密には固定費/変動費の切り分けが必要。）  

## 2-2. 主要コストの規模と比率（上半期）

* **人件費計：18,008千円（実績）／16,456千円（予算） → ＋1,552千円超過。人件費率：26.2%（実績）／19.4%（予算）。** 
* **地代家賃：3,883千円（実績）／3,570千円（予算） → ＋314千円。** 
* **業務委託料：17,474千円（実績）／49,023千円（予算） → ▲31,549千円（大幅未執行）。** 
* **広告宣伝費：1,797千円（実績）／1,926千円（予算） → ▲129千円。** 
* **支払手数料：358千円（実績）／3,170千円（予算） → ▲2,812千円。** 

**示唆**：

* 予算に対して**人件費は超過**、一方で**業務委託費・支払手数料は大幅に未執行**。この**費用ミックスの変化**が、売上の予算未達をカバーして**営業黒字を確保**した主因。

# 3. 追加の着眼点（全社P/Lの月次推移を踏まえて）

* **6–8月の各月**は、売上・販管費のブレが大きく、特に**業務委託費の月次実績が予算を大きく下回る**推移（Q2累計でも▲24,257千円）。**季節要因/SFFイベントの実施タイミング**を前提とした運用予算の見直し余地があると見ます。 
* **経常利益は上半期で予算超**（+1,270千円）だが、**雑収入・受取利息等の営業外も軽微**で、黒字は**本業のコストコントロールに依存**。下期に業務委託費を執行する場合、**利益率は低下**するリスク。 

---

# 4. “いま”判断できること（ポートフォリオ視点の素地）

* **売上はSFF/YouTubeがコア（計74%）**。この2事業の**計画・実施タイミング**が全社P/Lを大きく左右。 
* **費用面では人件費が増、委託が減**という**内製化寄りの実行**が黒字化に寄与（上半期時点）。  
* **短期的には黒字確保**（営業6.2%）だが、**予算未執行費目の下期執行**が起きれば利益は圧迫。下期の**SFF開催計画・制作外注計画**の実行有無で着地が変わる。
