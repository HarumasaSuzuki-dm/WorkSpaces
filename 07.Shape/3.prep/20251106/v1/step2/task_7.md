# 🧭 Shape Fit社：戦略的レバーマップ（定性 × 定量）

> 目的：
> 「理念（ビジョン）」を“どの数字”で実現するのかを明示し、
> 11/6 MTGで“経営を動かすポイント”を一目で語れるようにする。

---

## 🌱 全体構造イメージ

| レバー領域 | 現状（定量） | 課題（定性） | ビジョン方向A（理念寄り） | ビジョン方向B（効率寄り） | 財務影響／KPI |
| ----- | ------ | ------ | ------------- | ------------- | -------- |

---

### ① 広告宣伝費（マーケティング投資レバー）

| 観点                        | 内容                                                   |
| ------------------------- | ---------------------------------------------------- |
| **現状（定量）**                | 広告宣伝費：305千円（売上比 0.4%）<br>CPAやROAS未管理。集客は主にYouTube依存。 |
| **課題（定性）**                | 「ブランド認知を広げるべきか」「既存顧客深耕に集中すべきか」戦略不明確。                 |
| **ビジョン方向A：理念重視（ブランディング）** | 広告投資を3倍（900千円規模）に増やし、「SFFブランド」浸透を狙う。認知・共感重視。         |
| **ビジョン方向B：効率重視（CPA最適化）**  | 広告投資は抑制し、データドリブンに運用。CPA/ROASをKPI化してROA改善を狙う。         |
| **影響指標**                  | 売上成長率＋10〜15%、CPA、広告ROI、ROA改善＋1〜2pt                   |

---

### ② 人件費・委託費（組織構造レバー）

| 観点                        | 内容                                   |
| ------------------------- | ------------------------------------ |
| **現状（定量）**                | 人件費：20,447千円＋委託費：24,300千円＝販管費の約88%。  |
| **課題（定性）**                | 外注依存度が高く、固定費構造が硬直化。「成長しても利益が残らない」体質。 |
| **ビジョン方向A：理念重視（内製・人材共感）** | ビジョン共感型の社員採用を強化。報酬を「顧客変容連動」に設計。      |
| **ビジョン方向B：効率重視（外注・成果連動）** | 委託を成果報酬化し、変動費化。短期利益率を優先。             |
| **影響指標**                  | 労働分配率、営業利益率（改善＋2〜5pt）、固定費比率、社員定着率    |

---

### ③ SFF事業投資枠（成長ドメインレバー）

| 観点                       | 内容                                                    |
| ------------------------ | ----------------------------------------------------- |
| **現状（定量）**               | 売上32,347千円／営業利益19,466千円／利益率 約60.2%。<br>東京・大阪・福岡など大会拡大中。 |
| **課題（定性）**               | 人的・金的リソースが集中し、他事業（GYM・FC）への展開余地を圧迫。                   |
| **ビジョン方向A：理念重視（ブランド軸化）** | SFFを“内面変容の体験ブランド”に据える。大会価値を上げ、共感型コミュニティへ。             |
| **ビジョン方向B：効率重視（自走・分社化）** | SFFを分社化／自走化し、GYM・FCへの送客装置に位置づけ。                       |
| **影響指標**                 | 売上構成比、事業別ROA、SFF大会ROI、ブランド認知率                         |

---

### ④ YouTube活用（メディアレバー）

| 観点                        | 内容                                      |
| ------------------------- | --------------------------------------- |
| **現状（定量）**                | 売上20,329千円、営業利益5,211千円（利益率 約25.6%）。<br>案件単価下落傾向、送客貢献度不明。 |
| **課題（定性）**                | 収益事業か集客チャネルかが曖昧。ビジョン（発信の意義）との接続弱い。      |
| **ビジョン方向A：理念重視（発信＝価値創造）** | YouTubeを「理念発信装置」と定義。収益よりも“世界観の共有”を重視。   |
| **ビジョン方向B：効率重視（マーケ連携）**   | YouTubeをCPA改善装置としてKPI管理。送客効率と単価を最適化。    |
| **影響指標**                  | YouTube送客率、CPA、ROAS、チャンネル登録者増加率、SFF集客率  |

---

### ⑤ 貸付金・投資資産（資本配分レバー）

| 観点                      | 内容                                  |
| ----------------------- | ----------------------------------- |
| **現状（定量）**              | 長期貸付金 12,752千円（総資産の16%）。            |
| **課題（定性）**              | 遊休資本化のリスク。資本効率（ROA）を下げている可能性。       |
| **ビジョン方向A：理念重視（社会的投資）** | Shape Fitの理念に共感する他社・個人への支援資金と位置づける。 |
| **ビジョン方向B：効率重視（成長投資）**  | 貸付金を回収し、GYM/SFF/FC等の成長投資に再配分。       |
| **影響指標**                | ROA改善＋3〜5pt、貸付金回転率、内部投資リターン率（ROI）   |

## 🧩 総括：Shape Fit社の「5大経営ハンドル」

| YouTube | 世界観を発信   | 集客ROIを管理    |
| 投資      | 共感投資     | 資本効率最大化     |
