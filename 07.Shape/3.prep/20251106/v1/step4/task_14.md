# A案：経営基盤プラン（1→10確立型）

> **テーマ：経営の“型”をつくる**
> 目的：数字で語れる経営チームをつくり、会議とKPIを仕組み化する。
> 期間：3ヶ月（基盤整備フェーズ）

---

### ① 支援目的

経営会議・PL/BS・KPI管理の「型」を確立し、
**理念（中井）と数字（天池）を接続した“経営運営モデル”**を実装する。

---

### ② 実施内容（アクションプラン）

| ステップ  | 実施内容                         | クライアントの関与      |
| ----- | ---------------------------- | -------------- |
| Step1 | PL・BS・KPIを統合した「経営ダッシュボード」を設計 | 天池氏が数値提供／設計あなた |
| Step2 | 月次経営会議のテンプレ・アジェンダを設計         | 経営会議で試用・運用     |
| Step3 | Vision→KGI→KPI翻訳マップを作成       | 中井氏と共同設計       |
| Step4 | あなたがファシリで“会議1回分”を同席し運用支援     | 経営陣3名が実践参加     |

---

### ③ アウトプット（納品物）

| 成果物                  | 内容                  | 使用場面       |
| -------------------- | ------------------- | ---------- |
| 📊 **経営ダッシュボード**     | PL／BS／KPIを一枚に統合     | 月次報告・経営会議  |
| 🗂 **経営会議テンプレート**    | 今月の数値・課題・意思決定フォーマット | 経営会議の標準化   |
| 🧭 **Vision-KPIマップ** | 理念→KPI→PL構造を可視化     | 社内共有・判断基準  |
| 📘 **運営マニュアル**       | 会議運営・KPI更新の手順書      | 社内教育・引き継ぎ用 |

---

### ④ ゴール（3ヶ月後の理想状態）

| 項目   | Before   | After（A案完了後）   |
| ---- | -------- | -------------- |
| 経営会議 | 感覚・雑談型   | KPI・PLベースで意思決定 |
| 数字管理 | 各担当がバラバラ | ダッシュボードで統一管理   |
| 理念   | 言葉として存在  | KPIに翻訳され行動指標化  |
| 経営構造 | なーすけ氏中心  | 経営チームで回る“仕組み”化 |

* 営業利益率：0.2% → **5%**

# B案：戦略伴走プラン（10→100成長型）

> **テーマ：経営を“伸ばす”**
> 目的：事業ポートフォリオと中期成長シナリオを構築し、“理念で伸ばす経営”へ進化。
> 期間：6ヶ月（中期拡張フェーズ）

---

### ① 支援目的

経営基盤を前提に、**SFF／GYM／YouTube／FCの最適構成と中期投資方針**を設計。
理念起点で「どの事業を伸ばし、どこに再投資するか」を見える化する。

---

### ② 実施内容（アクションプラン）

| ステップ  | 実施内容                        | クライアントの関与     |
| ----- | --------------------------- | ------------- |
| Step1 | 事業ポートフォリオ再設計WS（PPM分析）       | 経営陣3名が議論      |
| Step2 | 投資・財務戦略シミュレーション（If-Thenモデル） | 天池氏主導／設計支援あなた |
| Step3 | 10年Visionロードマップ再構築          | 中井氏と共同設計      |
| Step4 | 拡張シナリオ（FC・教育・社会連携）設計        | 経営者判断＋あなたが構造化 |

---

### ③ アウトプット（納品物）

| 成果物                          | 内容                  | 使用場面          |
| ---------------------------- | ------------------- | ------------- |
| 📈 **事業ポートフォリオマップ**          | 各事業の収益性×成長性を可視化     | 投資判断・会議       |
| 💰 **財務シナリオシート（If-Thenモデル）** | 広告・人件費・投資パターン別の収益影響 | シミュレーション・経営判断 |
| 🧭 **10年Visionロードマップ**       | 理念→事業→財務→組織の中期構造    | 経営方針発表・社内共有   |
| 🤝 **中期成長プラン（3年アクションプラン）**   | 重点KPI・行動スケジュール・担当配置 | 実行フェーズ管理      |

---

### ④ ゴール（6ヶ月後の理想状態）

| 項目     | Before  | After（B案完了後）     |
| ------ | ------- | ---------------- |
| 投資判断   | 感覚的・単発的 | ROA・成長率ベースの構造的判断 |
| Vision | 言語レベル   | 財務・組織に接続された10年構想 |
| 財務戦略   | 短期PL管理  | 中期投資モデル＋再投資戦略    |
| 経営構造   | “会議体”中心 | “経営設計チーム”として自走化  |

---

### ⑤ 成果指標（KPI）

* 売上成長率：**＋20〜30%**
* ROA維持・改善：**27.4% → +1.0pt**
* Vision接続率（KPI→理念連動）：**80%以上**

---
