| 項目          | 内容                                                                   |
| ----------- | -------------------------------------------------------------------- |
| **目的**      | Shape Fit社の現状課題（属人経営・下期赤字化）を踏まえ、<br>「①経営基盤の再構築」と「②中期成長戦略の設計」を二段階で支援。 |
| **フェーズ構成**  | Phase1＝A案（経営基盤プラン） → Phase2＝B案（戦略伴走プラン）                              |

## 2️⃣ 財務基盤の現状（損益計算書より）

出典：「株式会社Shape Fit御中_20251023_経営会議資料(速報値）」

| 区分                          | 売上高（千円） | (償却後)営業利益（千円） | 営業利益率      |
| --------------------------- | ------- | ------------- | ---------- |
| **上期（2025年3〜8月 実績）**        | 68,650  | 4,274         | **＋6.2%**  |
| **通期（2025年3〜2026年2月 着地見込）** | 102,774 | ▲9,300        | **▲9.1%**  |
| **下期（2025年9〜2026年2月 見込）**   | 34,124  | ▲13,574       | **▲39.8%** |

📘 **現状分析**

* 上期は黒字（+6.2%）だが、下期で▲39.8%と大幅悪化。
* 固定費・外注費が集中しており、**「下期収益構造の崩れ」**が最大課題。
* よってA案では「赤字幅の半減（利益構造の正常化）」を、
  B案では「翌上期の成長回復（売上・利益率向上）」を目標とする。

## 4️⃣ 成果報酬条件の具体設計

### 💠 A案：経営基盤プラン（11〜2月）

| 項目            | 内容                                                                 |
| ------------- | ------------------------------------------------------------------ |
| **対象期間**      | 2025年11月〜2026年2月（3ヶ月）                                              |
| **基準**        | 下期営業利益率 ▲40%（通期データ差分より算出）                                          |
| **目標**        | ▲20%以上へ改善（赤字幅の半減）                                                  |
| **成果報酬**      | ＋¥200,000支給（3ヶ月平均で判定）                                              |
| **算定根拠（ROI）** | 売上約15,000千円×改善幅20pt＝＋3,000千円改善 → ROI＝(3,000−1,100)/1,100＝**約170%** |

🟢 **ポイント**：
短期間で黒字転換は現実的ではないが、「固定費調整＋KPI管理＋会議効率化」により**赤字幅半減**は十分実現可能。
支援終了後も継続効果が残る構造改善投資。

---

### 💠 B案：戦略伴走プラン（3〜8月）

| 項目            | 内容                                                               |
| ------------- | ---------------------------------------------------------------- |
| **対象期間**      | 2026年3〜8月（翌期上期）                                                  |
| **基準**        | 前年同期（2025年上期）売上68,650千円／営業利益率6.2%                                |
| **目標**        | 売上＋8%以上（74,142千円以上）または営業利益率＋2pt（8.2%以上）                          |
| **成果報酬**      | ＋¥300,000支給（いずれか達成で発生）                                           |
| **算定根拠（ROI）** | 売上増5,492＋利益率改善1,483＝＋6,975千円 → ROI＝(6,975−2,100)/2,100＝**約230%** |

🟢 **ポイント**：
A案で整えたKPI・PL構造を基盤に、事業戦略と再投資判断を実装。
**前年対比で確実に成長を実感できるライン**を成果基準に設定。

## ✅ 最終コメント（経営会議向け）

> Shape Fit社は現在、下期赤字（営業利益率▲40%）という明確な経営課題を抱えています。
> そのため、まずは **A案で「赤字幅を半減する基盤づくり」** を実行し、
> その後 **B案で「前年同期比＋8%成長を実現する戦略経営」** に進む構成が最適です。
>
> 成果報酬は「期間平均／累計」で客観評価し、ROIはいずれもプラス（A案170%、B案230%）。
> リスクを抑えつつ確実な経営改善が見込める二段階支援モデルです。
