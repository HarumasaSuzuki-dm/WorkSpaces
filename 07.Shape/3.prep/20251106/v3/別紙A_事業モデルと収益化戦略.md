## 1. Shape Fitの「収益化（マネタイズ）モデル」について

Shape Fitが実現したい世界観は"社会実装型"であり、**ビジネス × 文化 × 社会変革**の三層構造を持っています。

**「収益エンジン」と「価値拡張エンジン」を明確に分離**することで、持続可能な事業構造を構築できます。以下のように整理いたします。

---

### 1-1. 収益構造の3層モデル

| レイヤー                   | 内容                                           | 収益化の度合い      | 備考                                       |
| ---------------------- | -------------------------------------------- | ------------ | ---------------------------------------- |
| **① コア収益層（キャッシュエンジン）** | 企業向け組織変革・人材開発プログラム、研修、コンサル、マネジメント層支援         | ★★★ 収益性高     | 「管理職×挑戦文化」はお金を払う主体が明確（BtoB）。再現性が高く単価が高い。 |
| **② 拡張収益層（中期収益化）**     | 教育×行政連携プログラム、自治体協働、出張授業、教材・講座販売              | ★★ 中期収益化     | 学校は単価低いが採択されると大きい。寄付・補助金モデルも成立。          |
| **③ 文化醸成層（長期投資領域）**    | コミュニティ／カルチャー活動、イベント（SFF）、メディア、SNS発信、コンテンツ文化化 | ★ 収益性低い／投資寄り | ブランド認知・ファン化・カルチャー醸成 → 事業の“防御力”になる領域      |

---

### 1-2. 収益柱の具体案

| 収益モデル                       | 説明                              | 単価イメージ           | 備考                 |
| --------------------------- | ------------------------------- | ---------------- | ------------------ |
| **企業向け組織変革コンサル／研修**      | 「挑戦文化×自己理解×変容プロセス」を法人向けにパッケージ化  | 200〜800万円/社      | 最も利益率高く、年間契約化しやすい  |
| **管理職〜リーダー層トレーニング**      | “挑戦する組織をつくるマネジメント”研修            | 20〜100万円/回       | 1day〜3ヶ月伴走型など構造化可能 |
| **教育×自治体・学校導入プログラム**     | 出張授業・探究学習・キャリア教育・教材提供           | 10〜100万円/校       | 補助金/助成金もセットで取れる    |
| **オンライン講座／動画教材販売**       | Shapeness / 自己理解プロセス講座          | 1〜5万円／人          | BtoC型のスケーラブル商材     |
| **コミュニティ会費／イベント収益（SFF）** | フェス、挑戦発表会、スポンサー協賛               | スポンサー50〜300万 etc | 事業価値よりブランド／文化の拡張装置 |
| **出版／メディア／SNS広告／IP展開**   | “Shapeness”を言語化した思想書／番組／会員制メディア | 中〜長期             | “思想の社会実装”フェーズ      |

---

### 1-3. 収益構造のポイント

- **企業向けが最も即収益化しやすい（高単価×意思決定者が明確）**
- 教育／地域は「スケール型だが低単価」なので中期投資領域
- カルチャーは **直接収益より「ブランド／思想の土壌づくり」**

**結論：短期で稼ぐ → 企業領域、中期で社会インパクトを広げる → 教育／地域、長期で文化化 → カルチャー事業**

---

## 2. 「企業／教育／地域／カルチャー」の優先順位について

Shape Fitの4つの事業領域について、以下の2つの観点から整理いたします：
- **価値観・想いの優先度**
- **事業としてのインパクト（再現性／収益性）**

これらを明確にすることで、リソース配分の意思決定が可能になります。

---

### 2-1. 価値観・想いの優先度

| 領域    | 想いの強さ              | 根拠                      |
| ----- | ------------------------- | ----------------------- |
| カルチャー | ◎◎◎ 最も情緒的価値が強い            | Shapenessを"文化にする"と複数回明言 |
| 教育    | ◎◎ "未来の世代に伝える"という使命感あり    | 中高生・価値観形成層への執着が見える      |
| 地域    | ◎ "挑戦が見える街づくり"という語りの熱量が高い | 生活圏での挑戦文化＝理想像           |
| 企業    | ○ 内容は詳細・構造化されているが感情優先ではない | 使命より「実装のための装置」の位置づけ     |

---

### 2-2. 事業インパクト優先度（収益性＋再現性ベース）

| 領域        | 収益性 | 実装スピード | スケール性   | 総合優先度              |
| --------- | --- | ------ | ------- | ------------------ |
| **企業**    | ★★★ | 早い     | 中       | 最優先（現実解）         |
| **教育**    | ★★  | 中      | 高       | △ 中期（自治体/助成金必要）    |
| **地域**    | ★★  | 遅い     | 中       | △ ブランド・社会性は高いが収益弱い |
| **カルチャー** | ★   | 遅い     | 高（社会浸透） | ◎思想価値は高いが収益化/実行負荷高 |

---

### 2-3. 2軸マトリクスによる優先領域の整理

**縦軸：事業インパクト（収益性×実現スピード）**
**横軸：価値観・想いの強さ**

|     | 低インパクト   | 高インパクト               |
| --- | -------- | -------------------- |
| 価値観・想いが強い | カルチャー／地域 | 価値観と事業の一致領域（設計で最大化） |
| 価値観・想いが弱い | （該当なし）   | 企業（実装・収益化・再現性トップ）    |

**重要なポイント：**「価値観・想いが強いが収益が弱い領域」を **どう守りながら事業にするか？** が今後の設計における重要な検討事項です。

---

## 3. ご支援可能な領域と提供価値

Shape Fitの課題に対し、どの領域でご支援できるかを整理いたします。

### 3-1. Shape Fitが直面している課題（前提整理）

#### （1）ビジョンは壮大だが、事業化ロジックが未整理

・社会実装／文化醸成／挑戦の連鎖という思想は強いが、「事業としてどう伸ばすか」が未定義
・とくに短期収益モデルと中長期インパクトモデルの整理が必要

#### （2）4領域（企業／教育／地域／カルチャー）が並列で、優先順位が曖昧

・リソース投下判断軸がないため「全部やりたい状態」が続くリスク
・フェーズ分け、MVP観点、ロードマップ策定が必須

#### （3）思想→文化→事業の一体運営が必要だが、実装力が不足

・コンセプトはあるが、言語化・商品化・顧客化・拡張の設計はまだ途中段階
・特に「事業パッケージ化」「営業導線設計」「KPI設計」が存在しない

#### （4）カルチャー発信はしているが、ブランド/コミュニティがまだ資産化していない

・SNS・Web・イベントは点として存在するが、統合的戦略になっていない
・思想（WHY）と事業（WHAT）を橋渡しする「物語構造」が未整備

---

### 3-2. 課題に対する提供価値・役割

| Shape Fitの課題                   | 提供価値・役割                       |
| --------------------------- | -------------------------------- |
| ビジョンはあるが事業化が未構造化            | 事業モデリングと収益モデル設計が可能              |
| 4分野を同時に動かそうとしている            | PMスキルにより優先順位設計／ロードマップ化が可能       |
| 組織／教育／地域／カルチャーの横断実装が必要      | 領域横断のプロジェクト推進経験がある               |
| 思想は強いが市場への転換点が未定義           | マーケティング設計（顧客言語化／市場定義／ファネル構築）が可能 |
| カルチャー任せになっておりブランドが資産化されていない | Web/SNS/コミュニティ運営を戦略として統合可能      |
| 事業単位でのKPIや事業管理フレームが存在しない    | 経営ダッシュボード／指標体系づくりが可能            |

---

### 3-3. コア強みと、Shape Fit事業への活かし方

#### 強み1：プロジェクトマネジメント（PM）

・複数領域／複数ステークホルダー／抽象テーマを具体化する型を持つ
・Shape Fitの「企業／教育／地域／カルチャー」4事業をフェーズ分解し、実行計画に落とし込むことができる
・ビジョンベースではなく、ロードマップとKPIで“運べる組織”に変換可能

#### 強み2：マーケティング／ブランド設計

・思想を「誰に・何を・どう届けるか」に変換する領域が得意
・企業研修／教育プログラム／地域コミュニティ／カルチャー発信を“統合ブランド”として設計できる
・コミュニティ・SNS・メディア・Web・イベントをひとつの“成長導線”として構築できる

#### 強み3：IT / Web / デジタル構造化

・思想や体験を「仕組み」「プロダクト」「システム」に変換できる
・オンライン講座／会員サイト／データ基盤／ダッシュボード等、価値を定量的に蓄積できる設計が可能
・“文化で終わらせず、資産として残す”という思想と相性が良い

---

### 3-4. ポジションの定義

- 思想を「社会実装モデル」として構造化する役割
- 4領域を「個別運営」から「再現性ある事業」に変換する役割
- ビジョンを"語るだけ"で終わらせず、"動く仕組み"に変えるハブ役
