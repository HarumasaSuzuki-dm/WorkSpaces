## 1. 前提整理：Shape Fitの「言語化された目的」

ご提供いただいた資料から抽出できる、明示されている目的は以下です：

| レイヤー               | 明示されている目的                          |
| ------------------ | ------------------------------------- |
| 10年後VISION         | 「自分らしく生きられる社会」                        |
| 会社VISION           | 「1人ひとりが自分の人生を好きになる」                   |
| MISSION            | 「挑戦と物語を共にする／変化を生み出すプラットフォーム」          |
| VALUE              | 「どういう姿勢で登るか＝自分軸／自己理解／成長／他者貢献」         |
| CJM Before → After | 「他人基準 → 自分基準」「挑戦不可 →挑戦できる」「比較嫉妬→応援協力」 |
| 社会展開               | 企業／教育／地域へ波及し文化を変える                    |

## 2. インサイトの深掘り：**"WHYそれをやりたいのか？"** という問い

上記の目的から、以下の問いが重要になります：

- 「なぜ "フィットネス事業" に留まらず、社会／企業／教育／地域まで広げたいのか？」
- 「なぜ "挑戦できる人を増やしたい" ではなく、"挑戦文化を社会に広げたい" なのか？」
- 「なぜ "自分らしく生きる" がここまで強調されているのか？」

これらは「事業目的」ではなく **"存在理由（Existence Reason）"** に近い領域です。

---

## 3. 資料の構造から読み取れる「潜在的なキーワード」

資料に散らばる言葉を整理すると、反復している概念が見えてきます：

| 繰り返し使われている語         | 含意                                 |
| ------------------- | ---------------------------------- |
| 自分らしさ／自分の軸          | 「答えを外に求めない生き方への転換」がテーマ             |
| 比較・嫉妬 → 応援／共創       | “競争”より “共存” を肯定する世界観               |
| 挑戦／物語               | 「行動を通じてアイデンティティが形成される」という前提        |
| 社会に波及／文化になる         | “個人変化” では足りず “社会レベル” での変化を狙っている    |
| Before状態にある人の描写が具体的 | 「自己否定／停滞／他人基準で生きざるを得ない構造」への強烈な問題意識 |

特に **Before側の描写の具体さ** は重要な示唆を与えています。
「自分がよくわからない／不安／比較／挑戦できない／失敗が怖い／応援できない」
→ これは **"個人の問題" ではなく "社会構造によって生まれた状態"** である、という思想が前提にあると考えられます。

つまり、Shape Fitは **"フィットネス" を目的とするのではなく、変容プロセスを社会に実装したい** という意図があると読み取れます。

## 4. 推察される「根源的な動機（インサイト）」

### 4-1. インサイト仮説

> 「人は本来"自分の人生を生きる力"を持っているのに、
> 社会システムや文化がそれを奪ってしまっている。
> だから "挑戦できる個人" を増やすだけでは足りない。
> **"個人の変容 → 組織 → 社会まで波及させる装置" をつくらなければ、根本的な解決にはならない。**」

つまり、以下のような世界観の違いがあると考えられます：

| 一般的なパーソナルジムの世界観 | Shape Fitの世界観       |
| --------------- | ------------------- |
| 身体が変わることで人生が変わる | 人生が変わるプロセスを社会に伝播させる |
| 個人の成功で終わる       | 変容者が次の変容を生む連鎖をつくる   |
| 健康／美容産業         | 文化設計／社会変容産業         |

> 「自分が変われたプロセスは "運" ではなく "構造化すれば誰でも再現できるはず"
> → だから事業ではなく "社会実装" にしたい」

## 9. 各競合の "ビジネスモデル + 強み" の詳細分析

### キャリアbase（#1）

* **モデル**：学校・行政など教育機関と連携し、自己理解→職業理解→就労支援まで一貫支援を提供。中・長期的伴走型。 ([NPO法人キャリアbase][1])
* **強み**：教育現場をパートナーにしており「点で終わらない」継続的な支援が可能。特定層を逃さない「誰ひとり取り残さない」姿勢。

### Grow & Leap（#2）

* **モデル**：中高生対象にマンツーマン支援＆対話・交流機会提供、キャリアアンカー発見プログラム。 ([growandleap.com][2])
* **強み**：自己理解を核に据えており、意志形成までアプローチ。早期教育領域に特化。

### 日本ピーススマイル協会（#3）

* **モデル**：青少年対象に自己肯定／自己理解ワークショップ、居場所づくり、体験プログラム。 ([tvac.or.jp][3])
* **強み**：「生きづらさ」の文脈を抱える層にリーチしており、課題解決型・社会インパクト型。ブランドとしての信頼感。

### 日本アクティブキャリア開発協会（#4）

* **モデル**：キャリアコンサルタントの育成＋自己／職業理解支援を制度的に構築。 ([cc.j-acd.org][4])
* **強み**：専門職ネットワーク・制度的裏付けがあるためスケーラブル。個人支援＋制度支援の両輪。

### アイ・ケイ・ケイホールディングス（#5）

* **モデル**：企業内で「挑戦者として行動する人」を評価、人事制度にチャレンジ志向を組み込む。 ([ikk-grp.jp][5])
* **強み**：自社の“挑戦文化”を明文化し、制度・評価仕組みで支えている。事業モデルに“挑戦”がインテグレートされている。

### Microsoft Japan（#6）

* **モデル**：クラウド・ソフトウェア提供を通じて、個人・組織の能力を最大化。ミッションに“個人・組織の達成”を掲げる。 ([Microsoft][6])
* **強み**：グローバルリソース、技術基盤、スケール力。個人の“可能性を引き出す”支援をテクノロジー＋文化で実現。

### ネットマーブルジャパン（#7）

* **モデル**：ゲーム／エンタメ領域で“挑戦”をコアバリューに掲げ、若年層へのブランド訴求を行う。 ([recruit.netmarble.co.jp][7])
* **強­み**：文化的エンタメ領域で挑戦・イノベーションを語ることで、ブランド力／共感力が高い。対象層が明確。

### ONE JAPAN（#8）

* **モデル**：「挑戦する文化をつくる」ためのコミュニティ／イベント運営。企業・個人を巻き込む。 ([note（ノート）][8])
* **強み**：場をつくる、ネットワークをつなぐ機動力。“挑戦文化”を語り、広めるプラットフォーム型。

### み・らいず2（#9）

* **モデル**：若者・生きづらさを抱える人の支援。地域・社会という文脈で「自分らしく暮らせる社会」を目指す。 ([日本財団][9])
* **強み**：社会課題（発達障害・いじめ・貧困など）と直結しており、ニーズが緊急かつ本質的。地域ベースでの影響力。

### Goodpatch（#10）

* **モデル**：デザイン会社として「人に投資する」「人と向き合う」文化を掲げ、価値観を明文化。 ([株式会社グッドパッチ][10])
* **強み**：デザイン／クリエイティブという背景を持ち、文化・価値観を可視化・体現しており、内外への発信力が高い。

## 10. 分析：Shape Fitとの差別化・機会領域

上記を踏まえ、Shape Fitが"挑戦文化醸成／自己理解支援"領域で差別化できるポイント・機会を整理いたします：

1. **「身体（フィットネス）を手段にした変容ストーリー」**
   多くの事例がキャリア教育・組織文化変革・若者支援に集中しており、フィットネス＋挑戦文化というハイブリッドモデルは少数派。身体変化→自己理解→挑戦文化という流れを強みにできる。

2. **「挑戦→物語→社会波及」という"連鎖モデル"**
   どちらかといえば "個人支援" や "企業文化" が主流で、「支援者が次の挑戦者を生む構造」「変容者が発信者になる連鎖」というモデルはあまり典型的ではない。これがShape Fitの強みとなります。

3. **教育・企業・地域をまたぐ “3軸展開”**
   競合はどちらかに特化（教育 or 企業 or地域）が多いため、3つを統合的に展開するモデルは希少。教育→企業→地域という流れを設計できればポジショニング優位性あり。

4. **文化化・ライフスタイル化への寄与**
   “挑戦文化” を単なるプログラムではなく、ライフスタイルや文化として浸透させる視点が少ない。Shape Fitが価値観（Shapeness＝自分らしさ）を文化化するという話はこのギャップを突ける。

5. **“挑戦／自己理解”の対象層を拡張**
   若年層や企業内部の挑戦者支援が多い中、例えば「日常生活のなかで挑戦を起こしたい一般層」「ライフステージ転換期（30〜50代）」「地域コミュニティにおける挑戦」という領域にフォーカスすると、競合と重ならない領域もある。

## 11. 市場調査：ターゲット層別の「時代的な悩み」とトレンド

Shape Fitが解決しようとしている「人々が挑戦を恐れている」という課題は、現代の各セグメントが抱える中核的な悩みと強く一致しています。

---

### 11-1. 10代〜30代（若者層）：失敗を恐れ、「自分らしさ」を模索する世代

Shape Fitの資料にある「他人と比較し、劣等や苦痛で行動できない」状態を裏付けるトレンドが顕著です。

* **トレンド①：深刻な「タイパ（タイムパフォーマンス）」重視と失敗への恐怖**
    * Z世代はデジタルネイティブである一方、経済不安の中で育っており、安定志向が強いとされます。
    * 彼らにとって「挑戦の失敗」は、時間とコストを失う**最悪の「タイパが悪い」行為**と認識されがちです。これにより、挑戦そのものを回避し、あらかじめ成功が約束された（あるいは他人が良しとする）選択肢に流れる傾向があります。
* **トレンド②：「完璧」よりも「共感できる弱さ」の希求**
    * SNSでの「完璧な姿」に疲弊する一方、Z世代の多くが「自虐コンテンツ（ダメな自分）」に共感を示しています。
    * これは、Shape FitのCJMにおける「before（自分のことがこわい）」状態と、「after（ありのままの自分を肯定）」の間に存在する強いニーズを示唆します。彼らは**「弱さを見せても大丈夫」という安心感**と、それを肯定する「物語」を求めています。
* **トレンド③：自己肯定感の低さと「所属」のための自己分析**
    * 日本の若者の自己肯定感は他国と比較して依然として低い水準にあります。
    * MBTIなどの自己分析ツールが流行していますが、これはしばしば「自分だけの物語を見つける」ためではなく、「自分はどのタイプに属するか」を確認し、**他者との共通項を見つけて安心する**ために使われています。これは「挑戦」とは逆の「同調」への欲求とも言えます。

### 11-2. 企業：イノベーションを求めつつ、「挑戦」を阻む組織文化

Shape Fitの資料の「企業」セクションで指摘された「理念を信じ切れていない集団」や「挑戦を笑う空気」が、現代の企業課題の核心です。

* **トレンド①：「心理的安全性」の形骸化**
    * 多くの企業が「心理的安全性」の重要性を認識し、「高い」と回答する企業が半数を超えています。
    * しかし実態は、多くの現場で「言いたいことが言える」状態には至っていません。特に若手社員は「キャリアパスや成長機会の不足」を感じており、「リベンジ退職」（評価されないことへの反発としての退職）が2025年のトレンドとして予測されています。
* **トレンド②：イノベーションの停滞と「挑戦できる人材」の枯渇**
    * 日本企業のイノベーション停滞は大きな課題であり、「挑戦できる人材が社内にいない」ことが原因として挙げられています。
    * 多くの企業が「古い慣行」から抜け出せず、失敗を許容する風土がないため、挑戦意欲のある人材（特にZ世代）ほど早く見切りをつけて離職するという悪循環に陥っています。
* **トレンド③：低いエンゲージメントと「物語（パーパス）」の欠如**
    * 従業員エンゲージメントの低さは、企業の生産性や収益性に直結します。
    * その原因は、経営層のメッセージが従業員に「本気度」として伝わっておらず、自分の仕事と会社のビジョン（＝物語）が結びついていない点にあります。

### 11-3. 教育機関：「非認知能力」を重視しつつも、「同調圧力」が根強い現場

Shape Fitの資料の「教育」セクションにある「『結果』だけで人を評価する文化」と「『他者否定』の空気」が、教育現場の最大の課題として浮き彫りになっています。

* **トレンド①：最重要視される「非認知能力」**
    * 教育関係者が今最も注目しているトピックは「非認知能力」（忍耐力、協調性、自己肯定感など）です。
    * これは、従来の「学力重視」の教育では、社会で生き抜く力（＝挑戦し、失敗から立ち直る力）が育たないという強い危機感の表れです。
* **トレンド②：キャリア教育の歪みと「自分らしさ」の欠如**
    * 現場では「キャリア教育」が求められていますが、多くの場合、それは「どんな職業があるか」という知識教育に留まっています。
    * Shape Fitが目指す「"自分で"進路を選ぶ」という、主体的な「生き方」の教育（＝自己理解）が決定的に不足しています。
* **トレンド③：根強い「同調圧力」と「いじめ」の増加**
    * いじめの認知件数は増加傾向にあります。これは「他者と違うこと」を許容しない、強烈な同調圧力の存在を示しています。
    * この環境では、Shape Fitが指摘する「挑戦を笑う」空気が醸成され、「自分らしさ」を発揮することは極めて困難です。

## 12. 結論とインサイト

調査結果は、Shape Fitが解決しようとしている「挑戦を阻む文化」が、**日本社会の若者・企業・教育の全セグメントに共通する、根深く巨大なペイン（悩み）**であることを強力に裏付けています。

* **若者**は「失敗（＝タイパが悪い）」を恐れ、挑戦できずにいます。
* **企業**は「挑戦できる人材」が育たない・辞めてしまうことに悩んでいます。
* **教育**は「挑戦する心（非認知能力）」を育てたいのに、方法がわからず同調圧力に負けています。

Shape Fitの「Shapeness」という概念は、これらすべてのセグメントが求めている**「失敗を許容し、自己理解に基づいた挑戦を応援する」**というソリューションを提供する、極めて時流に適したポジショニングにあると言えます。
