---

### **エグゼクティブサマリー**

*   **背景と目的**
    *   現在貴社が取り組むべき3つの重要テーマ「**①経営体質最適化**」「**②組織基盤強化**」「**③法人研修事業の立ち上げ**」は、相互に連携させて推進することで、初めて最大の効果を発揮します。
*   **ご提案の核心**
    *   これら3つのプロジェクトを一つの**「包括アドバイザリー契約」**として統合的にご支援します。クライアント様主導のもと、私どもはアドバイザーとして伴走し、事業成長を加速させます。
*   **選べる2つの契約プラン**
    *   事業のフェーズとニーズに合わせ、**「プランA：3ヶ月集中プラン」**と**「プランB：6ヶ月伴走プラン」**の2つのご契約期間をご用意しました。
*   **費用体系**
    *   **月額費用:** 330,000円（税込）
    *   **成果報酬:** 各プロジェクトの成果に応じた変動報酬
*   **期待される成果**
    *   「守り」「組織」「攻め」の各領域で具体的な成果を創出し、次フェーズに向けた明確な道筋を確立します。

---

### **ご提案の全体像：3つのプロジェクトの統合推進**

本契約は、個別課題への対症療法ではなく、事業成長の土台を再構築するための統合的なご支援です。

| プロジェクト | テーマ | 概要と目的 |
| :--- | :--- | :--- |
| **① 経営体質最適化** | **守り** | 実質PL管理を導入し、コスト構造を徹底的に可視化する。データに基づいた「正しい経営判断」の土台を構築する。 |
| **② 組織基盤強化** | **組織** | 役割と責任(R&R)を明確化し、リソースの無駄を排除する。特に課題となっている業務のRACIチャートを作成し、迅速な意思決定を可能にする。 |
| **③ 法人研修事業** | **攻め** | SHAPE社の強みを活かした高付加価値な研修サービスを設計する。事業の骨子を固め、市場投入に向けた具体的なアクションプランを策定する。 |

---

### **支援スコープと主な活動内容**

3ヶ月間で以下の活動を並行して推進し、具体的な成果創出を目指します。

*   **① 経営体質最適化支援 (Phase 1)**
    *   実質PL管理の導入支援
    *   外注費および原価構造の分析・可視化
    *   コスト削減／交渉シナリオの仮説構築

*   **② 組織基盤強化支援 (Phase 0)**
    *   主要ステークホルダーへのインタビューと課題構造化
    *   課題を可視化した組織図（たたき台）の作成
    *   最重要業務プロセスにおけるRACIチャート（役割分担表）の作成

*   **③ 法人研修事業立ち上げ支援**
    *   サービス設計思想（価値・型）の具体化と壁打ち
    *   ターゲット顧客へのヒアリングとニーズ検証
    *   事業計画およびサービス提供マテリアル（骨子）の作成支援

### **投資対効果（ROI）シミュレーション**

本ご提案が、貴社の事業成長に対して合理的かつ効果的な投資であることをご理解いただくため、保守的な仮説に基づいたROI（投資対効果）シミュレーションを提示します。

*   **シミュレーションの前提**
    *   プロジェクト①「経営体質最適化」の成果として、主に**コスト構造の改善（委託費・人件費の変動費化など）**に焦点を当てます。
    *   参考資料「KPI\_インパクトテーブル」に基づき、**営業利益率が3.0〜4.5pt改善**されると仮定します。（6ヶ月プランは、より深い実行支援により高い改善率を想定）
    *   これにより、**年間で約310〜460万円の財務的インパクト**が創出されると試算します。
    *   ※組織強化や新規事業開発による中長期的なリターンは、保守的に見積もるため本計算には含めていません。

| 項目 | プランA：3ヶ月集中プラン | プランB：6ヶ月伴走プラン |
| :--- | :--- | :--- |
| **ご契約期間** | **3ヶ月** | **6ヶ月** |
| **期待される年間リターン** | **約 310万円** | **約 460万円** |
| | (営業利益率 +3.0pt改善を想定) | (営業利益率 +4.5pt改善を想定) |
| --- | --- | --- |
| **投資総額（シミュレーション）** | **約 240万円** | **約 332万円** |
| 月額費用 | 99万円 | 198万円 |
| 固定成果報酬 | 110万円 | 88万円 |
| 変動成果報酬 | 31万円 | 46万円 |
| --- | --- | --- |
| **ROI（年間）** | **約 29%** | **約 39%** |

**結論:** 上記は保守的な試算ですが、いずれのプランにおいても**投資を上回るリターン**が期待できます。特に6ヶ月プランは、より高いROIを見込める設計となっております。

### **もう一つのROI：経営資源（時間）の創出**

本契約の価値は財務改善だけでなく、社長およびコアメンバー皆様が**より高付加価値な業務に集中できる時間を創出する**点にもあります。分析・資料作成・会議準備等をアドバイザーが代替・効率化することで生まれる**「機会費用の削減効果」**を以下に試算します。

*   **シミュレーションの前提**
    *   **対象:** 社長およびコアメンバー3名
    *   **時間単価（機会費用）:** 平均 15,000円／時間 と仮定
    *   **創出時間:** プランAでは月35時間、プランBでは仕組みの定着により月平均45時間と仮定

*   **創出される時間価値と総合ROI**

| 項目 | プランA：3ヶ月集中プラン | プランB：6ヶ月伴走プラン |
| :--- | :--- | :--- |
| **創出時間（月平均）** | **35時間** | **45時間** |
| **創出される時間価値（年間）** | **約 630万円** | **約 810万円** |
| | (35h×15,000円×12ヶ月) | (45h×15,000円×12ヶ月) |
| --- | --- | --- |
| **A. 財務改善リターン（年間）** | 約 310万円 | 約 460万円 |
| **B. 時間創出リターン（年間）** | 約 630万円 | 約 810万円 |
| **リターン合計 (A+B)** | **約 940万円** | **約 1,270万円** |
| --- | --- | --- |
| **投資総額** | **約 240万円** | **約 332万円** |
| --- | --- | --- |
| **総合ROI（年間）** | **約 291%** | **約 282%** |

**結論:** 経営陣の「時間」という最も重要な資源の創出まで考慮すると、**投資額を大幅に上回るリターン**が期待できます。月額33万円の投資は、**月間で50万円以上の時間価値**を生み出す、極めて合理的な意思決定であるとご判断いただけます。
