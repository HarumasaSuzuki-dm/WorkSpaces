# 包括アドバイザリー契約 - 見積もり＆ROIロジック分析レポート

## 1. はじめに

本資料は、「事業成長加速に向けた包括アドバイザリー契約のご提案」に関する見積もりおよび投資対効果（ROI）のロジックを分解・整理し、客観的な意思決定を支援することを目的とします。

## 2. 料金体系の構造分析

提案されている料金体系は、**リスクを抑制**しつつ**成果へのコミットメント**を両立させる「ハイブリッドモデル」です。

*   **月額固定費用**: アドバイザーの基本的な活動時間と専門知識に対する対価。これにより、安定的な支援体制が確保されます。
*   **成果報酬**: 各プロジェクトの具体的な成果（コスト削減、事業開発など）と連動する変動報酬。これにより、アドバイザーとクライアントの目標が一致し、成果創出へのインセンティブが働きます。

## 3. 投資総額シミュレーション

提案されている2つのプランにおける投資総額は以下の通りです。

| 項目 | プランA：3ヶ月集中プラン | プランB：6ヶ月伴走プラン |
| :--- | :--- | :--- |
| **契約期間** | **3ヶ月** | **6ヶ月** |
| 月額費用 | 990,000円 | 1,980,000円 |
| 固定成果報酬 | 1,100,000円 | 880,000円 |
| 変動成果報酬（※） | 310,000円 | 460,000円 |
| --- | --- | --- |
| **投資総額（シミュレーション）** | **約 2,400,000円** | **約 3,320,000円** |

※変動成果報酬は、後述する財務改善リターン（年間削減効果額）の10%として試算されたものです。

## 4. 投資対効果（ROI）ロジックの分解

本提案では、ROIを「財務改善」と「時間創出」の2つの側面から算出しており、非常に説得力があります。

### 4-1. 財務改善ROI

これは、本契約によって直接的に生み出される財務的なリターンを評価するものです。

*   **算出ロジック**:
    *   `期待される年間リターン ÷ 投資総額`
*   **リターンの根拠**:
    *   `KPI_インパクトテーブル` に基づく「**営業利益率の3.0〜4.5pt改善**」を前提としています。これは、主に委託費・人件費の変動費化など、コスト構造の最適化によって達成されると試算されています。

| 項目 | プランA：3ヶ月集中プラン | プランB：6ヶ月伴走プラン |
| :--- | :--- | :--- |
| **期待される年間リターン** | **約 310万円** | **約 460万円** |
| 投資総額 | 約 240万円 | 約 332万円 |
| **財務改善ROI（年間）** | **約 29%** | **約 39%** |

### 4-2. 時間創出ROI（機会費用）

これは、経営陣が雑務から解放され、より付加価値の高い業務に集中できる時間を金銭価値に換算したものです。

*   **算出ロジック**:
    *   `創出時間（月） × 時間単価 × 12ヶ月`
*   **前提条件**:
    *   対象: 社長およびコアメンバー3名
    *   時間単価: 15,000円／時間
    *   創出時間: プランA（月35時間）、プランB（月45時間）

| 項目 | プランA：3ヶ月集中プラン | プランB：6ヶ月伴走プラン |
| :--- | :--- | :--- |
| **創出される時間価値（年間）**| **約 630万円** | **約 810万円** |

### 4-3. 総合ROI

上記2つのリターンを合算し、投資全体に対する総合的な効果を評価します。

| 項目 | プランA：3ヶ月集中プラン | プランB：6ヶ月伴走プラン |
| :--- | :--- | :--- |
| **リターン合計（年間）** | **約 940万円** | **約 1,270万円** |
| 投資総額 | 約 240万円 | 約 332万円 |
| **総合ROI（年間）** | **約 291%** | **約 282%** |

## 5. 結論と意思決定のポイント

*   **ロジックの妥当性**:
    *   提案されている見積もりおよびROIのロジックは、計算間違いもなく、`KPI_インパクトテーブル` という具体的な分析に裏付けられており、**極めて健全かつ合理的**です。

*   **意思決定のポイント**:
    1.  **前提条件の精査**: ROIの妥当性は「営業利益率の改善」という前提に依存します。契約前に、その達成に向けた具体的な施策（例：どの委託費を見直すか）について、双方で目線を合わせることが重要です。
    2.  **成果報酬の定義**: 「年間削減効果額」の定義や測定方法、「RACIチャートの承認」の基準など、成果報酬の支払い条件を契約書上で明確に定義しておくことで、将来的な認識齟齬を防ぐことができます。
    3.  **プランの選択**:
        *   **プランA（3ヶ月）**: 短期間で課題を構造化し、具体的なアクションプランを策定することに主眼が置かれています。**「まず現状を正確に把握し、次の一手を決めたい」**フェーズに適しています。
        *   **プランB（6ヶ月）**: 計画策定後の実行支援や施策の定着化までをスコープに含んでいます。**「計画倒れを防ぎ、着実に成果を創出したい」**フェーズに適しています。
