## 1. 本要求案の目的

* 本書は、プロジェクト「組織基盤強化」のマイルストーン①（〜3ヶ月）の全タスク（`pj_2.0.3.スコープ・ベースライン.md`）を実行するために、**理想として必要なリソース（役割と工数）**を定義するものです。
* 本書は、プログラム計画の「フェーズ2（個別計画）」 で作成する素案であり、他プロジェクト（経営体質最適化、法人研修）との**リソース競合は考慮していません**。
* 本書は、次フェーズ「フェーズ3（統合・調整）」 において、プログラム全体のボトルネックを特定し、リソースの優先順位付けを行うための**インプット（要求）**として使用されます。

## 2. 見積もりの前提条件

1.  **対象タスク:** `pj_2.0.3.スコープ・ベースライン.md` に記載された全タスク（0.1〜3.4）を対象とする。
2.  **対象期間:** プロジェクト開始（W1）からマイルストーン①完了（W13）までの3ヶ月間。
3.  **工数定義（案）:** 1人日 = （案）8時間 とする。
4.  **インタビュー対象者:** タスク 1.2 のインタビュー対象者は、主要ステークホルダー5名（なーすけ様、中井様、紫音様、的場様、大川様） と仮定する。

## 4. 潜在的ボトルネック（リソースリスク）

本要求案（素案）の時点で、以下のリソースがプログラム全体のボトルネックとなる可能性があります。

1.  **【R1】インタビュー対象者（STH）:**
    * **リスク:** タスク 1.2（インタビュー実施） は、本プロジェクトの最重要インプットです。対象者（なーすけ様、紫音様、的場様など） のスケジュールが確保できず W1-W3 で実施できない場合、後続の全タスクが遅延します。
2.  **鈴木（コンサル / 支援）: （案）25.75 人日**
    * **最大リスク:** 本プロジェクトは3つの中で最も鈴木のリソースを必要とします。特にPhase 0のインタビュー(1.2)と分析、Phase 2の設計(3.1, 3.2)に作業が集中します。
    * **影響:** 他プロジェクト（特に「経営体質最適化」）の支援作業と重複するため、**フェーズ3（統合・調整）における最優先の競合調整対象**となります。
3.  **【R2】中井様（PM）: （案）9.5 人日**
    * **リスク:** 中井様はプログラム全体のPMを兼務 しており、負荷が集中します。レビュー（2.1, 2.4）や意思決定（2.3, 3.3） が滞留すると、プロジェクトのボトルネックとなります。
4.  **なーすけ様（スポンサー / 承認者）: （案）2.0 人日**
    * **リスク:** 工数は小さいですが、Quick Win対象の選定（2.3） やRACIの承認（3.4） など、プロジェクトの方向性を決める重要な意思決定が集中します。スケジュール確保が必須です。
