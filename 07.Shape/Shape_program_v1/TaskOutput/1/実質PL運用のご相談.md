# 経営判断高度化に向けた「管理会計（実質PL）」運用のご相談

-----

## 1\. 本ご相談の背景と目的

### 1.1. 背景

現在、弊社の事業は「フィットネス（店舗）」「イベント（SFF）」「メディア（YouTube）」と多角化が進んでおります。
一方で、現在の「税務会計上のPL」と、経営陣が肌感覚で持っている「事業の実力値（キャッシュ創出力）」の間には、**費用の計上方針による大きな乖離**が生じております。

### 1.2. 目的

税務申告用の決算書とは別に、経営陣が毎月の意思決定を行うための\*\*「管理会計（実質PL）」の仕組み**を整備したいと考えております。
つきましては、**「以下の経営数値が見える状態」\*\*を実現するために、貴所の専門的な見地から、会計ソフトの設定や記帳ルールの変更・運用方法についてご提案をいただきたく存じます。

-----

## 2\. 実現したいゴール（あるべきPLの構造）

経営陣として、以下の4段階の利益管理を行いたいと考えております。
現在の会計データから、可能な限りスムーズにこの数値を算出できる状態を目指します。

| 階層 | 管理したい利益 | 算出ロジック | 経営判断の目的 |
| :--- | :--- | :--- | :--- |
| **Lv.1** | **限界利益** | 売上高 - **変動費** | 事業の「基礎収益力」の把握（撤退基準） |
| **Lv.2** | **貢献利益** | 限界利益 - **事業直接固定費** | 現場責任者の評価（本社費配賦前の実力） |
| **Lv.3** | **事業利益** | 貢献利益 - **戦略配賦費** | 事業投資のROI評価（役員・専門家コスト回収度） |
| **Lv.4** | **全社利益** | 事業利益 - **全社共通費**<br>（※政策的支出控除後） | 会社全体の最終着地 |

-----

## 3\. 具体的なご相談事項（3つの論点）

上記PLを作成するにあたり、現在課題となっているのが以下の3点です。これらを解決するための「仕訳ルール」や「補助科目/部門設定」の運用についてご助言ください。

### 論点①：「業務委託費」の性質別・目的別分解【最重要】

**【現状の課題】**
現在、「業務委託費」等の科目の中に、\*\*「A：経営政策的な支出（関連会社への支払い等）」**と、**「B：純粋な事業経費（外部への支払い）」\*\*が混在しています。
これにより、事業本来の収益力（＝本来これだけ稼ぐ力があるはず、という実力値）が見えなくなっています。

**【実現したい状態】**
管理会計上、\*\*「A：経営政策的な支出」を利益に足し戻して（経費から除外して）\*\*評価を行いたいため、会計データ上で以下の性質を明確に区別したいです。

1.  **経営政策費:** 節税・利益圧縮等を目的とした支出（関連会社間取引など）
2.  **実質人件費:** 店舗運営に不可欠なシフト制スタッフ・トレーナーへの報酬
3.  **制作原価:** YouTube動画編集費など、売上・制作本数に連動する変動費
4.  **純粋外注費:** 上記以外のスポット依頼（コンサル、デザイン等）

> **ご相談:**
>
>   * 特に「1. 経営政策費」と「それ以外」が明確に分かれるよう、補助科目や取引先タグ等で区別する運用が可能でしょうか？
>   * また、「2. 人件費性質」や「3. 原価性質」のものも、同様にタグ付けや科目設定で区別可能でしょうか？

### 論点②：事業固有コストの「直接費」化

**【現状の課題】**
現在、「本社負担金」として処理されている費用の中に、本来は特定の事業（SFFや店舗）が負担すべき「事業固有のコスト」が含まれており、事業ごとの正しいコストが見えていません。

**【実現したい状態】**
以下の費用を、本社共通費ではなく「各事業の直接経費」として紐付けたいです。

  * 特定事業のみで使用しているシステム利用料（例：Amelio等）
  * 特定事業専任のバックオフィス業務委託費
  * イベント開催ごとのプロジェクト原価（会場費、賞金等）

> **ご相談:**
>
>   * これらを「部門別会計」機能を使って紐付けるのが最適でしょうか？
>   * 特にイベント（SFF）のような「プロジェクト単位」の収支管理に適した方法はありますでしょうか？

### 論点③：共通費配賦のルール見直し

**【現状の課題】**
地代家賃や税務顧問料などが一律で各事業に配賦されており、現場の収益実態が歪んで見えています。

**【実現したい状態】**
費用対効果を明確にするため、以下のルールで管理を行いたいです。

1.  **役員報酬・専門家報酬:** 均等配賦ではなく、「関与工数（タイムチャージ）」に応じて配賦先を変えたい。
2.  **インフラコスト（家賃・通信・顧問料）:** 無理に配賦せず、「全社共通費（配賦不能費）」として本社部門で一括管理したい。

> **ご相談:**
>
>   * 会計ソフト上で無理に配賦を行わず、データ出力後にExcel等で集計する運用が良いか、会計ソフトの配賦機能を活用すべきか、運用負荷の観点からアドバイスをいただきたいです。
