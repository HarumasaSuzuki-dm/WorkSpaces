# 会計事務所打ち合わせ用：論点確認チェックリスト
**目的:** 経営判断に必要なデータ粒度を確保するための「運用ルール」を確定させる。

---

## 1. 最優先：業務委託費の「分解」ルール (Must Have)
*ここが曖昧だと、損益分岐点が出せず、コスト削減の打ち手が見えなくなります。*
【ご回答】
* [ ] **★「業務委託費」の内訳を、会計ソフト上で区別できる運用に合意できたか？**
    * *理想の回答:* 「補助科目を設定しましょう（例：業務委託費_人件費、業務委託費_制作費）」
    * *妥協案:* 「科目は増やせませんが、摘要欄の先頭に【人件費】【制作】等のタグを入力するルールにしましょう」
    * *NG:* 「摘要欄を見て都度判断してください（現状維持）」
【ご回答】
* [ ] **★「変動費（売上連動）」と「固定費」の区別が可能になったか？**
    * YouTube編集費（本数連動）と、定額顧問料（固定）が混ざらない状態を作れるか確認する。
【ご回答】
* [ ] **トレーナー報酬（歩合 vs 固定）の処理ルールは決まったか？**
    * 同じトレーナーでも「セッションフィー（変動）」と「固定給（固定）」がある場合、どう入力し分けるか確認する。

## 2. 事業別収支の「紐付け」ルール (Must Have)
*ここを握らないと、SFF（イベント）や特定事業の正しいROIが出せません。*
【ご回答】
* [ ] **★「部門別会計」の適用範囲について合意できたか？**
    * 現状の部門（フィットネス、YouTubeなど）の設定が正しいか、見直しが必要か確認する。
【ご回答】
* [ ] **本社経費として埋もれている「特定ベンダー」の部門紐付けは可能か？**
    * 具体例：システム費用（Amelio）、コーチング費用（Ideate）など。
    * これらを「全社共通」ではなく、「フィットネス部門」「SFF部門」の直接経費として計上するフローにできるか。
【ご回答】
* [ ] **イベント（SFF）の「プロジェクト別管理」の方法はあるか？**
    * 「月」をまたぐイベント費用（会場手付金など）を、どうやって1つのイベント収支としてまとめるか、会計ソフトの機能（タグ、プロジェクトコード等）で対応可能か聞く。

## 3. 本社費配賦・共通費の扱い (High Priority)
*ここは会計ソフトでやるか、Excelでやるかの「割り切り」が重要です。*
【ご回答】
* [ ] **「配賦しない」という選択肢（全社共通部門の活用）は可能か？**
    * 家賃や税務顧問料を、無理にYouTube等へ配賦せず、「全社共通（管理本部）」という部門に残したままで良いか確認する。
    * *※管理会計上で配賦したい場合は、CSV出力後にExcel側で行う方針で合意する。*
【ご回答】
* [ ] **役員報酬の部門按分は変更可能か？**
    * 「期中で配賦率を変えると税務上ややこしいか？」を確認。
    * 管理会計用と税務用で分ける場合、会計ソフト上はどう入力するのがベストか（毎月手動で振替伝票を切るのか等）を聞く。

## 4. 運用負荷の確認 (Feasibility)
*ルールを作っても、運用が回らなければ意味がありません。*
【ご回答】
* [ ] **★この変更による「毎月の入力工数」は許容範囲か？**
    * しおん様（経理担当）の作業時間が倍増しないか？
    * 会計事務所側で代行してくれる範囲はどこまでか？（例：補助科目の設定作業など）
【ご回答】
* [ ] **過去データ（今期分）の修正は可能か？**
    * 今期の期首（または直近3ヶ月分）に遡って、今回の新ルールでデータを修正してもらえるか？（これがないと、今期の分析ができないため）
