## 1. Why: 背景と目的
* **背景**: 法人研修事業の立ち上げにあたり、サービスの有効性を客観的データで証明する最初のパイロット導入が最重要課題です。その最優先ターゲットとしてVEATM社を位置付けており、具体的な協業の検討を開始するためには、まず担当者との対話の場を設ける必要があります。
* **目的**: VEATM社とのパイロット導入に向けた具体的な協議を開始するため、担当者（キーパーソン）との初回ヒアリングをセットアップする。
* **インパクト**: 本タスクは、法人研修事業の成功モデルを確立するための最初のマイルストーン「M1.1: VEATM社からパイロット導入の承認を得る」に直結する、極めて重要な第一歩です。

## 2. What: 成果物と完了定義 (DoD)
* **提出形式**: なし（行動が成果となります）
* **完了の定義**: VEATM社の担当者と、パイロット導入に関する初回ヒアリングの**日程が確定し、関係者のカレンダー登録が完了している状態**。
* **品質期待値**:
    * [x] スピード優先（60%の完成度でOK） / [ ] クオリティ優先（ミス許容ゼロ）
    * 迅速な日程調整を最優先とします。

## 3. How: スコープとリソース
* **スコープ内**:
    * VEATM社の研修導入における意思決定者、またはそれに繋がるキーパーソンへのアプローチ。
    * 初回ヒアリングの日程調整と確定。
* **スコープ外**:
    * 初回ヒアリングでの具体的な提案内容の説明や交渉。（これは次タスク`3.2.2`で扱います）
    * 契約条件に関する協議。
