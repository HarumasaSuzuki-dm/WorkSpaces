# デジタル化・AI導入補助金2026 ネクストアクション チェックリスト
> 2026年3月最新版 ｜ 申込から交付決定まで約4〜6週間（入金まで最大6か月）
> Shape Fit株式会社（法人）・通常枠 想定

---

## STEP 1｜事前準備（締切の3週間前までに完了）

- [ ] **gBizIDプライム**を取得する（公式：gbiz-id.go.jp ／ 発行まで約2週間かかるため最優先で着手）
- [ ] **SECURITY ACTION「☆☆二つ星」**を宣言する（IPA公式サイト ／ 発行まで2〜3日 ／ ☆☆は加点要素にもなる）
- [ ] **みらデジ経営チェック**を実施してPDFを保存する（miradigi.go.jp ／ 約10分で完了 ／ 加点要素）
- [ ] 自社が**申請対象（中小企業・小規模事業者）**に該当するか確認する
- [ ] **みなし大企業**に該当しないか確認する（大企業が株式の1/2以上保有 等）
- [ ] IT導入補助金2022〜2025の**過去受給歴**を確認する（交付決定から12か月以内は通常枠に再申請不可）
- [ ] 過去に賃上げ計画で加点を受けた場合、**賃上げ要件（3.5%以上向上）**の達成状況を確認する

> **ポイント：** gBizIDの取得が最もリードタイムが長い（約2週間）。これが遅れると申請自体ができないため、すべてに先行して動き出すこと。

---

## STEP 2｜IT導入支援事業者・ツールの選定（締切の2週間前まで）

- [ ] 公式の「**ITツール検索**」で、事務局に登録済みのITツールを探す（it-shien.smrj.go.jp/search/）
- [ ] 導入したいツールが**AI機能を持つか**確認する（2026年度より絞り込み検索が可能）
- [ ] 導入ツールが**汎用プロセスのみでないこと**を確認する（「顧客対応・販売支援」「会計・財務」等の業務プロセスを1種類以上含むこと）
- [ ] IT導入支援事業者の**採択実績・採択率**を確認する（実績件数・採択率を数字で公開しているか確認）
- [ ] IT導入支援事業者の**費用体系を書面で確認**する（成功報酬型 / 固定報酬型 / ハイブリッド型）
- [ ] 支援事業者が**実績報告・効果報告までフォロー**するか確認する
- [ ] キックバック・ポイント還元の提案がある事業者は**即断る**（不正受給に該当）

> **ポイント：** 自社の課題に合わないツールを「補助金ありき」で提案してくる事業者に注意。提案理由とROIを必ず自分で確認すること。

---

## STEP 3｜書類の収集・作成（締切の1週間前まで）

### ▼ 法人が準備する書類（公募要領 p.18 準拠・3種類のみ）

- [ ] **履歴事項全部証明書**（発行から3ヶ月以内）を取得する
      （法務局の窓口・オンライン・郵送で申請可能）

- [ ] **法人税の納税証明書「その1」または「その2」**（直近分）を取得する
      （「その1」と「その2」はどちらか一方でOK。両方は不要）
      （税務署・e-Tax で取得可能）

- [ ] **直近分の貸借対照表及び損益計算書**を準備する
      （確定申告書一式・勘定科目明細書は不要。IT補助金には不要）

### ▼ 個人事業主が準備する書類（公募要領 p.18 準拠・4種類）

- [ ] **本人確認書類**（運転免許証・運転経歴証明書・住民票のいずれか1点）
      ※住民票は発行から3ヶ月以内のもの

- [ ] **所得税の納税証明書「その1」または「その2」**（直近分）

- [ ] **確定申告書の控え**（税務署が受領したことが確認できるもの）
      ※収受印または受付番号・受付日時の印字が必要

- [ ] **青色申告決算書または収支内訳書**（直近分）

### ▼ 該当者のみ追加書類

- [ ] **賃金状況報告シート（補助率引上げ・加点措置①用）**
      ※補助率2/3の適用を受ける場合（最低賃金近傍の事業者）

- [ ] **IT戦略ナビwith の実施結果（IT戦略マップ）PDF**
      ※加点8）を受ける場合。申請に使ったgBizIDプライムが確認できること

- [ ] **賃金状況報告シート（加点措置②用）**
      ※最低賃金＋63円以上の加点を受ける場合

### ▼ 書類準備の落とし穴チェック（公募要領より追加）

- [ ] 確定申告書に**収受印または電子受付の受付番号**があるか確認した
      ※税理士の印のみの書類は受理されない
- [ ] マイナンバー・保険者番号が記載されている書類は**黒塗り**した
- [ ] 全書類をPDF形式に変換した

### ▼ 支払方法の事前確認（公募要領 p.26 準拠）

- [ ] ITツール代金の支払いを**銀行振込（法人口座から）**で行う準備ができている
      ※クレジットカード払いの場合は**1回（一括）払いのみ**
- [ ] **現金払いは一切不可**であることを支援事業者と共有した
- [ ] **法人申請の場合は法人名義のカード**で支払うことを確認した


---

### ⚠️ 特にShape Fitとして確認が必要な論点

1. 申請額が150万円未満か以上かを先に決める
   → 150万円未満（初回）なら賃上げは加点のみ・必須ではない
   → 150万円以上なら賃上げ計画の策定・表明が申請要件になる

2. SECURITY ACTIONの宣言タイミング
   → 第1次（5/12締切）に申請するなら現行IDで可
   → 第2次（6/15締切）以降は新システムでの宣言が必要な可能性あり
      → 早めに第1次申請を目指すか、新システム移行を確認する

3. 過去のIT導入補助金受給歴の確認
   → 2022〜2025年度に受給歴がある場合、賃上げ要件・減点・12ヶ月制限の
      3つのリスクが発生する

## STEP 4｜申請マイページから提出

- [ ] IT導入支援事業者から**申請マイページへの招待**を受け取る
- [ ] 申請マイページに基本情報・書類を入力・添付する
- [ ] IT導入支援事業者がITツール情報・事業計画値を入力したことを確認する
- [ ] 申請内容を最終確認し、**自社担当者（gBizID保有者）**が宣誓・提出ボタンを押す
  - 支援事業者への完全丸投げは禁止。最終提出は必ず自分で行う

> **注意：** 締切は**17:00厳守**。直前はシステムが混雑するため、余裕を持って申請する。  
> 第1次締切：2026年5月12日 17:00 ／ 第2次：2026年6月15日 ／ 第3次：2026年7月21日

---

## STEP 5｜審査・交付決定（締切から約4〜6週間）

- [ ] 審査中の**追加資料依頼に即日〜翌日中に対応**する（遅いと審査が大幅遅延）
- [ ] 申請マイページのステータスと**メールを毎日確認**する
- [ ] 交付決定通知を受け取るまで、**発注・契約・支払いは絶対に行わない**

> ⚠️ **最重要ルール：** 交付決定前の発注・契約・支払いは補助金ゼロ。悪質な場合は返還命令＋年利10.95%の加算金＋5年間申請禁止＋刑事罰のリスクがある。

---

## STEP 6｜発注・導入・実績報告（交付決定後〜実績報告期限まで）

- [ ] 交付決定通知を確認した後、ITツールを**発注・契約・支払い**する
  - **補助金は後払い（完全事後精算）のため、一度全額を自社で立替える必要がある**
  - 導入費用200万円・補助率1/2の場合、200万円を先払いし後から100万円が入金される
- [ ] ITツールの導入・運用を開始する
- [ ] 以下の**証憑書類を整理・保管**する
  - 請求書（ITツール名・数量・単価・請求日が必須）
  - 支払証憑（ネットバンキングは「送金完了ステータス」のみ有効。「承認待ち」はNG）
  - ソフトウェア利用確認のスクリーンショット（ログインIDが確認できるもの）
  - 補助金受取口座の通帳写し（表紙＋取引ページ）
- [ ] 申請マイページから**実績報告を提出**する（補助事業者側から開始すること）
- [ ] **補助金額確定後のSMS承認**を失念しない（承認しないと入金されない）

> **ポイント：** 実績報告から入金まで約1.5〜2.5か月かかる。交付決定〜入金まで最大5〜6か月の資金繰りを事前に手当てしておくこと。

---

## STEP 7｜効果報告（入金後も3年間継続）

- [ ] 事業終了後**3年間、年1回（計3回）**の効果報告を申請マイページから提出する
- [ ] 報告期間中は**ITツールを解約・利用停止しない**（解約すると返還請求の対象になる）
- [ ] 報告内容（給与支給総額・売上高・従業員数・ツール活用状況）を毎年集計する
- [ ] 賃上げ加点を受けた場合、**3年度目に賃上げ要件の達成状況**を報告する（未達の場合は補助金の一部返還）

---

## 覚えておきたい4つのポイント

1. **gBizIDは最優先で動く** — 取得まで約2週間。これが遅れると申請できない。
2. **交付決定前は発注・契約・支払い禁止** — 補助金ゼロ＋最悪刑事罰のリスクがある絶対ルール。
3. **全額立替が必要** — 補助金は後払い。導入費用の全額を一時的に自社で用意できるか事前に確認。
4. **3年間はツールを使い続ける義務がある** — 導入後の解約は返還リスクに直結。
