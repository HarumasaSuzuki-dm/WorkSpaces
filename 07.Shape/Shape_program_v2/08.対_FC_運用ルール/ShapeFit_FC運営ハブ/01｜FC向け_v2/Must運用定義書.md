## 4つのMust 一覧

| # | Must | ひとことで | 対象者 |
|---|------|----------|--------|
| 1 | セッションログ完全提出 | お客様との会話を記録 | トレーナー全員 |

## 1. セッションログ完全提出

### なぜやる？
お客様の変化を「チームの記録」にし、全員で学ぶため。

### いつ？
**毎セッション後**（当日中が基本、無理なら翌日午前中まで）

### 入力項目

| 項目 | 必須 |
|------|------|
| 日付・店舗名・トレーナー名 | ○ |
| お客様名（イニシャル可） | ○ |
| お客様の言葉（そのまま書く） | ○ |
| 本音・弱みの開示（あり/なし/分からない） | ○ |
| 開示の内容（「あり」の場合） | △ |
| 次の一歩 | ○ |

> **「本音・弱みの開示」とは？**
> 「本当はこう思ってた」「実はこれが怖い」など、普段言わないことを話してくれた状態。
> 迷ったら「分からない」でOK。

### 確認方法
週1回、管理シートで提出率をチェック → 目標：**100%**

### 例外
- システム障害 → 事務局Aに連絡、復旧後に提出
- 体調不良 → 事務局Aに連絡、復帰後に提出

### Q&A
**Q. 1日に何件もある場合は？** → 1セッションごとに1回提出  
**Q. お客様があまり話さなかった？** → 「なし」で出してOK

## 2. 24時間以内レスポンス

### なぜやる？
連絡が止まるとチーム全体の仕事が止まるから。

### 対象
**事務局（本部）からの連絡**（全体連絡、個別DM、確認依頼など）

### 24時間のカウント
連絡が届いた時刻から24時間以内（土日祝含む）

### 何をすればいい？

| OK | NG |
|----|----|
| 返信コメント / スタンプ（👍✅） | 既読スルー |
| 「確認します、◯日までに返信します」 | 24時間過ぎてから反応 |

> **ポイント**：完璧な回答じゃなくていい。「見ました」の意思表示が大事。

### 確認方法
週1回、未対応をチェック → 目標：**未対応ゼロ**

### 例外
- 長期離脱 → 事前に事務局Aへ連絡、代理対応者を決める
- 急病 → 復帰後すぐに事務局Aへ連絡

### Q&A
**Q. 深夜に届いた連絡も？** → 翌日中に反応すればOK  
**Q. すぐ回答できない？** → 「確認中、◯日までに返信」と一言入れればクリア

## 3. 状態を測るNPS実施

### なぜやる？
体重だけでなく「本当の変化」を測り、ShapeFitらしさを形にするため。

### いつ？
**月1回**、月末に実施（具体的な日程は各店舗で決定）

### 質問項目

| # | 質問 | 形式 |
|---|------|------|
| 1 | ShapeFitを友人にすすめたいと思いますか？ | 0〜10 |
| 2 | 最近、体や生活の変化を感じていますか？ | はい/少し/いいえ |
| 3 | トレーナーに話したいことを話せていますか？ | はい/少し/いいえ |
| 4 | 他に伝えたいことがあれば | 自由記述 |

### 対象
継続中の全お客様（新規は翌月から）

### 確認方法
月1回、実施有無をチェック → 目標：**全店舗で実施**

### 例外
- 回答拒否 → 無理強いせず「未回答」と記録
- 月末にセッションなし → LINEで送付、回答なしでも「送付済み」と記録

### Q&A
**Q. 結果が悪かったら？** → 2月は「実施したか」だけを見ます  
**Q. 紙とフォームどっち？** → お客様が答えやすい方でOK

## 4. オーナー現場対話義務

### なぜやる？
トレーナーの悩みを早めにキャッチし、理念と現場のズレを防ぐため。

### いつ？
**週1回以上**、自店舗のトレーナーと1対1で話す

### 最低ライン
- 1回15分以上
- 話した内容を記録シートに入力

### 話すテーマ例
- 最近のセッションで困っていること
- お客様のことで気になっていること
- 働き方や体調

> **ポイント**：「評価面談」ではなく「気軽に話せる時間」として設計

### 記録項目
日付 / 相手 / 主な話題 / 気になったこと / 次のアクション

### 確認方法
週1回、実施有無をチェック → 目標：**毎週1人以上と対話**

### 例外
- オーナー不在 → 事前に事務局Aへ連絡、免除 or リモート実施
- トレーナー急休 → 翌週にまとめて実施

### Q&A
**Q. 1人しかいない店舗は？** → その1人と毎週話す  
**Q. 立ち話でもOK？** → 15分以上、1対1なら可。記録は必須
