# 例外申請ガイド

> **2月はテスト期間**：申請があれば原則OKです。記録を残すことが目的です。

---

## 申請が必要なケース・不要なケース

| 不要（連絡だけでOK） | 必要（申請してください） |
|---------------------|------------------------|
| システム障害 | 左に当てはまらない想定外の事情 |
| 体調不良 | 例：店舗トラブルで1週間対応できない |
| 事前に伝えた出張・旅行 | 例：家庭の事情で2週間レスポンスが難しい |
| トレーナー・顧客の急な欠席 | |

> **迷ったら**：まず事務局A（大川さん）に相談 → 「申請して」と言われたら申請

---

## 申請の仕方

**連絡先**：事務局A（大川さん）へSlack DM

**伝えること**（これだけ）：
```
【例外申請】
・Must：（どのMust？）
・期間：（いつからいつまで？）
・理由：（なぜ？）
```

**回答**：24時間以内に届きます

---

## 結果の種類

| 結果 | 意味 |
|------|------|
| OK | そのまま例外を認めます |
| 条件つきOK | 条件（例：◯日までに対応）を守れば認めます |
| NG | 2月は基本的に出しません |

---

## 守ってほしいこと（2つだけ）

1. **結果が届く前に、勝手にMustを破らない**
2. **条件つきOKの場合、条件と期限を守る**

---

**何かあったら → 事務局A（大川さん）へSlack DM**
