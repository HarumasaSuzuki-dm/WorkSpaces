## 📌 設計メモ・引き継ぎ事項

### ▸ BGMについて
2025年では `03_台本/04_CHIBA` 配下に `BGM_2025千葉大会以降` フォルダが混入していた。
BGMは台本フォルダとは管理目的が異なるため、2026年では **台本フォルダには格納しない**。
BGM素材が必要な場合は別途「音響・映像素材」ドライブを用意すること。

### ▸ ファイル命名規則について
2025年は大会ごとに命名ルールが統一されておらず、以下の混在が発生していた。

| 大会                   | 記号の例            | ファイル名例                                 |
| ---------------------- | ------------------- | -------------------------------------------- |
| ALL / KANAGAWA / CHIBA | ✅（チェックマーク） | `✅全体共通`, `✅【受付】スタッフ用マニュアル` |
| TOKYO                  | ★（星印）           | `★【受付】スタッフ用マニュアル`              |
| OSAKA / FUKUOKA        | 記号なし            | `【受付】スタッフ用マニュアル`               |
| 内部                   | 絵文字              | `🎤大会当日 朝礼スクリプト（完全版）`         |

**2026年からの統一ルール：**
- 役割マニュアル本体：`【役割名】スタッフ用マニュアル`（記号なし・括弧統一）
- 配布用PDF：`【役割名】スタッフ用マニュアル.pdf`
- 大会固有ファイル：`ファイル名（関西）` / `ファイル名（関東）`
- テンプレート：`【テンプレ】ファイル名`
- 絵文字（✅ ★ 🎤）は **使用しない**（PDF・Slack共有時の文字化けリスクあり）

### ▸ コメント付きマニュアルについて
2025年の `マニュアルデータ（コメント付き）` フォルダは、当日スタッフからフィードバックをもらったマニュアルにGoogleドキュメントのコメントを残したもの。
`【参照】2025フィードバック版マニュアル` フォルダに格納し、2026年のマニュアル改訂時の参考資料として残す。

### ▸ 東京大会担当者修正版について
`※紫音修正※マニュアル【TOKYO】`（2025）は、TOKYO大会向けに内容を最新化した修正版。
2026年マニュアル改訂時に内容を参照すること。新ディレクトリ構造では独立ファイルとしては不要。
