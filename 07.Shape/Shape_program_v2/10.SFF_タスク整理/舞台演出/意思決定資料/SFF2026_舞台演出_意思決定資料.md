# SFF2026 舞台演出 意思決定資料
**作成日**: 2026年4月3日

---

## 1. まず確認：この資料でどんなことを決めるの？

SFF2026では、ステージに **「LEDビジョン（大型映像スクリーン）」** を設置するかどうかを検討しています。

LEDビジョンとは、コンサートやスポーツ大会などで見る**ステージ上の大型スクリーン**のことです。  
競技映像をリアルタイムで映したり、スポンサーロゴや演出動画を表示したりするために使います。

この資料では **「どのプランを選ぶか」** または **「そもそも導入するかどうか」** を決めることを目的としています。

## 3. どんな選択肢があるの？

ステージ上への設置場所と画面の数によって、以下の4つの選択肢があります。

---

### ❌ Option 0｜LEDなし

```
┌─────────────────────────────────────────────────┐
│  （暗幕・照明のみ。スクリーンなし）               │
│                                                 │
│            ▌司会者・選手▐                       │
└─────────────────────────────────────────────────┘
```

**どんなイメージ？**  
照明と音響だけのシンプルなステージ。  
2025年の大阪・神奈川大会と同じような演出規模感です。

**費用（税別）**：演出費合計 **250万円**

---

### ✅ Option 1｜プランB ← 推奨

```
┌─────────────────────────────────────────────────┐
│  ┌──────────┐              ┌──────────┐         │
│  │ W6m×H3m  │              │ W6m×H3m  │         │
│  │  スクリーン│             │  スクリーン│         │
│  └──────────┘              └──────────┘         │
│               ▌司会者・選手▐                     │
└─────────────────────────────────────────────────┘
```

**どんなイメージ？**  
ステージの左右に6m×3mのスクリーンを1枚ずつ設置。  
どの座席からも見やすく、左右対称でバランスの良い構成です。  
コンサートやフィットネス大会でよく見られるスタンダードな配置です。

**費用（税別）**：LED費 460万円 ＋ 基本演出 250万円 ＝ **演出費合計 710万円**  
**見積番号**：Q-0000000049

### Option 2｜プランA

```
┌─────────────────────────────────────────────────┐
```

**費用（税別）**：LED費 480万円 ＋ 基本演出 250万円 ＝ **演出費合計 730万円**  
**見積番号**：Q-0000000048

## 5. 費用が予算に与える影響

### 現在の予算（シミュレーションv7）との比較

現在の予算シートでは、**WESTにのみLEDが計上されており、EASTにはLEDが入っていません。**

| 現在の予算内訳 | 金額（税別） |
|---|---|
| WEST（PJT1）演出費 | 912万円（基本250万＋旧LED見積もり662万） |
| EAST（PJT2）演出費 | 250万円（LEDなし） |
| **2大会合計（現予算）** | **1,162万円** |

### 今回の新見積もりに切り替えた場合の比較

| パターン | 2大会合計 | 現予算との差 |
|---|---|---|
| 両大会ともLEDなし | 500万円 | **▲662万円のコスト削減** |
| **WESTのみLED・プランB（推奨）** | **960万円** | **▲202万円のコスト削減** |
| 両大会ともLED・プランB想定 | 1,420万円 | **＋258万円の予算超過** |

> ✅ **重要なポイント**：現在の予算には旧見積もり（662万円）が入っているため、  
> 今回の新見積もり（プランB・460万円）に変えるだけで **WESTだけで202万円削減**できます。

## 6. 財務インパクト（2大会合計）

| | 両大会LEDなし | **WESTのみLED（推奨）** | 両大会LED |
|---|---|---|---|
| 2大会 売上合計 | 5,453万円 | 5,453万円 | 5,453万円 |
| 2大会 演出費合計 | 500万円 | 960万円 | 1,420万円 |
| 演出費以外のコスト | 4,366万円 | 4,366万円 | 4,366万円 |
| **2大会 限界利益** | **587万円** | **127万円** | **▲333万円** |
| **限界利益率** | **10.8%** | **2.3%** | **▲6.1%** |

> ※ EAST の LED費はプランB相当（460万円）を使った想定値。実際の見積もりは別途取得が必要。  
> ※ 限界利益とは「売上からコストを引いた残り（粗利益）」のことです。両大会LEDにすると赤字になります。

## 7. 推奨と次のアクション

### 推奨：**WESTのみ・プランB（Option 1）で確定する**

| 判断軸 | 内容 |
|---|---|
| 費用 | 現予算より202万円削減でき、予算内で収まる唯一のLED導入案 |
| 演出 | パープルホール1,400名規模には十分な迫力。左右対称で全席から見やすい |
| 財務 | 両大会LEDにすると限界利益がマイナスに。2025年度赤字着地の状況では避けたい |
| 進め方 | WESTをまず確定 → EASTはひらしんの見積もりを取得してから改めて判断 |
