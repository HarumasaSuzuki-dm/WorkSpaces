## 収集してほしい情報

### 1. 基本情報

| 項目           | 調べてほしいこと                                     |
| -------------- | ---------------------------------------------------- |
| **正式社名**   | 株式会社○○など、登記上の正式名称                     |
| **会社HP**     | 公式サイトのURL                                      |
| **本社所在地** | 都道府県・市区町村レベルでOK                         |
| **地域区分**   | 上記から判断して「関東」「関西」「その他」のいずれか |

### 2. 事業内容

| 項目                   | 調べてほしいこと                                             |
| ---------------------- | ------------------------------------------------------------ |
| **業種**               | 主な事業領域（食品/アパレル/IT/美容/医療/フィットネス など） |
| **主力商品・サービス** | 代表的な商品名やサービス名を3つ程度                          |
| **ターゲット顧客層**   | BtoB/BtoC、年齢層、性別など分かる範囲で                      |
| **企業の強み・特徴**   | 他社と差別化しているポイント                                 |

### 3. 協賛適性の判断材料

| 項目                             | 調べてほしいこと                                     |
| -------------------------------- | ---------------------------------------------------- |
| **フィットネス・健康との関連性** | 健康食品、スポーツ用品、美容など関連があるか         |
| **過去のスポンサー実績**         | 他イベントへの協賛歴があれば（なければ「情報なし」） |
| **SNS/メディア露出**             | Instagram、X(Twitter)のフォロワー数や発信内容の傾向  |

### 4. 担当者・決裁者情報

| 項目                | 調べてほしいこと                         |
| ------------------- | ---------------------------------------- |
| **代表者名**        | 代表取締役の氏名                         |
| **広報/マーケ担当** | 問い合わせ先やプレスリリースから分かれば |
| **会社規模感**      | 従業員数、資本金など分かる範囲で         |

## 目標プランの推定

上記の情報をもとに、以下の観点から**推奨する協賛プラン**を提案してください。

### プラン判断の基準

| プラン       | 金額（税抜） | 向いている企業の特徴                                         |
| ------------ | ------------ | ------------------------------------------------------------ |
| **TOP**      | 500万円      | 大手企業、フィットネス業界のリーダー、全国展開している企業   |
| **DIAMOND**  | 70万円       | 中堅企業、健康・美容関連、地域密着で知名度向上を狙いたい企業 |
| **PLATINUM** | 30万円       | 小規模企業、新規参入企業、まずは試してみたい企業             |
| **物品協賛** | 1.5万円〜    | サンプリングしたい商品がある企業、現金支出を抑えたい企業     |

### 推定時に考慮してほしいポイント

1. **商品とSFF参加者の相性**: フィットネス愛好家（20〜40代、健康意識高い）に刺さるか
2. **企業規模と予算感**: 売上規模や従業員数から無理のない金額か
3. **協賛で得られるメリット**: ブランディング重視ならTOP/DIAMOND、販促重視ならブース出店可能なプラン
4. **物品協賛の可能性**: サンプリングに適した商品（プロテイン、健康食品、サプリ等）があるか
