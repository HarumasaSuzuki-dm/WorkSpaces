# UGC（投稿）ポリシー

---

## はじめに

Shapeコミュニティ（以下「本コミュニティ」）は、「1人ひとりの挑戦と物語を、共に育てる」というミッションのもと、挑戦が続く人を増やすための場として運営されています。

本ポリシーは、すべての参加者が安心して挑戦を続けられる環境を維持するため、投稿コンテンツおよびコミュニティ内での行動に関するルールを定めるものです。

**私たちが大切にしていること：**

- 挑戦を称える（結果より選択を尊重する）
- 物語を言語化する（感情を次の一歩に変換する）
- 比較を手放す（勝ち負けの場にしない）
- 混ざり合う（熱量が循環する）
- 戻ってこれる（途切れを責めない）

これらの価値観に基づき、本ポリシーを遵守いただくことで、全員にとって「火を灯し、守り、広げる」場を共に作っていきましょう。

## 第1章　適用範囲と定義

### 1.1 適用範囲

本ポリシーは、以下のすべてに適用されます。

- **オンライン活動**：コミュニティプラットフォーム上でのすべての投稿、コメント、ダイレクトメッセージ、プロフィール、チャット
- **オフライン活動**：本コミュニティが主催または連携するイベント、セッション、ワークショップ
- **関連プログラム**：語る時間、言語化時間、その他の公式プログラム
- **外部連携**：本コミュニティが公認する提携コミュニティ、グループ活動

### 1.2 用語の定義

| 用語 | 定義 |
| --- | --- |
| UGC（ユーザー生成コンテンツ） | 参加者が作成・投稿するすべてのコンテンツ（テキスト、画像、動画、音声、リンク、絵文字、リアクション等） |
| 禁止コンテンツ | 本ポリシーで投稿・共有が禁止されているコンテンツ |
| モデレーター | 運営から権限を付与され、コミュニティの健全性維持に協力する参加者 |
| Roots（リーダー層） | 自分の火で燃え、発信・伴走ができる参加者 |
| フォロワー層 | 誰かと一緒に燃え、共感・参加ができる参加者 |
| 火種（ワナビーズ）層 | まだ積極的な行動には移せないが、参加意欲のある参加者 |
| 未成年 | 18歳未満の者 |
| 個人情報 | 特定の個人を識別できる情報（氏名、住所、電話番号、メールアドレス、顔写真等） |
| 機微情報 | 病歴、信条、犯罪歴など特に配慮が必要な個人情報 |

### 1.3 外部リンク・第三者サービスについて

参加者が投稿する外部リンク先のコンテンツについて、運営は内容を保証せず、責任を負いません。ただし、明らかに有害なリンクを発見した場合は削除対応を行います。

## 第2章　コンテンツ基準

本コミュニティでは、以下のコンテンツの投稿を禁止します。

### 2.1 違法・危険なコンテンツ

- 違法行為の助長、教唆、または実行を促す内容
- 薬物（違法薬物、処方薬の不正使用含む）の売買、使用の推奨
- 武器、危険物の製造方法や入手方法に関する情報
- 詐欺、マルチ商法、違法な金融商品の勧誘
- 危険なダイエット方法、健康を害する極端な食事制限の推奨

### 2.2 暴力・残虐なコンテンツ

- 暴力行為を美化、推奨、または詳細に描写する内容
- 残虐な画像、動画、または詳細な描写
- 特定の個人または集団に対する脅迫、威嚇
- 身体的危害を加えることを示唆する内容

### 2.3 ヘイトスピーチ・差別

- 人種、民族、国籍、宗教、性別、性的指向、障がい、疾病、社会的身分に基づく差別的表現
- 特定の属性を持つ集団への憎悪を煽る内容
- 差別を正当化または助長する言説

### 2.4 ハラスメント

- 特定の個人に対する継続的な嫌がらせ、つきまとい
- 侮辱、中傷、名誉毀損に該当する表現
- 他の参加者を萎縮させる攻撃的な言動
- 荒らし行為（議論を妨害する目的での投稿）

### 2.5 性的コンテンツ

- 露骨な性的表現、ポルノグラフィ
- 性的な勧誘、出会い目的の投稿
- 未成年者を性的対象とするいかなる表現も（これは特に厳格に禁止します）
- 同意のない性的画像の共有

### 2.6 自傷・自殺に関するコンテンツ

- 自傷行為の方法を詳細に説明する内容
- 自殺を美化、推奨する内容
- 摂食障害を助長する内容

**重要：** 参加者またはその周囲の方が危機的状況にある場合は、速やかに専門機関への相談をお願いいたします。運営に相談いただいた場合は、適切な相談窓口をご案内します。

### 2.7 誤情報・なりすまし

- 健康・安全に関する虚偽の情報
- 他者になりすました投稿
- 意図的に誤解を招く内容
- 実在しない資格、実績の詐称

## 第3章　行動規範

### 3.1 基本姿勢：Shapeコミュニティの5つの約束

本コミュニティでは、以下の行動規範を遵守してください。

### 1. 挑戦を称える

- 誰かの挑戦を笑わない、否定しない
- 結果ではなく「選んだこと」を尊重する
- 小さな一歩も「成果」として認め合う

### 2. 比較しない

- 他の参加者との比較でマウントを取らない
- 自分のペースは自分のもの、他者のペースは他者のもの
- 「自分のものさし」で変化を測る

### 3. 安心して弱さを見せられる場を守る

- 弱っている人を責めない
- 火が小さい時期があることを前提とする
- 「離脱→復帰」は自然なこと

### 4. 熱量を循環させる

- 一方的に「引っ張る」のではなく「混ざる」姿勢
- 受け取るだけでなく、与えることも意識する
- 固定的な上下関係を作らない

### 5. 言葉にして、次の一歩へ

- 感情や詰まりを放置せず言語化する
- 批判ではなく建設的な対話を心がける
- 相手の言葉に耳を傾け、理解しようとする

### 3.2 禁止される行為

- **ストーカー行為**：特定の参加者を継続的に追跡、監視する行為
- **威圧的行為**：立場や経験を利用して他者を萎縮させる行為
- **過度な煽り**：議論の建設性を損なう挑発的な言動
- **運営妨害**：モデレーターや運営の指示に故意に従わない行為
- **勧誘活動**：他のサービス、商品、宗教、政治活動等への勧誘

### 3.3 参加者間のトラブル

- 参加者間でトラブルが発生した場合は、運営にご相談ください
- 当事者間でのDM上での解決を試みる前に、まず運営への報告を推奨します
- 報復行為は禁止します

## 第4章　知的財産権・権利保護

### 4.1 著作権の尊重

- 他者の著作物（文章、画像、動画、音楽等）を無断で転載しないでください
- 引用する場合は、出典を明記し、引用の範囲を超えないようにしてください
- 海賊版コンテンツの共有は禁止します

### 4.2 商標・ブランドの保護

- 他社の商標を不正に使用しないでください
- Shapeのロゴ、名称を無断で商業利用しないでください
- 公式と誤認させるような表現は禁止します

### 4.3 肖像権の尊重

- 他の参加者を撮影・録画する場合は、必ず事前に本人の許諾を得てください
- オフラインイベントでの撮影ルールは、各イベントの案内に従ってください
- 許諾なく撮影・公開された場合は、運営にご報告ください

### 4.4 UGCの権利帰属とライセンス

### 投稿者の権利

- 投稿したコンテンツの著作権は、原則として投稿者に帰属します
- 投稿者は自身のコンテンツについて責任を負います

### 運営への利用許諾

本コミュニティに投稿することにより、投稿者は運営に対し、以下の非独占的ライセンスを付与するものとします：

- **表示・配信**：本コミュニティのサービス提供に必要な範囲での表示・配信
- **プロモーション利用**：本コミュニティの紹介・宣伝目的での利用（SNS、ウェブサイト等）
- **編集・翻訳**：サービス向上のための軽微な編集、翻訳
- **アーカイブ**：サービス運営に必要なバックアップ・保存

※プロモーション利用の際は、可能な限り投稿者への事前連絡を行うよう努めます
※投稿者から削除依頼があった場合、合理的な期間内に対応します

### 4.5 権利侵害の申立て

著作権その他の権利侵害を発見した場合は、以下の情報とともに運営までご連絡ください：

- 権利を侵害していると思われるコンテンツの特定情報
- 権利者であることの証明
- 連絡先情報

## 第5章　プライバシー・個人情報保護

### 5.1 投稿禁止の個人情報

以下の情報は、ご自身のものであっても公開投稿での共有は推奨しません。他者の情報については投稿を禁止します。

- 住所（詳細な所在地）
- 電話番号、メールアドレス
- 勤務先の詳細情報
- 金融情報（口座番号、クレジットカード情報等）
- マイナンバー、パスポート番号等の公的識別番号
- リアルタイムの位置情報

### 5.2 ドキシング（晒し行為）の禁止

以下の行為は厳格に禁止します：

- 他者の個人情報を本人の同意なく公開する行為
- 他者を特定しようとする行為（「特定班」的行為）
- 個人情報を用いた脅迫、嫌がらせ

### 5.3 未成年者の個人情報

未成年者の個人情報については、特に厳格な保護を行います。未成年者の個人情報を含む投稿を発見した場合は、直ちに運営にご報告ください。

### 5.4 公開範囲と自己責任

- 投稿時の公開範囲設定は、投稿者の責任で適切に行ってください
- 一度公開した情報は完全な削除が困難な場合があることをご理解ください
- プロフィール情報は必要最小限の記載を推奨します

## 第6章　プラットフォームの健全性維持

### 6.1 スパム・宣伝行為

- 同一または類似の内容を繰り返し投稿する行為は禁止します
- 外部サービス、商品の宣伝は原則禁止です
- アフィリエイトリンクの投稿は禁止します
- 運営が許可した場合を除き、自己宣伝目的のリンク投稿は控えてください

### 6.2 不正行為

以下の行為を禁止します：

- 複数アカウントの作成・運用（なりすまし、BAN逃れ等）
- ボット、自動化ツールを用いた投稿
- 他者のアカウントへの不正アクセス
- システムの脆弱性を悪用する行為

### 6.3 フィッシング・詐欺的行為

- 個人情報を騙し取ろうとする行為
- マルウェアを含むリンクの投稿
- 詐欺的な投資、ビジネスへの誘導

## 第7章　安全設計

### 7.1 センシティブコンテンツの取り扱い

以下のコンテンツには適切な注意喚起を付けてください：

- 身体変化に関する画像（Before/After等）
- 食事内容、カロリーに関する詳細な情報
- トレーニング中の怪我、身体的負担に関する内容

### 7.2 危険投稿への対応

運営は、以下のコンテンツに対して露出制限を行う場合があります：

- 他の参加者に悪影響を与える可能性のあるコンテンツ
- 議論を呼ぶ可能性のあるコンテンツ
- 事実確認が必要なコンテンツ

### 7.3 位置情報・活動ログ

- ランニングルート、ジムの位置など、生活圏を特定できる情報の共有は慎重に行ってください
- リアルタイムでの位置共有は推奨しません

## 第8章　AI生成コンテンツ

### 8.1 許容範囲

- AI生成ツール（ChatGPT、画像生成AI等）を活用したコンテンツの投稿は、以下の条件で許容します：
    - AI生成であることを明示すること
    - 内容の正確性を投稿者が確認していること
    - 本ポリシーの他の規定に違反していないこと

### 8.2 禁止事項

- AI生成コンテンツを人間が作成したものと偽る行為
- AIを使用した大量スパム投稿
- 誤情報を含むAI生成コンテンツの無批判な共有

### 8.3 AI学習利用について

本コミュニティへの投稿が、運営以外の第三者によるAI学習に利用されることはありません。運営がAI学習目的で投稿を利用する場合は、別途同意を求めます。

## 第9章　モデレーション運用

### 9.1 監視体制

本コミュニティは、以下の方法でコンテンツを監視します：

- **自動検知**：特定のキーワード、パターンの自動検出
- **人的確認**：運営スタッフおよびモデレーターによる巡回
- **参加者からの通報**：通報機能による報告

### 9.2 判断基準

コンテンツの判断にあたっては、以下を考慮します：

- 投稿の文脈と意図
- コミュニティへの影響
- 本ポリシーの趣旨との適合性

### 9.3 例外的な取り扱い

以下の場合、通常は禁止されるコンテンツでも例外的に許容されることがあります：

- 教育・啓発目的での引用（適切な注意喚起付き）
- ニュース・報道の共有（出典明記）
- 体験談の共有（他者を傷つける意図がないもの）

### 9.4 措置の段階

違反が認められた場合、以下の段階で対応します：

| 段階 | 措置内容 | 適用場面 |
| --- | --- | --- |
| 第1段階 | 注意・警告 | 軽微な違反、初回違反 |
| 第2段階 | コンテンツ削除 | 明確な違反 |
| 第3段階 | 一時的機能制限 | 繰り返しの違反 |
| 第4段階 | 一時的参加停止 | 重大な違反、改善が見られない場合 |
| 第5段階 | 永久BAN | 極めて重大な違反、悪質な繰り返し |

### 9.5 即時措置

以下の場合は、警告なく即時に措置を講じることがあります：

- 法令に違反するコンテンツ
- 未成年者の安全を脅かすコンテンツ
- 他の参加者に重大な被害を与える可能性のあるコンテンツ
- なりすまし、詐欺行為

### 9.6 復帰条件

一時停止からの復帰には、以下が必要です：

- 停止期間の経過
- 違反内容の理解と再発防止の意思表示
- 運営との面談（必要に応じて）

## 第10章　通報・異議申立て

### 10.1 通報方法

違反を発見した場合は、以下の方法でご報告ください：

- **投稿単位**：各投稿の「報告」ボタン
- **ユーザー単位**：プロフィールの「報告」機能
- **運営への直接連絡**：指定の連絡窓口

### 10.2 通報時にご提供いただく情報

- 違反と思われるコンテンツの特定情報
- 違反の種類
- その他参考情報

### 10.3 対応目安

- 通報受領後、原則として72時間以内に初動対応を行います
- 複雑なケースについては、調査に時間を要する場合があります
- 対応結果の詳細は、プライバシー保護の観点からお伝えできない場合があります

### 10.4 異議申立て

措置に不服がある場合は、以下の手続きで異議申立てができます：

- **申立て期限**：措置の通知から14日以内
- **申立て方法**：指定の窓口への書面（メール可）
- **記載事項**：措置の内容、異議の理由、関連する証拠

異議申立てがあった場合、運営は再審査を行い、その結果を通知します。

### 10.5 虚偽通報への対応

悪意を持って虚偽の通報を繰り返す行為は、それ自体が違反行為として措置の対象となります。

## 第11章　法令遵守

### 11.1 準拠法・管轄

本ポリシーは日本法に準拠し、本ポリシーに関する紛争は東京地方裁判所を第一審の専属的合意管轄裁判所とします。

### 11.2 法的対応

以下の場合、運営は関係機関への通報、法的手続きへの協力を行うことがあります：

- 犯罪行為またはその疑い
- 裁判所、捜査機関からの正式な要請
- 他者の生命、身体に対する重大な危険

### 11.3 違法コンテンツへの対応

プロバイダ責任制限法その他の法令に基づき、違法コンテンツの削除、発信者情報の開示を行う場合があります。

変更は以下の方法で周知します：

- コミュニティ内での告知
- メール通知（重要な変更の場合）
- 本ポリシーの改定日の更新

### 12.3 継続利用による同意

ポリシー変更後も本コミュニティを継続利用された場合、変更後のポリシーに同意したものとみなします。

## 付録：具体的な事例ガイド

### これはOK ✓

- 自分の挑戦や目標を共有する
- トレーニングの成果や学びを投稿する
- 他の参加者の投稿に励ましのコメントをする
- 困っていることを相談する
- 自分の体験から得た気づきを言語化して共有する
- Before/After写真を適切な注意書きとともに投稿する

### これはNG ✗

- 他の参加者と自分を比較してマウントを取る
- 「○○さんより私の方が…」という投稿
- 科学的根拠のない極端なダイエット法の推奨
- 他の参加者の写真を無断で転載する
- コミュニティ外のサービスへ繰り返し誘導する
- 議論を攻撃的にする、相手を否定する発言
