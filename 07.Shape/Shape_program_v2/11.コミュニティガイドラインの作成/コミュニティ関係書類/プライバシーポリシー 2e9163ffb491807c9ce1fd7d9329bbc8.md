# プライバシーポリシー

Shape Fit（以下「当社」といいます）は、当社が運営するShapeコミュニティ（以下「本サービス」といいます）において、利用者の皆様（以下「ユーザー」といいます）の個人情報の保護を重要な責務と考えています。本プライバシーポリシー（以下「本ポリシー」といいます）は、当社がユーザーの個人情報をどのように収集、利用、共有、保護するかについて説明するものです。

本サービスをご利用いただくことにより、ユーザーは本ポリシーに同意したものとみなされます。

---

## 第1条（定義）

本ポリシーにおいて、以下の用語は以下の意味を有します。

1. **「個人情報」** とは、生存する個人に関する情報であって、当該情報に含まれる氏名、生年月日その他の記述等により特定の個人を識別できるもの（他の情報と容易に照合することができ、それにより特定の個人を識別することができることとなるものを含む）、または個人識別符号が含まれるものをいいます。
2. **「本サービス」** とは、当社が提供するShapeコミュニティおよびこれに付随する一切のサービスをいいます。
3. **「ユーザーコンテンツ」** とは、ユーザーが本サービス上で投稿、送信、共有するテキスト、画像、動画、その他のコンテンツをいいます。

## 第2条（収集する情報）

当社は、本サービスの提供にあたり、以下の情報を収集することがあります。

### 1. アカウント・本人情報

ユーザーが本サービスに登録する際、以下の情報をご提供いただきます。

- 氏名（ニックネーム含む）
- メールアドレス
- 電話番号（任意）
- 生年月日または年齢
- 性別（任意）
- プロフィール写真（任意）
- 自己紹介文（任意）
- 挑戦の目標・関心分野

### 2. コミュニティ活動データ

本サービスのご利用に伴い、以下の情報を収集します。

- 投稿・コメント・リアクションの内容および履歴
- メッセージの送受信履歴
- 共有された画像・動画
- 「語る時間」「言語化時間」等のプログラムへの参加記録
- 他のユーザーとの交流履歴
- コミュニティ内での役割（Roots/フォロワー/火種等）に関する情報

### 3. イベント・参加データ

- オンライン・オフラインイベントへの参加申込情報
- 出欠記録
- アンケート・フォームへの回答内容
- ワークショップ等での言語化・目標設定の記録

### 4. デバイス・ログ情報

本サービスへのアクセス時に、以下の情報が自動的に収集されます。

- IPアドレス
- デバイスの種類、OS、ブラウザの種類
- アクセス日時
- 参照元URL
- Cookie情報および類似の識別子
- サービス利用状況（閲覧ページ、滞在時間等）
- アプリケーションのクラッシュログ・エラーログ

### 5. 位置情報

以下の場合に位置情報を収集することがあります。

- オフラインイベントへの参加時（おおよその地域情報）
- ユーザーが位置情報の共有を許可した場合
- IPアドレスから推定される概略的な位置情報

位置情報の収集は、ユーザーの端末設定により制御することができます。

### 6. 健康・フィットネス関連情報

本サービスの性質上、以下のセンシティブな情報を取り扱う場合があります。

- 運動・フィットネスに関する目標や記録
- 身体の変化に関する自己申告情報
- 健康状態や体調に関する任意の共有情報
- 挑戦の過程における心理的な変化の記録

これらの情報は、ユーザーの明示的な同意に基づいてのみ収集・利用されます。

### 7. 決済・購買情報

有料サービスをご利用の場合、以下の情報を収集します。

- 購入・課金履歴
- 請求先情報
- 決済方法に関する情報（クレジットカード情報は決済代行会社が管理し、当社は保持しません）

## 第5条（情報の共有・提供）

当社は、以下の場合を除き、ユーザーの個人情報を第三者に提供することはありません。

### 1. 業務委託先への提供

当社は、本サービスの運営に必要な業務を外部に委託する場合があります。この場合、当社は委託先との間で適切な契約を締結し、委託先における個人情報の管理を監督します。

主な委託先のカテゴリー：

- クラウドサービス提供事業者
- 決済代行事業者
- カスタマーサポート業務委託先
- メール配信サービス提供事業者
- 分析ツール提供事業者

### 2. コミュニティ内での公開

本サービスの性質上、以下の情報はコミュニティ内で他のユーザーに表示されます。

**すべてのユーザーに公開される情報：**

- ニックネーム
- プロフィール写真
- 自己紹介文（公開設定の場合）
- コミュニティへの投稿・コメント
- 「語る時間」での共有内容

**限定的に共有される情報：**

- イベント参加者リスト（参加者間で共有）
- メッセージ（送信相手のみ）
- 「言語化時間」での記録（本人および担当者のみ）

ユーザーは、プロフィールおよびプライバシー設定により、情報の公開範囲を調整することができます。

### 3. 法執行機関等への開示

以下の場合、当社は法令に基づき個人情報を開示することがあります。

- 法令に基づく開示請求があった場合
- 裁判所の命令があった場合
- 捜査機関からの法令に基づく照会があった場合
- 人の生命、身体または財産の保護のために必要な場合
- 公衆衛生の向上または児童の健全な育成の推進のために特に必要な場合

### 4. 事業譲渡等

合併、会社分割、事業譲渡その他の事由により本サービスに関する事業が承継される場合、当社は承継先に対し、本ポリシーに従い個人情報を引き継ぐことがあります。この場合、当社は事前にユーザーに通知いたします。

## 第6条（データの保存・越境移転）

### 1. データの保存場所

ユーザーの個人情報は、主に日本国内のサーバーに保存されます。ただし、本サービスの運営に利用するクラウドサービス等の関係で、以下の国・地域にデータが保存される場合があります。

- アメリカ合衆国
- その他、クラウドサービス提供事業者のサーバー所在地

### 2. 越境移転の保護措置

日本国外への個人情報の移転にあたっては、個人情報保護法その他関係法令に従い、適切な保護措置を講じます。

### 3. 保存期間

当社は、ユーザーの個人情報を以下の期間保存します。

| 情報の種類 | 保存期間 |
| --- | --- |
| アカウント情報 | アカウント削除後6か月 |
| 投稿・コメント | アカウント削除後3か月（削除処理期間として） |
| ログ・アクセス記録 | 取得から1年間 |
| 決済関連情報 | 法令上必要な期間（最大7年間） |
| お問い合わせ記録 | 対応完了後3年間 |

なお、法令上の保存義務がある場合、または正当な事業上の必要性がある場合は、上記期間を超えて保存することがあります。

## 第8条（ユーザーの権利）

ユーザーは、ご自身の個人情報について、以下の権利を有します。

### 1. アクセス権

当社が保有するご自身の個人情報の開示を請求することができます。

### 2. 訂正権

ご自身の個人情報に誤りがある場合、その訂正を請求することができます。

### 3. 削除権

一定の条件のもと、ご自身の個人情報の削除を請求することができます。

### 4. 利用停止権

一定の条件のもと、ご自身の個人情報の利用停止を請求することができます。

### 5. データポータビリティ

一定の条件のもと、ご自身の個人情報を構造化された一般的に使用される形式で受け取ることを請求することができます。

### 6. 権利行使の方法

上記の権利を行使される場合は、第14条に記載の問い合わせ窓口までご連絡ください。本人確認の上、合理的な期間内に対応いたします。

なお、以下の場合は、権利行使に応じられないことがあります。

- 本人確認ができない場合
- 法令上の保存義務に抵触する場合
- 当社の正当な業務遂行に著しい支障が生じる場合
- 他のユーザーの権利を侵害する場合

## 第10条（未成年者の取り扱い）

### 1. 年齢制限

本サービスは、18歳以上の方を対象としています。18歳未満の方は、本サービスをご利用いただけません。

### 2. 未成年者の登録

当社が18歳未満の方の登録を認識した場合、当該アカウントおよび関連する個人情報を速やかに削除します。

### 3. 保護者からの申し出

保護者の方が、ご自身の子どもが本サービスに登録していることを発見された場合は、第14条の問い合わせ窓口までご連絡ください。速やかにアカウントの削除および個人情報の消去を行います。

## 第12条（Cookieポリシー）

### 1. Cookieとは

Cookieとは、ウェブサイトがユーザーのブラウザに保存する小さなテキストファイルです。当社は、以下の目的でCookieを使用しています。

### 2. 使用するCookieの種類

| 種類 | 目的 | 保存期間 |
| --- | --- | --- |
| 必須Cookie | サービスの基本機能の提供、ログイン状態の維持 | セッション終了まで |
| 機能Cookie | ユーザー設定の記憶、利便性の向上 | 最大1年間 |
| 分析Cookie | 利用状況の分析、サービス改善 | 最大2年間 |
| 広告Cookie | 広告の表示・効果測定（使用する場合） | 最大2年間 |

### 3. Cookieの管理

ユーザーは、ブラウザの設定により、Cookieの受け入れ・拒否を選択できます。ただし、必須Cookieを無効にした場合、本サービスの一部または全部をご利用いただけなくなる場合があります。

## 第13条（本ポリシーの変更）

### 1. 変更の方法

当社は、法令の改正、サービス内容の変更、その他必要に応じて、本ポリシーを変更することがあります。

### 2. 通知方法

本ポリシーを変更する場合、当社は以下の方法によりユーザーに通知します。

- 本サービス上での告知
- 登録メールアドレスへの通知
- その他当社が適切と判断する方法

### 3. 効力発生

変更後のポリシーは、当社が別途定める場合を除き、本サービス上に掲載した時点から効力を生じるものとします。重要な変更については、効力発生日の14日前までに通知いたします。

### 4. みなし同意

本ポリシーの変更後、ユーザーが本サービスを継続して利用した場合、変更後のポリシーに同意したものとみなします。
