# フィットネスコンテスト「大会当日サポートプラン」費用見積もりテンプレート作成のための情報調査レポート

## 【1】APF（Asia Pacific Federation）大会の費用情報

### 1-1. バックステージパス

**■ 2026年度の料金体系（ルール変更あり）**

APFは2026年度より、バックステージパスのルールを大幅に変更しました。

- **事前購入（オンライン）**: 13,000円（座席なし）
- **当日購入（会場販売）**: 15,000円（座席なし）

**従来価格**: 15,000円（最低価格）  
**変更点**: 事前購入で2,000円値下げ、観戦座席は含まれない

**■ 購入方法**
- 選手本人がコンテスト登録時に同時購入
- バックステージパスのみでは会場に入場不可
- 別途「観戦チケット」の購入が必須
- 大会当日、選手受付時にサポーターに引き渡し
- 1選手につき最大2名まで

**■ キャンセル・返金ポリシー**
- **観戦チケットが売り切れの場合でも、バックステージパスの返金は不可**
- 決済完了時点でキャンセル・返金は一切不可

**■ 重要な注意事項**
- サポーターは選手と同じ集合時間に同行必須（同行できない場合はパスを受け取れない）
- 大会当日は「観覧用リストバンド」「バックステージパスリストバンド」「バックステージパスホルダー」の3種類を常時装着必要
- 撮影のみ目的での利用は不可

**情報ソース**: 
- [APF公式サイト - 2026年度バックステージパスルール変更](https://apf.fitness/information/22003/)
- [APF公式サイト - バックステージパス詳細](https://apf.fitness/common/backstage-pass/)

### 1-2. 観戦チケット

**■ 価格帯**
- 事前購入: **完売が非常に早い**（価格は大会ごとに異なる）
- 当日券: 販売可否は当日の座席状況により決定（確約なし）

**■ 購入方法**
- 各大会の観戦チケットは「[チケットぴあ](https://t.pia.jp/pia/search_all.do?kw=APF)」にて購入
- 直前で売り切れになる可能性が高いため、事前購入推奨

**■ 大会による価格変動**
公開情報では具体的な価格帯の記載はありませんが、会場規模や大会グレードにより価格は異なる模様。

**情報ソース**:

## 【2】APF以外の主要フィットネスコンテスト団体の費用情報

### FWJ（Fitness World Japan）

**■ バックステージパス**
- **料金**: 10,000円/枚（2025-2026年度）
- **2025年12月以降の重要な変更**: NPC Worldwideルール改定により、バックステージに入場するサポーターは以下が必須
  - バックステージパス購入
  - **NPC Worldwideメンバーシップ登録**（年間費用別途必要）
  - 観戦を希望する場合のみ観戦チケット購入

**■ 購入方法・特徴**
- 選手が出場エントリー時に同時購入
- 追加購入は選手プロフィールページから可能
- 当日追加購入は選手ミーティング時のみ受付

## 【5】費用見積もりテンプレートの設計参考

### 5-1. 記載すべき項目

**■ 基本構成**

```
【見積書タイトル】
フィットネスコンテスト 大会当日サポートプラン 費用見積書

【基本情報】
- 見積書番号
- 作成日
- 有効期限
- お客様名
- 大会名・開催日
- 担当トレーナー名

【費用明細】
1. サービス料金（固定費）
   - 大会当日サポート基本料金: XX,000円
   
2. 実費精算項目（変動費）
   - バックステージパス代: XX,000円
   - 観戦チケット代: XX,000円～（価格未確定の場合は概算）
   - 交通費（往復）: XX,000円（概算）
   - その他経費: XX,000円

【小計・合計】
- 小計: XX,000円
- 消費税（10%）: XX,000円
- 合計金額: XX,000円

【按分に関する特記事項】
※複数のお客様をサポートする場合の按分方法

【注意事項・免責事項】
- 実費精算項目の確定金額について
- キャンセルポリシー
- 返金不可項目について
```

---

### 5-2. フォーマット設計

**■ LINEで送付する前提の場合**

**推奨フォーマット**: PDF

**理由**:
1. **正式な見積書としての体裁**が整う
2. 金額変更の履歴管理が容易
3. 印刷・保管が可能
4. 改ざん防止

**代替案**: 
- 簡易版として画像（PNG/JPG）も可
- テキストのみは非推奨（誤解を招きやすい）

---

**■ 同意の記録方法**

**ベストプラクティス**:

1. **PDF送付 + LINEでの承認返信**
   - 例: 「上記見積内容を確認し、同意いたします。（お客様氏名・日付）」
   - LINEのトーク履歴が証拠となる

2. **電子署名サービスの活用**（より正式な場合）
   - DocuSign、Adobe Sign、クラウドサインなど

3. **金額確定のタイミングでの再確認**
   - 概算から確定金額に変わる際の通知＋再同意

---

**■ 概算表記のベストプラクティス**

**表記例**:
```
■バックステージパス代: 13,000円～15,000円（概算）
※事前購入13,000円、当日購入15,000円
※確定金額は購入時期により決定

■観戦チケット代: 3,000円～8,000円（概算）
※大会・座席により異なります
※完売の場合、当日券購入となり価格が変動する可能性があります

■交通費（往復）: 約2,000円（概算）
※最寄り駅からの実費精算
※実際の経路・交通手段により変動します
```

**重要ポイント**:
- 「概算」「見込み」と明記
- 変動する理由を簡潔に説明
- 確定時期を明示

---

### 5-3. トラブル防止の観点

**■ 事後精算でトラブルになりやすいポイント**

1. **想定外の費用追加**
   - 防止策: 見積書に「その他経費」項目を設け、発生可能性のある費用を列挙

2. **按分人数の変動**
   - 防止策: 「按分対象人数が変更になった場合、XX日前までにご連絡ください。それ以降の変更は按分計算の対象外となります」

3. **チケット代の高騰**
   - 防止策: 「事前購入ができなかった場合、当日券価格での精算となる場合があります」

4. **交通費の解釈相違**
   - 防止策: 具体的な計算方法を明記（例: 「○○駅から会場最寄り駅までの往復実費」）

---

**■ キャンセル不可商品を含む場合の同意取得**

**記載例**:
```
【キャンセル・返金に関する重要事項】

以下の項目は、一度購入・決済した後のキャンセル・返金ができません。
□ バックステージパス代（APF規定により返金不可）
□ 観戦チケット代（各チケットサイト規定により返金不可）

お客様都合によるキャンセルの場合も、上記費用は請求対象となります。
あらかじめご了承の上、お申し込みください。

□ 上記内容を理解し、同意します
　（お客様氏名・日付）
```

---

**■ 按分人数変動時の事前説明ベストプラクティス**

**記載例**:
```
【按分に関する重要事項】

本プランでは、バックステージパス代・観戦チケット代・交通費を
複数のお客様で按分することで、お一人あたりの費用を抑えています。

■按分対象費用
- バックステージパス代: 15,000円 ÷ サポート人数
- 観戦チケット代: 実費 ÷ サポート人数
- 交通費: 実費 ÷ サポート人数

■按分人数確定のタイミング
- 大会XX日前（○月○日）をもって按分対象人数を確定します
- それ以降のキャンセルの場合も、按分計算の対象となります
```
