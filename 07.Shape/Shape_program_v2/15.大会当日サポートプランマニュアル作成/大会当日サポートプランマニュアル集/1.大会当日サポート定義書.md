# 大会当日サポート定義書

**対象：現場トレーナー・FCオーナー向け**

---

## このドキュメントの目的

「大会当日サポートって、結局何をするの？」  
「どこまでやればいいの？やらなくていいことは？」

**この疑問にハッキリ答えるための資料です。**

トレーナーが迷わず動けるように、  
お客様との認識がズレないように、  
**サービスの中身を明確にします。**

---

## 1. 大会当日サポートとは？

### 一言で言うと

> **お客様が「最高の状態」でステージに立てるように、大会1週間前〜当日をサポートするサービス**

### 「最高の状態」とは？

- **身体のコンディション**が整っている
- **気持ち（メンタル）** が整っている
- **「自分らしくステージに立てた」と思える状態**

つまり、単に「付き添う」だけではなく、  
**身体面・精神面の両方で安心して大会に臨める状態をつくること**が目的です。

---

## 2. サポートの目的（なぜこのサービスがあるのか）

### お客様視点

- 初めての大会、または経験が浅い方でも **「不安なく当日を迎えられる」**
- 当日の流れが分かり **「何が起きるか分からない」という不安がなくなる**
- 近くに頼れる人がいることで **「一人じゃない」と思える**

### トレーナー視点

- お客様の不安を事前に把握し **「先回りしてケア」できる**
- サポート範囲が明確なので **「どこまでやるか」に迷わない**
- 複数人サポートでも **ルールがあるから公平に対応できる**

---

## 3. サポート範囲（やること）

### 一覧表

| タイミング | やること | 目的 |
|------------|----------|------|
| **大会1週間前〜前日** | 食事内容の確認 | 当日に向けた調整の把握 |
| | 体重・身体の質感の確認 | コンディションの最終チェック |
| | メンタル面の確認 | 不安・緊張の早期発見 |
| | 当日の流れ・持ち物の共有 | 「何が起きるか分からない」をなくす |
| **大会当日** | チェックイン〜表彰まで同席 | 一人にしない安心感 |
| | パンプアップのサポート（必須） | ステージ前の最終仕上げ |
| | 控室・招集前・ステージ直前のメンタルケア | 不安を和らげ自信を持たせる |
| | ステージ後のケア | 結果に関わらず気持ちの整理をサポート |
| | トラブル・イレギュラー時の相談対応 | 想定外の事態でも冷静に対処 |

### 補足：カラーリング対応について

- **大会ルールに準ずる**（APFの場合、指定業者のみ・証明書必須）
- 対応可否は**事前にオーナーへ確認**
- 自己塗りカラー・オイルは**失格リスクがあるため絶対にNG**

---

## 4. サポート適用外（やらないこと）

### 一覧表

| 項目 | 理由 |
|------|------|
| 写真・動画の撮影 | 確約不可（余裕があれば対応することもある） |
| 観戦席の確保・席取り | サポート業務に専念するため |
| 自宅・宿泊先からの過剰な付き添い | 会場でのサポートが本来の業務 |
| 他コーチ担当領域への最終判断介入 | 責任範囲の明確化のため |
| 結果・順位・評価の保証 | 大会結果は保証できない |

### 例外が発生した場合

- **「特別対応」として事前にオーナーへ共有・確認**
- 勝手に判断しない

---

## 5. お客様との期待値のすり合わせ

### なぜ大切か

大会への出場は、お客様とトレーナーが数ヶ月〜長い時間をかけて準備してきた大切な舞台です。

その分、お客様の中には期待だけでなく、不安や心配も生まれやすくなります。

そうした気持ちを事前にすり合わせ、お互いが同じ方向を向いて大会当日を迎えるために、**事前の期待値調整はとても重要です。**

### 申し込み時に必ず確認すること

| 確認項目 | 内容 |
|----------|------|
| サポート内容の説明 | できること・できないことを明確に伝える |
| 出場大会の概要 | 日程、エントリーカテゴリー |
| 他トレーナーとのサポート状況 | ポージング・トレーニング等で別コーチがいるか |
| 複数人サポートの可能性 | マンツーマンになれない可能性を事前に伝える |

### 複数人サポートのルール

- **同時サポートは最大3名まで**
- **ステージ前の招集時点で必ず合流する**（これだけは絶対守る）
- 優先順位：月のセッション回数が多い方 → 同数なら先着順
- **お客様に事前に了承を得る**（快諾を得ていても、認識がズレていないか再確認）

---

## 6. 男性トレーナーが女性選手をサポートする場合

女性控室には入れないため、以下を徹底します。

| 対応 | 具体的にやること |
|------|------------------|
| 連絡を取り続ける | LINE等で細かくやり取りし、放置しない |
| 時間管理を徹底 | 招集・パンプアップには**必ず**間に合わせる |
| 不安を感じさせない | 「今どこにいるか」「何時に戻るか」を伝える |

---

## 7. APF大会でのバックステージパス利用ルール

APF大会でサポートに入る際は、以下のルールを必ず守ってください。

### 必須ルール

| 項目 | 内容 |
|------|------|
| バックステージパスの目的 | **選手サポート目的のみ**（撮影だけはNG） |
| 観戦チケット | **バックステージパスだけでは会場に入れない**（別途必要） |
| 受付 | **選手と同じ集合時間に同行しないと渡してもらえない** |
| 装着物 | リストバンド等を**常時着用**（提示できないと退場） |
| 購入後のキャンセル | **不可**（変更・返金もNG） |

### 撮影ルール

| OK | NG |
|----|----|
| スマホ・タブレット | 三脚 |
| | 販売目的の撮影 |
| | ビブス着用者以外の一眼レフ等 |

### カラーリングルール

| 項目 | 内容 |
|------|------|
| 指定業者 | **指定オフィシャルのみ**（自己塗りカラー禁止） |
| オイル類 | **ワセリン主成分含む製品は禁止** |
| 証明書 | **カラーリング証明書の提出が必須** |
| 違反した場合 | **失格** |

> **参照元**：[APF公式 バックステージパスに関して](https://apf.fitness/common/backstage-pass/)

## 8. トラブルを防ぐためのチェックリスト

### 申し込み時

- [ ] サポート内容（できること・できないこと）を説明した
- [ ] 複数人サポートの可能性を伝えた
- [ ] 他コーチのサポート状況を確認した
- [ ] 費用について説明した（交通費・バックステージパス代・観戦チケット代の扱い）
- [ ] 観戦チケット代はお客様負担であることを説明した（APFの場合はコンテスト登録時に購入）

### 大会1週間前

- [ ] 食事・体重・メンタルの確認を開始した
- [ ] カラーリングの手配を確認した（指定業者か？証明書は？）
- [ ] 当日の集合場所・時間を共有した
- [ ] 当日の持ち物を共有した
- [ ] パンプアップの流れを確認した

### 大会当日

- [ ] 選手と同時に受付を完了した
- [ ] 装着物（リストバンド等）を確認した
- [ ] 控室でお客様と連絡を取り続けている（女性選手の場合）
- [ ] 招集前に合流した
- [ ] パンプアップをサポートした
- [ ] ステージ後のケアを行った

---

## 9. 判断に迷ったときのルール

| 状況 | どうするか |
|------|------------|
| サポート範囲外のことを頼まれた | 「特別対応」としてオーナーに確認 |
| 複数のお客様の対応が重なった | ステージ前の招集を最優先 |
| お客様が不安を訴えている | まず話を聞く → 具体的な対応は後から |
| 自分で判断できない | **勝手に決めず、オーナーに相談** |

---

## 10. まとめ：このサービスで守りたいこと

### 3つの約束

1. **お客様を「一人ぼっち」にしない**
2. **何があるか分からない不安をなくす**
3. **サポート範囲を超えることは勝手にやらない**

### トレーナーとして意識すること

- **できることは全力でやる**
- **できないことは正直に伝える**
- **迷ったらオーナーに相談する**
