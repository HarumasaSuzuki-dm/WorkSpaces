# 大会当日サポートプラン ご案内

---

## このプランについて

大会当日サポートプランは、あなたが大会当日に**最高の状態でステージに立つこと**をサポートするプランです。

「最高の状態」とは、身体のコンディションだけでなく、**気持ちの面も整っている状態**のこと。「自分らしい姿」で、「自信を持って」ステージに立てるよう、私たちがサポートします。

---

## 私たちがサポートすること

### 【大会1週間前〜前日】

大会当日を安心して迎えられるよう、以下を一緒に確認していきます。

| 確認項目               | 内容                                           |
| ---------------------- | ---------------------------------------------- |
| 食事内容の確認         | 今の状態を把握するための確認です               |
| 体重・身体の質感の確認 | コンディションの最終チェック                   |
| メンタル面の確認       | 不安や緊張感があれば一緒に整理します           |
| 当日の流れの共有       | 集合場所・時間、持ち物、パンプアップの流れなど |

**※ 食事を「管理する」ことが目的ではなく、今の状態を一緒に把握するための時間です。**

---

### 【大会当日】

| サポート内容                   | 詳細                                                         |
| ------------------------------ | ------------------------------------------------------------ |
| **チェックイン〜表彰まで同席** | 基本的にあなたと一緒にいます                                 |
| **パンプアップのサポート**     | 必ずサポートします                                           |
| **メンタルケア・声掛け**       | 控室、招集前、ステージ直前など、不安を感じやすいタイミングで |
| **コンディションチェック**     | 当日の体調や状態を確認・ケア                                 |
| **トラブル時の相談対応**       | 何か困ったことがあればすぐにご相談ください                   |

---

## サポートに含まれないこと

以下については、本プランの対象外となります。あらかじめご了承ください。

| 項目                                 | 備考                                               |
| ------------------------------------ | -------------------------------------------------- |
| 写真・動画の撮影                     | 対応できる場合もありますが、確約はできません       |
| 観戦席の確保・席取り                 | 座席チケットは別途ご購入が必要です                 |
| ご自宅・宿泊先からの付き添い         | 会場集合が基本となります                           |
| 他のコーチが担当している領域への介入 | ポージング指導など、他のコーチが担当されている場合 |
| 結果・順位・評価の保証               | サポートは全力で行いますが、結果の保証はできません |

## 費用について

### 【プラン料金に含まれるもの】

- 大会1週間前〜当日のトレーナーサポート

### 【お客様にご負担いただくもの】

| 項目                     | 備考                                           |
| ------------------------ | ---------------------------------------------- |
| **バックステージパス代** | 10,000円/枚（選手登録時に購入）                |
| **観戦チケット代**       | サポーターも別途購入が必要です                 |
| **交通費**               | 会場までの往復交通費を実費でご負担いただきます |

**【複数のお客様を同時にサポートする場合】**

同じ大会で複数のお客様をサポートする場合、交通費・チケット代は人数で按分（割り勘）となります。

例）2名の場合 → 交通費・チケット代をそれぞれ1/2ずつご負担

---

## 複数のお客様を同時にサポートする場合について

同じ大会で複数のお客様をサポートさせていただく場合があります。

### 【事前にお伝えすること】

- 他のお客様のサポートがある場合は、**必ず事前にお伝えします**
- マンツーマンでのサポートが難しくなる可能性があります

### 【お約束すること】

- **ステージ前の招集時には、必ずあなたのそばにいます**
- **パンプアップは、必ずサポートします**
- 同時サポートは**最大3名まで**としています

### 【優先順位について】

- 月のセッション回数が多い方を優先します
- セッション回数が同じ場合は、先着順となります

## チケットについての重要なお知らせ

APF大会では、以下のルールがあります。ご注意ください。

### 【バックステージパス】

- **選手登録時に、お客様ご自身で購入が必要**です
- サポーターは選手と同じ集合時間に、一緒に受付を行います
- **選手と一緒に受付できない場合、パスを受け取れません**

### 【観戦チケット】

- バックステージパスとは**別に**購入が必要です
- バックステージパスだけでは会場に入れません
- 売り切れる場合がありますので、**早めのご購入**をおすすめします

### 【ご注意】

- **購入後のキャンセル・返金はできません**
- 観戦チケットが売り切れても、バックステージパスの返金はできません

参考：[APF バックステージパスに関して](https://apf.fitness/common/backstage-pass/)

## 当日の装着物について

大会当日、バックステージに入るサポーターは以下の**3点すべて**を常に身につける必要があります。

1. **観覧用リストバンド**（観戦チケットで受け取り）
2. **バックステージパス用リストバンド**
3. **バックステージパスホルダー**

**どれか1つでも確認できない場合、バックステージから退場となります。**

## カラーリングについて

APF大会では、カラーリング（タンニング）に厳しいルールがあります。

| ルール                           | 詳細                                 |
| -------------------------------- | ------------------------------------ |
| **指定業者のみ利用可能**         | 指定外の業者は使用できません         |
| **カラーリング証明書が必要**     | 当日のチェックイン時に提出           |
| **自己塗りは禁止**               | 自分で塗ったカラーリングは失格の対象 |
| **ワセリン主成分のオイルは禁止** | こちらも失格の対象                   |

**カラーリングは大会1ヶ月前までに予約・実施をおすすめします。**

参考：[APF カラーリング・オイルに関して](https://apf.fitness/common/coloring/)

## サポートを受ける前にお伝えください

### 【必ずお伝えいただきたいこと】

- [ ] 大会の日程・会場
- [ ] エントリーするカテゴリー
- [ ] 他のトレーナー・コーチのサポート状況（ポージング、食事指導など）

当日に向けて、どんな不安がありますか？（該当するものにチェック）

- [ ] 当日の流れがわからない
- [ ] ステージに立つこと自体が不安
- [ ] 他の選手との比較
- [ ] 結果への不安
- [ ] その他（                    ）

## やむを得ず離席する場合

大会当日、やむを得ず一時的にあなたのそばを離れる場合があります。

その際は、

- **なぜ離席するのか**
- **何時頃に戻れるのか**

を必ず事前にお伝えします。

**あなたが会場で「一人ぼっち」になって不安を感じることがないよう、十分配慮します。**

---

## 男性トレーナーが女性選手をサポートする場合

男性トレーナーは女性控室に入ることができません。

そのため、

- 控室にいらっしゃる間も、**LINEなどでこまめに連絡**を取ります
- **招集・パンプアップには必ず間に合うよう**、時間管理を徹底します

不安なことがあれば、いつでもご連絡ください。

---

## 困ったとき・判断に迷ったとき

大会当日に何か困ったこと、判断に迷うことがあれば、**すぐにトレーナーにご相談ください。**

トレーナーで判断が難しい場合は、オーナーに確認の上、対応いたします。
