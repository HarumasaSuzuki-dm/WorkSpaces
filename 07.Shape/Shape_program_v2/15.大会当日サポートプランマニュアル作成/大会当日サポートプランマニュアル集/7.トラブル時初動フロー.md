# トラブル時初動フロー

**対象：現場トレーナー・FCオーナー**

---

## このドキュメントの目的

大会サポート中にトラブルが起きた時、**誰が・何を・どの順番でやるか**を明確にし、初動の遅れや判断ミスを防ぐためのフローです。

**大原則：迷ったらオーナーに連絡。自己判断で進めない。**

---

## 1. トラブルの種類と対応レベル

トラブルは大きく3つのレベルに分けて対応します。

| レベル | 内容 | 対応者 | 対応スピード |
|--------|------|--------|--------------|
| **レベル1** | 現場で解決できる小さなトラブル | トレーナー | その場で対応 |
| **レベル2** | お客様の不満・費用トラブル | トレーナー → オーナー | 当日中に報告 |
| **レベル3** | クレーム・大きなトラブル | オーナー → 本部 | 即時連絡 |

---

## 2. レベル別：具体例と対応フロー

### 【レベル1】現場で解決できるトラブル

**具体例**
- パンプアップ器具の不足・故障
- 予定していた場所が使えない
- タイムスケジュールの軽微な変更
- お客様の忘れ物（軽微なもの）

**対応フロー**
```
① トラブル発生
　　↓
② その場で解決策を考え、対応する
　　↓
③ お客様に状況を説明し、安心させる
　　↓
④ 当日中にオーナーへ「こういうことがあった」と共有
```

**ポイント**
- お客様を不安にさせないことが最優先
- 解決後でいいので、必ずオーナーに報告する

---

### 【レベル2】お客様の不満・費用トラブル

**具体例**
- 「思っていたサポートと違う」と言われた
- 費用の認識が食い違っている
- 複数サポートで対応が薄いと感じられた
- お客様の表情・態度が明らかに不満そう

**対応フロー**
```
① トラブル発生（不満の兆候に気づく）
　　↓
② まずお客様の話を最後まで聞く（否定・言い訳しない）
　　↓
③ 「ご不安にさせてしまい申し訳ありません」と伝える
　　↓
④ その場で解決しようとせず「確認させてください」と伝える
　　↓
⑤ 当日中にオーナーへ連絡（電話 or LINE）
　　↓
⑥ オーナーの判断を仰いでから、お客様に回答する
```

**絶対にやってはいけないこと**
- ❌ その場で金額の減額や返金を約束する
- ❌ 「自分の判断ではないので」と責任逃れする
- ❌ 他のトレーナーやスタッフのせいにする
- ❌ お客様の話を遮って説明・反論する

**オーナーへの連絡内容（テンプレ）**
```
【トラブル報告】
■ 発生日時：○月○日 ○時頃
■ お客様名：○○様
■ 内容：（何が起きたか簡潔に）
■ お客様の状態：（怒っている/困っている/不安そう 等）
■ 現在の対応：（確認中とお伝えしている 等）
■ 判断が必要な点：（○○について、どう回答すべきか）
```

---

### 【レベル3】クレーム・大きなトラブル

**具体例**
- お客様から「もう来ない」「訴える」等の発言
- 本部・会社宛にクレームの連絡が入った
- SNSに書かれそう/書かれた
- APFルール違反（カラーリング等）で失格になった
- 費用の未払いが発生している

**対応フロー**
```
① トラブル発生（クレーム発覚）
　　↓
② 即座にオーナーへ電話連絡（LINEではなく電話）
　　↓
③ オーナーが本部へ連絡・相談
　　↓
④ 本部とオーナーで対応方針を決定
　　↓
⑤ オーナーからトレーナーへ指示
　　↓
⑥ 指示に基づき対応（トレーナーからの謝罪連絡等）
　　↓
⑦ 対応完了後、報告書を作成（オーナー）
```

**トレーナーがやること**
1. **すぐにオーナーに電話する**（LINEだと気づかない可能性あり）
2. **事実を正確に伝える**（自分に不利なことも隠さない）
3. **お客様には「上に確認します」と伝え、勝手に約束しない**
4. **オーナーの指示を待つ**

**オーナーがやること**
1. **トレーナーから状況を聞き取る**
2. **本部に第一報を入れる**（詳細は後でOK、まず発生の報告）
3. **本部と対応方針を決める**
4. **トレーナーに具体的な対応を指示する**
5. **対応完了後、報告書を作成・提出する**

## 3. 連絡先・エスカレーションライン

```
【緊急度：高】電話 → LINE → メール の順で連絡

トレーナー → FCオーナー → 本部

※ オーナーに連絡がつかない場合は、本部に直接連絡してOK
```

| 連絡先 | 電話番号 | LINE | 備考 |
|--------|----------|------|------|
| 所属店舗オーナー | ※各店舗で記入 | ※各店舗で記入 | 第一連絡先 |
| 本部（ななこ） | ※記入 | ※記入 | オーナー不在時 |
| 本部（なーすけ） | ※記入 | ※記入 | 予備 |

## 4. よくあるトラブルと対応例

### ケース①：「期待していたサポートと違う」と言われた

**起きやすい状況**
- 複数サポートでマンツーマンになれなかった時
- 控室に入れず、LINEでのやり取りが中心だった時
- お客様の不安が大きかった時

**初動対応**
1. まず「ご不安な気持ちにさせてしまい、申し訳ありません」と伝える
2. 具体的に何が不安だったか、話を聞く
3. その場で言い訳せず「確認してご連絡します」と伝える
4. オーナーに報告し、対応を相談する

---

### ケース②：費用の認識が違う

**起きやすい状況**
- 交通費・チケット代の按分説明が不十分だった
- 複数サポート時の費用分担が伝わっていなかった
- 事前に金額を伝えていなかった

**初動対応**
1. 「ご説明が不十分で申し訳ありません」と伝える
2. **金額の減額・返金はその場で約束しない**
3. 「オーナーに確認して、正式にご連絡します」と伝える
4. オーナーに報告し、対応を決める

---

### ケース③：APFルール違反で問題が起きた

**起きやすい状況**
- カラーリングが指定業者以外だった
- バックステージパスの運用ルールに違反した
- 撮影禁止エリアで撮影した

**初動対応**
1. まずお客様の安全・精神面をケアする
2. 大会スタッフの指示に従う
3. オーナーに即時報告（電話）
4. お客様には「対応を確認します」と伝え、落ち着かせる

**予防が大事**
- 事前にAPFルールをお客様と一緒に確認する
- カラーリング証明書の有無を確認する
- 撮影ルールを事前に共有する

## 5. トラブル対応の3つの鉄則

### 鉄則① 初動は「聞く」こと

お客様が不満や不安を感じている時、最初にやるべきは**話を聞くこと**です。

- 言い訳しない
- 否定しない
- 遮らない
- 最後まで聞く

「そうだったんですね」「ご不安でしたよね」と受け止めるだけで、状況が落ち着くことも多いです。

---

### 鉄則② 約束は「その場で」しない

費用の減額、返金、特別対応などは、**絶対にその場で約束しない**でください。

「確認してご連絡します」が正解です。

勝手に約束すると、会社として対応できなくなったり、後から覆すことでさらに信頼を失います。

---

### 鉄則③ 報告は「早く・正確に」

トラブルの報告で大事なのは、**スピードと正確さ**です。

- 悪い報告ほど早くする
- 自分に不利なことも隠さない
- 事実と推測を分けて伝える
- 「自分はこう思う」という意見も添える

報告が遅れると、対応の選択肢が減ります。早ければ早いほど、打てる手が増えます。

## 6. 報告書フォーマット（レベル3用）

レベル3のトラブルが発生した場合、対応完了後にオーナーが報告書を作成します。

```
【トラブル対応報告書】

■ 報告日：
■ 報告者：
■ 店舗名：

■ 発生日時：
■ お客様名：
■ 担当トレーナー：

■ トラブル概要：
（何が起きたか、時系列で簡潔に）

■ お客様の主張・訴え：
（お客様が何を不満に思っていたか）

■ 事実確認結果：
（実際に何があったか、トレーナーへの確認結果）

■ 対応内容：
（どのような対応をしたか）

■ お客様の反応：
（対応後、お客様はどうだったか）

■ 再発防止策：
（同じことを防ぐために、何をするか）

■ 備考：
```

## 8. 過去のトラブル事例から学ぶ

### 前川様のケースから学ぶ教訓

**何が起きたか**
- 複数サポートでマンツーマンの対応ができなかった
- LINEでやり取りしていたが、期待するサポートが得られなかった
- 費用（交通費・チケット代）の説明が不十分だった
- 過去の大会でもサポート内容の擦り合わせが不十分だった

**防げたポイント**
1. **複数サポートになる場合は、事前にしっかり説明する**
   - 「マンツーマンにならない時間がある」と正直に伝える
   - それでもOKか、お客様に確認する

2. **期待値を事前に揃える**
   - 「何をする/しないか」を申し込み時に説明する
   - 不安なことを事前にヒアリングする

3. **費用は事前に明確にする**
   - 按分ルールを説明し、金額を伝える
   - 請求前に内訳を共有する

4. **不満の兆候に早く気づく**
   - お客様の表情や態度の変化に注意する
   - 「大丈夫ですか？」とこまめに声をかける

## まとめ

| ポイント | 内容 |
|----------|------|
| 判断に迷ったら | オーナーに連絡。自己判断しない |
| 初動で大事なこと | お客様の話を聞く。約束はしない |
| 報告のコツ | 早く・正確に。隠さない |
| 予防が一番 | 事前の説明と期待値合わせ |

**トラブルは起きてから対応するより、起きる前に防ぐ方が100倍楽です。**

このフローを頭に入れつつ、普段から「期待値を揃える」「こまめに声をかける」を意識してください。
