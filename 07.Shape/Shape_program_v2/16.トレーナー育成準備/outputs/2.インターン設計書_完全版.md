# 【SFG】佐久間さん インターン設計書

---

## 1. インターンの目的

**面接だけでは見えなかった部分を、実際の行動を通して確認する期間**

| 確認したいこと |
|---|
| 複数人の場での発言・質問ができるか |
| 自分の考え（自分軸）を言葉にできるか |
| フィードバックを受け止めて改善できるか |
| 本店メンバーとの相性 |
| 他者に流されないか |
| 基本的な業務姿勢（時間管理、報連相、丁寧さ） |

## 7. フェーズごとの確認ポイント

| フェーズ | 時期 | 確認ポイント |
|---|---|---|
| 序盤 | 最初の2週間 | 基本姿勢（挨拶、時間管理、言葉遣い）、本店メンバーとの初期コミュニケーション |

## 10. 採用判断の基準

### 必須基準

- シェイプのVALUEが行動に表れている
- 違和感を溜め込まず伝えられる
- 複数人の場で発言できる
- 他者に流されない「自分軸」がある
- 本店メンバーとの相性が良い
- 基本的な業務姿勢が安定している

### プラス評価

- 小さな提案ができる
- 他店舗とも自然に馴染める
- 物事を構造的に捉えられる
- 組織を前に進める視点がある
