# SFF2026 協賛プロジェクト概要

> このドキュメントは、Shape Fit Festival 2026（SFF2026）の協賛企業プロジェクトに関わるメンバーが、全体像をすばやく理解するためのまとめ資料です。専門用語はできるだけ使わず、初めての方にもわかるように書いています。

---

## 1. SFFってなに？

**Shape Fit Festival（SFF）** は、株式会社Shape Fitが主催する **フィットネスコンテスト** です。ボディビルのような「順位や結果」だけを追うのではなく、**出場者一人ひとりの「挑戦の物語」を大切にする** のが最大の特徴です。

**大会のコンセプト：**「挑戦」と「物語」を共に

- 「挑戦」＝ 自分で決めた目標に向かって行動すること
- 「物語」＝ その挑戦の中で何を感じ、どう成長したかというプロセス
- 結果だけが価値ではなく、挑戦とその過程にこそ価値がある、という考え方

**2026年は「広く」ではなく「深く」** ── 人数を増やすことより、一人ひとりとの関わりを深める「ファンベース戦略」に転換しています。

## 5. 各特典のかんたん説明

### ロゴ掲載
大会の公式パンフレット、SNS投稿、公式HPに企業ロゴを載せます。TOPは大きく目立つ位置、DIAMOND・PLATINUMはそれより控えめなサイズです。

### オープニング映像
大会の冒頭に流れる映像に企業名やロゴが表示されます。TOP・DIAMONDが対象で、ランクによって表示の大きさや時間が異なります。企業側はロゴのPNGデータと社名テキストを大会の約4ヶ月前までに提出します。

### 企業メッセージ映像（TOPのみ）
TOPスポンサーだけの特別枠。企業が伝えたいメッセージを映像にして大会中に流します。

### 企業ストーリー取材・発信（TOPのみ）
SFF側が企業を取材し、その企業の「挑戦の物語」をコンテンツとして発信します。

### 席招待
大会当日、企業関係者を特別席（VIP席）に招待します。

### 司会からの紹介
大会の司会者が、協賛企業の名前や商品を口頭で紹介します。

### 物品協賛
企業の商品をサンプルや景品として大会で配ります。入賞者への景品、または来場者への無料配布ができます。提出期限は大会の約2ヶ月半前で、配布はSFF側が当日手渡しするか、企業が後日郵送するかを選べます。

### 過程イベント
大会本番だけでなく、準備期間中（「過程」）にも企業と選手をつなぐイベントを開催できます。例えばSNSでのチャレンジ投稿キャンペーンなど。企画・運営は基本的に企業側が主導し、SFFは初回の告知と公式SNSでの拡散をサポートします。

### ブース出店（TOP・DIAMONDのみ）
大会会場に企業ブースを出して、商品の展示・販売・試食・体験などができます。来場者と直接コミュニケーションがとれる貴重な機会です。ブースの位置希望やアナウンス文章、物販申請書は大会の約2ヶ月半前に提出が必要です。

## 6. 協賛企業にとってのメリット（4つ）

| メリット | 説明 |
|----------|------|
| **ブランド価値の向上** | 「挑戦を応援する企業」として社会的な信頼が高まる。社会貢献イメージ×共感が得られる |
| **採用・人材ブランディング** | 挑戦志向の若者層に企業をアピールできる。「こんな会社で働きたい」と思ってもらえる機会 |
| **販促・商品訴求** | ブースでのサンプリングや試食を通じて、来場者に商品を直接体験してもらえる |
| **関係構築** | 健康や自己成長に関心の高いユーザーとの接点が持てる。長期的な顧客との関係づくり |

## 8. 協賛の進め方（全体の流れ）

```
① 商談・資料共有
   └ 提案資料と議事録をお送りする

② ご回答（1週間以内）
   └ 協賛するかどうか、どのプランにするかを回答いただく

③ 契約手続き
   └ 協賛契約書を交わす
   └ ロゴデータ・紹介文・広告素材などを提出いただく

④ お支払い
   └ 契約後に請求書を発行
   └ 入金確認後、公式HP・SNSにロゴ掲載開始

⑤ 当日までの準備
   └ 協賛特典に合わせて素材を準備（映像・パンフ原稿・ブース備品など）
   └ 当日の動線やVIP席のご案内

⑥ 大会当日
   └ 協賛特典の履行（ブース・映像・紹介など）

⑦ 大会後
   └ 活動レポートの送付（来場者数・SNSリーチなど）
   └ 次年度に向けた振り返り・ご相談
```

## 10. プロジェクトのスケジュール

| 時期 | フェーズ | やること |
|------|---------|---------|
| 1月下旬〜2月初旬 | Phase 0：体制構築 | 役割分担の決定、担当アサイン、全企業の状況把握 |
| 2月 | Phase 1：リターン設計・リストアップ | 協賛パッケージの内容見直し、新規候補企業のリスト作成 |
| 2〜3月 | Phase 2：既存先クロージング | すでに話が進んでいる企業との契約を確定させる |
| 3〜4月 | Phase 3：追加開拓 | 足りない分を新しい企業へのアプローチで補う |
| 4〜5月 | Phase 4：目標達成・最終調整 | 全17社の確定を目指すラストスパート |
| 3〜5月 | Phase 5：契約・素材収集 | 全企業との契約締結、ロゴや映像素材の回収 |
| 5〜7月 | Phase 6：制作・準備 | パンフレット・オープニング映像の制作、ブース調整 |
| 8〜9月 | 大会当日 | 協賛企業対応、全特典の履行 |
| 大会後 | Phase 7：フォロー | 活動レポート送付、次年度に向けた関係構築 |

## 12. 重要な用語集

| 用語 | 意味 |
|------|------|
| 協賛（きょうさん） | 企業がお金や物品を提供して大会を支援すること。スポンサーと同じ意味 |
| リターン | 協賛してくれた企業に対してSFF側が提供するお返し（ロゴ掲載、ブース、映像など） |
| クロージング | 商談を進めて最終的に契約を決めること |
| 過程イベント | 大会本番前の準備期間中に実施するイベント（SNSキャンペーンなど） |
| 物品協賛 | お金ではなく商品（サンプル・景品）で協賛すること |
| ファンベース戦略 | 広く浅くではなく、熱量の高いファンとの深い関係を重視する方針 |
| フェーズ | プロジェクトの段階のこと。Phase 0〜7まである |
| マイルストーン（MS） | 「ここまでに達成すべき」という重要な節目のこと |
