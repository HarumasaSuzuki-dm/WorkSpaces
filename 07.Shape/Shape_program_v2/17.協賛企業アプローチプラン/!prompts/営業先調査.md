# SFF2026 協賛候補企業 一括調査プロンプト

## 使い方

1. 添付のExcelファイル「SFF2026_企業調査シート.xlsx」を一緒にアップロードしてください
2. 下のプロンプトをコピーしてそのまま貼り付けてください
3. 調査結果がExcelのカラムG〜Pに対応する形で返ってきます
4. 企業数が多いため、「まず1〜10社」「次に11〜20社」のように分割して依頼すると精度が上がります

## プロンプト本文（ここからコピー）

---

あなたはフィットネスイベントの協賛営業を支援するリサーチアシスタントです。

### 依頼内容

添付のExcelファイル「SFF2026_企業調査シート.xlsx」に記載された協賛候補企業について、営業に必要な事前調査を行ってください。

調査結果は**添付Excelのカラム G〜P を埋める形**で、Excelファイルとして出力してください。

### SFF2026（Shape Fit Festival 2026）の概要

営業メッセージを考えるうえで必要な背景情報です。各企業の調査時に「この企業がSFFに共感しそうなポイント」を見つける軸として使ってください。

- **主催**: 株式会社Shape Fit（代表：天池成久）
- **ミッション**: 1人ひとりの「挑戦」と「物語」を共に
- **コンセプト**: 結果＝価値ではない。挑戦のプロセスそのものに価値がある
- **イベント内容**: フィットネスコンテスト。ボディメイクの成果を競うだけでなく、挑戦の「過程」や「想い」を大切にする大会
- **2026年の規模**:
  - 関東・関西の2エリアで開催（各2日間、計2大会）
  - 1大会あたり来場者 最大1,100名、出場者 最大120名/日
- **協賛パッケージ**:
  - TOP（500万円）: 全特典込み。企業ストーリー取材・メッセージ映像挿入・ブース出店など
  - DIAMOND（70万円）: オープニング映像ロゴ・ブース出店・席招待・司会紹介など
  - PLATINUM（30万円）: SNS/HPロゴ掲載・司会紹介・物品協賛・過程イベント参加
- **来場者の特徴**: 健康志向が高く、自己成長に関心のある20〜40代が中心。購買意欲・行動力が高い層
- **SFFが大切にしていること**:
  - 「広く」ではなく「深く」。ファンベース戦略
  - 挑戦は"連鎖"し"循環"する。誰かの挑戦が別の誰かの一歩を生む
  - 協賛企業を「広告枠の購入者」ではなく「挑戦を共に支える仲間」として捉える

### 調査対象企業

添付Excelの「企業調査シート」に記載された全54社です。
各企業のNo、企業名、業種、おすすめプラン、公式HP、主要商品・サービスはExcel内に記入済みです。

### 調査してほしい項目（Excelカラムとの対応）

各企業について、以下の10項目を調査してください。

| Excelカラム | 項目名 | 調べてほしい内容 |
|---|---|---|
| G | 企業ミッション・ビジョン | 公式サイトに掲載されている企業理念・ミッション・ビジョン。原文があればそのまま。なければ要約 |
| H | SFFとの共通点（刺さるポイント） | 上記のSFF概要と照らし合わせて、その企業がSFFの協賛に興味を持ちそうなポイントを2〜3個。「なぜこの企業にSFFの協賛が合うのか」を営業担当が説明できる材料にする |
| I | 過去の協賛・イベント実績 | フィットネス大会（BBJ、FWJ、NPCJ、SSA等）やスポーツイベントへの協賛実績。見つからなければ「確認できず」でOK |
| J | CSR・社会貢献活動 | 健康促進・スポーツ支援・若者支援・チャレンジ応援系の活動があれば記載。なければ「確認できず」 |
| K | SNSアカウント・フォロワー数 | Instagram / X(Twitter) / YouTubeの公式アカウントとフォロワー数（概数でOK）。発信内容の傾向も一言 |
| L | 直近のニュース・プレスリリース | 直近6ヶ月の新商品発表・イベント参加・事業拡大などのニュース。特に営業の切り口になりそうなもの |
| M | 問い合わせメールアドレス | 公式サイトに記載のメールアドレス。なければ「サイト上に非掲載」 |
| N | 問い合わせフォームURL | 問い合わせフォームのページURL |
| O | 担当部署・担当者名 | 広報・マーケティング・協賛担当など、わかる範囲で |
| P | アプローチ時の一言メモ | この企業に最初のメッセージを送るとき、冒頭に入れると効果的な一言。例:「FWJでの協賛実績を拝見し〜」「〇〇という御社の理念にSFFのコンセプトと通じるものを感じ〜」など、具体的に |

### 出力ルール

1. **出力形式**: 添付Excelのカラム G〜P を埋めた状態で、Excelファイルとして出力してください
2. **文量**: 各セルは1〜3文程度で簡潔に。営業担当がパッと読んで使える分量にする
3. **見つからない情報**: 「確認できず」「サイト上に非掲載」など明記する。空欄にしない
4. **SFFとの共通点（カラムH）を特に重視**: ここが営業メッセージの核になるので、具体的かつ説得力のある内容にしてください
5. **Web検索を活用**: 各企業の公式サイト・SNS・プレスリリースを実際に確認してください

### 分割実行の指示（企業数が多い場合）

54社すべてを一度に調査するのが難しい場合は、以下のように分割してください：

- **第1回**: No.1〜No.12（12社）
- **第2回**: No.13〜No.27（11社）
- **第3回**: No.29〜No.46（12社）
- **第4回**: No.49〜No.78（11社）

各回の完了後に「次のグループを調査してください」と指示します。
最終的に全54社分の調査結果を1つのExcelファイルにまとめてください。
