### 2. コシコーヒー店：落選者へのコーヒー郵送サポート ★要判断

**目的：** 2/12の打ち合わせでコシコーヒー店から出た「落選者にコーヒーを郵送したい」という提案について、方針を決める。

**背景：** 落選した応募者に、黒「アクティブ」＋白「リラックス」の1週間分コーヒーセットを郵送したいという提案。郵送はコシコーヒー店が対応してくれる。ただし、落選者の個人情報（名前・住所）をShape側から渡す必要がある。なーすけは前向き。しおんは「社内で検討します」と即答は避けている。

**推奨案：受ける方向でOK。ただし個人情報は直接渡さず、落選通知時に「希望者のみ」で対応する。**

落選通知に「協賛企業からコーヒーセットをお届けできます。希望する方は住所を入力してください」というフォームを入れれば、同意取得と情報取得が同時にできる。

**決めたいこと：**

| 論点 | 推奨 |
|---|---|
| 企画を受けるか？ | → **Yes**（なーすけも前向き） |
| 個人情報の渡し方 | → **落選通知時にオプトイン方式**（希望者のみ） |
| コストは誰が持つ？ | → しおんが先方に確認（おそらく先方全額負担だが明確にしておく） |

### 3. コシコーヒー店：QRコード・看板露出 ★要判断

**目的：** 2/12に出た以下2つの提案について、回答方針を決める。

- **A）ドリップパック裏面にQRコード掲載**（なーすけYouTubeへ誘導。販売先：自社EC・店舗・東急ハンズ）
- **B）店舗外看板になーすけ露出**（協賛している旨の掲出）

**背景：** どちらも先方のスタンスは「載せさせてください」ではなく「載せられますよ」。こちらにデメリットがほぼない提案。なーすけも「別にいいんじゃないの」という反応。

**推奨案：どちらも条件付きでOK。**

- QRコード → OK。飛び先はYouTubeチャンネルTOPが無難（特定動画だとリンク切れリスク）。TOP協賛（ステディジャパン等）の権利と被らないか一応確認。
- 看板 → OK。ただし**掲出前にデザイン確認させてもらう**条件をつける。

**決めたいこと：**

| 論点 | 推奨 |
|---|---|
| QRコード掲載OK？ | → **Yes** |
| QRコードの飛び先 | → **YouTubeチャンネルTOP** |
| 看板露出OK？ | → **Yes（デザイン事前確認が条件）** |

### 4. NASK・OneTies Japan：イレギュラー協賛の方針 ★メイン議題

**目的：** 2/12の打ち合わせで出た、OneTies Japan（お兄さん）からの「70〜100万出せる」という話について、どう対応するか方向性を決める。

**背景：** 村田屋バウムクーヘン（NASK・弟）とワンピースカード限定トレカショップ（OneTies Japan・兄）の兄弟と合同打ち合わせを実施。お兄さんが積極的に資金提供の意志を示してくれているが、事業との親和性が低く、通常のリターン（ブース・ロゴ等）が機能しない。なーすけのお父さん繋がりなので丁寧な対応が必要。弟側（NASK）の協賛意向は未確定。

**接点になりうるポイント：**
- YouTubeインサイト（男性20代後半〜30代前半）が一致
- なーすけがルフィ好き → ワンピースカードとの接点あり
- 筋トレ×経営者の相関性（トレカ購入層に経営者が多い）

**推奨案：受ける方向で検討。「純粋応援型」をベースに、YouTube連携を追加オプションとして提案する。**

無理にブースやロゴでリターンを組むと双方しっくりこない。「応援協賛」として大会パンフ・SNS・司会紹介のみのシンプルなリターンをベースにし、YouTubeのインサイト一致を活かした動画企画を追加オプションで提案するのが現実的。

**決めたいこと：**

| 論点 | 推奨 |
|---|---|
| イレギュラー枠で受けるか？ | → **Yes**（金額・関係性の両面から） |
| 提案の方向性 | → **応援協賛（ロゴ・紹介のみ）＋YouTube連携オプション** |
| 弟側（NASK）はどうする？ | → しおんが意向確認してから判断 |
| カスタム提案書は誰が作る？ | → **鈴木が骨子作成 → しおんが先方に提示** |
