## 3. ぼんの推奨案（議論のたたき台）

### ✅ 推奨：「条件付きで受ける」＋「SFF主導に切り替える」

バイタスの提案をそのまま受けるのではなく、**SFF側が主導権を握る形に設計し直す**という方針です。

### 具体的にどうするか

| 項目 | やること |
| --- | --- |
| **映像素材の管理** | バイタスに素材を直接渡すのではなく、SFF側（おしりちゃん）が使用可能な素材を選定・提供する。どの映像を使うかはSFFが決める。 |
| **編集のチェック体制** | バイタス側で編集したものは、公開前に必ずSFF側で最終チェックを行う。SFFの世界観に合わないものは差し戻しできるルールにする。 |
| **ロゴの扱い** | 「バイタスのロゴ」ではなく「SFF × バイタス」の共同クレジット表記にする。SFFが主役、バイタスは応援者という見え方にする。 |
| **他社への説明** | 「編集リソースをバイタスさんが提供してくれている」という位置づけにすることで、他社から見ても「バイタスだけ特別扱い」に見えにくくする。 |
| **費用** | DIAMOND枠内の対応とする。追加費用は取らない代わりに「SFF側の最終承認権」を条件にする。これが実質的な対価。 |

### なぜこの案がいいのか

**理由① バイタスとの関係を壊さない**
先方は「SFFを盛り上げたい」という善意からの提案。断ると関係が冷える可能性がある。条件付きで受けることで、その気持ちに応えつつSFF側の主導権も守れる。

**理由② SFFのブランドを守れる**
素材選定も最終チェックもSFF側が行うので、世界観が崩れるリスクを最小限にできる。「SFF × バイタス」の共同クレジットにすれば、バイタスだけが目立つ形にもならない。

**理由③ 他社への説明がつく**
「バイタスさんが編集リソースを無償提供してくれている」という形なので、バイタスだけ広告枠が増えているようには見えない。他社から同じ提案が来ても「リソース提供してくれるなら検討します」と言える。

### ⚠️ この案のリスク

- おしりちゃんのチェック工数が増える。本業（YouTube）の負担にならないか確認が必要。
- バイタス側が「それなら自由にやらせてほしい」と言ってきた場合のフォールバック（代替案）も考えておく必要がある。

**基本の考え方：** SFFの映像素材を、バイタスの自社マーケティング活動に貸し出す。バイタスが自社アカウントで発信する形なので、SFFのSNS上にバイタスのロゴが載るわけではなく、ステディとの衝突リスクがなくなる。

### 確認ガイドライン（おしりちゃん用）の作成方針

おしりちゃんの工数を減らすため、以下のような**チェックリスト形式**で確認ポイントを事前に決めておく。

| チェック項目 | OKの基準 | NGの基準 |
| --- | --- | --- |
| SFFの世界観 | 「挑戦と物語」のトーンに沿っている | バラエティ的・おふざけ的な編集になっている |
| 映像の使い方 | 提供した素材の範囲内で使用 | 提供していない素材が勝手に使われている |
| テロップ・テキスト | 事実に基づいた内容 | 誤情報や過度な煽り文句が入っている |
| 音楽・効果音 | SFFの雰囲気を壊さない | 世界観と全く合わないBGMや効果音 |
| 出典表記 | 「映像提供：Shape Fit Festival」が入っている | 出典表記がない、またはバイタスだけの表記になっている |

※このガイドラインは、ルール文書化の担当者がおしりちゃんと一緒に詳細を詰める。

### なぜこの方式がいいのか

1. **バイタスの希望どおりなので交渉がスムーズ** — 先方のモチベーションを下げずに条件を付けられる
2. **ステディとの衝突がゼロ** — SFFのSNS上の話ではないので、TOPスポンサーの立ち位置を一切侵害しない
3. **しおんさんの説明が簡単** — 「バイタスさんが自社アカウントで大会の様子を紹介してくれているだけです」で済む
4. **他社にも同じルールを適用できる** — 「素材をお渡しするので、同じようにどうぞ」と言える
5. **SFF側の負担が最小限** — やることはチェックだけ
6. **おしりちゃんのチェック工数も管理可能** — ガイドラインがあるので「毎回考える」負担がなくなる
7. **品質リスクをコントロールできる** — 打ち切り条項があるので、最悪の場合でも撤退できる
