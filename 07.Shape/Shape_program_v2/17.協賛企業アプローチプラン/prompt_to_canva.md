なるほど、Reactで図解を作成して、それをCanvaで清書する際のベースにしたいということですね。ロゴやアイコンを活用した視覚的な資料で、配置・メッセージライン・メッセージ内容にこだわりたいと。

---

## **プロンプト（Canva清書前提のReact図解用）**

```
添付した資料「[ファイル名]」の内容を、Canvaで清書することを前提とした図解資料をReactで作成してください。

【目的】
- 資料の核心を視覚的に伝える
- Canvaでの再現・調整がしやすい構成にする

【デザイン要件】

■ ロゴ・アイコン
- 各要素（概念、ステークホルダー、プロセスなど）を表すシンプルなアイコンやロゴを作成
- SVGまたはシンプルな図形ベースで、Canvaで再現可能なレベルのシンプルさ
- 色は3〜4色程度に抑え、役割ごとに色分け

■ 配置
- 情報の流れが直感的に分かるレイアウト（左→右、上→下、または中心→外側）
- 余白を十分に取り、要素同士が窮屈にならないこと
- グリッドベースで整列し、Canvaでの位置調整がしやすい構成

■ メッセージライン（矢印・接続線）
- 要素間の関係性を示す矢印や線を明確に
- 線の種類で関係性の違いを表現（実線＝直接的、点線＝間接的、など）
- 矢印の向きで因果関係や時間軸を示す

■ メッセージ内容（テキスト）
- 各要素には短いラベル（2〜5文字程度）
- 補足説明は別途キャプションエリアに配置
- キーメッセージは1つに絞り、図の上部または中央に大きく配置

【出力に含めてほしいもの】
1. React図解コンポーネント
2. 使用した配色リスト（カラーコード）
3. Canva清書時の配置ガイド（どの要素をどこに置くかの簡易説明）
4. 各アイコン/ロゴの意味と使用意図の説明

【補足】
- 元資料の論理構造や意図を最初に整理してから図解設計に入ること
- 「この図で何を伝えたいか」を1文で定義してから作成すること
```

---

## **オプション：追加で指定すると良いこと**

用途に応じて以下を追加すると、より意図に合った出力になります：

```
【追加指定（必要に応じて）】

■ 図解のタイプ
- フロー型（プロセスや時系列）
- 構造型（階層や関係性）
- 比較型（Before/After、A vs B）
- サイクル型（循環するプロセス）

■ トーン・世界観
- ビジネス向け（シンプル・クリーン）
- フィットネス/エネルギッシュ（Shape Fitブランドに合わせる場合）
- 親しみやすい（コミュニティ向け説明）

■ 想定する閲覧者
- 経営陣向け（意思決定に必要な情報を強調）
- チームメンバー向け（具体的なアクションが分かる）
- 外部パートナー向け（協業メリットが伝わる）
```

---

これでベースとしては十分だと思いますが、実際に使う資料の性質（企画書なのか、条件整理なのか、提案書なのか等）に合わせて微調整すると精度が上がります。

実際に試してみて調整したい部分があれば、フィードバックをもらえればブラッシュアップしますね。
