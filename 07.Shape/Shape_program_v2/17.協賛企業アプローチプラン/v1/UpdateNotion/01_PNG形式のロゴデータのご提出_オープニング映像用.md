# PNG形式のロゴデータのご提出

チェックボックス: No
パッケージ名: 大会オープニング映像に企業名・ロゴ
対応方法: ファイル提出
対応期日: 2026年4月20日

### ■ご提出場所

大会の開幕時に上映されるオープニング映像内に、協賛企業様のロゴを掲載します。

下記データを格納先へご提出をお願いいたします。

**👉格納ドライブはこちら**

### ■提出条件

| 項目 | 内容 |
| --- | --- |
| ファイル形式 | PNG形式（背景透過推奨） |
| 推奨サイズ | 長辺 1,920px 以上 |
| 最低サイズ | 長辺 800px 以上 |
| 備考 | 映像内で拡大表示される場合があるため、できるだけ高解像度のデータをご提出ください |

※ロゴデータの使用に指定がある場合（例：背景色に合わせた白抜きロゴ等）は、ファイル名の末尾に「_指定あり」と記載の上、別途Slackにてご連絡ください。
