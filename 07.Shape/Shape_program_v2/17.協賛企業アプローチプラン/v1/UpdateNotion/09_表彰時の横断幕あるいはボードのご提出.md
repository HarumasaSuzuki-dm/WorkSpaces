# 表彰時の横断幕あるいはボードのご提出

チェックボックス: No
パッケージ名: 記念撮影
対応方法: Slack対応
対応期日: 2026年8月15日

### ■概要

表彰時にステージにて写真撮影を行なう際に使用するボードあるいは横断幕のご提出をお願いいたします。

**事前のご提出物はございません。**

**横断幕 or ボードを当日お預かりさせていただきます。**

### ■仕様・サイズについて

| 項目 | 内容 |
| --- | --- |
| 推奨サイズ | 横 約〇m × 縦 約〇m 以内（※ステージ上で撮影に収まるサイズ） |
| 形式 | 自立式パネルボード または 横断幕（吊り下げ式）のいずれか |
| 素材 | 特に指定なし。ただし自立式の場合はスタンド付きでお持ちください |
| 備考 | ステージ照明の関係上、光沢が強い素材は反射する場合があります。マット仕上げを推奨します |

※仕様についてご不明な点があれば、Slackにてお気軽にご相談ください。

### ■当日のお預かりについて

| 項目 | 内容 |
| --- | --- |
| お預かりタイミング | 大会当日の受付時（開場時刻〜） |
| お預かり場所 | 弊社スタッフが受付にてお預かりいたします |
| 返却 | 大会終了後、受付にてご返却いたします |

※事前に郵送での納品をご希望の場合は、大会の1週間前までに弊社宛にお送りください。送付先はSlackにてご案内いたします。
