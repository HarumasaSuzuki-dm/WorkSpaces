## 💪 4. フィットネス・スポーツとの親和性

MiRiceのライスケーキは、**フィットネス業界との親和性が非常に高い**商品です。noteの公式記事には以下のように明記されています：

> 「モデル・アスリート・ボディビルダーなどからも支持されています。例えば**ベストボディ・ジャパン**に出場する方々にも、食事管理の一環として使われている」

### ✅ フィットネス訴求ポイント一覧

| ポイント | 詳細 |
|----------|------|
| **カーボアップに最適** | 水分・脂質ゼロで純粋な糖質のみを補給。大会前の炭水化物チャージに最適 |
| **むくみ対策** | 無塩・低水分で体のシャープなラインを維持。コンテスト直前に重宝 |
| **低GI食品** | 血糖値の急上昇を防ぎ、エネルギーを持続的に供給 |
| **高携帯性** | 袋から取り出してそのまま食べられる。ジムバッグ・大会の控室でも利用可能 |
| **カロリー計算が簡単** | 1枚7.7kcalと明確。PFCバランス管理がしやすい |
| **グルテンフリー** | 食物アレルギーを持つアスリートにも対応 |
| **ISO22000認証工場製造** | 食品安全性の国際認証を取得済みの高品質製造 |

> **糖質カットだけが正解じゃない。「質の良い糖質を、必要なタイミングで摂る」ことが大切。ライスケーキはその理想を叶える食材です。** — MiRice公式note

## 🌱 8. 社会的価値・ブランドの深み

MiRiceが他のライスケーキブランドと差別化される独自性として、**社会的貢献活動**が挙げられます。

- **里山保全への貢献**: 里山の農地・景観を維持する活動に関連
- **就労支援への貢献**: 障がいのある方などの就労機会創出（Instagramバイオに明記）

これは単なる食品販売を超え、「購入することで社会貢献につながる」という**エシカル消費**の文脈でブランド価値を高めています。
