### 過程イベントの企画について

**■ Belisse側からの草案発表**

Belisse側で検討した月別スケジュール案を共有いただいた。

| 時期 | 内容 | 形式 |
| :---- | :---- | :---- |
| **4月上旬**（選手確定後） | コンテストサポート専門サロンが教える大会準備セミナー（サロン紹介・役割・大会スケジュール・カラーリング効果の概要説明）。参加者への割引クーポン配布も検討 | オンライン（Zoom）1〜1.5時間 |
| **5月** | 最高の身体でステージに立つためのカラーリング・ハイパーナイフ施術説明会（座学＋カラーリング・ハイパーシェイプのデモンストレーション）。参加者向けにハイパーナイフ回数券等の特典も検討 | オフライン（都内を想定） |
| **6月** | 衣装選びとヘアメイクのポイント（衣装の選び方・着方・入手先のアドバイス。ヘアメイク相談）。フィジーク・ビキニ経験者のスタッフやプロ選手も同席予定 | オフライン |
| **7月** | 減量停滞打破のための施術体験会（ハイパーナイフ・ハイパーシェイプ・リラクゼーション・整体など複数メニューから選択）。サロン（福生）または都心レンタルスペースへ出張で実施も検討 | オフライン（場所は要調整） |
| **8月上旬**（関西大会前） | 大会直前グループポージングセミナー①。SFFトレーナー派遣またはBelisse提携のフィジーク・ビキニ選手が講師として対応 | オフライン |
| **9月上旬**（関東大会前） | 大会直前グループポージングセミナー②（同上） | オフライン |

- **毎月何かしら開催**するスタンスで、選手が都合に合わせて参加できる機会を確保したいとのこと。  
- 認知拡大（オンラインセミナー）→ 深化・体験（オフラインセミナー・体験会）→ 直前サポート（ポージングセミナー）という段階的な設計を双方で確認。

**■ ポージングセミナーの開催主体について**

- SFF側で自社トレーナーの派遣の可否を確認し、難しい場合はBelisse提携の選手が対応する方向で調整する。

### Luxy Bronze（ラグジュアリーブロンズ）のカラーリング協賛について

- SFF側よりLuxy Bronzeからカラーリング協賛について打診があることをお伝えした。  
- 以下の懸念点をご共有いただいた。  
  - サロンにより使用するリキッドや技術水準が異なり、色ムラが生じやすい。  
  - Luxy Bronzeは業務委託形式で対応するため、全スタッフの研修・品質統一が保証しにくい。  
  - 複数カラーリング業者が混在すると、審査時に選手間の色の差が生じ、選手・審査員双方に不利益となる。  
  - 現在、遠方選手向けには信頼するサロンを個別紹介しており、予約・協賛金の管理も一元化して運用している。  
- 以上を踏まえ、カラーリングでの協賛参入はSFF・選手・既存スポンサーいずれにとってもリスクが高いと判断。  
- **結論：Luxy Bronzeへのカラーリング協賛は断る方向**とし、脱毛や筋膜リリース（ジュラシック）など別カテゴリーでの協賛を案内する。
