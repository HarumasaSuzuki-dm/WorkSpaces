# スケジュール設計（MTG用たたき台）

---

## 前提：このスケジュールが確定するために、MTGで決めること

### 決定事項①：オンライン併用の要否

- パターンA（オフライン単体）→ すぐ始められる
- パターンB（オンライン併用）→ 準備期間が伸びる＋工数増
- **提案はA（オフライン単体から）**
  - 判断の「原型」を取り出す目的にはAが最適
  - Bは後からでも検証できる。逆順は不可
- → **ここが決まらないとPhase 1の期間が確定しない**

### 決定事項②：対象者の方針

- 既存顧客から選ぶ or 新規募集か
  - 既存 → すぐ動ける。関係性がある分、初見の原型は取りにくい
  - 新規募集 → 準備に2〜3週間かかる。初見×対面の原型が出やすい
- 選定基準の確認
  - 「やりやすい人」ではなく「判断の分岐が多く出そうな人」
  - ペルソナ像との一致度
- 「実験を兼ねている」と伝えるか否か
- → **ここが決まらないと1人目の日程が決まらない**

### 決定事項③：実施の進め方

- **月に何回セッション枠を確保できるか？**
  - 月2回 → 順番型で約3ヶ月 / 並行型で約2ヶ月
  - 月3〜4回 → 順番型で約2ヶ月 / 並行型で約1.5ヶ月
- **3人を並行でやるか、順番にやるか？**
  - 順番 → 1人目の学びを2人目に活かせる。時間はかかる
  - 並行 → 短期で完了。ただし振り返り・FBの負荷が集中する
- → **ここが決まらないとPhase 2の期間が確定しない**

---

## たたき台：全体タイムライン

> 以下は「パターンA（オフライン単体）・月2回枠・順番型」を仮置きした場合のスケジュール。
> MTGでの決定事項に応じて調整する。

### Phase 0：準備（MTG後 〜 2週間）

- **ゴール：「1人目の日時・場所・相手・記録方法・FB体制」がすべて決まった状態**
- なーすけさん
  - 対象者の確定・声かけ（or 募集開始）
  - 1人目の日程・場所の確定
  - 対象者への事前説明（実験を兼ねている旨の伝達方針に沿って）
- ぼんちゃん
  - フィードバック観点の確定版を共有
  - 振り返りフォーマットの確定版を共有
  - かおりさんへの協力依頼・役割共有
  - 動画共有の手段確定（Googleドライブ等）
  - 契約まわりの整理（参加者との合意事項）

### Phase 1：実施（約2〜3ヶ月）

- **ゴール：3人分のセッション完了＋判断基準の素材が蓄積された状態**

#### 1人目

- セッション実施（1〜2回）
- 当日：なーすけさん振り返り記入
- 動画共有 → ぼんちゃん・かおりさんFB（1週間以内）
- すり合わせMTG
  - 振り返り × FBの突き合わせ
  - 「型／裁量／思想」の仮仕分け（初回）
  - **1人目の学びを踏まえて、2人目の選定・進め方を微調整**

#### 2人目

- セッション実施（1〜2回）
- 同上のFB・すり合わせサイクル
- 1人目との比較で判断基準の共通点・差異を確認

#### 3人目

- セッション実施（1〜2回）
- 同上のFB・すり合わせサイクル
- 3人分を通して判断基準のパターンを確認

### Phase 2：翻訳（Phase 1完了後 〜 約1ヶ月）

- **ゴール：判断基準が「思想／型／裁量」の3層に整理され、言語化された状態**
- やること
  - 蓄積した判断基準の整理・分類
  - コーチ設計の土台としてドキュメント化
- この先に繋がるもの
  - コーチ育成の基準
  - レビュワーの判定基準
  - 思想拠点で語られる内容

## 実施サイクル（毎回のオペレーション）

```
セッション当日
  └→ なーすけさん：振り返りフォーマット記入（当日中）
  └→ 動画をぼんちゃん・かおりさんに共有

〜1週間以内
  └→ ぼんちゃん・かおりさん：動画確認＋FB作成
      ※なーすけさんの振り返りは先に読まない

FB完了後
  └→ すり合わせMTG
      ├ 振り返り × FBの突き合わせ
      ├ ズレの確認（＝無意識の判断の発見）
      └ 判断基準の蓄積・仮分類
```

## MTGでの確認チェックリスト

- [ ] オンライン併用の要否 → A or B、この場で決定
- [ ] 対象者の方針 → 候補名 or 募集方法を決定
- [ ] 月のセッション確保枠 → 月○回
- [ ] 並行 or 順番 → どちらで進めるか
- [ ] 1人目の実施目標日 → ○月○日頃
- [ ] 3人完了の目標月 → ○月末
- [ ] 動画共有の手段 → ツール確定
- [ ] かおりさんへの依頼タイミング → いつ・誰から
- [ ] 契約・合意事項の範囲 → どこまで必要か確認
