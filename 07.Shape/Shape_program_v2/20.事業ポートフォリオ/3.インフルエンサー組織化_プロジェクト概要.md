# Shape Fit｜インフルエンサー組織化構想ブリーフィング

## 1. 背景

Shape Fitはフィットネス系YouTuber・なーすけが率いる事業体。主力コンテンツ「ノーストレス減量」を軸に、YouTube（視聴者の約90%が男性・ジム契約者）、SFF（フィットネス大会）、SFGジム（フランチャイズ）、コミュニティ事業を展開。現在、ノーストレス減量のカリキュラム化を最優先プロジェクトとして進行中（ローンチ目標：約1年後）。

**構造的課題：** 認知・集客・ブランドのすべてが「なーすけ個人」に依存しており、「なーすけを知っている人しか買わない」状態の突破が経営課題。一方で、ブランドの希薄化やコモディティ化への強い警戒がある。

---

## 2. 山本さんからの提案

外部パートナー・山本氏より2つの協業提案を受領。

### 提案A：インフルエンサー組織化（なーすけ側の優先候補）

- 料理系・暮らし系・ダイエット系インフルエンサーを「認定講師」として組織化
- LINE/DM誘導で教材を紹介、紹介報酬10〜30%のアフィリエイト構造
- Instagram女性市場へのアクセスを獲得（現在リーチできていないセグメント）
- 狙い：なーすけ依存脱却 × ライト層・女性層への拡張 × カリキュラムの販路構築

### 提案B：リアル店舗（関節整体×フィットネス）

- 山本氏の関節整体サロン（19店舗）との連携による新レーベル
- 低コスト出店（内装費約50万円）、レベニューシェア型
- なーすけ側は提案Aを先行させる意向。店舗は後回しの認識

---

## 3. 核心的トレードオフ

本案件の意思決定を難しくしている構造的な対立軸は以下の3つ。

| 対立軸 | 得るもの | 失うリスク |
|---|---|---|
| **拡張 vs 濃度** | ライト層・女性市場へのリーチ | ブランドの世界観の希薄化 |
| **脱・なーすけ vs なーすけらしさ** | スケーラビリティ、本人の稼働軽減 | 「直接関わってくれる感」の喪失 |
| **協業 vs 自律** | 販路・マーケ機能の獲得、ローンチ不安の軽減 | 意思決定権、撤退の自由度 |

なーすけ本人は「乗りたい」方向に傾いているが、「解像度が荒い」と自覚しており、好意的印象が判断を曇らせるリスクも認識されている。

## 4. 未解決の論点

### A. 事業設計に関する論点

| # | 論点 | 判断に必要な情報 |
|---|---|---|
| A1 | インフルエンサーの選定権は誰が持つか | 属人化リスク（なーすけ選定）vs 質の担保 |
| A2 | 認定講師の役割範囲と品質管理 | 稼働強度、発信チェック体制、修正権限の所在 |
| A3 | 山本氏側の収益モデル | 利害一致の検証。Shape Fitが成長しなくても山本氏が得をする構造でないか |
| A4 | 撤退設計 | 契約解除条件、インフルエンサー個別契約の取扱い、ブランド毀損時の対応 |
| A5 | 既存事業とのカニバリ/シナジー | SFF「挑戦と物語」路線とライト層拡張の整合性、SFGジムとのターゲット重複 |

### B. リソース・優先順位に関する論点

| # | 論点 | 判断に必要な情報 |
|---|---|---|
| B1 | カリキュラムローンチとの関係 | 競合するのか加速手段なのか。山本氏と組むことでリソース不足は解消されるか |
| B2 | 社内リソースの限界ライン | 現チームで同時に回せるプロジェクト数。マネジメント工数の爆発リスク |
| B3 | 時間軸の設計 | 今すぐ組む必要があるか。1年後でも関係性維持は可能か |

### C. ガバナンス・組織に関する論点

| # | 論点 | 判断に必要な情報 |
|---|---|---|
| C1 | 意思決定権の配分 | レベニューシェア条件次第で決定権が実質移転するリスク |
| C2 | メンバー共有の設計 | タイミング（ヒアリング前/後）、対象（全員/コア）、目的（意見収集/合意形成） |
| C3 | 「なーすけブランド」と「Shape Fitブランド」の分離設計 | 組織化は実質的にブランド分離の第一歩。その覚悟と設計方針 |

## 5. なーすけが「譲れない」と明言した条件

1. インフルエンサーとの関係性（仲間であり、使い捨てにしない）
2. Shape Fitの未来とブランドの毀損防止
3. 組織の熱量の維持
4. 組織が崩れないこと
5. 内部メンバーの余白（疲弊させない）
