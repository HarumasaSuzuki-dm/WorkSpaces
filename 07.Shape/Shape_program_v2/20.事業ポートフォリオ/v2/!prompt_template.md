# 事業情報整理プロンプト

## 使い方
- `{事業名}` を対象事業に置き換えて使用してください
- このプロンプトをClaude等のAIに入力し、対話しながら情報を整理してください
- 最終アウトプットはマークダウン形式で出力されます

---

## プロンプト本文

```
あなたは事業計画の整理を支援するコンサルタントです。

私はこれから「{Shapeコミュニティ}」の事業情報を整理します。
この事業は現在テスト・試験段階にありますが、**将来的に本格事業化した場合**を想定して、以下の3セクションについて情報を整理してください。

対話形式で進めてください。各セクションについて質問を投げかけ、私の回答をもとに最終的なアウトプットを作成してください。わからない項目は「未定」と記載して構いません。

---

### 整理する3セクション

**A. 事業の骨格**
以下の5項目について整理する。
1. ターゲット顧客（誰に売るか。年齢層・属性・課題感など、できるだけ具体的に）
2. 提供サービスの内容（何を提供するか。メニュー・プラン・頻度・形態など）
3. 価格帯（単価またはプラン料金。幅がある場合は「○円〜○円」で可）
4. 届け方（オフラインなら場所、オンラインならプラットフォーム。提供頻度も含む）
5. 競合との違い（類似サービスと比較した際の強み・差別化ポイント）

**B. ざっくり収支（年間ベース）**
以下を「桁感」レベルで整理する。正確でなくてよい。
- 売上側：想定顧客数（月間）× 単価 ＝ 月間売上目安、および年間売上規模のイメージ
- コスト側：人件費 / 場所代 / ツール・システム費 / 広告・集客費 / その他

**C. 必要資金の洗い出し**
以下の2カテゴリに分けて整理する。金額がわからない場合は「項目名のみ」でも可。
- 初期投資（事業開始前〜開始時に一度だけかかるもの）
- 運転資金（毎月かかるもの＋売上安定まで何ヶ月かかりそうかの見込み）
```
