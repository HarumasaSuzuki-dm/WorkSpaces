## 5. フレアネストへの報酬設計（たたき台）

### 構造の整理

フレアネストへの支払いは**2層構造**で設計する。

```
Layer 1｜定常費（人件費として毎月計上）
  → しょうまくんのPM稼働に対する固定の業務委託費

Layer 2｜変動費（案件ごとに発注）
  → MYS・システム・HPなどの案件をフレアネスト or 他社に外注
  → しょうまくんが発注判断。フレアネストが受けた分がそのまま売上に
```

### Layer 1｜定常費の考え方

- 2027年2月期計画の業務委託料欄（月約211〜246万円）の中に位置づける
- しょうまくんへの固定報酬として月□万円を設定（Excelで試算）
- フルコミット前（〜9月）は現状21万円を基準に業務範囲を再定義した上で継続

### Layer 2｜変動費の考え方

- 案件単価の目安：HP制作50〜100万円・運用保守3〜5万円/月・システム案件都度
- しょうまくんが市場価格と比較して発注先を判断（フレアネスト or 他社）
- フレアネストへの外注はShapeFitの売上原価に計上

### Layer 3｜利益分配（2027年2月期以降）

- **発動条件：** 2027年2月期の営業利益が黒字で確定した時点
- **分配対象：** 営業利益の□%（下記「確認が必要な論点」参照）
- **対象者：** しょうまくん（フレアネストを通じて、または個人として）
- **支払タイミング：** 決算確定後（年1回）
- **2028年以降：** MYS収益本格化に合わせて分配率・上限を再設計
