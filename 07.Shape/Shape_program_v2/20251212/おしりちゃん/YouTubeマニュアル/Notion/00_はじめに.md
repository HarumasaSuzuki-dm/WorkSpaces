# 📂 00_はじめに

## 【必読】このマニュアルについて・編集者の皆様へ

「なーすけFitness」の動画編集チーム（または採用トライアル）へようこそ。
本マニュアルは、なーすけ様の動画制作における **「基準（スタンダード）」** と **「心構え（マインド）」** をまとめたものです。

作業に入る前に、必ずご一読ください。

---

### 1. マニュアル作成の目的

このマニュアルの最大の目的は、**「チーム全員が、迷わずに『なーすけFitness』のクオリティを再現できる状態を作ること」**です。

これまで口頭や個別のメモで共有されていたルールを統合し、以下の状態を目指します。

1.  **「正解」の可視化：** 誰が編集しても、なーすけ様の魅力や動画の意図がブレないようにする。
2.  **リスクの排除：** チャンネルの存続に関わる「映り込み」や「コンプライアンス」のリスクを確実に防ぐ。
3.  **役割の明確化：** 技術的な細部を編集者様が完結させることで、より良いコンテンツ作りに集中できる体制を作る。

---

### 2. あなたに期待する役割とゴール

私たち編集チームが目指すゴールは、単に「素材をつないでテロップを入れること」ではありません。
**「視聴者がストレスなく最後まで見られるVlog」**を作り上げ、なーすけ様の想いを視聴者に届けることです。

そのために、編集者の皆様には以下の役割（責任範囲）をお願いしています。

#### ✅ 編集者様の責任範囲（完結させてほしいこと）
おしりちゃん（クライアント）に提出する前に、以下の項目は**100%完璧な状態**にしてください。

* **リスク管理の徹底：** 住所、下着、肌の露出などの「映り込み」に対するモザイク処理。
* **基礎クオリティの担保：** 誤字脱字、音声レベルの調整、ノイズの除去、指定フォーマット（フォント・色）の遵守。
* **スムーズな納品：** 指定されたファイル形式、フォルダ構成でのデータ共有。

#### 🤝 おしりちゃんの役割（最終チェック）
編集者様が技術的な基盤を完璧にしてくださることで、おしりちゃんは以下のチェックに全力を注ぐことができます。

* **演出面の確認：** なーすけ様の想いが正しく伝わるか。
* **全体俯瞰：** 動画としての面白さや、構成の流れに違和感がないか。

> **Point:**
> 「モザイク漏れがないか」「誤字がないか」をおしりちゃんが探す時間は、コンテンツの質を高めるための時間ではありません。**「技術的なミスは編集段階でゼロにする」**というプロ意識を持って作業にあたってください。

---

### 3. このマニュアルの構成と使い方

本マニュアルは、大きく分けて以下のセクションで構成されています。

| セクション | 内容 | 重要度 |
| :--- | :--- | :--- |
| **01_編集方針・マインド** | 「誰に」「何を」伝えるか。編集判断の核となる考え方。 | ★★★ |
| **02_重要ルール・リスク管理** | **【最重要】** チャンネルを守るためのNG事項と表記ルール。 | ★★★ |
| **03_実務・テクニカルガイド** | VREWやFCPXの具体的な設定値、デザインの数値など。 | ★★☆ |
| **04_納品・運用フロー** | データの受け渡し方法やコミュニケーションのルール。 | ★★☆ |
| **05_採用トライアル** | 新規応募者向けのトライアル課題と、その評価観点。 | ★★☆ |
| **06_編集者評価制度** | 編集者グレード・評価基準・インセンティブ/昇格フローの整理。 | ★★★ |

作業中に迷ったときは、まずこのマニュアルの該当箇所を確認してください。それでも解決しない場合は、遠慮なく質問してください。

それでは、素晴らしい動画を一緒に作っていきましょう！
