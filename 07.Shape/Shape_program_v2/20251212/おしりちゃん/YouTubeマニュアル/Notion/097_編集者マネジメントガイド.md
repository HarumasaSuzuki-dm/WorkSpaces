# 📂 097_編集者マネジメントガイド (Internal Use Only)

**⚠️ 注意：このページは管理者（おしりちゃん・チーフ）専用です。**
編集者には共有しないでください。

このガイドは、編集チームの「モチベーション維持」と「リスク管理」を行うための運用マニュアルです。
**「属人化を防ぎ、誰が管理してもチームが回る状態」**を目指します。

---

## 🌱 1. オンボーディング（導入〜定着まで）

採用直後の2週間が、編集者の定着率を決める最も重要な期間です。
「放置」せず、以下のステップでチームに馴染ませてください。

### Step 1: 環境セットアップ（初日）
* [ ] **権限付与:** Googleドライブ（素材フォルダ）、Notion（マニュアル）へのアクセス権を付与。
* [ ] **LINEグループ招待:** 挨拶を促す。
    * *管理者からの紹介メッセージ例:* 「新しく編集チームに入ってくれた〇〇さんです！以前は××系の動画を作っていたそうです。これからよろしくお願いします！」（※既存メンバーが反応しやすい空気を作る）

### Step 2: 「赤ペン先生」期間（1本目〜3本目）
最初の3本は、クオリティの基準（目線合わせ）を行う期間です。
ここで妥協すると、後で修正コストが膨らみます。

* **方針:** 細かい点（0.1秒の間、テロップの位置ズレ）まで徹底的にFBする。
* **伝え方:** 「最初は細かく見ますが、これは〇〇さんと長く付き合いたいからこそ、最初に基準を合わせたいからです」と前置きする。

### Step 3: 独り立ち（4本目以降）
* 大きなミスがなければ、「ここからは修正なしでお任せします！」と伝え、信頼していることを示す。
* **権限委譲:** 簡易的な判断（BGM選定など）は編集者の裁量に任せる。

## 🗣 2. フィードバックとコーチングの技術

修正指示は、編集者のやる気を削ぐ最大の要因になり得ます。
「否定」ではなく「改善」として伝えるテクニックを使ってください。

### 🍔 サンドイッチ話法（Good - More - Good）
修正点を伝える際は、必ず「良い点」で挟んで伝えます。

1.  **Good（承認）:** 「今回の冒頭の掴み、テンポ良くてすごく良かったです！」
2.  **More（修正）:** 「ただ、中盤の食事シーンの間が少し長いので、あと2秒詰められますか？」
3.  **Good（期待）:** 「そこさえ直れば完璧です！よろしくお願いします。」

### 🚫 使ってはいけない「NGワード」
抽象的な指示は、編集者を混乱させます。

| NGワード（抽象的・感情的） | OKワード（具体的・建設的） |
| :--- | :--- |
| 「なんか違う」「変」 | 「BGMがシーンの感情と合っていない気がします」 |
| 「つまらない」 | 「会話の間延びが気になります。もっとテンポアップできますか？」 |
| 「普通にやって」 | 「マニュアルの〇〇ページのデザイン通りにお願いします」 |
| 「前も言ったよね？」 | 「再発防止のため、チェックリストに追加してもらえますか？」 |

### 📊 数字（ファクト）を使ったフィードバック
感覚論で納得しない編集者には、YouTubeアナリティクスのデータを共有します。
* 「〇〇さんの編集した動画、**視聴維持率が平均より5%高かったです！**」
* 「ここの離脱が多いので、次はもっと早めにカットしてみましょう」

## 🤝 3. チームビルディング（空気作り）

リモートワークの編集者は孤独です。「自分はチームの一員だ」と思わせる仕掛けが必要です。

### 褒める時は「公開（全体）」で
* 良い編集や、素晴らしい気遣い（報告の速さなど）があった場合は、個別のDMではなく**「LINEグループ全体」**で褒める。
    * *効果:* 本人の承認欲求が満たされる ＋ 他のメンバーへの良い手本になる。

### 叱る時は「裏（DM）」で
* ミスの指摘や注意は、必ず**「個人宛のDM」**で行う。
    * 公開処刑は絶対に避ける（チーム全体の士気が下がります）。

### 横の繋がりを作る
* 雑談歓迎の雰囲気を作る。「最近買ってよかったプラグインある？」「このフォントおすすめ」など、編集者同士で情報交換できる空気感を、管理者が率先して作る。

## ⚠️ 4. 代謝・リスク管理（イエロー/レッドカード）

パフォーマンスが低い、またはルールを守れない編集者に対し、**感情を挟まずに**対応するための基準です。

### 🟨 イエローカード（警告）基準
以下の行動が見られた場合、DMで注意勧告を行います。
1.  **連絡不通:** 平日に24時間以上レスポンスがない。
2.  **納期遅延:** 事前連絡なしに納期を過ぎる。
3.  **同じミスの繰り返し:** 3回以上同じ修正（誤字、モザイク漏れ）を指摘されている。

### 🟥 レッドカード（契約解除）基準
以下に該当する場合、アカウント権限を剥奪し、契約を終了します。
1.  **イエローカード累積:** 改善が見られず、警告が3回溜まった場合。
2.  **重大なコンプライアンス違反:** 住所情報の流出（未遂含む）、素材の私的流用。
3.  **音信不通:** 48時間以上連絡が取れず、業務に支障が出た場合。

### 🗑 契約終了時のTODOリスト（セキュリティ）
お別れが決まったら、速やかに以下のアクセス権を削除してください。
* [ ] Googleドライブの共有解除（素材の持ち出し防止）。
* [ ] Notionの閲覧権限解除。
* [ ] LINEグループからの退会処理。
* [ ] VREW等の共有アカウントがあればパスワード変更。
