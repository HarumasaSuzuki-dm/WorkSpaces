# 📝 00-2. [DB] 運用シート集

このセクションは、読み物ではなく\*\*「現場ですぐに使うためのツール（データベース）」\*\*です。
Shape Fitが重視する「体重以外の変化（メンタル・睡眠・自律度）」を可視化し、お客様とトレーナーの共通言語にするために使用してください。

-----

## 1\. 目的と運用ルール

従来のジムのように「体重と体脂肪率」だけを管理すると、お客様は必ず「数字の奴隷」になります。このシート集の目的は、**「数値化しにくい変化」をあえて数値化し、お客様の視点を「人生の質（QOL）」へ移動させること**です。

### 運用の鉄則

1.  **診断ではなく「鏡」として使う**
      * 「点数が低いからダメ」とジャッジするのではなく、「なぜ低いのか？」を一緒に考える材料にしてください。
2.  **体重計の前にこれを見る**
      * セッション時は、必ず体重測定の**前**に「朝の勝利シート（睡眠・メンタル）」を確認します。「体重は減っていないが、睡眠の質は最高」であれば、それは勝利です。

-----

## 2\. ツール一覧

### ① 【入会前】準備状態チェックシート（Readiness Check）

**目的：** 「他責思考（お金を払えば痩せさせてくれる）」のお客様を見極め、入会前にマインドセットを「自律型」へチューニングする。
**タイミング：** カウンセリング時・契約前

| チェック項目 | 質問の意図・確認ポイント |
| :--- | :--- |
| **□ 目的の解像度** | 「痩せたい」の先にある\*\*「どんな生活・感情を手に入れたいか」**が語れるか？<br>（NG例：なんとなく自信をつけたい / OK例：服を選ばず着て、友人の誘いを断らない自分になりたい） |
| □ 責任の所在 | 「トレーナーは地図を渡すが、歩くのは自分」という合意ができているか？<br>（「忙しいから管理してほしい」という丸投げ発言がないか確認） |
| □ 優先順位の変更 | 短期的な体重減少よりも、「睡眠」や「メンタル」の安定を優先する勇気があるか？<br>（「生理が止まってもいいから痩せたい」という危険思想がないか） |
| □ 失敗の開示 | 食べたことやサボったことを**「怒られる」と思わず正直に報告できる\*\*約束ができるか？<br>（「怒らないので、隠さず教えてください」と伝え、反応を見る） |

> **トレーナーの判断基準：**
> 全てにYESがつかない場合、すぐに契約せず「プレセッション（意識合わせ）」を挟むか、勇気を持って「今はタイミングではない」とお断りすることも検討してください。

-----

### ② 【日常】朝の勝利シート（Morning Victory）

**目的：** 「痩せたかどうか」ではなく\*\*「今日1日を気分よく始められたか」\*\*を最重要KPI（成果指標）にする。
**タイミング：** 毎朝（LINE報告または手帳記入）

**【スコアリング基準（10点満点）】**
お客様に以下の基準を渡し、毎朝点数をつけてもらいます。

  * **10点（完全勝利）：** 目覚ましなしでスッキリ起きた。身体が軽く、今日やることにワクワクしている。
  * **7点（通常勝利）：** 普通に起きられた。日中の眠気もなく、頭はクリア。
  * **4点（イエローカード）：** 起きるのが辛い。二度寝した。身体が重い。
  * **1点（レッドカード）：** 全く起きられない。絶望感がある。何もしたくない。

**【トレーナーの声かけ例】**

> 「今週は体重は横ばいですが、**『朝の勝利』スコアが平均4点から7点に上がっていますね。**
> これは脂肪が燃える準備が整った証拠であり、何より生活の質が上がっている素晴らしい成果です」

-----

### ③ 【運動】ストレス・ポートフォリオ（負荷調整マトリクス）

**目的：** その日のストレスレベルに合わせて、運動強度を最適化する（＝やりすぎによる「Distress（悪いストレス）」を防ぐ）。
**タイミング：** セッション開始時・メニュー作成時

**【運用フロー】**
ご来店時、お客様に「今日のライフスタイル（仕事・家庭）のストレス度」を申告してもらい、メニューを決定します。

| ストレス度 | 状態（サイン） | 推奨メニュー（処方箋） |
| :--- | :--- | :--- |
| **低（Green）** | 余裕がある。<br>睡眠十分、意欲的。 | **通常トレーニング**<br>（負荷をかける、新しい種目に挑戦する） |
| **中（Yellow）** | 少し疲れている。<br>仕事が忙しいが動ける。 | **現状維持・調整**<br>（重量を追わず、フォーム確認やストレッチ種目を優先） |
| **高（Red）** | 余裕がない。<br>睡眠不足、イライラ、頭痛。 | **グラウンディング・ジャーニー（外歩き）**<br>またはストレッチのみ。<br>※「今日はジムに来ただけで100点」とし、早めに帰宅させる。 |

> **重要：**
> ストレス「高」の日にハードなトレーニングをさせると、コルチゾール（ストレスホルモン）が増加し、逆に痩せにくい体になります。**「今日は歩くだけにしましょう」と提案できるのがプロのトレーナーです。**

-----

### ④ 【食事】直感食べ・昇格検定シート（Intuitive License）

**目的：** アプリでの食事管理（カロリー計算）から卒業し、なーすけ流の「直感食べ」へ移行しても良いか判定する。
**タイミング：** 減量開始から2〜3ヶ月目（フェーズ移行期）

以下の「行動プロセス」と「感覚精度」がクリアできれば、アプリ記録の頻度を減らします。

#### A. 行動ベースの確認プロセス（3ステップ）

食欲が湧いた時、無意識に以下の行動が取れていますか？

1.  **水分補給（Water First）**
      * [ ] 食べる前に、まず水や炭酸水を飲んで一息ついている。
2.  **意識の切り離し（Environment Change）**
      * [ ] 食材の目の前から一度離れ、立ち上がって数分間別のことをしている。
3.  **意識の切り分け（Self-inquiry）**
      * [ ] 「これは本当の空腹か？ それともストレスや退屈によるニセの食欲か？」と自問できている。

#### B. 感覚精度の確認（誤差測定ゲーム）

  * **テスト方法：** 食事記録をつける前に「これは約○○kcalで、脂質はこれくらい」と予想し、答え合わせをする。
  * **合格ライン：** \*\*「予想と実測の誤差が±10%以内」\*\*の状態が1週間以上続いている。

> **判定後のアクション：**
>
>   * **合格：** 週1〜2回の記録のみに変更（残りは直感でOK）。
>   * **不合格：** まだ「感覚」と「事実」がズレています。「自分を責める必要はありません。まだ補助輪（アプリ）が必要なだけです」と伝え、記録を継続します。
