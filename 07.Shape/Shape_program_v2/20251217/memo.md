### **A. 山頂の景色（1年後、最高と言える状態）**

#### **Q1. 1年後に何を見て「最高だった」と言いますか？**

*  **組織全体の一体感:** 業務委託のトレーナーを含め、全員が「ShapeFit（シェイプイット）」のことを深く理解し、同じ方向を向いてサービスを提供できている状態 。


*  **トレーナーの自律性:** 受け身ではなく、トレーナー自身が自分の強みを理解し、「もっとこんなことができるのではないか」と自主的に提案・創造している状態（「みんなで作っている感覚」がある） 。


*  **事業間の連携強化:** SFF（ShapeFit Festival等のイベント事業と推察）とジム事業のつながりが強くなり、情報共有が活発化している 。


*  **PDCAの高速化:** サービスがある程度固まり、お客様の声を常に拾い上げ、即座に改善・反映できる体制ができている 。


*  **季節性の克服:** 冬場（SFFのオフシーズン）であっても、ジム単体でお客様の熱量やフィードバックを感じられ、活発に動けている状態 。



#### **Q2. その“最高の状態”が実現している1日を（朝〜夜で）どんなふうに描きますか？**

* **朝の習慣（儀式）:**
* 全員で「昨日の成功事例」や「顧客フィードバック」を共有する朝会を実施 。


*  **表彰文化:** 「昨日の声掛けが良かった」「自分らしさが出ていた」など、些細なことでも拍手し合い、ポジティブな空気で一日をスタートさせる 。




* **日中の動き（フィードバックの循環）:**
* お客様からのフィードバックに対し、「全部やる」のではなく「自分たちの軸（なぜそうしているか）」と照らし合わせて判断し、ネクストアクションを決めるという意思決定が、各所のミーティングで自律的に起きている 。


* 単にセッションをこなすだけでなく、「もっとどうしたら良くなるか」を考える時間が確保されている 。




* **各メンバーの理想的な動き:**
*  **話者1（リーダー層）:** メンバーやお客様に対し、ビジョンや「太い哲学の軸」を伝え続け、コミュニケーションを取り、視座を引き上げている。同時に、外部からのインプットも積極的に行っている 。


*  **話者3（クリエイティブ/企画層）:** 「これをやったら素敵そう」というアイデアを発散させる時間や、熟考する時間が確保されており、ワクワクしながら働いている 。


*  **話者2（マネジメント/ハブ層）:** 全体の状況を把握し、メンバーの話を聞いて整理し、「このまま進んで大丈夫」と安心感を与え、人をつないでいく役割を果たしている 。





#### **Q3. その状態を壊す「絶対にやらないこと」は何だと思いますか？**

*  **サービス品質の妥協:** サービスの質や理念浸透を犠牲にしてまで規模拡大（FC展開など）を急ぐこと。自信を持って友人に紹介できない状態になること 。


*  **数字ファースト:** 理念を置いてきぼりにして、売上や集客数だけを追うこと（「売上上がったけど、これ何だっけ？」という状態） 。


*  **価値不明瞭な提供:** 提供価値が明確でないものをサービスとして提供すること 。


*  **中途半端な多忙:** 目の前のことに追われて、全てのタスクが60点くらいの出来（妥協）になってしまうこと 。

### **B. 顧客価値（誰が、どう変わる）**

#### **Q4. どんな人をいちばん救いたいですか？（その人はどんな状態からどう変わりますか？）**

*  **ターゲット:** 「自分らしく生きられていない人」。自分の可能性を信じているが、環境や仕事、家庭などに引っ張られ、やりたい気持ちを抑え込んでいる人 。


*  **変化:** フィットネス（ボディメイク）を通じて「自分はここまで努力できる」「変われる」という成功体験を得て、そこから人生全体の「自分らしい挑戦」へと踏み出し、自分らしさを言葉にできるようになる 。



#### **Q5. お客さんは、友達に何と言ってこの体験を勧めますか？（セリフでOK）**

*  **「とりあえず行って、やってみようよ。絶対変わるから」** 。


*  **「あなたに紹介したかったんだよ」** 。


*  **「一回ここで悩みを全部ぶつけて（吐き出して）みたら？」** 。


*  **ニュアンス:** 単に「体が痩せるよ」ではなく、「人生やあなた自身が変わるよ」という、相手の人生に影響を与えるような熱量の高い紹介 。紹介する側（既存客）も、自分に自信を持ち、目が輝いている状態で紹介してくれる 。



#### **Q6. 「それはうちの提供範囲じゃない」と線を引くなら、何を引きますか？**

*  **結果・数字への固執:** プロセス（過程）を無視して「とにかく痩せたい」「結果だけ出したい」というニーズ 。


*  **単なる知識提供（How-to）:** 知識だけを教えること。それは他社やYouTubeでもできる。ShapeFitの価値は、その人がライフスタイルの中でどう動くかを問い続け、自走させることにある 。


*  **軸が通っていない事業:** 「Shapeness（シェイプネス）」という考え方や理念に繋がっていない、全く別の儲け話のような新規事業 。

### **C. 体験の核（選ばれる理由）**

#### **Q7. お客さんは、初回のどの瞬間に「ここ、他と違う」と感じますか？**

*  **圧倒的な自己開示と心理的安全性:** トレーナー自身が「すごい人」としてではなく、「同じ人間として悩み、挑戦している物語」を語る（自己開示する）ことで、お客様が「ここでは自分も話していいんだ」「この人には全部言える」と感じる瞬間 。


*  **組織全体から滲み出る空気感:** 特定のトレーナーだけでなく、ジム全体、組織全体が「Shapeness」の理念を体現しており、その熱量や空気感が、入店した瞬間や会話の端々から伝わってくること 。



#### **Q8. 継続の決め手になる“習慣・儀式・仕掛け”を何にしたいですか？**

*  **ワクワクする「ネクストアクション」の設定:** トレーナーが決めるのではなく、本人が「やりたい」と心から思えるアクションを設定する。さらに「いつ、どこで、誰と」やるかまで解像度を高め、一歩目を軽くする 。


*  **「語る時間」の創出:** お客様自身が自分の変化や過程を誰かに語り、認められる場を作ることで、自己肯定感を高める仕組み 。


*  **可能性を信じ抜くクロージング:** セッションの最後に、必ず未来の可能性を信じる言葉（「絶対できますよ」等）をかけ、自分を信じられる状態で帰ってもらう 。



#### **Q9. 規模を増やすと品質が落ちるポイントを何だと思いますか？**

*  **トレーナーの「体現」レベルの低下:** 理念を頭で理解しているだけでなく、自身の行動で「体現」できていないトレーナーが増えること 。


*  **熱量の希薄化:** 組織の熱量や変化のスピードについていけないメンバーが増え、一人ひとりのお客様に向き合う深さが浅くなること 。

* **「SFF（イベント）」と「ジム」の統合:** これまではSFFが熱量の中心だったが、ジム事業（日常）でも同等の熱量とフィードバックループを作りたいという強い意志があります。
