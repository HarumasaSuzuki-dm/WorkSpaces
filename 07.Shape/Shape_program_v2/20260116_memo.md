## 落選メール（鈴木）
```
SFFにエントリーいただきありがとうございます。

エントリー確定メールにつきましては7/15付けで当選者様全員にお送りが完了しております。
エントリー確定メールが届かなかった選手につきましては、今回は残念ながら落選となり、現在はキャンセル待ちの状態となっております。

なお、キャンセル待ちから当選の可能性もございますため、
その流れをご説明させてただきます。
当選された方の入金期限は7/22(火)までとなっており、
この期日までに入金が確認できない場合は、キャンセル扱いとし、
順次キャンセル待ちの方へ繰り上げのご案内をさせていただきます。
そこで当選された場合は、出場の可能がございますため、
もうしばらくお待ちいただけますと幸いです。

その他ご不明な点がございましたらお気軽にご質問ください。

何卒、よろしくお願いいたします。
```
