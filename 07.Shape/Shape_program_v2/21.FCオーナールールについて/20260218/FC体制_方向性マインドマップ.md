# FCオーナー 1on1を受けて - FC体制の方向性

## 現状認識

- なーすけさんの情熱はFC展開より本店・SFF・SFC・その他のサービス開発に向いている
- FCオーナー側も「自由にやりたい」「ロイヤリティを下げたい」という声がある
- 本部メンバーのリソースがFC支援に食われ、コア事業に集中できていない
- → **双方の望みは実は一致している**（本部：解放したい ↔ FC：自由にやりたい）

## 判断の根拠（3つの事実）

- **経済合理性**：5%に下げても本部の支援コストに見合わない可能性が高い
- **1年間の実績**：気持ちを動かそうとしてきたが、温度差は埋まらなかった
- **Mustの結果**：できない原因の大半は「やり方」だが、腹落ちしていない構造は変わらない

## 方向性

- 「解約」ではなく **「独立・卒業」** として設計する
- 本部が切るのではなく、オーナーが自分の店として自由にやれるようになる話
- 全店舗一律ではなく、店舗ごとに最適な形を選んでもらう

## 選択肢の検討結果

### 社内MTGでの結論：実質「完全独立」一択

- **A. 完全独立**：自分の屋号で運営／ロイヤリティなし／移行期間あり → **採用**
- **B. ブランドライセンス**：ShapeFit名は継続／ロイヤリティ縮小／支援は最低限 → **却下**
    - 理由：ロイヤリティを下げても結局システム管理・問い合わせ対応のコストが発生する
    - 離れきれないことで不満が再燃するリスクがある
    - 「ロイヤリティ払ってる意味あるんですか？」が繰り返される構造は変わらない
- **C. 卒業ネットワーク**：契約終了／勉強会・イベントには参加可能な緩やかな関係 → **却下**
    - 理由：「甘いところだけ吸われる」リスク（SFF等のサービスだけ利用される）

### 伝え方の方針

- 本部としては「完全独立」一択
- ただし、オーナーに「自分で決めた」と感じてもらえるよう伝え方を設計する
- 一緒にやるか、独立するかの **二択** として提示する案も検討
    - 一緒にやる場合：Mustへの完全合意が前提。これまでの説明を繰り返すことはしない
    - 独立する場合：自分の屋号で自由に運営できる

## 進め方（3ステップ）

### ステップ1：事実を揃える（2月中）

- 各店舗のロイヤリティ収入 vs 本部の支援コスト
- 1on1での各オーナーの本音の温度感
- FC契約の解約条項・ブランド使用の取り決め確認
    - **要確認**：正式な契約書が巻けていない可能性がある
    - 金銭面の取り決め（未払い分の精算等）を整理する
- 事業ポートフォリオ・事業計画の整理

### ステップ2：なーすけさんと方針を固める（2月末〜3月初め）

- 店舗ごとにどの形が現実的か整理
- 本店へのリソース集中プランを設計
- オーナーへの伝え方・公式発表の内容を設計

### ステップ3：オーナーと個別に対話（3月）

- 柏店から開始（自らロイヤリティ相談をしており、前向きな対話が期待できる）
- 「本部の反省 → あなたたちの力の承認 → 選択肢の提示」の順で伝える
- 一人ずつ、1on1の場で丁寧に進める

## ケアすべきこと

### コミュニティへの影響

- SFFコミュニティにジム利用者が多く、変な噂や誤解が広まらないよう配慮が必要
- 公式発表の内容は丁寧に詰める

### リソース解放後の集中先

- 本店の経営（集客・紹介の強化）
- オフライン構想の言語化とテスト
- SFF・SFC等のサービス開発
