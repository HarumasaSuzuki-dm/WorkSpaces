## タスク① 1.2.14｜貸付金残高を返済台帳で正確に確認する

### 目的
現在の精算シナリオ資料は概算値のため、合意書ドラフトに使える正確な数字を確定させる。

### 対象店舗
- 柏店（概算：残高約812万・残約80回）
- 上板橋店（概算：残高約452万・残約43回）

### 要件
- [ ] 返済台帳から各店舗の「現在の正確な残高」「残返済回数」「利息の有無・金額」を確認する
- [ ] 月々の返済額・返済スケジュール（いつ完済か）を整理する
- [ ] 全額回収方針に基づき、合意書に載せる「返済スケジュール案」を作成する
- [ ] 概算値との乖離がある場合は、その差分と理由をメモしておく

### アウトプットイメージ
- 店舗別の貸付金サマリー表（残高・残回数・月返済額・完済予定）
- 合意書に記載できる返済スケジュール案（日付・金額の一覧）

### 備考
- このタスクが完了するまで合意書ドラフト（タスク③）の金銭条件欄は確定できない
- なーすけ確認不要、ぼんちゃん判断で進めてよいタスク

## タスク② 1.2.10｜店舗別合意内容の文書化フォーマットを作成する

### 目的
全店舗の合意内容を統一フォーマットで記録するための「器」を先に作る。
上板橋店（すでに合意済み）の文書化にすぐ使用する。

### 要件
- [ ] 以下の項目を含む共通フォーマットを設計する
  - 独立時期（予定日・確定日）
  - 金銭精算条件（貸付金残高・返済スケジュール・ロイヤリティ終了日）
  - システム移行期間（猶予期間の開始・終了・完了条件）
  - ブランド使用停止日（屋号・SNS・Googleビジネス等）
  - 本部のサポート範囲（提供するもの・しないものをリスト化）
  - 署名欄（オーナー・本部それぞれ）
- [ ] 店舗ごとに内容を差し替えられる構成にする（共通部分と個別部分を分ける）
- [ ] 読んだだけでオーナーが内容を理解できる平易な文章・構成にする

### アウトプットイメージ
- Word or PDFで提出可能な合意確認書テンプレート（1〜2ページ程度）
- 上板橋店の内容を入れた記入済みサンプルも同時に作成する

### 備考
- このフォーマット完成が、上板橋店の文書化（2.2.0）と全店舗の合意記録の前提になる

## タスク③ 1.2.6-1｜合意書ドラフトを作成する

### 目的
弁護士レビュー前の合意書の下書きを作成する。
システム関連条項は「宮崎さん回答後に追記」として空欄にしておいてよい。

### 要件
- [ ] 以下の条項を含む合意書ドラフトを作成する
  - 第8条（競業禁止の免除）：独立後の競業を免除する条件を明記
  - 第30条（顧客データの引渡し）：引渡し方法・期限・責任範囲を明記
  - 金銭精算条件：全額回収方針に基づく返済スケジュール（タスク①の数字を使用）
  - 本部のサポート範囲：猶予期間中に提供すること・しないことをリスト化
  - 猶予期間の定義：契約締結後1〜2ヶ月、完全独立までの移行支援期間を明記
  - システム関連条項：「宮崎さん回答後に追記」として空欄・プレースホルダーを設置
- [ ] 法律用語を多用せず、オーナーが読んでも理解できる文体にする
- [ ] 弁護士に渡してレビューしやすい構成にする（条項番号・見出しを明確に）

### アウトプットイメージ
- 合意書ドラフト（Word形式、全条項入り・システム欄は空欄）
- 弁護士への依頼メモ（レビューしてほしいポイントを箇条書きで添付）

### 備考
- なーすけ確認不要。ぼんちゃん判断で弁護士に依頼可能
- タスク①（貸付金確認）完了後に金銭条件欄を記入する

## タスク⑤ 1.2.5｜移行期間を設計する（システム・顧客・ブランド）

### 目的
方針（Pattern C：段階的切り離し）が確定しているため、具体的な移行スケジュールのテンプレートを作成する。
宮崎さんの回答で実態値が入れば即完成できる状態にしておく。

### 要件
- [ ] 移行完了の定義（下記6項目）をチェックリスト形式で整理する
  1. システム切替完了
  2. 契約書面の最終化
  3. ロイヤリティ終了
  4. ブランド使用停止（屋号・SNS・Googleビジネス等）
  5. 顧客データの引き渡し完了
  6. 貸付金の返済スケジュール確定・書面化
- [ ] 猶予期間（1〜2ヶ月）の中で「本部がやること」「FCがやること」を整理する
- [ ] 店舗ごとに時期をずらして適用できる構成にする（上板橋→柏→大阪天六の順）
- [ ] システム期間の実態値はプレースホルダーにしておき、宮崎さん回答後に埋める

### アウトプットイメージ
- 移行スケジュールテンプレート（ガントチャート or 表形式）
- 本部・FC店舗それぞれの「猶予期間中のやることリスト」

### 備考
- このタスクの完成度は3/11 MTG後に上がる。今は「骨格を作る」フェーズと割り切る
- なーすけ確認不要。ぼんちゃん判断で進めてよい
