## ❸ Must運用規定の遵守義務

継続の場合、以下に定める4つのMust運用規定の遵守を契約上の義務とする。

| \# | Must | 内容 | 対象 |
| :---- | :---- | :---- | :---- |
| 1 | セッションログ提出 | 毎セッション後、当日中（翌日午前中まで）に所定フォームへ提出 | トレーナー全員 |
| 2 | 24時間以内レスポンス | 本部からの連絡に1営業日以内に返信またはスタンプで反応 | 全員 |
| 3 | NPS実施 | 月1回、継続中の全会員を対象にアンケートを実施・記録 | 各店舗 |
| 4 | オーナー現場対話 | 週1回以上、自店舗トレーナーと1対1で15分以上対話し、所定シートに記録 | FCオーナー |

### ■ 強制卒業条項

上記Must運用規定のいずれかについて、**3週間以上の未実施が確認された場合**、本部はオーナーに対して書面にて通知のうえ、フランチャイズ契約を終了し、独立・卒業の手続きへ移行する。この場合の手続きは、別途締結する合意確認書に基づくものとする。

**【法的根拠】**

本条項は以下の法令および判例に基づく。

- **民法第541条（催告による解除）**：債務者が履行しない場合、相当の期間を定めて催告したうえで契約を解除できる旨を定める。Must運用規定はフランチャイズ契約上の重要な債務であり、継続的な不履行は解除事由にあたる。  
  - 参照：[e-Gov 民法第541条](https://elaws.e-gov.go.jp/document?lawid=129AC0000000089#Mp-At_541)  
- **現行フランチャイズ契約書第25条（約定解除権）**：「本契約および個別契約に基づく債務を履行せず、相当の期間を定め催告したにもかかわらず、なお履行しないとき」を解除事由として規定している。  
- **裁判例（東京地判平成11年5月11日 判タ1026号211頁）**：フランチャイズ契約は当事者間の信頼関係を基礎とする継続的契約であり、運用規定の継続的な不遵守は「信頼関係の破壊」に該当し、契約解除の正当事由となると判示している。  
  - 参照：[フランチャイズ契約解除の有効性｜弁護士Yのブログ](https://lawleaf-memorandum.com/fc-cases019/752/)

なお、3週間という基準は、一度の未実施ではなく継続的・反復的な不履行を要件とするものであり、直ちに解除に至らないよう加盟者の利益に配慮した設計としている。

## ❹ 本部が提供するサポートの範囲

### ■ 経営責任・売上責任に関する原則

大阪天六店の経営判断、売上・収益の確保、スタッフ管理、顧客対応、および その他店舗運営に関わる一切の責任は、オーナー（株式会社RESH）に帰属する。

本部はブランド・商標・ノウハウの使用権を付与する立場にあり、各店舗の 売上向上や集客結果について責任を負うものではない。

**【法的根拠】**

- **公正取引委員会「フランチャイズ・システムに関する独占禁止法上の考え方」 （令和3年4月28日改正）**：「加盟者は法律的には本部から独立した事業者」 であることを明示しており、本部と加盟者間の取引に独占禁止法が適用される 根拠としている。加盟者が独立事業者である以上、経営判断および収益確保の 責任は加盟者側に帰属する。  
    
  - 参照：[公正取引委員会 フランチャイズ・ガイドライン](https://www.jftc.go.jp/dk/guideline/unyoukijun/franchise.html)


- **中小企業庁「特定連鎖化事業（フランチャイズ）について」**：「フランチャイズ 契約は、加盟店と本部事業者がそれぞれ独立した事業者として、各々の責任に おいて締結するもの」と明記しており、経営責任が加盟者側にあることを 公的機関が確認している。  
    
  - 参照：[中小企業庁 特定連鎖化事業について](https://www.chusho.meti.go.jp/shogyo/shogyo/franchise.html)


- **弁護士法人クラフトマン「フランチャイズシステムの法律解説」**：加盟店は 「独立した事業者」として事業リスクを負担するという考え方が法的解釈の 基礎とされており、本部が加盟店の売上・収益結果に対して法的責任を 負わないことが実務上確立している。  
    
  - 参照：[弁護士法人クラフトマン フランチャイズシステムの法律解説](https://www.ishioroshi.com/biz/kaisetu/m_and_a/manda/franchise/)

**【法的根拠・妥当性】**

フランチャイズ契約の法的性質は「商標・ノウハウ・経営システムの使用権付与」を本質とするものである。

- **公正取引委員会「フランチャイズ・システムに関する独占禁止法上の考え方」（令和3年4月28日改正）**：フランチャイズ契約の基本は「加盟者が本部の商標・商号等を使用して営業することの許諾」および「加盟者の統制・指導」であるとしており、直接的な集客支援や広告制作の実施は本部の義務として規定されていない。  
  - 参照：[公正取引委員会 フランチャイズ・システムに関する独占禁止法上の考え方](https://www.jftc.go.jp/dk/guideline/unyoukijun/franchise.html)  
- **現行フランチャイズ契約書の定義**：「フランチャイズ権」とは、本部が開発したノウハウ・商標・サービスマークを使用し経営する非独占的な権利と定義されており、集客施策の直接提供は契約上の義務に含まれない。  
- **加盟者の独立事業者性**：同ガイドラインは「加盟者は法律的には本部から独立した事業者」であると明記しており、各店舗の集客・経営は独立した事業者たるオーナーが責任を負う領域である。  
  - 参照：[フランチャイズシステムの法律解説｜弁護士法人クラフトマン](https://www.ishioroshi.com/biz/kaisetu/m_and_a/manda/franchise/)

ロイヤリティは上記のブランド使用権・ノウハウ提供の対価であり、個別の集客支援・広告費用については別途の見積もり・費用負担による対応となる。

## ❻ 独立を選択した場合の本部対応（参考）

継続ではなく独立・卒業を選択した場合、本部は以下の対応を行う。

| 項目 | 内容 |
| :---- | :---- |
| **違約金の免除** | 契約書第25条・第27条に定める違約金（金10,000,000円）を全額免除 |
| **競業避止の免除** | 契約書第8条第2項に定める契約終了後5年間の競業禁止規定を免除 |
| **システム移行支援** | 独立日から2026年8月末日まで、新システム選定・移行に関する技術的サポートを提供 |
| **ブランド使用猶予** | 独立日から2026年8月末日まで、名称・ロゴの切り替え猶予期間を設ける |
