# Shapeコミュニティ 優先事項マップ

## 🔴 最優先①：本からの流入をどう受け入れるか

### なぜ最優先か
- 本が出たら約30人が「入りたい」と来る
- 今のコミュニティは15人。一気に倍になると空気が壊れる
- 準備なしで受け入れると「熱い場」が「ただの集まり」になるリスク

### STEP1：本にどんな導線が載っているか確認する
- 本はもう印刷済み → 中身は変えられない
- だから「何が載っているか」を正確に把握して、それに合った受け皿をつくる
- なーすけに確認すること
    - 本の中にどんなリンク・QRコードが載っているか？
    - それはどこに飛ぶのか？
- 載っている可能性があるもの（例）
    - LINEオープンチャットのリンク
    - Instagram / SNSのアカウント
    - 公式サイトやLP（ランディングページ）のURL
    - メルマガやLINE公式アカウントの登録リンク
    - イベント案内ページのURL
    - 特に何も載っていない（口頭やSNSで案内する想定）

### STEP2：導線の先に「受け皿」をつくる
- 本を読んだ人が最初にたどり着く場所＝「受け皿」
- 受け皿の役割は「いきなりコミュニティに入れる」ではなく「一旦ここで受け止める」
- 導線ごとの受け皿の考え方
    - LINEオープンチャットに飛ぶ場合
        - → オープンチャットが最初の接点になる
        - → ここを「無料の入口」として整理し、有料コミュニティへは別の案内を出す
    - Instagramに飛ぶ場合
        - → プロフィールやハイライトに「コミュニティに興味がある方へ」の案内を用意する
        - → ストーリーズや投稿で段階的に告知する
    - 公式サイトやLPに飛ぶ場合
        - → ページ内に「まずはこちらから」と応募フォームや説明ページへの導線を置く
    - 特に導線がない場合
        - → 本の感想をSNSに投稿した人に対して、こちらからアプローチする
        - → 解説動画の最後にコミュニティへの案内を入れる
- どのパターンでも共通で大事なこと
    - 「興味ある人」と「本気で入りたい人」を分ける仕組みをつくる
        - 興味ある人 → 無料の場（オープンチャット・SNS）で繋がり続ける
        - 本気の人 → 応募フォームに進んでもらう
    - 一気に30人がコミュニティに流れ込まないようにする

### STEP3：「誰を入れるか」の基準をつくる
- 受け皿に来た人を、全員入れるのではなくフィルタをかける
- 第1フィルタ：かんたんな応募フォーム
    - 聞くこと（3問くらい）
        - なぜ入りたいのか？
        - 今どんな挑戦をしているか？
        - Shapeとの接点は？（本？Instagram？イベント？）
    - 目的：「熱量」と「相性」を見る（正解・不正解ではない）
- 第2フィルタ：熱量が高い人だけ1on1
    - 最初はなーすけが担当 → 慣れたらコミュニティ担当へ
    - 1on1の目的はセールスではなく「一緒につくる仲間かどうか」の確認
    - 「この人はリーダーになりそうか」を見るポイント
        - 自分から動ける人か
        - 他の人を応援できる人か
        - 自分の言葉で語れる人か

### STEP4：通過した人数を見て「一気に入れるか・分けるか」を判断する
- STEP2（受け皿）とSTEP3（フィルタ）がちゃんと機能していれば、そもそも大量に通過してくることは少ない
- なので、まず通過した人数を見てから判断する
- 判断の目安
    - 5人以下 → 一気に入れてOK。既存15人の空気が崩れるリスクは低い
    - 6〜10人 → 一気に入れてもいいが、受け入れ体制（STEP5のバディなど）を先に整えてから
    - 10人超 → 分けた方がいい。既存メンバーが新規に「数で負ける」と文化が薄まる
- 分ける場合の進め方
    - 第1期と第2期の間に2〜3週間は空ける（新しい人が馴染む時間が必要）
    - 3月中にイベント（ロゴづくり・感想会など）を挟むと自然にタイミングを分けられる
        - イベントに参加してもらうことで「この場の空気」を体感してもらう効果もある
- 大事な考え方
    - 最初から人数を固定するのではなく、STEP2・3の結果を見て柔軟に決める
    - 「何人入れるか」より「既存の空気を守れるか」で判断する

### STEP5：入ってきた人が「お客さん」にならない仕組み
- 新しい人が「見てるだけ」にならないための工夫
    - ウェルカムスレッドで自己紹介＋「今の挑戦」を書いてもらう
    - 既存メンバーが「迎える役割」を持つ（バディ制度）
        - 例：1人の新メンバーに1人の既存メンバーがつく
        - 最初の1週間だけでOK。負担を軽くする
    - 入って最初の2週間で「小さな挑戦」を1つ宣言してもらう
- 既存メンバーへの事前共有
    - 「新しい人が入ってくるよ」と伝えておく
    - 「迎える側」としての意識をつくる（＝既存メンバーの成長機会にもなる）

### STEP6：入れなかった人・時期をずらした人のケア
- 「断られた」と感じさせない伝え方
    - 「今は定員のため、次の募集をお待ちください」
    - 「まずはイベントやオープンチャットで繋がりましょう」
- 無料のオープンチャットを「待合室」として残す
    - ここで発信を見たり、イベント情報を受け取れる状態にする
- 次の募集のタイミングを明示する（「○月頃に第2期募集します」）

## 🔴 最優先②：チームの役割を整理する（誰が何をやるか）

### なぜ最優先か
- 今「誰がどのボールを持っているかわからない」状態
- しおんさんに負荷が集中している
- これが解消されないと、何をやっても進みが遅くなる

### やること①：今動いている仕事を全部並べる
- まず「今やっていること」を一覧にする
    - 有料化の告知と決済の仕組みづくり
    - Discordの準備
    - 本からの流入対応
    - コミュニティの全体構想づくり
    - 3月のイベント企画
    - 日々の運営（オープンチャットの管理、挑戦を広げる会の運営）
- それぞれについて「誰が持っているか」「今どうなっているか」「次にやることは何か」を書く

### やること②：「決める人」と「動く人」を分ける
- なーすけ＝「方向を決める人」
    - コミュニティの構想・方針
    - 流入の設計・基準づくり
    - 有料化の伝え方（確定済み）
- しおん・チーム＝「形にして動かす人」
    - Discordの構築・設定
    - イベントの実行
    - 日々の運営タスク
- 大事なルール
    - なーすけが方向を決めるまで、チームは動けない → だから「いつまでに決める」を約束する
    - 逆に、チームが動き始めたら、なーすけは口を出しすぎない

### やること③：しおんさんの負荷を見直す
- 今しおんさんが抱えているものを全部書き出す
- その中で「なーすけに戻すもの」「他の人に渡せるもの」を仕分ける
- 1人に集中しすぎない体制をつくる

### やること④：共有の場所をつくる
- 全員が見える場所にタスク一覧を置く（Notion、スプレッドシートなど）
- 週1回、5分でいいから「今どうなってる？」を確認する時間をつくる

## 🟠 高優先③：コミュニティの「最初のかたち」を決める

### なぜ高優先か
- 有料化する＝「お金を払って何が得られるか」を約束すること
- 全部を決めてからだと遅い。でも何も決めずに走ると方向がバラバラになる
- 「最低限これだけ決めれば走れる」と「後から決めていいこと」を分ける

### 今すぐ決めること（有料化の前に必要）
- 月1,000円で何が得られるのか
    - 有料メンバー限定のDiscordに入れる
        - Discordでできること＝有料の特典として伝える
        - 例：挑戦宣言の部屋、アイデア投稿、イベント企画への参加など
    - 語る時間（2週に1回）、言語化時間への参加
    - 日常的な仲間との交流・応援
    - ここを「約束」として明文化する
- LINEオープンチャットとDiscordの役割分け
    - 有料メンバー → Discordに移動（ここが「本拠地」）
    - 無料の人・まだ入っていない人 → LINEオープンチャットに残る（情報を受け取れる場所）
    - つまり「有料化＝Discordに入れる」がセットになる
- 最初の1ヶ月のスケジュール
    - 最低限「いつ、何があるか」がわかる状態にする
    - 例：第1・3週＝語る時間、第2・4週＝言語化時間
- 3つの層（火種・フォロワー・Roots）の「入口だけ」決める
    - 最初は全員「火種」からスタートでOK
    - 「フォロワーやRootsにどうなるか」は後で決める

### 3月以降に決めていいこと（走りながら調整）
- 層の移動ルール（火種→フォロワー→Rootsの条件）
- リーダーの育て方・見つけ方
- 数字の目標（KPI）の細かい設計
- 半年〜1年先のロードマップ
- 「文化の火」（人がいなくても回る状態）の具体的な仕組み
- 外部への発信戦略（Instagramなど）の詳細

### 決める順番のコツ
- まずは「有料メンバーへの約束」を1枚にまとめる
- その約束を守れる「最初の1ヶ月の動き方」を決める
- それ以外は、やりながら「これも決めた方がいいな」と気づいたら決める
