## 全体の流れ

```
Phase 0（3月）        5ヶ月分の準備を終わらせ、4月から動き出せる状態をつくる
Phase 1（4月）        Shapeの考え方をみんなで読んで、「自由に話せる場所」をつくる
Phase 2（5月）        「なぜ自分がここにいるか」を自分の言葉で言えるようにする
```

## Phase 0｜5ヶ月分の準備を終わらせ、4月から動き出せる状態をつくる
**期間：2026年3月**

> 3月は動き出す月ではなく、準備する月。4月以降に「どうすればいいんだっけ？」と迷う時間をなくすために、5回分のイベント内容・担当・会場・備品をこの月にすべて決めきる。準備が全部終わった状態で4月を迎えることが、この月の唯一のゴール。

### やること

**内容を決める**
- 企画会議①：5回分のテーマ・ゴール・タイムスケジュールを決める（運営メンバー全員・2〜3時間）
- 企画会議②：各回のグループワークの内容・問い・発表の方法を詰める（運営メンバー全員・2〜3時間）
- 決まった内容をワークフックに書き込み、4月以降は全員がそこを見れば動ける状態にする

**担当を決める**
- 運営メンバー全員の5ヶ月分のスケジュールを確認する
- 毎月ずっと担当することと、月によって変わる担当を整理した担当表を作り、全員に共有する

**4月イベントの準備を終わらせる**
- 会場を予約する（土曜14:00〜16:00、レンタルスペースまたはジム）
- 声をかけるメンバーのリストを作る
- 備品リストを作る（名札・付箋・ペン・飲み物・「Shapeの証し」クリアファイル16冊）
- LINEオープンチャットへの告知文を用意する

### 成果物
| 成果物 | 内容 |
|---|---|
| ① 5ヶ月分のイベント設計書 | テーマ・ゴール・タイムスケジュール・グループワーク内容が決まったもの |
| ② 担当表 | 5ヶ月分の担当が一覧になっていて、運営メンバー全員に共有済みのもの |
| ③ 4月イベントの準備完了確認 | 会場・声かけリスト・備品・告知文がすべて揃った状態 |

### Phase 0の完了条件（4月に入る前にすべて終わっていること）
- [ ] 5回分のテーマ・ゴール・タイムスケジュールがワークブックに書き込まれている
- [ ] 5ヶ月分の担当表が作られ、運営メンバー全員に共有されている
- [ ] 4月イベントの会場が押さえてある
- [ ] 4月イベントで声をかけるリストが準備されている
- [ ] 備品リスト（毎回使うもの＋その回だけ必要なもの）が作られている
- [ ] 初回イベントの告知文が用意されている

## Phase 1｜Shapeの考え方をみんなで読んで、「自由に話せる場所」をつくる
**テーマ：「こんなに自由に話していいんだ、と思える最初の体験をつくる」**
**期間：2026年4月【Month 1・立ち上げ期】**

> 新しいメンバーを集めることはしない。この月のゴールは「安心して話せる空気をつくること」だけ。運営メンバーも自分の意見をフラットに話すスタンスで場に入り、「正解を教える場じゃない」という雰囲気を最初の1回でつくる。ここで生まれた安心感が、残りの5ヶ月の深さを決める。

### やること

**イベント本体（第1回：Shapeの考え方に触れる会）**
- 事前に本か解説動画を読んでくることをマストとして周知する
- 10分間の自分を振り返る時間 → 話すときのルール確認 → 4人グループで対話（40分）の順で進める
- 運営メンバーも自分の意見をフラットに話すスタンスで参加する
- 「自分にとってのShapeを一言で言う」時間を最後に設ける

**イベントとイベントの間をつなぐ**
- イベント後：オープンチャットで感想を共有するよう声をかける
- 翌週のオンライン定例：イベントの余韻を10分共有する
- 次のオンライン定例：来月の内容を予告する
- 「Shapeの証し」クリアファイルを来月配布できるよう準備しておく

### 月のリズム
```
第1水曜　オンライン定例（先月の余韻を共有 / 来月の予告）
第3土曜　オフラインイベント【第1回】
第3水曜　オンライン定例（今回の余韻を共有）
```

### ゴール
| ライン | 内容 |
|---|---|
| 最低ライン | 「来てよかった」と思えた。顔と名前がわかる仲間が3人以上できた |
| 目標ライン | 本の中で一番響いた部分と自分の体験を重ねてグループで話せた。自分にとってのShapeを一言で言えた |

### 来月に向けて確認すること
- また来たいと思えた人はどれくらいいたか
- 自分から発言できた人の割合
- イベント後にオープンチャットへの投稿が増えたか

### 成果物
| 成果物 | 内容 |
|---|---|
| ① 振り返りメモ | よかった点・改善したい点・来月に活かすこと |
| ② オープンチャット観察記録 | 投稿数・内容の変化のメモ |

## Phase 2｜「なぜ自分がここにいるか」を自分の言葉で言えるようにする
**テーマ：「自分がここにいる理由を、自分の言葉で言えるようにする」**
**期間：2026年5月【Month 2・立ち上げ期】**

> Phase 1でShapeの考え方を一緒に読んだ土台の上で、今度は「自分自身」に目を向ける。バランスホイールを使って自分の人生の各分野を整理し、「なぜフィットネス・トレーニングをするのか」を3段階で掘り下げる。この月のポイントは「自分の話を誰かにちゃんと受け止めてもらえた体験」をつくること。その体験が、後半の自分から動く気持ちにつながる。

### やること

**イベント本体（第2回：自分のことを言葉にするワークショップ）**
- バランスホイール作成（15分）→ グループ内で深掘りワーク（35分）→ 全体で共有・褒め合う時間（15分）
- 各グループに運営メンバーを1人入れ、「それってどういうこと？」「本当はどうなりたいの？」と優しく掘り下げる
- 「Shapeの証し」クリアファイルを全員に配布し、今日のワークシートを1枚目として綴じる
- イベント終わりに、反応がよかったメンバーに声をかける：「次回のアイスブレイク、ちょっとやってみてくれない？」

**次のリーダー候補を見つける準備**
- Phase 1で積極的に動いていたメンバーにグループ進行の補助をお願いする
- イベント後に自然に残って話す文化をつくる（「今日どうだった？」と一緒に振り返る）

**イベントとイベントの間をつなぐ**
- イベント後：「ワークシートの写真をオープンチャットで共有してもOKだよ」と案内する
- 翌週のオンライン定例：「あれから何か変化あった？」と声をかけて余韻をつなぐ

### 月のリズム
```
第1金曜　オンライン定例（先月の振り返り / 今月の予告）
第3土曜　オフラインイベント【第2回】
第3金曜　オンライン定例（余韻をつなぐ / リーダー候補に来月の役割をお願いする）
```

### ゴール
| ライン | 内容 |
|---|---|
| 最低ライン | 「今やっていること」と「なぜやっているか」をセットで話せた。バランスホイールで自分の生活を整理できた |
| 目標ライン | 自分の変化をみんなの前で話せた。「よかったよ」と受け止めてもらえた体験ができた。「本当はどうなりたいか」まで言葉にできた |

### 来月に向けて確認すること
- 「なぜ」まで話せた人の数
- グループ進行の補助を頼んだメンバーの反応
- 「自分もやってみたい」という声が出たか

### 成果物
| 成果物 | 内容 |
|---|---|
| ① 振り返りメモ | よかった点・改善したい点・来月に活かすこと |
| ② リーダー候補リスト（初期版） | 声をかけた人・反応・気づいたこと（名前で記録） |
| ③ 「Shapeの証し」ファイル配布完了 | 16名全員分 |

## Phase 3｜「私たちのコミュニティ」をみんなの言葉で定義する
**テーマ：「個人の言葉を、みんなの言葉に育てる」**
**期間：2026年6月【Month 3・巻き込み期】**

> Phase 1でShapeの考え方を一緒に読み、Phase 2で自分自身の言葉を見つけた。このフェーズでは「私たちのコミュニティってこういう場所だ」という、みんなの言葉をつくる。ビジョン・Do/Don't・コミュニティの定義の3つを柱に進める。この回で出た言葉が、来月のロゴやデザイン制作の材料になる。

### やること

**イベント本体（第3回：コミュニティの「大切にしたいこと」を言葉にする）**
- ビジョンワーク「3〜5年後、このコミュニティがどうなっていたら最高？」（25分）
- Do/Don'tワーク「こんな場所にしたい / こんなことはやりたくない」（25分）
- 各グループの発表 → まとめ → コミュニティの定義を一言で言葉にする（10分）
- アイスブレイクの進行はリーダー候補に任せる。運営メンバーは全体の進行に専念する

**3月 → 4月の間に必ず行うこと（重要）**
- 運営メンバーがこの回で出たキーワードを整理する
- デザイナー（IDAさん）にロゴ候補とDo/Don'tスライドの制作をお願いする（ななこちゃんが窓口）
- メンバーにゼロから作らせるのではなく、候補を用意して「選んでもらう」形にする

**中間確認の実施**
- 5ヶ月後にどうなっていたいかとの中間確認を行う（なーすけさんと運営担当）
- 1on1を始める（リーダー候補や積極的なメンバー、1人30分）
- 「コミュニティが自分たちで動ける状態かどうか」のチェックリストを記入し始める

**イベントとイベントの間をつなぐ**
- イベント後：まとまった言葉をオープンチャットに投稿する
- 翌週のオンライン定例：追加のアイデア共有・デザイン制作に入ることを伝える

### 月のリズム
```
第1水曜　オンライン定例（先月の振り返り / 今月の予告）
第3土曜　オフラインイベント【第3回】
第3水曜　オンライン定例（言葉の補足 / デザイン制作に入ることを伝える）
月中〜末　1on1スタート（対象者×30分・順次）
```

### ゴール
| ライン | 内容 |
|---|---|
| 最低ライン | コミュニティで大切にしたいことが3つ以上出せた |
| 目標ライン | 「私たちのコミュニティはこういう場所」を全員が自分の言葉で言えた。ビジョンとDo/Don'tが具体的に出てきた |

### 来月に向けて確認すること
- 自分からアイデアを出した人がいたか
- リーダー候補を名前で言えているか
- 5ヶ月後の目標と比べて今どのあたりにいるか

### 成果物
| 成果物 | 内容 |
|---|---|
| ① キーワード整理メモ | この回で出た言葉をIDAさんへの依頼用に整理したもの |
| ② 振り返りメモ | よかった点・改善したい点・来月に活かすこと |
| ③ 中間確認の記録 | 5ヶ月後の目標と今の状態を比べたメモ |
| ④ 1on1の記録 | 話した人ごとのメモ・変化の記録 |

## Phase 4｜ロゴや行動指針を、みんなで選んで形にする
**テーマ：「自分たちで選んで決めた、という実感をつくる」**
**期間：2026年7月【Month 4・巻き込み期】**

> このフェーズのポイントは「ゼロから作らせない」こと。前月で出た言葉を元に運営メンバーとIDAさんが作ったデザイン候補をメンバーが選ぶ形にすることで、「参加したら何かが決まった」という達成感をみんなが感じられる。リーダー候補がイベント進行の大半を担う月でもある。

### やること

**イベント本体（第4回：デザイン候補への投票・決定会）**
- 運営メンバーから「前回の言葉をどうデザインに変換したか」のプロセスと候補を説明（20分）
- グループで話し合う「こっちの方がしっくりくる」「この部分を入れてほしい」（30分）
- 全体投票で決める（20分）。決まりきらなければグループ宿題で持ち帰りもOK
- Do/Don'tスライドの内容を確認・最終調整（15分）
- **リーダー候補がイベント進行の大半を担当。運営メンバーは安全に進行できているかの確認と投票の管理のみ**

**リーダー候補を育てる**
- リーダー候補を企画の打ち合わせに招く（月の最初）
- イベント進行の半分以上をリーダー候補に任せる
- 「自分たちで動ける状態かどうか」のチェックリストを引き続き記入する

**イベントとイベントの間をつなぐ**
- イベント後：決定したデザインをオープンチャットで共有する
- 翌週のオンライン定例：微調整があれば確認する
- IDAさんへ最終デザインの制作をお願いする

### 月のリズム
```
第1水曜　オンライン定例（先月の振り返り / 制作物のお披露目準備）
第3土曜　オフラインイベント【第4回】
第3水曜　オンライン定例（決定内容の共有 / 来月の予告）
```

### ゴール
| ライン | 内容 |
|---|---|
| 最低ライン | ロゴ・Do/Don'tスライドの方向性が投票で決まった |
| 目標ライン | ロゴ・Do/Don'tの最終案が決まり、「自分たちが選んで決めた」という実感がある |

### 来月に向けて確認すること
- リーダー候補がスムーズに進行できていたか
- 自分から参加・発言した人の割合
- チェックリストの進み具合

### 成果物
| 成果物 | 内容 |
|---|---|
| ① ロゴ確定版（IDAさん制作） | メンバーの投票で選ばれた最終デザイン |
| ② Do/Don'tスライド確定版 | コミュニティの行動指針を見える形にしたもの |
| ③ 振り返りメモ | よかった点・改善したい点・来月に活かすこと |

## Phase 5｜5ヶ月を振り返り、SFFに向けてみんなで気持ちを合わせる
**テーマ：「5ヶ月の変化を実感し、SFFに向けて一緒に気持ちを高める」**
**期間：2026年8月【Month 5・自走期】**

> これまでの4ヶ月で積み上げてきたもの（言葉・仲間・ロゴ・会員証）を手に取り、5ヶ月の変化を確かめる月。イベントの企画と進行の大半はリーダー候補に任せる。Part 2のご飯会では、SFF以降の方向性について運営メンバーとリーダー層で気軽に話し合う。

### やること

**イベント本体（第5回：SFFに向けた決起会 + 5ヶ月の振り返り）**
- 「挑戦を広げる会」の記録を見返し、半年間の自分の挑戦を書き出す
- 半年間一緒にいたグループ（4〜5人）でお互いの成長や良いところを伝え合う
- SFFの説明と協力のお願い「一緒に作っていきたい」
- 成果物（ロゴ・会員証）の完成版をお披露目する
- **イベントの企画と進行の大半はリーダー候補が担当。運営メンバーは全体の見守りのみ**

**Part 2：別会場でご飯会（参加自由）**
- SFF以降の方向性について気軽に話し合う
- 「こういうの考えてるんだけどどう思う？」のスタンスで、正式発表前に意見を聞く
- 有料化に向けた感触をメモしておく（なーすけさんが管理）

**5ヶ月のまとめ**
- 5ヶ月後にどうなっていたいかと今の状態の最終照合
- 「コミュニティが自分たちで動ける状態かどうか」のチェックリストを記入（8個以上○なら自走OK）
- SFF以降の運営メンバーの関わり方の方針を決める

### 月のリズム
```
第1水曜　オンライン定例（先月の振り返り / 決起会の予告）
第3土曜　オフラインイベント【第5回 決起会 + Part 2 ご飯会】
第3水曜　オンライン定例（振り返り / SFFの告知 / 次のフェーズの予告）
```

### ゴール
| ライン | 内容 |
|---|---|
| 最低ライン | 5ヶ月の自分の変化を一言で言えた |
| 目標ライン | SFFで「自分はこう関わりたい」というイメージが持てた。コミュニティのことを外の人にも説明できる状態になっている |

### 自走できているかの確認ポイント
- 運営メンバーがいなくても進んだ場面があったか
- 5ヶ月後の目標と今の状態の最終照合
- チェックリスト：8個以上○ → 自走OK / 5〜7個 → 一部サポートしながら移行 / 4個以下 → もう1〜2ヶ月継続

### 成果物
| 成果物 | 内容 |
|---|---|
| ① 5ヶ月振り返りレポート | 目標との照合結果・チェックリスト・リーダー候補の記録 |
| ② SFF以降の運営メンバーの関わり方の方針 | どの業務を誰に引き継ぐかの確定版 |
| ③ 有料化に向けたメモ | メンバーの声・行動変化の記録（なーすけさんが管理） |

## Phase 6｜深めてきた土台の上で、コミュニティを広げる
**期間：SFF以降（2026年9月〜）**

> SFFが終わったタイミングで、コミュニティは「深める期」から「広げる期」に入る。16名が自分の言葉でShapeを語れる状態になった土台の上に、新しいメンバーを迎え入れる。有料化の判断はこのフェーズの最初に行う。

### Phase 6に入るための前提条件
| 項目 | 状態 |
|---|---|
| 16名のうち過半数が「なぜここにいるか」を自分の言葉で話せる | 確認済み |
| リーダー候補が2〜3名、行動で特定されている | 確認済み |
| コミュニティの定義・ロゴ・行動指針が完成している | 完成済み |
| 自走チェックリストが5個以上○ | 確認済み |

### やること

**有料化の判断（なーすけさん × 鈴木 のみ）**
- Phase 5で集めた「メンバーの声の記録」を元に有料化するかどうかを判断する
- 確認ポイント：無料だから来ているのか、価値があるから来ているのか
- 価格・プランの内容・告知の方針を決める

**新しいメンバーを受け入れる**
- SFFに参加したシェイパーを既存のコミュニティイベントに招待する
  （コミュニティメンバーは無料、それ以外は1,500円）
- 1,500円払って来るかどうかが「本当に関わりたい人かどうか」の目安になる
- 初めて参加する人向けの案内や、バディをつける仕組みを整える

**リーダー層に少しずつ任せていく**
- 運営メンバーの引き継ぎ方針に沿って、業務を順番に移していく
- 「やってみて」と任せる機会を意図的につくり、実績を積ませる

## 「Shapenessを、仲間と共に生き続ける場所をつくる」の完了定義

今期のテーマが達成できたと言える状態の定義。

| レベル | 完了条件 |
|---|---|
| **最低ライン** | 16名のうち過半数が「なぜ自分がここにいるか」を一言で言える |
| **今期の本当のゴール** | メンバーが自分の言葉でShapeのことを語れるようになり、友人や家族に「Shapeってこういう場所」と説明できる |
| **深めきった状態** | メンバーが仲間の前で自分の挑戦を話し、それをお互いに受け止め合う文化が、運営メンバーがいなくても自然に続いている |

---

## ⚠️ 注意事項・SFFまでやらないこと

### SFFまでは絶対にやらないこと
- 新しいメンバーを集めること・外への発信を強化すること
- 有料化の判断・発表
- メンバーが自分でイベントを企画・主催する状態を目指すこと

### 大切にしていること
- ワークショップは「その場で全部出しきる」設計。持ち帰り課題は設けない
  （持ち帰ると考えすぎて話せなくなるリスクを防ぐため）
- 毎回グループの組み合わせを変え、リーダーが自然に生まれる場をつくる
- 「Shapeの証し」ファイルを5ヶ月かけて積み上げ、自分の変化を手で感じられるようにする
