2段階調査用のプロンプトです。

**プロンプト①：広く事例を収集する**

```
あなたは企業スポンサー戦略の専門アナリストです。

以下の条件で、スポンサー契約として成功している事例をできるだけ多く調査・列挙してください。

【調査対象】
- 海外企業（地域不問）
- 年間契約・複数年契約・単発契約、いずれも対象
- スポーツ・フィットネス・エンタメ・メディア・テック・ライフスタイルなど業界不問
- スポンサーする側（出資企業）とされる側（イベント・団体・個人等）の両視点を含める

【各事例に含めてほしい情報】
1. スポンサー企業名 ／ スポンサード対象（イベント・団体・選手・コンテンツ等）
2. 契約の種類（年間／単発／複数年／カテゴリー独占 等）
3. スポンサーの目的（認知拡大／販売促進／採用／コミュニティ形成 等）
4. 主なリターン内容（露出・体験・データ・コンテンツ統合 等）
5. 成功のポイント（なぜうまくいったか、どこが特徴的か）

【特に注目してほしいスポンサー手法】
- 体験型・参加型スポンサー
- コンテンツ統合型（"Presented by"形式など）
- データ提供・アンケート権を含むスポンサー
- コミュニティ型・Naming Rights型
- BtoB接点創出型（スポンサー企業同士をつなぐ）
- デジタル・SNS連動型
- スポンサーが複数年契約に至った事例

数は多ければ多いほど良いです。網羅性を優先し、各事例は簡潔にまとめてください。
最後に「特にユニークだと思う事例トップ5」を選んで理由とともに提示してください。
```

**プロンプト②：気になった事例を深掘りする**

```
あなたは企業スポンサー戦略の専門アナリストです。

以下の企業／事例について、スポンサー戦略を可能な限り詳しく調査・分析してください。

【調査対象】
{①で気になった事例名をここに入れる}

【調査してほしい内容】
1. スポンサー戦略の全体像
   - どんな相手と、何件くらい、どんな種類の契約を結んでいるか
   - スポンサー収益は全体売上のどの程度を占めるか（推定でも可）

2. スポンサーパッケージの設計
   - 階層構造（Title / Gold / Silver 等）があるか
   - カスタム型か標準パッケージ型か
   - スポンサーに提供しているリターンの具体的な内容

3. スポンサー獲得・維持のアプローチ
   - どのように新規スポンサーを獲得しているか
   - 継続・複数年契約に持ち込むための工夫
   - スポンサー企業の「成功体験」をどう設計しているか

4. 差別化ポイント
   - 他社にはないスポンサー手法・仕掛けは何か
   - スポンサーが「ここに出したい」と思う理由は何か

5. 失敗・課題・批判（あれば）
   - うまくいかなかった点、改善された点

6. Shape Fitへの示唆
   - フィットネスイベント・オンラインコーチング・ジムFC・コミュニティ運営など
     複数事業を持つ会社がこの事例から学べることは何か
   - 特に「年間スポンサー契約の拡大」という観点で転用できるアイデアを具体的に提案してほしい

できるだけ一次情報（公式発表・インタビュー・IR資料等）に近い情報を使い、
推定の場合はその旨を明記してください。
```

---

使い方のイメージとしては、①を回した後に気になった事例名を②の `{}` 部分に入れて複数回まわす想定です。②は1事例ずつ入れると深度が出ます。
