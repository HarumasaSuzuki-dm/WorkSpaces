# MYS 要件定義｜What × Who

---

## WHO｜対象顧客の定義

### 確定していること
| 項目 | 内容 |
|---|---|
| 表層的な状態 | 世間的には成功している・結果を出している |
| 内面の状態 | 余白がない・自分軸を持てていない・疲れている |
| フィットネスとの関係 | 結果にとらわれている（表面的）／本当はとらわれたくない（潜在的） |
| 変化前の兆候 | 笑顔が少ない・口癖がネガティブ・他人軸で動いている |

### 要検討の論点

**Q1. 属性の絞り込み**
- 年齢・性別・職業のターゲットはあるか？
- 例：20-30代会社員？子育て中の女性？起業家？

**Q2. フィットネス経験レベル**
- 初心者も対象か、ある程度経験がある人が前提か？

**Q3. 顕在層 vs 潜在層の優先度**
- 「余白がない」と自覚している人を狙うか
- 「痩せたい」など結果目的で来た人を転換する設計にするか

---

## WHAT｜提供内容の定義

### 確定していること
| 項目 | 内容 |
|---|---|
| 最重要テーマ | 自己効力感を高めること |
| 手段① | 自己把握・内省（コーチによる言語化支援） |
| 手段② | ビジョンへの熱量を育てる（変化した人の実例・道筋の提示） |
| 行動変容の方向 | 1%の積み重ね・完璧主義の解放・バランスホイールの充足 |
| やらないこと | 極端な減量・短期結果主義・焦りを煽る関わり方 |

### 要検討の論点

**Q4. プログラム期間**
- 何ヶ月のプログラムとして設計するか？
- 短すぎると変容が起きない、長すぎると継続しにくい

**Q5. コンテンツの構成要素**
以下のどれを組み合わせるか優先度をつけたい：

```
[ ] 個別コーチングセッション
[ ] 内省・ワークシート
[ ] フィットネスプログラム（食事・運動）
[ ] コミュニティ（仲間との交流）
[ ] 変容事例のコンテンツ（動画・ストーリー）
```

**Q6. 「シェイプネスを体現した状態」の可視化**
- 数字に頼らない成果をどう本人に実感させるか？
- バランスホイールのスコア？内省ジャーナル？

**Q7. フィットネス要素の比重**
- あくまでフィットネスが主軸か
- マインド・ライフスタイル変容がメインでフィットネスは手段か

---

## 優先度の高い論点

上記の中で、**以下3つを先に固めると全体設計が一気に進む**と思います。

1. **Q3**（顕在層 vs 潜在層）→ 入口設計・メッセージに直結
2. **Q5**（コンテンツ構成）→ How・コスト・工数に直結
3. **Q7**（フィットネスの位置づけ）→ サービスのアイデンティティに直結

---

## HOW｜提供方法の定義

### 確定していること
| 項目 | 内容 |
|---|---|
| コーチの姿勢 | 責めない・言語化を助ける・このままでいけると伝えられる存在 |
| 伴走スタイル | 内省の深掘り・失敗を責めない関わり方 |
| メッセージ方針 | 結果・焦りを煽らない。シェイプネスという在り方で届ける |
| 理想の連鎖 | 受講者が体現者になり、次の誰かを変えていく構造 |

### 要検討の論点

**Q8. 提供形態**
- オンライン・オフライン・ハイブリッドのどれか？
- 地域制約をどう考えるか（全国展開 vs 首都圏集中）

**Q9. セッション形式**
以下のどの組み合わせにするか：
```
[ ] 1on1コーチング（個別・深い関与）
[ ] グループセッション（少人数・共感効果）
[ ] コミュニティ型（非同期・自走）
[ ] 上記の組み合わせ（どの比率か）
```

**Q10. コーチ要件の定義**
- シェイプネスを体現していることは必須条件か
- MYS受講者からコーチを育成する仕組みにするか
- コーチのトレーニング・認定をどう設計するか

**Q11. 顧客の入口（導線）**
- SNS・紹介・SFF経由など、主要チャネルはどこか
- 「結果を求めて来た人」をどのタイミングでシェイプネス文脈に転換するか

**Q12. コミュニケーション・言語設計**
- 「結果を歌わない」を守りながら、どんな言葉で興味を引くか
- SFやShapeFitの既存ブランドとMYSの関係をどう整理するか

---

## SUSTAIN｜継続・拡張の定義

### 確定していること
| 項目 | 内容 |
|---|---|
| スケールの理想像 | 体現者が講師・コミュニティ貢献者になる連鎖構造 |
| 広め方のNG | 短期成果の強調・派手なビフォーアフター・焦りを燃料にする |
| 社会的ゴール | 自分軸で生きる人を増やし、社会・家族・仕事を変えていく |

### 要検討の論点

**Q13. ビジネスモデルの設計**
以下の収益構造のどれを主軸にするか：
```
[ ] 月額サブスク（コミュニティ・コンテンツ）
[ ] 期間プログラム型（3ヶ月・6ヶ月の一括）
[ ] コーチング単発課金
[ ] 上記の組み合わせ（どのフェーズで何を主軸にするか）
```

**Q14. 価格帯の設計**
- ターゲット（余白のない成功者）の支払い許容額はどのくらいか
- 安すぎると価値が伝わらない・高すぎると入口が狭まるバランスをどう取るか

**Q15. フェーズ設計**
- ローンチ時のMVPをどこに絞るか（最小構成は何か）
- SFF2026後のタイミングで本格始動する前提か

**Q16. 品質担保しながらのスケール**
- コーチが増えたとき、シェイプネスの質をどう維持するか
- 認定制度・定期レビューなどの仕組みが必要か

**Q17. KPI・評価軸の設計**
数字に頼らない思想を持ちつつ、事業として何を測るか：
```
例）
・受講者のバランスホイールスコアの変化
・継続率・コミュニティ残存率
・体現者からコーチへの転換率
・NPS（推奨意向）
```

---

## 優先度の高い論点

Howの中では：

1. **Q9**（セッション形式）→ コスト・工数・体験品質の根幹
2. **Q10**（コーチ要件）→ スケールの可否に直結
3. **Q11**（顧客導線）→ SFF等の既存アセットをどう活かすか

Sustainの中では：

1. **Q15**（フェーズ設計・MVP定義）→ いつ・何から始めるかの意思決定
2. **Q13**（ビジネスモデル）→ 収益構造が決まると全体設計が固まる
3. **Q17**（KPI設計）→ 「数字に頼らない」思想とビジネス評価の両立方法
