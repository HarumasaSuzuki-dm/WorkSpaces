# 🏢 IT導入補助金（デジタル化・AI導入補助金）完全解説 2025〜2026年度版

> 調査日：2026年3月15日 / 対象制度：IT導入補助金2025 & デジタル化・AI導入補助金2026

---

## ❶ 制度名・スキームの変化

### 🔄 名称変更の経緯

| 年度 | 制度名 | 実施主体 |
|------|--------|---------|
| ～2024年度 | IT導入補助金2024 | 中小機構（SMRJ）・TOPPAN株式会社（事務局） |
| 2025年度 | IT導入補助金2025 | 同上（2025年1月7日をもって公募終了） |
| **2026年度〜** | **デジタル化・AI導入補助金2026** | 同上（2026年3月末より受付開始） |

**名称変更の理由：** 単なるITツール導入にとどまらず、「より踏み込んだデジタル化の推進とAIの活用」が中小企業の生産性向上に不可欠であることを広く周知するためと、公式に説明されています。[情報源](https://it-shien.smrj.go.jp/news/40013)

---

### ✅ IT導入補助金2025からの主な変更点（2026年度）

**変更点①：補助金名称の変更**
- 「IT導入補助金」→「**デジタル化・AI導入補助金**」に改称

**変更点②：2回目以降の申請に追加要件（重要！）**
IT導入補助金2022〜2025の期間中に交付決定を受けた事業者が再申請する場合、以下の賃上げ要件が**新たに必須**となりました。

| 要件 | 内容 |
|------|------|
| ① 給与総額の向上 | 全従業員（非常勤含む）の1人当たり給与支給総額の年平均成長率を「日銀の物価安定目標（2%）＋1.5%＝3.5%以上」向上させること |
| ② 賃上げ計画の表明 | 交付申請時点で、賃金引上げ計画を従業員に対して表明すること |
| ③ 効果報告 | 翌事業年度以降3年間の事業計画を策定・実行し、効果を報告すること |

> ⚠️ 未達成・未報告の場合は**補助金の全額または一部返還**が求められます。

**変更点③：AI機能を有するツールの明確化**
- ITツール検索においてAI機能を持つツールの**絞り込みが可能**に
- 検索結果にAIツールであることが**明示**されるよう改善

**スキーム全体の変化：**  
申請枠の構成・補助率・対象経費の考え方はIT導入補助金2025の枠組みを**概ね踏襲**しており、枠名称と詳細要件の小変更にとどまります。[情報源](https://it-shien.smrj.go.jp/news/40013) / [補助金ポータル](https://hojyokin-portal.jp/columns/degital_ai_summary)

## ❷ 申請「枠」の種類と詳細

### 枠の全体像

```
デジタル化・AI導入補助金2026
├── ① 通常枠
├── ② インボイス枠（インボイス対応類型）
├── ③ インボイス枠（電子取引類型）
├── ④ セキュリティ対策推進枠
└── ⑤ 複数者連携デジタル化・AI導入枠
```

---

### ① 通常枠

| 項目 | 内容 |
|------|------|
| **補助対象** | 自社の課題に合ったITツール（ソフトウェア・クラウド利用料等）の導入 |
| **補助率** | 原則 **1/2以内**。令和6年10月〜令和7年9月の3か月以上、最低賃金未満で雇用している従業員が全体の30%以上の場合は **2/3以内** |
| **補助上限額** | 1プロセス以上：**5万〜150万円未満** ／ 4プロセス以上：**150万〜450万円** |
| **補助対象経費** | ソフトウェア購入費、クラウド利用料（最大2年分）、機能拡張、データ連携、セキュリティ、導入コンサル、保守サポートなど |
| **ツール要件** | 「顧客対応・販売支援」「決済・債権管理」「供給・在庫・物流」「会計・財務」「総務・人事・給与」などのプロセスを1種類以上含むこと。**汎用プロセス（分析ツール等）単体は不可** |
| **申請制限** | 交付申請期間中1法人1申請のみ。IT導入補助金2025通常枠等で交付決定を受けた場合、**決定日から12か月以内は再申請不可** |

---

### ② インボイス枠（インボイス対応類型）

| 項目 | 内容 |
|------|------|
| **補助対象** | インボイス制度対応の「会計」「受発注」「決済」機能を持つソフトウェア＋PC等ハードウェア |
| **補助率（ソフト50万円以下）** | 中小企業：**3/4以内** ／ 小規模事業者：**4/5以内** |
| **補助率（ソフト50万円超）** | **2/3以内**（50万円以下部分は上記の高率が適用） |
| **補助率（ハードウェア）** | **1/2以内** |
| **補助上限額** | ソフトウェア：最大 **350万円** ／ PC・タブレット等：**10万円** ／ レジ・券売機等：**20万円** |
| **機能要件** | 50万円以下：「会計」「受発注」「決済」のうち**1機能以上** ／ 50万円超：**2機能以上** |
| **特記事項** | ハードウェアのみの申請は不可。ハードはソフトウェア使用に資するものに限る |

### ⛔ 申請不可（みなし大企業）の定義

形式上は中小企業でも、以下に該当する場合は申請不可です。

1. 発行済み株式の**1/2以上**を1つの大企業が保有
2. 発行済み株式の**2/3以上**を複数の大企業が合計で保有
3. 大企業の役員・職員を兼ねる者が役員総数の**1/2以上**を占める
4. 上記①〜③に該当するみなし大企業が株式を**全て**保有する企業
5. 上記①〜③の役員・職員を兼任する者が役員**全員**を占める企業
6. 直近3年分の**課税所得年平均額が15億円超**

## ❹ 導入ツール側の要件

### 登録ITツールリストとは

**IT導入支援事業者が事務局ポータル（IT事業者ポータル）を通じて登録申請を行い、事務局の承認を受けたツールのみが補助対象**となります。このリストは公式サイトの「ITツール検索」機能で誰でも確認できます。

> 📎 ITツール検索：https://it-shien.smrj.go.jp/search/

**未登録のITツールでは交付申請ができません。**  
2026年度からは検索機能が強化され、AI機能を持つツールを絞り込み・識別できるようになりました。

---

### IT導入支援事業者との共同申請が必須な理由と仕組み

**なぜ共同申請が必要か？**

本制度は「中小企業がツールを選んで自ら申請する」のではなく、**IT導入支援事業者が導入後のサポートまで一体的に責任を持つ仕組み**として設計されています。これにより、ツールの有効活用・アフターフォローを担保し、補助金の効果を最大化することを目指しています。

**共同申請の実際のフロー：**

```
①  中小企業が「ITツール検索」でIT導入支援事業者を選定
       ↓
②  IT導入支援事業者から「申請マイページ」への招待を受ける
       ↓
③  申請者（中小企業）が代表者氏名等の基本情報を入力
       ↓
④  申請に必要な情報入力・書類添付
       ↓
⑤  IT導入支援事業者がツール情報・事業計画値を入力
       ↓
⑥  申請者が最終確認・宣誓を行い、事務局へ提出
       ↓
⑦  交付決定通知（締切から約5週間程度）
```

**IT導入支援事業者の役割：**
- 補助対象ITツールの事務局への登録
- 申請者への適切なITツールの提案・導入・アフターフォロー
- 事業実施効果報告まで一貫したサポート

### ツールが補助対象となるための条件

| 条件 | 内容 |
|------|------|
| **事業者登録** | IT導入支援事業者として事務局に正式登録されていること |
| **ツール登録・承認** | IT事業者ポータルで登録申請を行い事務局の承認を受けていること |
| **業務プロセスの充足** | 「顧客対応・販売支援」「決済・債権管理」「供給・在庫・物流」「会計・財務」「総務・人事・給与」「業種特化型」プロセスのいずれか1種類以上を有すること |
| **汎用プロセスのみは不可** | 「汎用・自動化・分析ツール」（AI分析ツール等）のみでの申請は不可。必ず上記の業務プロセスを伴うこと |
| **カテゴリー要件** | ソフトウェア・オプション・役務・ハードウェア（インボイス枠のみ）・セキュリティサービス、各カテゴリーに定められた要件を満たすこと |

## ❺ 採択審査の評価軸・基準

### 審査で重視される項目

| 評価軸 | 具体的な審査ポイント |
|--------|-------------------|
| **事業計画の具体性・説得力** | IT導入による効果や将来展望が明確か。「なぜそのツールが必要か」を論理的に説明できているか |
| **定量的な効果目標** | 導入後の効果測定方法・実行スケジュール・数値目標（コスト削減率、時間削減率等）が記載されているか |
| **課題とツールの整合性** | 自社の経営課題と導入ツールが明確に結びついているか。課題にマッチしたツール選定がされているか |
| **国の政策目標との整合性** | 「企業の成長と国の政策目標（デジタル化・AI活用・賃上げ等）にどう貢献するか」が示されているか |
| **費用対効果** | 投資額に見合った改善効果が説明されているか |
| **社会的意義** | 人材確保、労働力不足対策、地域活性化等への寄与があるか |

### ➖ 減点・不採択要因

| 要因 | 内容 |
|------|------|
| 過年度の交付決定実績 | 2022〜2025年度の受給歴がある場合（重複プロセスは特に重減点） |
| プロセスの完全一致 | 過去導入ツールと今回導入ツールのプロセスが完全一致の場合は**不採択** |
| 賃上げ計画未達成 | 過去に賃上げ計画で加点を受けたにもかかわらず要件未達の場合 |
| 特定の組織形態 | 財団法人・社団法人・社会福祉法人は採択率が低い傾向 |
| 書類不備 | 要件不満足・誤字脱字・押印漏れ・提出期限遅れ |
| 対象外経費の申請 | ハードウェア単体購入費、業務外ソフト開発費等 |
