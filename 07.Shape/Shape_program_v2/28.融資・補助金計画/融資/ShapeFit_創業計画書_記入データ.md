## １．創業の動機

> 📌 **記入ポイント**：「なぜこの事業か」を熱意＋論理でまとめる。自身の実体験を入れると説得力が増す。

**【記入案】**

```
フィットネス指導を通じて、「体を変えること」だけでなく「自分で選び続ける力」を
持てる人を増やしたいという想いから、事業を拡大することを決意した。

既存のパーソナルジム（Shape Fit）の運営を通じ、身体の変化が人生全体に
波及することを多くのクライアントで実感してきた。しかし、対面でのパーソナル
指導は提供できる人数に限界があり、より多くの人に価値を届けるためには
オンラインでの仕組み化が不可欠だと判断した。

また、2026年に開催するShape Fit Festival（SFF）を軸としたブランド構築により、
フィットネスを入口とした「人生を整えるための伴走サービス」という新カテゴリで
事業を展開する。具体的には以下の3事業を展開する。

①MYSオンラインコーチング（新規）：6ヶ月間の伴走型オンラインプログラム
②Shapeコミュニティ（SFC）（新規）：継続的な自走を支えるコミュニティ
③Shape Fit Festival（SFF）（既存拡大）：年次フィットネスイベントのブランド強化
```

フィットネス×ライフコーチング×コミュニティの3事業を展開。「体を変えることを入口に、人生全体を整える伴走サービス」を提供する。

### 各サービスの詳細

#### ① MYSオンラインコーチング（主力商品）
- **ターゲット**：20代後半〜40代男性。会社員・経営者。フィットネスを手段として人生を前向きに変えたい人
- **サービス内容**：6ヶ月（24週間）の伴走型オンラインプログラム。3プラン構成
- **価格**：
  - セルフコース：12.5万円／半年（月換算 約2.1万円）
  - スタンダードコース：38.5万円／半年（月換算 約6.4万円）★主力
  - プレミアムコース：77万円／半年（月換算 約12.8万円）
- **開始時期**：2026年10〜11月（テスト受講者10〜20名）→ 2027年〜本ローンチ

#### ② Shape Fit Festival（SFF）
- **ターゲット**：フィットネスに関心のある20〜40代・全国のSFF参加希望者
- **サービス内容**：年次フィットネスイベント（関西大会・関東大会）の開催
- **収入源**：参加費・スポンサーシップ・グッズ販売
- **特記事項**：2026年度のブランディング強化のためブランディング会社に投資済み（SFF認知拡大・集客力強化が目的）

#### ③ Shapeコミュニティ（SFC）
- **ターゲット**：「挑戦したいのに日常で折れてしまう人」20〜40代男女
- **サービス内容**：月額会費制コミュニティ。月1回オフラインイベント・隔週オンラインスペース・1on1フォロー含む
- **価格**：月額5,000円（本格化後。2026年10月〜有料化予定）
- **現状**：テスト期として既存16名で無料運用中（〜2026年9月）

### セールスポイント（自社の強み）

```
【3つの差別化ポイント】

① 「卒業」を目指す設計（vs パーソナルジム）
パーソナルジムは通い続けることで収益を得るモデルだが、
Shape Fitは約6ヶ月で「自分で選び続けられる力」を身につけ自走することを
ゴールとする。依存させない設計が顧客の信頼につながる。

② フィットネス×ライフコーチングの融合（vs オンラインコーチング）
既存のキャリアコーチング（GOAL-B等）は「仕事・年収」が主語。
Shape Fitは「身体を起点にした人生全体の変化」が主語。
体が動く→生活が変わる→人生が変わる、という具体的な変化のループを提供。

③ SFFという実証実績を持つブランド（vs 他のオンラインサービス）
年次イベント「Shape Fit Festival」の開催実績により、
サービスの信頼性・コミュニティの熱量を対外的に証明できる。
SFF参加者が見込み顧客リストとして機能し、広告に頼らない集客が可能。
```

### 競合・市場など自社を取り巻く状況

```
【市場規模】
日本のフィットネス市場：約4,000億円規模（2023年）。コロナ後に回復・拡大傾向。
オンラインフィットネス・コーチング市場は急成長中。

【主要競合との比較】
・パーソナルジム（RIZAP等）：月10〜30万円。通い続ける前提モデル。「卒業」概念なし
・オンラインコーチング（GOAL-B等）：33〜110万円一括。キャリア特化。身体要素なし
・オンラインフィットネス（SOELU等）：月1,000〜10,000円。動画中心。伴走なし
・コミュニティ型サロン：月500〜5,000円。交流中心。継続設計が弱い

【Shape Fitのポジション】
「フィットネス × ライフコーチング × 自走卒業」という独自カテゴリで競合なし。
価格帯はRIZAPと同等だが、「6ヶ月で卒業できる」点と「コミュニティによる継続支援」で差別化。
SFFというリアルイベントを通じた顧客体験の提供も独自の強み。
```

## 📌 面談で聞かれる想定Q&A（準備用）

**Q1. なぜ今、この3事業を立ち上げるのですか？**
> SFFというブランドと、パーソナルジムでの指導実績によって、提供できる価値の広がりを確信したから。特にオンライン化することで、対面指導の限界（時間・場所）を超えて多くの人に届けられる。

**Q2. MYSとSFCはまだ売上ゼロですが、本当に見込みはありますか？**
> SFFを通じて既にLINEリスト・SNSフォロワーという温かい見込み客が存在する。テスト段階は10〜20名の小規模からスタートし、品質を確認してから拡大する保守的な設計をしている。

**Q3. SFFのブランディング費用（支払済み）はなぜ今回の融資に含めるのですか？**
> SFFのブランド強化は、MYS・SFCという新事業の集客基盤を整備するための先行投資。そのブランディング費用の一部（150万円）を新事業の立ち上げに紐づけた運転資金として申請している。請求書・領収書を添付している。

**Q4. 返済の見通しは？**
> 1年後の月利益70万円から算出すると、年間840万円の利益から年間返済額（550万円・10年返済で月約4.6万円）は十分に支払可能。テスト段階の創業当初（月▲10万円の赤字）については、自己資金と既存事業（パーソナルジム）の売上でカバーする計画。
