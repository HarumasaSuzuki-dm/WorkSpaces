## ❷ 2024年以降の制度変更・廃止・新設まとめ

### 🔴 廃止（2024年3月31日）

> #### ~~新創業融資制度~~
> 日本政策金融公庫を代表する創業融資として長年機能してきた「**新創業融資制度**」が、2024年3月31日をもって取扱終了。

---

### 🟢 新設・拡充・変更（2024年4月〜2026年現在）

| 時期 | 変更内容 | 主なポイント |
|---|---|---|
| **2024年4月** | 新規開業資金の大幅拡充 | ① 自己資金要件（1/10以上）の**完全撤廃** ② 無担保・無保証の申請が可能に ③ 運転資金の返済期間が7年→**10年**に延長 ④ 据置期間が2年→**5年**に延長 |
| **2024年4月** | 創業支援貸付利率特例制度 | 創業者への一律**▲0.65%**引下げ（雇用拡大なら**▲0.9%**） |
| **2025年3月** | 「新規開業資金」を**「新規開業・スタートアップ支援資金」に改称** | スタートアップ支援を明確化。VC出資を受けた企業も対象に整理 |
| **継続拡充** | 経営者保証免除特例制度 | 一定要件を満たせば代表者個人保証が不要。上乗せ利率なし |
| **継続拡充** | 再挑戦支援関連 | 廃業歴がある再創業者向け。運転資金の返済期間**15年以内** |
| **継続拡充** | 中小企業経営力強化関連 | 認定支援機関（税理士等）による指導を受けた創業者に**特別利率A**を適用 |

## ❸ 固定金利と変動金利の選択基準（2026年の金利環境をふまえて）

### 🏛️ 公庫融資の基本：「原則・全期間固定金利」

日本政策金融公庫の融資は、**申込時の金利が全返済期間にわたって固定される「全期間固定金利」** が原則です。これは民間銀行の変動金利型ローンと根本的に異なる特徴です。

### 📈 2026年の金利環境

```
日銀政策金利の推移
─────────────────────────────────────
2024年3月  0.10%（マイナス金利解除）
2024年7月  0.25%（利上げ）
2024年12月 0.50%（利上げ）
2025年12月 0.75%（利上げ ← 1995年以来の高水準）
2026年1月  0.75%（据え置き）
2026年3月  現時点でも0.75%水準
─────────────────────────────────────
```

### ✅ 2026年における推奨スタンス

| ケース | 推奨選択 | 理由 |
|---|---|---|
| **公庫融資**（返済期間5年以上） | **固定金利を選択・今すぐ申込** | 今後も追加利上げが予想される局面。現在の固定金利を早期に確定することが有利 |
| **民間銀行・変動金利ローン** | **固定金利への切替を検討** | 2026年4月以降、基準金利がさらに上昇する見通し |
| **据置期間の活用** | **最長5年の据置をフル活用** | 創業初期のキャッシュフロー安定まで元金返済を猶予。利息のみ払う形で資金繰りを守る |
| **資本性ローン** | **業績連動型の活用も** | 業績が悪い時期は金利が下がる構造。財務体質強化が目的なら選択肢 |

> ⚠️ **注意点**：公庫の「固定金利」は申込・実行時点で確定します。審査から実行までのタイムラグ（約1〜1.5ヶ月）がある点に注意。金融情勢によって申込後に利率が変わる場合があります。

## ❹ 信用保証協会との違い・使い分けの目安

### 📐 仕組みの根本的な違い

```
【公庫（直接融資）】
  事業者 ←── 資金 ──── 日本政策金融公庫
         ──── 返済 ──→

【信用保証協会（保証付き融資）】
  事業者 ── 申込 → 民間銀行 ─保証依頼→ 信用保証協会
         ←── 資金 ─── 民間銀行
         ── 保証料 ──────────────→ 信用保証協会
```

### 📊 7項目詳細比較表

| 比較項目 | 🏦 日本政策金融公庫（国民生活事業） | 🛡️ 信用保証協会付き融資（制度融資） |
|---|---|---|
| **融資形態** | 公庫が直接融資（二者間） | 民間銀行が融資、保証協会が保証（三〜四者間） |
| **金利水準** | 固定：3.30～4.70%（基準）/ 特利適用で2%台前半も | 変動：銀行が設定（制度融資なら自治体の利子補給あり） |
| **保証料** | **不要（ゼロ）** | 年0.5～2.0%程度（別途負担） |
| **審査期間** | **約1〜1.5ヶ月** | 約2〜3ヶ月（銀行＋保証協会の二段階審査） |
| **担保・保証人** | 無担保・無保証人の制度が豊富 | 原則代表者連帯保証（ガイドライン運用で緩和傾向） |
| **審査の重点** | 事業計画の妥当性・経営者の資質 | 財務内容・返済能力・財務実績 |
| **融資上限** | 7,200万円（国民生活事業） | 8,000万円（無担保保証限度額） |

## 📋 FPとしての総合アドバイス（2026年3月時点）

> ### 🎯 創業者へのアクションプラン
> 
> **STEP 1** — 女性・若者・シニアに該当するか確認 → **特別利率A + ▲0.65%** の組合せ金利を狙う
>
> **STEP 2** — 認定経営革新等支援機関（税理士・中小企業診断士）に事業計画書の添削を依頼 → 中小企業経営力強化関連でさらに**特別利率A**を確保
>
> **STEP 3** — 金利上昇局面であるため、**今すぐ申込・固定金利を確定**させる
>
> **STEP 4** — 据置期間は**最大5年**をフル活用し、事業が軌道に乗るまで元金返済を猶予
>
> **STEP 5** — 公庫での返済実績が1〜2年蓄積したら、信用保証協会付き融資（制度融資）で**民間銀行との関係構築**を開始し、将来のプロパー融資へのルートを整備
