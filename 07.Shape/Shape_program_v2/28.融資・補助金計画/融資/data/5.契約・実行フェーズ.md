# 📋 公庫融資「契約・実行フェーズ」完全ガイド【2026年3月版】

## ❶ 融資決定通知から契約締結までの手順

### 🔔 STEP 1｜融資決定通知の受け取り

面談から**約2週間**（土日祝含む）で審査結果が通知されます。

| 通知方法 | 内容 |
|----------|------|
| 📞 電話 | 最初に合否のみ口頭で連絡 |
| 📮 郵送 | 融資額・金利・返済期間・据置期間などの条件が記載された書面 |

> **チェックポイント：** 通知書に記載された条件（融資額、金利、返済年数、据置期間）を必ず確認し、当初希望と相違がある場合は担当者に問い合わせましょう。

---

### 📝 STEP 2｜契約書類の受け取りと返送（紙契約の場合）

融資決定通知から**5〜10日**で、以下の書類が郵送されます。

| 書類名 | 対応内容 |
|--------|----------|
| **借用証書** | 借主の署名、収入印紙の貼付、送金先口座の指定が必要 |
| ご契約に関する重要なご案内 | 契約条件・同意事項の確認 |
| 団体信用生命保険申込書兼告知書 | 加入希望の場合のみ記入 |
| 預金口座振替利用届 | 返済口座の指定（届出印と通帳が必要） |
| 印鑑証明書 | 法人→法務局、個人→市区町村役場で取得 |

---

### 💻 STEP 2'｜電子契約サービスを利用する場合（国民生活事業）

2026年3月現在、**国民生活事業では電子契約サービスが標準的な選択肢**として整備されています。記入・押印不要で、スマートフォンがあれば手続き可能です。

**電子契約サービスの利用フロー（5ステップ）：**

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  【事前準備】申込時に以下2点を提出                       │
│  ① 電子契約サービス（国民生活事業）利用申込書             │
│  ② 送金先口座の預金通帳の写し（表紙・見開き1ページ目）      │
│                          ↓                          │
│  STEP 1  ユーザーID・初期パスワードがメールで届く           │
│                          ↓                          │
│  STEP 2  「メールアドレス到達確認」メールを受信             │
│          → 到達確認コードを入力してメールアドレス確認         │
│                          ↓                          │
│  STEP 3  「電子契約手続きのお願い」メールを受信             │
│          → URLからログイン → 本人確認書類をアップロード      │
│                          ↓                          │
│  STEP 4  認証コード（SMS）を入力 → 「署名実行」クリック      │
│          ＝ 電子署名完了                               │
│                          ↓                          │
│  STEP 5  「電子署名完了のお知らせ」メール受信               │
│          → 融資予定日に指定口座へ送金                    │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**電子契約の利用可能時間：** 月曜〜土曜 9:00〜23:00（祝日含む）

> ⚠️ **電子契約が利用できないケース：**
> - 担保設定手続きを伴う融資
> - 代表者以外の連帯保証人の提供を伴う融資
> - 法人の場合は法人・代表者それぞれのメールアドレスが必要（異なるアドレスが必須）

[日本政策金融公庫 電子契約サービス（国民生活事業）](https://www.jfc.go.jp/n/service/econtract/kn/index.html)

## ❷ 担保・保証人が求められるケース／不要なケース

### ✅ 担保・保証人が「原則不要」なケース

| 制度・状況 | 内容 |
|-----------|------|
| **新規開業・スタートアップ支援資金** | 創業者・開業後7年以内が対象。原則**無担保・無保証人** |
| **マル経融資（小規模事業者経営改善資金）** | 商工会議所等の推薦を経た場合は原則**無担保・無保証人** |
| **経営者保証免除特例制度** | 下記の要件を満たす法人（上乗せ金利なし） |

**📌 経営者保証免除特例制度の主な適用要件（7パターン）：**

```
①【既存法人】税務申告2期以上完了しており…
   ✔ 法人と代表者の一体性解消が確認できること
   （= 事業上必要のない法人→代表者への貸付金等がないこと）
   かつ次のいずれか：
   ✔ 直近2期連続で減価償却前経常利益が赤字でない
   ✔ 直近決算期に債務超過でない

②【創業者・税務申告2期未満】上記一体性解消要件を満たす

③【物的担保提供あり】一体性解消要件を満たす

④【スタートアップ】開業後5年以内 ＋ 技術・ノウハウに新規性あり
  （知財権活用、ものづくり補助金等、VC出資を受けた事業など）

⑤【他行での保証免除実績あり・協調対応見込み】

⑥【事業承継・集約・活性化支援資金の利用者】

⑦【NPO法人向けソーシャルビジネス支援資金】
```

> 令和6年度実績：国民生活事業の法人・個人向け融資のうち**件数の71.3%、金額の61.9%が無保証**で融資されています。（公庫公式データ）

[保証人に依存しない融資｜日本政策金融公庫](https://www.jfc.go.jp/n/company/national/guarantor.html)

---

### ⚠️ 担保・保証人が求められやすいケース

| 状況 | 内容 |
|------|------|
| 大型融資（中小企業事業） | 数千万〜数億円規模では不動産担保等を求められることが多い |
| 債務超過・連続赤字の法人 | 保証免除特例の適用外となり経営者保証が求められる |
| 第三者保証人が必要な場合 | 代表者以外の保証人が要求される場合は電子契約も利用不可 |
| 優遇金利目的の担保提供 | 任意で担保を提供することで金利優遇を受けることも可能 |

## ❸ 振込実行のタイミングと口座に関する注意点

### ⏱ 振込実行までのタイミング

```
【紙契約の場合】
  契約書類を公庫へ発送
       ↓（郵送日数：1〜2日）
  公庫に書類が到着
       ↓
  【おおむね2〜4営業日以内に振込！】

【電子契約の場合】
  全署名者の電子署名完了
       ↓
  「電子契約手続きのお願い」メールに記載の【融資予定日】に送金
```

| 状況 | 入金までの目安 |
|------|-------------|
| 通常時（郵送・書類不備なし） | 書類到着後 **2〜4営業日** |
| 電子契約 | **融資予定日** に送金 |
| 繁忙期（年度末：3月末等） | **5〜6営業日**かかる場合も |
| 書類不備・差し戻しあり | 再提出後さらに数日〜1週間 |

> 💡 **実務Tips：** 3月26日（木）〜27日（金）に書類が到着すると、振込が年度をまたいで4月3日（金）になることが2026年は案内されています。**資金が必要な日から逆算**して余裕を持って手続きしましょう。

---

### 🏦 口座に関する重要な注意点

| 項目 | 内容 |
|------|------|
| **ネット銀行（送金先）** | ❌ 基本的に**不可**（楽天銀行・PayPay銀行等） |
| **ネット銀行（返済口座）** | ✅ 一部対応（事前に確認が必要） |
| **法人の場合** | ✅ **法人口座が必須**。未開設だと入金が遅延する（開設審査に時間がかかるため、融資申込と並行して開設手続きを開始すること） |
| **個人事業主の場合** | ✅ 通常の個人名義口座でOK（屋号付き口座は不要） |
| **振込入金額** | 融資額から①送金手数料 ②団信1年目掛金（加入の場合）が差引かれるため、借用証書記載額と**実際の振込額が異なる** |
| **返済引落日** | 「5日」「10日」「15日」「25日」「月末」などから選択 |

## ❹ 融資実行後に公庫へ提出が必要な書類

### 📌 事業区分別の提出義務まとめ

| 事業区分 | 提出義務の内容 |
|---------|-------------|
| **国民生活事業（小規模事業者・個人）** | 原則として**定期的な報告義務なし**。毎年の決算書提出も通常は求められない |
| **中小企業事業（中堅〜中小法人）** | **毎決算期**に税務申告書・勘定科目明細を含む決算報告書の提出が必要 |
| **資本性ローン** | **毎決算期**の決算書 ＋ **毎四半期ごとの試算表（業況報告書）** の提出が義務 |

---

### 🔍 国民生活事業の「実質的な後フォロー」

形式上の定期報告義務はなくても、実務上は以下の対応が発生します。

```
✔ 設備資金を借りた場合
  └ 設備の購入・設置が完了したことの確認（領収書・設置写真等の提出を
    求められるケースあり）

✔ 公庫担当者からの業況確認の連絡・訪問
  └ 融資後しばらく経つと担当者から「その後いかがですか？」という
    業況確認の電話・訪問が入ることがある

✔ 2回目の融資申請時
  └ その時点の決算書・試算表・確定申告書を提出し、1回目融資後の
    実績を報告することになる
```

### 🚨 融資実行後に絶対に守るべきルール

| ルール | 違反した場合のペナルティ |
|--------|----------------------|
| **資金用途を守る** | ①融資金の一括返済要求 ②金利引き上げ ③新規融資の停止 |
| 経営の重大変化（代表交代・業種変更等）は速やかに報告 | 契約違反となる可能性あり |
| 返済困難になった場合は早期相談 | 早期相談であればリスケジュール等の対応が可能 |

> ⚠️ **資金用途違反の具体例：**
> - 「設備購入資金」として借りたのに運転資金（人件費・家賃等）に充てる
> - 「運転資金」として借りたのに既存の借入返済に充てる
> - 申告した使途と全く異なる用途での支出

[創業融資の使い道とNG例｜資金用途違反のリスク](https://biz.moneyforward.com/establish/basic/75446/)

## 📊 全体タイムライン（審査通過〜返済開始まで）

```
【審査通過（面談から約2週間）】
        ↓ 通知メール・電話（合否）
        ↓ 融資条件通知書が郵送（5〜10日後）
【条件確認・契約手続き】
        ↓ 書類返送 or 電子契約（署名）
        ↓ 公庫が書類受領
【融資実行（書類到着後2〜4営業日）】
        ↓ 指定口座に振込
【据置期間（設定がある場合）】
        ↓ 利息のみ返済
【元本返済開始】
        ↓ 毎月口座振替
【2回目融資の相談 → 実績報告】
```

## ✅ 契約・実行フェーズの重要チェックリスト

- [ ] 融資条件（金額・金利・期間・据置）を通知書で確認した
- [ ] 電子契約希望の場合、申込時に「利用申込書」と通帳写しを提出した
- [ ] 送金先口座が**ネット銀行でないこと**を確認した
- [ ] 法人の場合、法人口座の開設が完了している
- [ ] 借用証書の署名・収入印紙の貼付・押印に漏れがない
- [ ] 返済引落日を指定し、口座振替の手続きが完了した
- [ ] 資金の使途を申告通りに守ることを確認した
- [ ] 繁忙期の場合は余裕を持ったスケジュールで書類提出した
