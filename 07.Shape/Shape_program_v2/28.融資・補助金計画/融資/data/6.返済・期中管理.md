# 🏦 日本政策金融公庫｜返済・期中管理 完全ガイド（2026年3月時点）

---

## 📋 概要マップ

```
借入実行後の管理フロー
│
├─ 1. 毎月の返済（口座振替が原則）
├─ 2. 返済困難時 → リスケ・借換制度の活用
├─ 3. 余裕資金がある時 → 繰り上げ返済の検討
├─ 4. 事業拡大時 → 追加融資・増額申請
└─ 5. 完済後 → 抵当権抹消・証明書取得
```

---

## 1. 返済方法の種類と手続き

### 🔵 返済スキームの種類（返済方式）

| 方式 | 内容 | 向いているケース |
|------|------|----------------|
| **元金均等返済** | 毎月の元金額が一定、利息は逓減 | 早期に元本を減らしたい |
| **元利均等返済** | 毎月の支払総額が一定 | 月々の資金計画を立てやすい |
| **ステップ（段階）返済** | 創業初期は返済額を少なく設定し、後半に増額 | 創業間もなく収益が安定していない |

> 据置期間（元金の返済を猶予する期間）を別途設定することも可能。通常1〜5年。

---

### 🔵 返済手段（お金の払い方）

**① 口座振替（原則的な方法）**

公庫の標準的な返済手段。毎月指定日に登録口座から自動引き落とし。

**手続き方法：**
- 公庫支店から送付される「**預金口座振替依頼書兼申込書**（2枚綴り）」に必要事項を記入
- 振替口座がある**金融機関の窓口に提出**するか、**WEB（オンライン）で申込**可能
- 対応ネット銀行（2023年以降拡大）：楽天銀行、PayPay銀行などの一部ネット銀行も対応

**② 銀行振込（繰り上げ返済・一時的な入金に利用）**

通常の月次返済は口座振替だが、繰り上げ返済時などは振込対応となる。

**③ 窓口払い**

公庫支店窓口での直接払いも対応しているが、現在は口座振替の移行が推奨されている。

> 📌 **ポイント**：返済日は毎月特定日（通常月末）に設定されることが多い。引き落とし残高不足に注意。残高不足が続くと延滞扱いとなり信用情報に影響する。

[日本政策金融公庫 公式サイト](https://www.jfc.go.jp/)

## 2. 返済困難になった場合の対応策

### ⚠️ 基本姿勢：「早期相談」が最重要

返済が困難になった場合、**延滞してから相談するのは最悪のパターン**。延滞が発生した時点で信用情報に傷がつき、以降の条件変更交渉も不利になる。**返済が厳しくなる兆候が出た段階で、すぐに担当支店に連絡することが最善策。**

---

### 🔵 ① リスケジュール（返済条件変更）

**内容：** 月々の返済額を一時的に減額、または元金返済を猶予してもらう条件変更。

**手続きの流れ：**

```
① 担当支店に「条件変更の相談をしたい」と電話連絡
       ↓
② 直近の試算表・資金繰り表・借入一覧表を準備
       ↓
③ 窓口面談（経営状況の説明）
       ↓
④ 経営改善計画書の作成・提出
       ↓
⑤ 審査・条件変更の決定（通常2週間〜1か月）
       ↓
⑥ 条件変更後の返済開始
```

**実態と注意点：**
- 公庫は政策金融機関のため、民間銀行と比較して**柔軟に対応しやすい**傾向
- 条件変更期間中は原則として**新規・追加融資を受けることができない**
- 「リスケ中」は信用保証協会の保証が使えなくなる場合があるため注意
- 利息は変更後も払い続ける必要あり（元金のみ猶予が多い）

---

### 🔵 ② 危機対応後経営安定貸付（セーフティネット貸付）【2025年2月〜】

コロナ特別貸付（2024年12月終了）の後継として新設された制度。

| 項目 | 内容 |
|------|------|
| **対象** | 過去の災害・感染症等で借入が増大した中小・小規模事業者 |
| **目的** | 既往債務の借換による返済負担軽減 |
| **融資限度額** | 直接貸付：**20億円**、代理貸付：2億2,500万円 |
| **返済期間** | **20年以内**（うち据置期間5年以内） |
| **金利** | 基準利率（信用リスク・融資期間により変動） |

[危機対応後経営安定資金（公庫公式）](https://www.jfc.go.jp/n/finance/search/keieiantei.html)

---

### 🔵 ③ 公庫融資借換特例制度

公庫内での既存融資を新たな融資に借り換える制度。返済期間の延長により月々の返済額を軽減できる。

**ポイント：**
- 原則として**既往の公庫融資の借換のほか、新規融資も利用**することが必要
- 一部借換の対象にできない融資口もあり
- 借換部分の融資金額は、対象口ごとに10万円未満を切り捨て

[公庫融資借換特例制度（公庫公式）](https://www.jfc.go.jp/n/finance/search/58.html)

---

### 🔵 対応策の選択フロー

```
返済が困難
     │
     ├─【一時的な資金不足】→ リスケジュール（元金猶予）
     │
     ├─【コロナ等の影響で借入過多】→ 危機対応後経営安定貸付で借換
     │
     ├─【公庫内の複数融資を整理したい】→ 公庫融資借換特例制度
     │
     └─【事業継続困難】→ 中小企業活性化協議会・弁護士等に相談
```

## 3. 繰り上げ返済の手順・手数料・メリデメ

### 🔵 手順

```
① 取引支店（担当支店）に電話で繰り上げ返済の意思を連絡
       ↓
② 繰り上げ返済可能額・残高・手続き方法の案内を受ける
       ↓
③ 指定された公庫口座に送金（金融機関経由）
       ↓
④ 完了（一部繰り上げの場合は残債の返済計画を更新）
```

> ⚠️ コールセンターでは繰り上げ返済の手続き**不可**。必ず**取引支店**に直接連絡すること。

---

### 🔵 手数料（送金手数料のみ、違約金なし）

| 送金方法 | 手数料の目安 |
|---------|------------|
| 窓口送金 | **800〜1,000円** |
| ATM | **300〜600円** |
| インターネットバンキング | **150〜400円** |

> ✅ **違約金（期前弁済手数料）は原則不要**。ただし、融資条件によっては稀に発生する場合があるため、事前に担当者に確認すること。

---

### 🔵 メリット・デメリット

| メリット ✅ | デメリット ❌ |
|-----------|------------|
| 総支払利息を削減できる | **手元の自己資金（キャッシュ）が大幅に減少する** |
| 早期に負債をなくせる | **完済後、次の融資が受けにくくなる可能性**（返済実績がなくなるため） |
| 心理的な負担が軽減される | 低金利融資の場合は利息削減効果が限定的 |
| B/S（貸借対照表）が改善する | 一括返済後は緊急時の資金余力が失われる |

> 💡 **税理士・専門家の視点**：公庫の金利は民間より低い場合が多く、「低金利で借りたまま自社の資金効率を上げる」戦略の方が有利なケースも多い。繰り上げ返済は**キャッシュフローに余裕がある場合に限定**するのが賢明。

[繰り上げ返済に関する詳細解説](https://www.sogyo-support.biz/kouko/prepayment-kouko/)

## 4. 追加融資（増額・借換え）のタイミングと条件

### 🔵 2回目以降の追加融資

**ベストタイミング：**

| タイミング | 理由 |
|-----------|------|
| **決算書確定直後** | 利益・売上実績が証明しやすく、返済能力が客観的に示せる |
| **初回融資から1年以上経過後** | 返済実績の積み重ねが最重要評価ポイント |
| **返済遅延ゼロの実績がある時** | 延滞歴が一度でもあると審査は大幅不利 |
| **新たな資金需要が明確な時** | 事業拡大・設備投資など具体的な目的がある場合 |

**審査で重視されるポイント（2回目以降）：**

| 評価項目 | 詳細 |
|---------|------|
| **返済実績** | 遅延・延滞がないか→最重要 |
| **事業の業績推移** | 売上・利益が計画通りに推移しているか |
| **資金繰りの現状** | 直近の資金繰り表・試算表で実態を確認 |
| **追加融資の使途明確性** | 新規設備・運転資金など目的が具体的か |

---

### 🔵 他行借入の公庫での借換え（原則不可）

**重要：** 銀行・信用金庫など**他の金融機関からの借入を公庫融資で返済する「借換え」は原則として不可**。公庫は補完的な政策金融機関であり、民間金融機関との競合を避ける立場のため。

**例外的に対応可能なケース：**
- 公庫内の複数融資間での借換え（公庫融資借換特例制度）
- 災害・感染症後の経営安定資金としての借換え（上記「危機対応後経営安定貸付」）

[追加融資タイミングの詳細](https://jfc-center.co.jp/jfc-additional-loan-timing/)

## 5. 完済後の手続き

### 🔵 手続きの全体像

```
完済
 │
 ├─【無担保融資の場合】→ 実質的に手続き完了（残高証明等が必要なら支店に依頼）
 │
 └─【担保（抵当権）設定あり融資の場合】→ 抵当権抹消手続きが必要
```

---

### 🔵 ① 抵当権抹消手続き（担保設定融資の場合）

完済後、公庫から**抵当権抹消登記に必要な書類**が交付される。これを用いて法務局で登記手続きを行う。

**手続きの流れ：**

| ステップ | 内容 |
|---------|------|
| ① | 公庫が完済を確認し、抵当権抹消書類を交付 |
| ② | 受け取った書類を管轄**法務局**に持参または郵送 |
| ③ | 登記申請（自分でも可能、司法書士への依頼も一般的） |
| ④ | 登記完了→不動産の登記簿から抵当権が削除される |

> ⚠️ **重要注意事項**：
> - 完済時に受け取った書類（登記識別情報または登記済証）は**絶対に紛失しないこと**
> - 紛失した場合は「事前通知制度」での抹消手続きとなり、**日数が大幅にかかる**
> - 抹消手続きは速やかに行うことが推奨（放置すると将来の不動産売却等に支障）
> - 抹消費用（登録免許税等）は**お客様負担**

---

### 🔵 ② 完済証明書・残高証明書の取得

| 書類名 | 取得方法 |
|-------|---------|
| **残高証明書** | 取引支店に依頼（有料の場合あり） |
| **返済完了の確認** | 通帳・口座振替の履歴、公庫からの通知書で確認 |
| **抵当権抹消書類の再交付** | 紛失時は公庫支店へ依頼（様式は公庫HPよりDL可） |

[抵当権等抹消書類の再交付案内（公庫公式PDF）](https://www.jfc.go.jp/n/service/pdf/teitouken_guide_250328.pdf)

## 🔍 2026年3月時点の重要トレンド

**① コロナ特別貸付後の新局面（2025年〜）**
コロナ特別貸付が2024年12月末に終了。2025年2月から「危機対応後経営安定貸付」が始動。最大20年・最大20億円で既往コロナ債務の借換が可能となり、月次返済額の大幅軽減が期待される。

**② ネット銀行への口座振替対応拡大**
2023年以降、楽天銀行・PayPay銀行等のネット銀行でも公庫融資の口座振替が可能に。利用者の利便性が向上している。

**③ ゼロゼロ融資返済ピーク対応**
民間・公庫合わせた「ゼロゼロ融資」の返済本格化が進行中。返済が厳しい企業はリスケ・借換制度を積極的に活用することが重要。
