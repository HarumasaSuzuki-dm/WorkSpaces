# 🏦 日本政策金融公庫（国民生活事業）融資の全体フロー【2026年3月最新版】

> **全体の目安：申込から融資実行まで平均 約1ヶ月〜1.5ヶ月**（書類準備期間を含めると2ヶ月程度を想定しておくと安心です）

---

## 📊 全体スケジュール概観

```
事前相談 → 申込・書類準備 → 面談 → 審査 → 結果通知 → 契約 → 融資実行 → 返済
  任意       約1週間        申込後    面談後   面談後     通知後   契約後    開始
                          1〜2週間  1〜2週間  1週間〜   1〜2週間  2〜3営業日
```

## ✅ STEP 1｜事前相談（任意・強く推奨）

| 項目 | 内容 |
|------|------|
| **目安日数** | いつでも（融資希望日の2〜3ヶ月前が理想） |
| **相談方法** | ①来店窓口（全国支店）②電話③**Microsoft Teamsオンライン面談**（🆕） |

### 相談窓口
- **事業資金相談ダイヤル：0120-154-505**（平日9時〜17時／創業者・個人・小規模事業者は〜19時）
- **オンライン創業相談**：Teams招待メールによるビデオ会議（希望日2営業日前16時までに要予約）
- **ビジネスサポートプラザ**：東京・名古屋・大阪（土曜・一部日曜も対応）

### ⚠️ 注意点
- 創業開業予定がある場合は**開業予定日の2〜3ヶ月前**には相談を始めること
- 「相談」と「申込」は別プロセス。相談のみでは申込は完了しない

## ✅ STEP 2｜申込・書類準備（目安：約1週間）

| 項目 | 内容 |
|------|------|
| **目安日数** | 書類準備に3〜7日程度 |
| **申込方法** | **インターネット申込（主流・24時間365日受付）** or 郵送・窓口持参 |

### 主な必要書類（創業者の場合）

| 書類 | 備考 |
|------|------|
| 📄 **創業計画書** | 最重要書類。公庫HPよりDL |
| 🪪 本人確認書類（顔写真付き） | 運転免許証・マイナンバーカード等 |
| 💰 送金先口座の通帳コピー | 電子契約利用者は必須 |
| 🏢 登記簿謄本（法人のみ） | 法務局・登記ねっとで取得 |
| 📋 許認可証・資格証明 | 飲食・美容・宅建業等の許認可業種 |
| 📐 設備の見積書 | 設備資金申請時 |
| 📑 電子契約利用申込書 | 電子契約希望者のみ（印鑑不要） |

### ⚠️ 注意点
- **インターネット申込が現在の主流**（書類はPDF/画像でアップロード）
- 書類の不備・不足が最も審査遅延の原因になる（追加提出のたびに数日〜数週間のロスが発生）
- 取得に時間がかかる書類（登記簿謄本等）は先行して手配すること

---

## ✅ STEP 3｜担当者との面談（申込から約3日〜1週間後に日程調整）

| 項目 | 内容 |
|------|------|
| **連絡まで** | 申込後3日〜1週間程度で公庫から電話が来る |
| **面談形式** | **対面（支店窓口）** or **オンライン（Microsoft Teams）**（🆕） |
| **所要時間** | 30分〜1時間程度 |

### 面談で確認される主なポイント
- 創業の動機・経緯
- 事業内容・強み・差別化ポイント
- 売上予測の数値的根拠
- 自己資金の出所（通帳履歴で証明）
- 返済計画の具体性
- 代表者の業界経験・スキル

### ⚠️ 注意点
- **代理人による面談は原則不可**（本人が必ず出席すること）
- その場で答えられない質問は「後日回答します」でOK。誠実な対応が重要
- 事業計画書の数字を自分の言葉で説明できるよう事前準備を
- オンライン面談を希望する場合は**申込時または事前に担当者に申し出る**こと

---

## ✅ STEP 4｜審査（面談後 約1〜2週間）

| 項目 | 内容 |
|------|------|
| **目安日数** | 面談後1週間〜2週間程度 |
| **審査方法** | 公庫内部での書面審査（信用情報照会・事業計画の精査等） |
| **結果通知** | 電話＋郵送（書面）の両方で通知 |

### 審査で重視される項目
- 💼 事業計画書の実現可能性・根拠の妥当性
- 💴 自己資金の額と出所の明確さ
- 📊 返済能力（収支予測の合理性）
- 🏆 代表者の業界経験・専門知識
- 📋 個人信用情報（延滞・滞納がないこと）
- 🔍 税金・社会保険の納付状況

### ⚠️ 注意点
- 審査中も追加資料の提出依頼が来ることがある→**レスポンスを素早く**
- 審査に落ちた場合、**再申請まで実務上6ヶ月程度の期間**を空けることが推奨される
- 繁忙期（**3月末・12月末**）や大型連休前後は審査が通常より長引きやすい

---

## ✅ STEP 5｜融資決定・結果通知（面談後 約1週間〜10日）

| 項目 | 内容 |
|------|------|
| **通知方法** | 電話連絡＋郵送書面 |
| **否決の場合** | 理由説明はなし。6ヶ月目安で再申請可（計画の見直しが必須） |

## ✅ STEP 6｜契約手続き（結果通知後 約1〜2週間）

**📌 2025年以降の大きな変更点：電子契約サービスが本格普及**

### 【A】従来の郵送契約

| 必要書類 | 内容 |
|----------|------|
| 借用証書 | 借入金額・返済方法・金利等の記載。本人署名＋実印押印 |
| 収入印紙 | 融資額に応じた印紙税額を貼付 |
| 印鑑証明書 | 法人：法務局 / 個人：市区町村役場で取得 |
| 返済口座の通帳コピー | 名義・支店・口座番号が確認できるページ |
| 償還金等の預金口座振替利用届 | 本人名義のもの |

### 【B】🆕 電子契約サービス（国民生活事業）

> **公庫より「記入・押印レス・来店不要・スピーディー」を標榜する新方式**

| 特徴 | 内容 |
|------|------|
| **利用時間** | 月〜土 9:00〜23:00（祝日含む） |
| **手続き方法** | Web上で全て完結。スマートフォンで本人確認書類を撮影・アップロード |
| **電子署名** | SMS認証コードで完了（来店・押印・印紙が不要） |
| **期限** | 融資予定日の**4営業日前**までに電子署名を完了 |
| **非対応の取引** | 担保設定を伴う融資、代表者以外の連帯保証人がいる融資等は利用不可 |

**電子契約の流れ：**
```
利用申込書提出 → ID/PW発行メール受信 
→ メールアドレス到達確認 → 電子契約手続きメール受信 
→ ログイン → 本人確認書類アップロード → SMS認証コード入力 
→ 電子署名完了 → 融資予定日に振込
```

### ⚠️ 注意点
- **ネット銀行（楽天銀行・PayPay銀行等）は送金先口座に指定不可**
- 電子契約希望者は**申込時に利用申込書を同時に提出**しておくとスムーズ
- 書類の誤記・印鑑の不備は手続きを止める最大の原因

## 🆕 2024〜2025年以降の主な制度変更・新機能まとめ

### ① 新創業融資制度の廃止（2024年3月31日）

> 最大の制度変更。「新創業融資制度」が廃止となり、**「新規開業・スタートアップ支援資金」**（2025年3月に改称）として機能が統合・拡充されました。

| 比較項目 | 旧：新創業融資制度 | 新：新規開業・スタートアップ支援資金 |
|----------|-------------------|--------------------------------------|
| 融資上限 | 3,000万円 | **7,200万円**（運転資金4,800万円） |
| 自己資金要件 | 総額の1/10以上 | **撤廃（不要）** |
| 金利優遇 | 制度利率 | **基準金利から一律0.65%引下げ**（雇用拡大は0.9%引下げ） |
| 据置期間 | 最大2年 | **最大5年に延長** |
| 無担保・無保証 | 条件付きで可 | **希望に応じて相談可（より柔軟に）** |

[新規開業・スタートアップ支援資金（日本政策金融公庫公式）](https://www.jfc.go.jp/n/finance/search/01_sinkikaigyou_m.html)

## ⚡ 審査を長引かせる「5大原因」と対策

| 原因 | 対策 |
|------|------|
| ①書類の不備・不足 | 専門家（税理士等）に事前にダブルチェックを依頼 |
| ②事業計画書の根拠が不十分 | 市場データ・競合調査・数値根拠を必ず添付 |
| ③自己資金の出所が不明瞭 | 通帳履歴を提示し、タンス預金・カードローンは自己資金として認められない |
| ④担当者との連携が遅い | 着信・メール確認を徹底。質問には即日〜翌日中に回答 |
| ⑤繁忙期・大型連休 | **3月末・12月は申込が集中**。なるべく避けるか余裕を持ったスケジュールで申込む |

## 📅 全体スケジュール早見表

```
Week 1-2  ：事前相談 → 書類収集・事業計画書作成
Week 3    ：インターネット申込（書類アップロード）
Week 3-4  ：担当者より面談日程の連絡 → 面談実施
Week 4-6  ：審査（内部審査）
Week 6    ：結果通知（電話＋郵送）
Week 6-8  ：契約手続き（電子契約 or 郵送）
Week 8    ：融資実行（口座振込）

```

## 💡 まとめ・アドバイス

- **開業を急ぐ場合は、開業予定日の2〜3ヶ月前には動き出す**のが鉄則
- 電子契約を活用することで**契約ステップを最大1〜2週間短縮**できる
- 自己資金要件は撤廃されたものの、**自己資金が多いほど審査に有利**なことに変わりはない
- 2025年3月に「新規開業資金」から**「新規開業・スタートアップ支援資金」に改称**されているため、古い情報に注意
- 不安な場合は**認定支援機関（税理士・中小企業診断士等）に相談**することで、書類の質が上がり審査通過率も高まる
