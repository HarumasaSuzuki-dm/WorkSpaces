## 📋 まず押さえておくべき最重要制度変更

> **⚠️ 制度変更アラート（2024〜2025年）**
> - **2024年3月**：「新創業融資制度」廃止 → 「新規開業・スタートアップ支援資金」に統合
> - **2025年3月**：「新規開業資金」から「新規開業・スタートアップ支援資金」に改名
> - **自己資金10%要件は撤廃済み**（2024年4月以降）
> - 担保・保証人なしでも相談可能な体制に

[新規開業・スタートアップ支援資金（公庫公式）](https://www.jfc.go.jp/n/finance/search/01_sinkikaigyou_m.html)の概要は以下の通りです。

| 項目 | 内容 |
|---|---|
| 対象者 | 新たに事業を始める方 or 開業後おおむね7年以内 |
| 融資限度額 | 7,200万円（うち運転資金4,800万円） |
| 設備資金返済期間 | **20年以内**（据置5年以内） |
| 運転資金返済期間 | **10年以内**（据置5年以内）※旧制度より延長 |
| 金利優遇 | 創業初期は一律**▲0.65%** 引下げ |
| 担保・保証人 | 希望に応じて相談可（無担保・無保証も対応） |

## 1️⃣ 事業計画書の審査ポイント ── 「何をどう書くか」

### 📌 審査の根本的な考え方：「減点方式」で見られる

公庫の審査官は「この融資を通してよいか」ではなく、「融資を通せない理由があるか」という視点でチェックします。したがって、**疑念を持たれない計画書**が合格ラインです。

---

### ✅ 高評価される事業計画書の5要素

**① 売上予測は「根拠ある数字」で構成する**

最も重視される項目です。「月商300万円を目指す」という一文では不十分で、以下の計算式を明示することが必須です。

```
例：飲食店の場合
客単価 2,500円 × 1日来客数 40人 × 営業日 25日 = 月商 250万円
→ 近隣競合店の客数を実地調査した数値と比較し、控えめに設定
```

過大な予測は信憑性を損ない逆効果。**最悪シナリオでも返済できる**数値設定が評価される。

**② 競合分析と差別化を具体的に記載する**

「競合はあまりいません」は最悪の回答。市場を理解していないと判断されます。

```
✅ 良い例：「半径500m以内に同業3店舗あり。当店は〇〇という特徴で
         差別化を図り、△△層を主ターゲットにすることで月商□□万円を見込む」
```

**③ 資金使途と資金計画を一致させる**

「何に・いくら使うか」が設備費・運転資金・各種費用ごとに明確であることが必要です。見積書の添付も効果的です。

**④ 経費の内訳を正確に記載する**

家賃・人件費・仕入原価・光熱費・広告費などを月次で詳細に記載し、**損益分岐点売上高**を明示すると担当者の信頼を得やすい。

**⑤ 代表者の「経験・スキル」を職歴と紐付ける**

事業計画書に経営者のキャリアとの関連性を明記することで、「この人は事業を知っている」と評価されます。

> 「前職で10年間 × 業界経験を有し、顧客基盤・仕入れルートはすでに確保済み」などの記載は強力なアピールになります。

---

### ❌ よくある落とし穴

| NG例 | なぜダメか |
|---|---|
| 売上を希望ベースで設定 | 根拠のない楽観予測は信憑性ゼロ |
| 競合ゼロと記載 | 市場調査不足・無知と判断される |
| 経費の記載が「その他」で誤魔化す | 資金使途の不透明感を与える |
| 計画書を専門家に丸投げし内容を把握していない | 面談で矛盾が発覚し致命的 |

[参考：審査に通る事業計画書の書き方](https://maeda-zeirishi.com/blog/shikinguri/sogyo-yushi-jigyo-keikaku/)

## 2️⃣ 面談（ヒアリング）での頻出質問と望ましい回答

面談は通常 **60〜90分程度**。担当者は「社内稟議を通す説明材料」を集めるために質問します。つまり**担当者はあなたの味方**です。

---

### 🎯 必ず聞かれる6大質問と回答の方向性

| # | 質問内容 | 担当者が見ているポイント | 望ましい回答の方向性 |
|---|---|---|---|
| ① | **なぜこの事業を始めるのですか？** | 熱意・論理性・実現可能性 | PREP法（結論→理由→具体例→結論）＋自身の実体験を交える |
| ② | **売上・経費の根拠は？** | 返済能力・計画の現実性 | 計算式を明示。「客単価×来客数×営業日」など数値ベースで説明 |
| ③| **自己資金はどう準備しましたか？** | 信頼性・資金管理能力 | 通帳の積立履歴で証明。見せ金は絶対NG |
| ④ | **競合他社との差別化は？** | 市場理解・独自性 | 「競合はいません」はNG。具体的な比較表＋自社の強みを示す |
| ⑤ | **事業のリスクとその対策は？** | 現実的な思考・準備力 | 「リスクはありません」はNG。代替策・数字で対応策を提示 |
| ⑥ | **融資額の使い道は？** | 資金使途の明確性 | 「〇〇費に××万円、△△費に□□万円」と項目別に明確に説明 |

---

### 💡 面談成功の5か条

1. **書類と発言の整合性**：計画書を専門家に作成してもらった場合でも、自分の言葉で説明できること
2. **専門用語を使わない**：担当者が家族に説明できるレベルの言葉で話す
3. **「いくら借りられますか？」はNG**：資金使途と金額の根拠をセットで伝える
4. **厳しい質問にも冷静に**：感情的にならず対立姿勢を見せない
5. **数字だけでなくストーリーで語る**：「なぜこの事業か」という想いが担当者の心を動かす

---

### 🚨 面談でのNG行動

```
❌ 事業内容を「よくわかりません」と答える
❌ 計画書の数値と説明が食い違う
❌ リスクを一切想定していない
❌ 税金・社会保険料の滞納を隠す
❌ 自己資金の出所を説明できない
```

## 3️⃣ 審査通過率を高めるための準備事項

### 📊 創業融資の審査通過率の実態

```
一般申請（自己申請）：通過率 約50〜60%
税理士など専門家サポートあり：通過率 約80〜90%
```

---

### 💰 自己資金比率の目安（2026年現在）

> 旧制度では「1/10（10%）以上」の自己資金要件がありましたが、**2024年以降は撤廃**されています。ただし自己資金は依然として審査で重視されます。

| 自己資金比率 | 審査への影響 |
|---|---|
| **0%（ゼロ）** | 法律上申請可能だが通過は非常に困難 |
| **10〜20%** | 最低ライン。計画の精度が相当高くないと厳しい |
| **24%前後** | **公庫公式統計の平均値**（実際の採択者平均） |
| **30%以上** | **推奨ライン**（コンサルタント各社の目安） |
| **50%以上** | 融資希望額に対する信頼性が大幅に向上 |

> 自己資金の「見せ金」（他人から一時的に借りた資金）は通帳履歴で判明します。**積立の過程を示せる資金**が有効です。

---

### 📋 通過率アップのための準備チェックリスト

**【信用情報の整備】**
- [ ] クレジットカード・ローンの延滞がないか確認（信用情報3機関で開示請求）
- [ ] 税金（所得税・住民税・消費税）の滞納がないこと
- [ ] 社会保険料・公共料金の滞納がないこと

**【事業計画の精度】**
- [ ] 売上予測に計算根拠を明示
- [ ] 損益分岐点売上高を試算済み
- [ ] 資金繰り表（月次）を作成
- [ ] 競合分析・差別化要因を記載

**【その他の強化ポイント】**
- [ ] 業種・業界経験年数を職歴として明記
- [ ] 認定支援機関（税理士等）の指導・助言を受ける → **金利▲優遇の対象にも**
- [ ] 開業前準備費用（研修費・資格取得費等）を自己資金に計上

## 4️⃣ 近年（2024〜2026年）の審査傾向の変化

### 🔄 制度的な変化（2024〜2026年）

```
2024年3月：新創業融資制度 廃止
     ↓
2024年4月：新規開業・スタートアップ支援資金 強化スタート
     ↓
2025年3月：「新規開業資金」から改名・制度整理
     ↓
2026年現在：無担保・無保証人対応が標準化、返済期間延長済み
```

---

### 📈 審査スタンスの変化：「創業支援の積極化」と「事業計画の精度要求の高まり」が同時進行

**① 国の方針として「創業支援の積極化」が継続**

政府が「開業率の向上」を政策目標に掲げており、公庫も創業融資を積極的に実行する流れが続いています。特に以下の属性は優遇制度あり：

| 対象者 | 優遇内容 |
|---|---|
| 女性・35歳未満・55歳以上 | 特別利率A（低金利）適用 |
| 廃業歴ありの再チャレンジ者 | 運転資金返済期間15年以内 |
| 認定支援機関の指導を受けている方 | 特別利率A適用 |

**② 「条件変更・代位弁済の増加」を背景に、与信判断が高度化**

公庫2024年度業績評価報告書では「**与信判断の難易度が高まる中での的確な貸付審査**」「**条件変更残高や代位弁済が増加傾向**」が言及されており、審査官の目が厳しくなっていることを示唆しています。

```
→ 現実的な返済シミュレーションを事業計画に盛り込むことが
  これまで以上に重要になっています
```

**③ 不動産投資・一部業種への慎重化**

現場の税理士・コンサルタントの声として、**不動産投資や一部ジャンルへの融資判断が慎重**になっている傾向があります（通常の創業融資への影響は限定的）。

**④ デジタル申請・オンライン面談の普及**

2024〜2026年にかけてオンライン申請・面談対応が標準化。書類のデジタル提出が一般的になり、**ネットバンク口座の入出金履歴PDF提出**も認められています（Excelは不可）。

## 📌 コンサルタントからの総合アドバイス

> **「審査は総合点です」**

公庫の審査は、自己資金・信用情報・事業計画・業種経験・面談の印象という5つの要素を総合的に判断します。一つが弱くても、他の要素で補完できることが多いです。

```
弱点の補い方の例：
✅ 自己資金が少ない → 業種経験・事業計画の精度を上げる
✅ 業種経験が短い → 自己資金比率を高め、研修・資格で補う
✅ 事業計画に不安 → 認定支援機関（税理士等）のサポートを受ける
                    （金利優遇にも繋がる）
```

**最も重要な一言：「担当者が社内稟議を通せる材料を提供すること」が審査通過の本質です。**
