#### 事例A｜株式会社サニーサイドアップ「32の制度」
**PR・スポーツ・エンタメ系 / 約100〜200名　→ Shape Fitへの参考度：◎**
🔗 https://www.ssu.co.jp/corporate/32rule/

**何をやっていたか**
スローガン「たのしいさわぎをおこしたい」を、制度の名前・金額・条件のすべてに刻み込んだ32種類の福利厚生。制度を使うたびに会社の哲学を身体で感じられる設計。

| 制度名 | 具体的な内容 |
|--------|-------------|
| **幸せは歩いてこない** | 月平均1万歩達成で¥3,200支給（"32"=社名にかけた金額） |
| **目指せ！A身体** | 健康診断でA判定を取ると¥32,000支給＋提携ジムを¥1,000で利用可 |
| **寝る子は育つ** | 月平均7時間以上の睡眠を記録したスタッフに¥3,200支給 |
| **スマイル給・MVT表彰** | 社員同士が「よいところ」を伝え合い、優秀者は賞与上乗せ |
| **たのしいさわぎ創造支援** | ライブ・イベントの体験費用を一部補助（感性インプットの推奨） |
| **失恋休暇・恋愛勝負休暇** | 人生の感情にも寄り添う独自の特別休暇 |

**なぜ機能するか**
金額（3,200円・32,000円）が社名にかけられており、制度の隅々まで哲学が宿っている。制度自体がブランドとなり、採用にも機能している。

**Shape Fitで出せそうなこと**

| アイデア | 内容 |
|---------|------|
| **「Beなヒト手当」** | 月次で自分が設定した行動変容チャレンジ（トレーニング・食習慣等）を達成したスタッフに報奨金 |
| **「Do記録ボーナス」** | トレーニング・健康習慣の継続記録を会社と共有し、継続達成で小額インセンティブ |
| **「Have体験補助」** | SFF出場・大会エントリー費・フィットネス体験に使えるバウチャーを月次支給 |
| **制度名にShape Fitを刻む** | 金額・名前・条件のどこかにBe→Do→Haveの哲学が見えるよう設計する |

#### 事例B｜ピアボーナス制度（Unipos型）
**複数スタートアップで導入 / 月額数万円〜　→ Shape Fitへの参考度：◎**
🔗 https://media.unipos.me/peer-bonus-demerit

**何をやっていたか**
スタッフ同士が「感謝・称賛のメッセージ」と「少額の報奨ポイント」を相互に贈り合う仕組み。

- 毎月1人あたり¥1,000相当のポイントを付与し、誰かに必ず贈る
- 贈る側は「なぜ贈るか」のメッセージを添える（メッセージなしは不可）
- 全員の贈り合いが社内で可視化され、称賛文化が根付く
- 導入企業では社員満足度85%超の実績あり

**なぜ機能するか**
人事評価では拾えない「縁の下の貢献」にスポットが当たる。正社員・業務委託が混在する組織では関係性が希薄になりやすいが、テキストで感謝を可視化することでチームの一体感が生まれやすい。

**Shape Fitで出せそうなこと**

| アイデア | 内容 | コスト |
|---------|------|--------|
| **「Shape Fit GIVE」チャンネル** | Slackに専用チャンネルを設置。毎週金曜に感謝メッセージを投稿する文化をつくる | 0円 |
| **月次「Be賞」表彰** | もっとも多くGIVEされたスタッフをMTGで表彰 | ほぼ0円 |
| **業委スタッフにも全員適用** | 雇用形態に関係なく全員参加にすることで組織の一体感を実現 | ― |

#### 事例C｜株式会社タニタ「日本活性化プロジェクト」
**健康機器メーカー / 約500名　→ 業委混在組織の設計思想として参考になる。参考度：○**
🔗 https://note.com/newwork_newlife/n/nbad0acad10ce

**何をやっていたか**
「自律的に健康的に生きる」という哲学を、働き方にも適用。希望する正社員が「雇用契約→業務委託（個人事業主）」へ任意で転換できる制度。

- 移行後の報酬＝給与＋残業代＋社会保険会社負担分＋退職金積立分を合算（手取りは増える設計）
- 移行前の担当業務をそのまま「基本業務」として委託し、新たな仕事は成果報酬で上積み
- 契約は原則3年・1年ごとに業務内容と報酬を見直し
- 会社と個人が「対等な事業者」として関係を再構築することで双方のモチベーションが向上

**なぜ機能するか**
報酬設計が透明かつ元の待遇より有利なため移行への不安が少ない。「主体性を持って働く」という哲学と業務委託化が完全に一致している点が鍵。

**Shape Fitで出せそうなこと**（現状の業委設計の思想的参考として）

| アイデア | 内容 |
|---------|------|
| **「Shape Fitパス」全員付与** | 業委含む全スタッフにジム・オンラインコーチングの無料利用権を付与（原価ほぼ0） |
| **業委向け報酬の透明化** | 基本報酬（担当クラス数保証）＋成果報酬（継続率・紹介数連動）の構造を明示 |
| **コミュニティ参加権の付与** | 社内勉強会・SFF等への無償スタッフ参加権を業委にも付与し、哲学の体現者として巻き込む |

#### ⚠️ 事例D｜NOT A HOTEL「ワンチーム評価（OKR連動賞与）」
**高級別荘・会員権サービス / 50〜100名規模　→ 賞与設計の考え方は参考になるが、現フェーズでの直接導入は要検討。参考度：△**
🔗 https://note.com/saimaru/n/n7f62f4a98c5c

**何をやっていたか**
個人評価を完全廃止し、全社OKRの達成度だけで全員の賞与を一律決定するシンプルな仕組み。

```
年度初めに全社で「カンパニーベット（会社として賭けること）」を1本設定

達成（OKR 90%以上） → 全員に月給の20%賞与
部分達成（70%以上） → 全員に月給の10%賞与
未達（70%未満）     → 全員に月給の5%賞与（ゼロにしないことが重要）
```

- 制度導入後、全ての期で達成・20%支給の実績あり
- 「月給×10 or 20%」と誰でも即計算できるシンプルさが心理的安全性につながる

**懸念点（Shape Fit視点）**
現状6名規模では個別事情（入社タイミング・産休等）への対応が難しく、そのまま導入するには少し早い可能性がある。「全社KPIで賞与の原資を決め、配分は別途調整する」という**部分採用**が現実的な落とし所。
