# クリエイティブディレクター採用：進め方・要件まとめ
**作成日：2025年4月2日（木）**
**目的：採用の進め方・要件について合意する**

---

## 1. なぜ今、採用するのか

Shape Fitには「伝えたい思い」がある。
だが、それをデザインに落とし込む人が社内にいない。

グラフィック・映像・Web・空間デザインなど、
幅広い領域で**Shape Fitの世界観を表現してくれる人**が必要。

---

## 2. どんな人を採りたいか

### 役割
- Shape Fitの思いをヒアリングして、デザインの言葉に翻訳する
- 外部クリエイター（デザイナー・映像など）をディレクションする
- 必要に応じて、自分でも手を動かせる

### 重視すること（ソフト面）

| 資質 | 具体的に言うと |
|---|---|
| ヒアリング力 | 曖昧な思いや言葉を、デザイン要件に整理できる |
| 文脈理解力 | Shape Fitの世界観を短期間でつかめる |
| 変化への柔軟性 | 要件が変わることを前向きに受け入れ、対応できる |
| パートナー感 | 受注者ではなく、経営と対話できるスタンス |

### ハードスキルについて
- 選考要件には含める（最低ラインの確認として）
- ただし、評価のメインはソフト面

### 雇用形態
- **正社員・フルコミット**
- **試用期間：1ヶ月**（詳細は後述）
