### 議題④ 4月末打合せのゴールとその後の流れ（10分）

`全体像を共有`

**4月末打合せで目指す状態**

| 理想 | 最低ライン |
| --- | --- |
| 施策内容・契約条件に合意 → 5月から実行開始 | 施策の方向性に合意 → 契約条件は次回確定 |

**打合せの流れ（当日イメージ）**

```
① 前回の振り返り（5分）
② なぜ売上に繋がっていないかの診断を共有（10分）
③ 施策ロードマップの提案（15分）
④ 施策後の売上シミュレーション・契約条件の提示（10分）
⑤ 合意・次のアクション確認（5分）
```

**打合せ後の動き（5月以降）**

合意できた場合：

```
5月前半   計測の仕組みをつくる（まとばっち主導）
5月後半   施策の実行開始（おしりちゃん・まとばっち）
6月〜     月次レポート ＋ 定例MTG（月1回）
7〜8月    コンテンツ施策の実行（年間スケジュール連携）
3ヶ月後   施策の効果検証 → 契約条件の再評価
```

合意できなかった場合：

```
5月上旬   条件を再提案
5月中旬   最終合意 or 撤退判断（なーすけさん決裁）
```
