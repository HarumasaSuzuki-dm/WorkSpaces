**キーメッセージ**：

**構成要素・レイアウト**：

- 中央に矢印で「ここが問題」と強調

**具体的な数値・文言**：

```
【ステップ別の課題度】
① インプレ → ② 接触          ：低（業界上位水準）
② 接触 → ③ プロフ訪問        ：中〜高（CTA設計の改善余地）
③ プロフ訪問 → ④ EC流入      ：高（リンク・導線が最適化されていない可能性）
④ EC流入 → ⑤ 購買            ：最高（LP・商品訴求・計測すべて未整備）

【課題カテゴリと仮説】
・CTA設計不足：動画内のCTA頻度・タイミングが最適化されていない

・商品訴求の薄さ：視聴エンタメ寄りで購買起点になりにくい

```

> 「ファネルを見ると、ボトルネックは明らかです。①と②、つまり『インプレッション』と『接触』までの数字は圧倒的ですが、③の『プロフィール訪問』から先で数字が急落しています。
> これが意味するのは、コンテンツ自体、つまりリーチを生み出す力には問題がないということ。課題は『視聴した人を、どう商品ページに連れて行くか』の動線設計にあります。
