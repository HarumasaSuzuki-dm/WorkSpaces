## フロー全体図

```
候補者決定
　↓
【Phase 1】締結
　↓
【Phase 2】受入れ・オンボーディング　← 稼働開始日
　↓
【Phase 3】報酬振り込み　← 毎月
　↓
【Phase 4】更新 or 解約
```

> 雇用形態（業務委託 / パートタイム / 正規雇用）による差分は各フェーズ内に記載しています。
詳細フローは [採用・受入れフロー マニュアル](%E6%8E%A1%E7%94%A8%E3%83%BB%E5%8F%97%E5%85%A5%E3%82%8C%20%E6%A5%AD%E5%8B%99%E3%83%9E%E3%83%8B%E3%83%A5%E3%82%A2%E3%83%AB/%E6%8E%A1%E7%94%A8%E3%83%BB%E5%8F%97%E5%85%A5%E3%82%8C%E3%83%95%E3%83%AD%E3%83%BC%20%E3%83%9E%E3%83%8B%E3%83%A5%E3%82%A2%E3%83%AB%2033c163ffb4918019bd3bfdad712045f2.md) を参照してください。
>

## Phase 1 ｜ 締結

**担当者：採用担当者 → 上長（最終確認）**

- [ ]  雇用形態に応じた契約書ひな形でドラフトを作成し、Google Driveに保存
- [ ]  上長へ内容確認・承認を依頼
- [ ]  上長がクラウドサインの電子締結リンクを作成
- [ ]  候補者へクラウドサインリンクを送付・締結
- [ ]  [請求書フォーマット_完全版](https://docs.google.com/spreadsheets/d/1ITJuA333Wv5T4pAvUtpAXNDSvNgDTiCiHbVUuhs0804/edit?gid=1170628518#gid=1170628518) に契約情報を登録

> 雇用形態別の必須確認項目など詳細は [採用・受入れフロー マニュアル](%E6%8E%A1%E7%94%A8%E3%83%BB%E5%8F%97%E5%85%A5%E3%82%8C%20%E6%A5%AD%E5%8B%99%E3%83%9E%E3%83%8B%E3%83%A5%E3%82%A2%E3%83%AB/%E6%8E%A1%E7%94%A8%E3%83%BB%E5%8F%97%E5%85%A5%E3%82%8C%E3%83%95%E3%83%AD%E3%83%BC%20%E3%83%9E%E3%83%8B%E3%83%A5%E3%82%A2%E3%83%AB%2033c163ffb4918019bd3bfdad712045f2.md) を参照。

## Phase 2 ｜ 受入れ・オンボーディング

**担当者：採用担当者**　**タイミング：稼働開始日**

### 前日までにやること

- [ ]  [オンボーディング情報シート](%E6%8E%A1%E7%94%A8%E3%83%BB%E5%8F%97%E5%85%A5%E3%82%8C%20%E6%A5%AD%E5%8B%99%E3%83%9E%E3%83%8B%E3%83%A5%E3%82%A2%E3%83%AB/%E3%83%89%E3%82%AD%E3%83%A5%E3%83%A1%E3%83%B3%E3%83%88%E4%B8%80%E8%A6%A7/%E3%80%90%E3%80%87%E3%80%87%E6%A7%98%E3%80%91%E3%82%AA%E3%83%B3%E3%83%9C%E3%83%BC%E3%83%87%E3%82%A3%E3%83%B3%E3%82%B0%E6%83%85%E5%A0%B1%E3%82%B7%E3%83%BC%E3%83%88%20341163ffb49180a4a4f3ee07d3f9b1af.md) を複製して記入を完成させる
- [ ]  [ツールアクセス管理表](https://docs.google.com/spreadsheets/d/1MLbD82IAzkonQ7PldowX3LGHNuBAA3is1vq83Ub69rY/edit?gid=55416742#gid=55416742) に該当者の行を記入する
- [ ]  [キックオフMTG資料](%E6%8E%A1%E7%94%A8%E3%83%BB%E5%8F%97%E5%85%A5%E3%82%8C%20%E6%A5%AD%E5%8B%99%E3%83%9E%E3%83%8B%E3%83%A5%E3%82%A2%E3%83%AB/%E3%83%89%E3%82%AD%E3%83%A5%E3%83%A1%E3%83%B3%E3%83%88%E4%B8%80%E8%A6%A7/%E3%80%90%E3%80%87%E3%80%87%E6%A7%98%E3%80%91%E3%82%AD%E3%83%83%E3%82%AF%E3%82%AA%E3%83%95MTG%20341163ffb491806c97f3dd3d36c96068.md) を複製して当日の内容に合わせて編集する

### 当日やること（キックオフMTG内で実施）

- [ ]  Slack・Canva・Notionへの招待を送信し、本人のログインを確認する
- [ ]  [ツールアクセス管理表](https://docs.google.com/spreadsheets/d/1MLbD82IAzkonQ7PldowX3LGHNuBAA3is1vq83Ub69rY/edit?gid=55416742#gid=55416742) に付与日・付与担当者を記録する
- [ ]  請求書の提出方法・支払いサイト（月末締め 翌月末払い）を説明する
- [ ]  秘密保持・情報管理ルールを説明する

> 雇用形態別の追加対応など詳細は [採用・受入れフロー マニュアル](%E6%8E%A1%E7%94%A8%E3%83%BB%E5%8F%97%E5%85%A5%E3%82%8C%20%E6%A5%AD%E5%8B%99%E3%83%9E%E3%83%8B%E3%83%A5%E3%82%A2%E3%83%AB/%E6%8E%A1%E7%94%A8%E3%83%BB%E5%8F%97%E5%85%A5%E3%82%8C%E3%83%95%E3%83%AD%E3%83%BC%20%E3%83%9E%E3%83%8B%E3%83%A5%E3%82%A2%E3%83%AB%2033c163ffb4918019bd3bfdad712045f2.md) を参照。

## Phase 3 ｜ 報酬振り込み

**担当者：採用担当者 → アールハープさん**　**サイクル：月末締め 翌月末払い**

### 業務委託の場合

- [ ]  月末までに業務委託者から請求書を受け取る
- [ ]  [請求書フォーマット_完全版](https://docs.google.com/spreadsheets/d/1ITJuA333Wv5T4pAvUtpAXNDSvNgDTiCiHbVUuhs0804/edit?gid=1170628518#gid=1170628518)「使い方ガイド」のチェックリストに沿って内容確認
- [ ]  管理シートに請求情報を記入・請求書のGoogle DriveリンクをURL欄に貼付
- [ ]  アールハープさんへSlack連絡（必要に応じてしおんさんをメールCC）

### パートタイム・正規雇用の場合

- [ ]  月末までに本人の勤怠を確認・承認する
- [ ]  アールハープさんへSlack連絡（しおんさんをメールCC）し、給与計算・振込を依頼する

> しおんさんへのエスカレーション条件など詳細は [採用・受入れフロー マニュアル](%E6%8E%A1%E7%94%A8%E3%83%BB%E5%8F%97%E5%85%A5%E3%82%8C%20%E6%A5%AD%E5%8B%99%E3%83%9E%E3%83%8B%E3%83%A5%E3%82%A2%E3%83%AB/%E6%8E%A1%E7%94%A8%E3%83%BB%E5%8F%97%E5%85%A5%E3%82%8C%E3%83%95%E3%83%AD%E3%83%BC%20%E3%83%9E%E3%83%8B%E3%83%A5%E3%82%A2%E3%83%AB%2033c163ffb4918019bd3bfdad712045f2.md) を参照。
>

## Phase 4 ｜ 更新・解約

**担当者：採用担当者 → 上長（承認）**

> 承認期限は **契約終了37日前**（解約通知30日前のさらに1週間前）。期限を必ず確認してください。
> 

### 更新の場合

- [ ]  [更新・解約起案フォーマット](%E6%8E%A1%E7%94%A8%E3%83%BB%E5%8F%97%E5%85%A5%E3%82%8C%20%E6%A5%AD%E5%8B%99%E3%83%9E%E3%83%8B%E3%83%A5%E3%82%A2%E3%83%AB/%E3%83%89%E3%82%AD%E3%83%A5%E3%83%A1%E3%83%B3%E3%83%88%E4%B8%80%E8%A6%A7/%E3%80%90%E3%80%87%E3%80%87%E6%A7%98%E3%80%91%E6%9B%B4%E6%96%B0%E3%83%BB%E8%A7%A3%E7%B4%84%E8%B5%B7%E6%A1%88%E3%83%95%E3%82%A9%E3%83%BC%E3%83%9E%E3%83%83%E3%83%88%20341163ffb491800f85ffd9a870c1cb58.md)を複製して記入する（根拠・理由の明記必須）
- [ ]  上長へ承認を依頼する（期限：契約終了37日前）
- [ ]  条件変更がある場合は新契約書を作成しクラウドサインで再締結
- [ ]  [請求書フォーマット_完全版](https://docs.google.com/spreadsheets/d/1ITJuA333Wv5T4pAvUtpAXNDSvNgDTiCiHbVUuhs0804/edit?gid=1170628518#gid=1170628518) の契約情報を更新する

### 解約の場合

- [ ]  [更新・解約起案フォーマット](%E6%8E%A1%E7%94%A8%E3%83%BB%E5%8F%97%E5%85%A5%E3%82%8C%20%E6%A5%AD%E5%8B%99%E3%83%9E%E3%83%8B%E3%83%A5%E3%82%A2%E3%83%AB/%E3%83%89%E3%82%AD%E3%83%A5%E3%83%A1%E3%83%B3%E3%83%88%E4%B8%80%E8%A6%A7/%E3%80%90%E3%80%87%E3%80%87%E6%A7%98%E3%80%91%E6%9B%B4%E6%96%B0%E3%83%BB%E8%A7%A3%E7%B4%84%E8%B5%B7%E6%A1%88%E3%83%95%E3%82%A9%E3%83%BC%E3%83%9E%E3%83%83%E3%83%88%20341163ffb491800f85ffd9a870c1cb58.md)を複製して記入する（根拠・引き継ぎ方針の明記必須）
- [ ]  上長へ承認を依頼する（期限：契約終了37日前）
- [ ]  契約終了30日前までに対象者へ解約通知をメール送付
- [ ]  [ツールアクセス管理表](https://docs.google.com/spreadsheets/d/1MLbD82IAzkonQ7PldowX3LGHNuBAA3is1vq83Ub69rY/edit?gid=55416742#gid=55416742) に解約・終了日を記入する
- [ ]  契約終了日に全ツールのアクセスを剥奪し、管理表の「アクセス剥奪」を「剥奪済」に更新する
- [ ]  [請求書フォーマット_完全版](https://docs.google.com/spreadsheets/d/1ITJuA333Wv5T4pAvUtpAXNDSvNgDTiCiHbVUuhs0804/edit?gid=1170628518#gid=1170628518) の契約ステータスを「契約終了」に更新する

## 基本ルール集

| ルール | 内容 |
| --- | --- |
| 支払いサイト | 月末締め 翌月末払い（全雇用形態統一） |
| 解約通知期限 | 契約終了の30日前・メールで実施 |
| 更新/解約承認期限 | 契約終了37日前までに上長承認を完了 |
| 契約書保管 | Google Drive → クラウドサインで締結 |
| アールハープさんへの連絡方法 | Slack連絡 ＋ 必要に応じてメールにしおんさんをCC |
| ツールアクセス付与タイミング | 稼働開始日（キックオフMTG内で実施） |
| ツールアクセス剥奪タイミング | 契約終了日（解約時に必ず実施） |

## ステークホルダー別役割

| 役割 | 担当 | 主な責務 |
| --- | --- | --- |
| 採用担当者 | 各BU担当者 | 業務委託者との一次窓口・契約書作成・請求書受領・ツールアクセス管理・起案 |
| 上長 | 採用担当者の直属上長 | 契約書の最終確認・クラウドサイン作成・更新/解約起案の承認 |
| しおんさん | 経理内部 | 特定業務種別の請求確認・例外・トラブル対応 |
| アールハープさん | 外部経理 | 振込執行・給与計算 |
