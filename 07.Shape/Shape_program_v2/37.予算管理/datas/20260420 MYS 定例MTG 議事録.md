## 3. 鈴木マター：業務委託契約の整備

### 本質①：契約形態

| 項目 | 決定内容 |
| --- | --- |
| **契約主体** | **法人契約**（個人でも柔軟に対応可だが御社のやりやすい形で） |
| **報酬形態** | **月額固定30万＋売上25%（成果報酬／圭佑のマーケ費込み）**／A案を第一希望 |
| **契約期間** | **1年契約ベース**（成果報酬の率の見直しタイミング次第では半年契約も選択肢） |
| **インボイス** | 登録済み |

### たくちゃんの設計思想（共有）

- 「協力会社という感覚ではなく、**同じチーム・同じ会社の中として**やっていきたい」
- **売上指標だけの報酬にすると、"売ること"が前に出て事業を良くする判断が抜けやすくなる**
- 営業代行・営業支援会社が数字に追われ、必要以上の集客・効果的でないタイミングでの広告費投下をしがちになるリスクがある
- だから**固定費があった方が、一気通貫で営業ルールを作る上で良い判断ができる**
- 売上25%は役割・責任範囲に照らせば妥当、むしろ安いぐらい（4人以上が関わる）
- **55〜60%を超えて支払ってしまうと、ShapeFit側が成り立たなくなる**——この上限ラインは意識する

### ぼんちゃんからの提案（合意済み）

- 成果報酬率は**段階的にアップ**させる設計にしたい
    - 例：売上のベースラインを超えたら成果報酬率を引き上げる
    - → 設計上OK。売上シミュレーションを踏まえて具体数字は後日連携
- 契約期間は報酬率見直しのタイミング次第。契約書ドラフト時にあわせて提案

### 今後の進め方

- 売上シミュレーション・報酬率の段階設計・契約期間 → ぼんちゃんが箇条書きで整理 → たくちゃんに共有 → 契約書ドラフトへ

## 6. ペルソナ・売ってはいけない人の言語化（壁打ち）

※ ぼんちゃんは5〜6節以降、一旦退出。本セクションは**なーすけさん×たくちゃん×みしぴー**の壁打ち記録。

### 6-1. なーすけさんが準備してきた「売ってはいけない人」の言語化

> **一言でいうと：「自分の内側を変えるつもりじゃなく、外側を変えようとしている人」**
> 

### 具体像

- HowTo狙い・最短ルート狙いで来る人
    - 「なーすけさんみたいな体を作りたい」「なーすけさんの筋トレ方法を知りたい」「なーすけさんの食事方法を知りたい」
    - **自分は既にマインドセットできている前提で、知識だけ取りに来る層**
- 既にフィットネスを楽しんで習慣化できている人（筋トレ行く習慣がすでにある層）
- 過去、実際にパートナートレーニングで出会った：「結局、なーすけさんってどうやって三頭筋鍛えてるんですか」→ 20分話しても結局トレーニングの話に戻ってくる
    - → **「僕じゃなくてもよくない？」感を強く持ったタイプ**

### 補助的な切り口：優先順位・心理的安全性

- **仕事・お金・生活の心理的安全性が低い人はフィットネスにコミットできない**
- 「仕事も変わります」を訴求するよりも、**ある程度仕事・お金・生活が安定している人の方が価値提供できる**
- ターゲットは「生活は困らないが、自分の望む未来ではない」という人（＝4/13で言語化したペルソナとも整合）

### 6-2. みしぴーからの問い：需要と供給はマッチしているのでは？

- グルメ系SNSコンサルでも似た層はいる。知識狙いの層は**受注自体はできる**。
- 「なーすけさんに知識を教わりたい」と「なーすけさんの知識を提供する（実働はバディや会社）」は**需要と供給として成立している**のでは？
- → 「会社として教育したバディがその知識を教える」で成立するなら、**売ってもよい層なのでは**

### 6-3. たくちゃんの整理：売ってよいが、サービスの主目的ではない

> **「見た目を良くしたい」と「人生を良くしたい」のどちらが先行するか** → なーすけさん即答：**「人生を良くしたいが先行する」**
> 

これを前提にすると：

- **人生を最高にするために体を変えるのは最短最速の手段**（コントロール100%可能）
- **知識提供はプログラム内に必然的に含まれる**——動画講座でも伴走者の指導でも、そこに需要はある
- **ただし、知識そのものを主商品にはしない**。「見た目を良くするための知識は本質ではない」と明確に打ち出す
- 営業・教育フェーズで「**なぜ今あなたの元に知識がないのか**」と問い返す → マインドに話を移す
- 結論：**「売ってはいけない」ではなく「売りたくない」寄り。なーすけさん自身がそこに価値を感じづらいから**

### 6-4. なーすけさんの合意ポイント

- 知識を教えること自体は嫌ではない。**手段として機能するなら全然あり**
- ただし「**超1級品のフィジーカーになりたい、それだけが目的**」の層は**別プログラム（Proコース的な位置づけ）**として切り分けるのが良さそう
- 理由：営業・教育段階では成約できても、**2ヶ月後に「違う気がする／僕はなーすけさんに教わりたかった」といったクレームになる**リスクが高い

### 6-5. 差別化の議論

- なーすけさん：ストレスフリーで減量しながら大会に出ているという**体現者としての象徴性**は差別化要素
- みしぴー：「差別化は本質じゃない気がする」——営業トークでの見せ方としては必要だが、勝負する場所ではない
- たくちゃん：**「差別化の一番のポイントは人。営業マンが売るのは"覚悟"。覚悟を誰よりも決めていく組織にすれば顧客満足度は必ず上がる」**
- → 差別化を考えるというより、**「俺たちは何をするのか／価値提供はどこにできるのか」**を掘り下げる方向で合意

### 6-6. 売ってはいけない人（最終結論）

- **結論：売ってはいけない人はほぼ思いつかない**
- お金と同じぐらい「見た目」はインフラ的な関心事。**「自分を変える覚悟はない」という覚悟がある人であれば、ほぼ誰にでも価値提供できる**
- 協調性のない人・攻撃性を持った人などは紛れ込みはするが、**未然に売らない仕組みを作るのは難しい**（営業マンは数字を追う体制にしたい）→ ある程度は中で対処する方針
- **最低20人、できれば30人ぐらい集めて1クラス分回せば見えてくる**

### 7-2. 年間サイクルのイメージ

```
7〜8月：プログラム開始（4ヶ月プラン）
9月：次期集客ローンチ
10月：セールス
11月：次期プログラム開始（4ヶ月）
```

→ **売上も立ち、事業も回せ、満足度も高く保てるサイクル**

### 7-3. サービスのゴール（顧客の最終状態）

**定量：**

- 体重減（例：10kg減）
- 写真で見た目の変化が出ている

**定性：**

- フィットネスの習慣化ができている
- **「自分もこの考え方で仕事・人生を動かしていきたい」と感じる状態**
- 自分が変わった感覚を他者にも広げたくなる
- 紹介発生／講師側（バディ候補）に回りたいと感じる
- SFF（オフライン）にボランティアとして関わってくれる／ShapeFitのビジョンを体現する存在になる

### 7-4. ビジョンからの逆算（なーすけさん言語化）

> **ShapeFitビジョン：1人1人が自分の人生を好きになれる世界**
> 
- 自分を知り、自分の軸で、自分の未来に一歩踏み出し、その結果他者にも貢献している状態＝幸せの定義
- フィットネスで自分の人生を好きになれる理由 ＝ **「自分の挑戦と物語を大切にできるようになるから」**
- 結果主義ではなく、**挑戦そのものに意味がある**という価値観を、フィットネスという"努力が裏切らない"領域でまず体感してもらう
- 仕事・家族・人間関係の捉え方にもこの考え方が波及する

### 7-5. サービスの"効く"構造（たくちゃん整理）

- 本質：**ゴール設定 × 意味づけ × 覚悟**
- **ゴール設定は過去のビリーフに紐づく**（勝たないといけない／認められたい等の強烈な感情）
- 平和で衣食住に困らない日本ではそのゴールを忘れてしまう → **覚悟を持たせる仕組み**が必要
- ただし、本質的なゴール設定を現場コーチに任せると**提供者に"ムラ"が出る**（定義が難しいから）→ ここをどう仕組み化するかが今後の論点
- フィットネスで成功体験を積ませる → その考え方は仕事・家族・人間関係に転用できる、というストーリー

### 7-6. 継続のためのキーファクター（なーすけさん）

1. **「自分はフィットネスをやる人間だ」というアイデンティティ（あり方）の書き換え**
2. **フィットネスを踏み出しやすく・継続しやすい環境を自分で整えられる状態**

## 補足：本MTGで出た重要な視点

### たくちゃんから

- **「覚悟を売る」**——営業マンが売るのは体でも知識でもなく"覚悟"。覚悟の解像度を間違えるとエネルギーの押し売り・クレームになる
    - 差別化の核は「人」。そして売り手側の「覚悟」——差別化のために何かをやるのではなく、組織として誰よりも覚悟を決めていること自体が差別化であり、それが顧客満足に直結する。営業マンが顧客に"覚悟"を押し付ける話ではなく、"覚悟の解像度を間違えるとエネルギーの押し売り・クレームになる"とたくちゃん自身が警告している点にも注意。
