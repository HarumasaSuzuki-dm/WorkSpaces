# 01. 共通コミュニケーションガイドライン

## 1. 根本的な考え方： “トレーナーとのバトンタッチ”
私たちのカスタマーサポート（CS）は、単なる事務的な返信作業ではありません。お客様とのやり取りは、店舗にいる「トレーナーへのバトンタッチ」であると考えてください。

お客様が問い合わせをしてきた背景にある悩みや期待を汲み取り、次につながる良好な関係性を築くことが、私たちの最大のミッションです。

## 2. 言葉遣いと表現のルール（Do / Don't）
ブランドの価値観を守り、お客様に誠実な印象を与えるために、以下のルールを徹底してください。

### 【Don't】控えるべき表現
* **結果を断定する言葉**: 「絶対に痩せさせます」「必ず結果が出ます」といった、結果にフォーカスしすぎた断定的な表現は使用しないでください。
* **事務的な一問一答**: 値段や所要時間を聞かれた際、数字だけを答えて終わらせないようにします。

### 【Do】推奨する姿勢・表現
  * **不安や希望に寄り添う**: 「どのようなお悩みがありますか？」「どのような姿を目指されていますか？」と一歩踏み込んで問いかけます。
  * **プラスアルファの提案**: お客様の質問の意図を想像し、その方に合ったプランや解決策をこちらから提案する姿勢を大切にしてください。

## 3. 対応スピードの基準
お客様を待たせないことが、信頼構築の第一歩です。
* **24時間以内の返信**: 全てのお問い合わせに対し、24時間以内に何らかの回答を送ります。
* **「確認中」の連絡**: 調査や確認に時間がかかり、すぐに結論が出せない場合でも、「お問い合わせを承りました。現在確認中ですので、改めてご連絡いたします」といった一次返信を必ず入れてください。

## 4. 媒体別のトーン＆マナー
連絡ツールによって、お客様が求める「距離感」が異なります。以下の通り使い分けてください。

### メール（SFG事業 / SAL事業）
* **トーン**: フォーマルで誠実な対応。
* **ルール**: 原則として絵文字や「！」（びっくりマーク）は使用しません。

### Lステップ（SFG公式LINE）
* **トーン**: 親しみやすく、少し柔らかい対応。
* **ルール**: 公式LINEの温度感に合わせ、適度に絵文字や「！」を使用しても問題ありません。

## 5. 報告・相談の判断基準
以下のようなケースは、自分の判断で返信せず、必ず一度「大川」へ報告・相談してください。

* 返金、キャンセル、返品交換の希望
* クレームや、トレーナー・サービスに対する厳しい意見
* 企業からの営業メールや、出演・取材等の依頼
* FC（フランチャイズ）オーナーやトレーナー個人に関わる内容
* その他、少しでも「どう返すべきか」迷った場合

※業務に慣れるまでの最初の1〜2ヶ月間は、すべての返信文案を作成した段階で、一度確認依頼を出してください。
