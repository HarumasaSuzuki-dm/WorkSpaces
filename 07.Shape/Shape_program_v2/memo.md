## 観点① ビジョン・文化と福利厚生が連動している事例

### 事例1：株式会社サニーサイドアップ「32の制度」
**業種：PR・スポーツ・エンタメ / 従業員数：約100〜200名**

#### 制度の内容（具体的に）

スローガン「**たのしいさわぎをおこしたい**」を文字通り"制度化"した、32種類のユニークな福利厚生。ビジョンと制度名・内容が完全に一致しているのが特徴。

| カテゴリ | 制度名 | 内容 |
|---|---|---|
| 体験・感性 | たのしいさわぎ創造支援 | ライブ・イベント体験費用を一部補助。感性のインプットが「さわぎ」を創ると位置づけ |
| フィットネス | 幸せは歩いてこない | 月平均1万歩で **3,200円**の報奨金を支給 |
| フィットネス | 目指せ！A身体 | 健康診断でA判定を取得すると **32,000円**支給＋提携ジムを**1,000円**で利用可能 |
| フィットネス | 汗活 | ジム・サウナ利用費の補助 |
| 睡眠・回復 | 寝る子は育つ | 月平均7時間以上の睡眠で **3,200円**支給 |
| 社員間称賛 | スマイル給・MVT表彰 | 社員同士がよいところを伝え合い、優秀者には賞与上乗せ |
| ライフイベント | 失恋休暇・恋愛勝負休暇 | 人生の感情にも寄り添う制度 |
| 学習 | サニー文庫・スキルアップ支援 | 書籍補助・セミナー費用補助 |

#### ✅ なぜ機能しているか（成功要因）

制度の1つ1つが「たのしいさわぎ」という言葉に紐づいており、社員が制度を使うたびに**会社の哲学を体感できる**仕組みになっている。健康=身体の楽しさという解釈が、PR会社らしい感性を磨く文脈に接続されている。報奨金の金額（3,200円、32,000円）も企業名「サニーサイドアップ=32（three-two）」にかけており、ブランドとの一体感がある。

[株式会社サニーサイドアップ「32の制度」](https://www.ssu.co.jp/corporate/32rule/)

#### 🔥 ShapeFitへの応用可能性

> **「Be→Do→Have」を福利厚生の名前・設計に刻み込む**

ShapeFitの哲学をそのまま制度名にすることで、働くたびに哲学を体感できる。

| ShapeFit版アイデア | 内容例 |
|---|---|
| **「Beなヒト手当」** | 月次の行動変容チャレンジ（自分が設定した習慣目標）を達成したメンバーに報奨金 |
| **「Do記録ボーナス」** | 自身のトレーニング・健康習慣を記録・継続したスタッフへ小額インセンティブ（コーチが率先垂範） |
| **「Have体験補助」** | 大会エントリー費・フィットネス体験に使える月○千円のバウチャー支給 |

### 🌟 事例2：株式会社カヤック「文化と直結した制度群」
**業種：Web制作・ゲーム / 本社：鎌倉 / 従業員数：約200名**

#### 制度の内容（具体的に）

カヤックの哲学「**つくる人を増やす**」と「**鎌倉に生きる**」が全ての制度に貫かれている。

| 制度名 | 内容 |
|---|---|
| **鎌倉職住近接手当** | 鎌倉・逗子周辺居住者に家賃補助 |
| **まちの社員食堂** | 鎌倉の企業合同で運営する食堂を利用可能 |
| **旅する支社** | PCと回線があればどこでも勤務可能。海外1か月滞在実績あり |
| **誕生日休暇** | 誕生月に1日休暇 |
| **旅するかばん持ち** | 20代若手が社長の採用・講演に同行し経営視点を学ぶ |
| **漫画名刺** | 自分の顔をモチーフにした漫画風名刺（個性を尊重する文化の表現） |

#### ✅ なぜ機能しているか（成功要因）

「鎌倉という場所に暮らすこと」がクリエイターとしての感性を磨くという思想が一貫している。また、若手が社長の行動に同行する「かばん持ち制度」は低コストでありながら最高の成長機会になっている。

[カヤック 制度・行事](https://www.kayac.com/company/institution)

### 🌟 事例3：ピアボーナス制度（Unipos等）
**業種：複数スタートアップで導入 / 導入コスト：月額数万円〜**

#### 制度の内容（具体的に）

従業員同士が「感謝・称賛のメッセージ」と「少額の報奨ポイント」を相互に送り合う仕組み。ポイントは一定期間後に金銭や商品と交換可能。

- 毎月のポイント付与上限を設定（例：1人月1,000円相当）
- 贈る側は「なぜ贈るか」のメッセージを添える
- 全員が送った/受け取ったメッセージが可視化される（称賛文化の醸成）

#### ✅ なぜ機能しているか（成功要因）

「人事評価では拾えない貢献」にスポットが当たる。特にリモート・業務委託混在組織では関係性が希薄になりやすいが、テキストで感謝を可視化することで**組織エンゲージメントが大幅向上**。導入企業のアンケートでは社員満足度が85%超を記録した例もある。

業務委託トレーナー・コーチにもポイントを付与できるよう設計することで、**雇用形態に関係なくチームの一体感**を作れる。既存のSlack等に無料ツールを連携するだけで始められる。

## 観点③ 業績連動賞与の設計が透明・シンプルで納得感が高い事例

---

### NOT A HOTEL「ワンチーム評価（OKR連動賞与）」
**業種：高級別荘・会員権サービス / 従業員数：50〜100名規模**

#### 制度の内容（具体的に）

個人評価を**完全廃止**し、全社のOKR達成度で全員の賞与・昇給率を決定するシンプルな仕組み。

```
賞与 = 基本給の「10% or 20%」 （一律、全社OKR達成度で切り替え）
昇給率 = カンパニーベット（全社OKR）の達成度に応じて全員同率
```

- 年度初めに全社で「カンパニーベット（会社として賭けること＝OKR）」を設定
- 達成：全員に**基本給の20%賞与**
- 未達：**10%賞与**（0ではない点が重要）
- 制度導入後、**過去すべての期で達成・20%支給**の実績あり

#### なぜ機能しているか（成功要因）

① 「評価に費やす時間をプロダクト・顧客のために使う」という哲学が制度に反映されている。② 個人評価がないため、**心理的安全性**が高まり、チームワークを最優先する行動が自然に生まれる。③ 賞与率が「10% or 20%」と**誰でも即計算できる**シンプルさ。④ 全員が同じ方向を向いているため、部門間の対立が起きにくい。

[NOT A HOTELの組織文化（note）](https://note.com/saimaru/n/n7f62f4a98c5c)

#### ShapeFitへの応用可能性

> **「Be→Do→Have全体の進捗」を賞与トリガーにする**

ShapeFitの4事業（ジム・オンラインコーチング・コミュニティ・大会）を束ねる全社KPIを1本設定し、それの達成率で賞与を決定するモデルが応用できる。

```
ShapeFit賞与設計（仮案）
────────────────────────
目標：四半期の「行動変容人数（Be→Do達成者数）」●名
達成（90%以上）→ 賞与：月給の20%
部分達成（70%以上）→ 賞与：月給の10%
未達（70%未満）→ 賞与：月給の5%（0は作らない）
────────────────────────
✔ 数字が1つだけなので全員に見せられる
✔ 業務委託スタッフにも「達成ボーナス」として設計応用可能
```

### 🌟 補足事例：株式会社カヤック「さいころ給＋月給ランキング（相互評価）」

#### 制度の内容

| 制度 | 内容 |
|---|---|
| **さいころ給** | 毎月給料日前に全社員がサイコロを振り「月給×出目%」を賞与に上乗せ。1〜6%のランダム要素を導入 |
| **月給ランキング** | 半年に1度、全社員が「自分が社長なら誰をどう評価するか」を投票。結果が月給改定に反映 |
| **プロデューサー賞与** | 四半期ごとに事業責任者が事業貢献度を考慮して賞与を配分 |

#### ✅ なぜ機能しているか

「完璧に公平な評価は存在しない」ということを**制度として認めている**誠実さが特徴。「天が采配するさいころ」というユーモアが、評価への不満を和らげる。「社長目線で投票する」という問い方が、全員に経営視点を持たせる。

#### 🔥 ShapeFitへの応用

> 相互投票型の小さな表彰制度として応用可能。「四半期ごとにメンバー全員が、最もBe→DoをDid（体現した）人を1票投票する」仕組みは低コストで一体感を生む。

## 観点④ 正社員と業務委託が混在する組織での制度設計事例

---

### 🌟 事例6：株式会社タニタ「日本活性化プロジェクト」
**業種：健康機器メーカー / 従業員数：約500名（プロジェクト参加者：数十名）**

#### 制度の内容（具体的に）

希望する正社員が「**雇用契約→業務委託契約（個人事業主）**」に任意で切り替える制度。健康機器メーカーとしての「自律的に健康的に生きる」哲学を働き方にも適用。

**業務委託契約の骨子：**

| 項目 | 内容 |
|---|---|
| **業務内容** | 移行前の担当業務をそのまま「基本業務」として委託 |
| **報酬** | 移行前の「給与＋残業代＋社会保険会社負担分＋退職金積立分」を合算した額が基本報酬（≒手取りは増える） |
| **追加業務** | 新たな仕事は「成果報酬」として上積み可能 |
| **契約期間** | 原則3年間、1年ごとに業務内容・報酬を見直し |

#### ✅ なぜ機能しているか（成功要因）

① 「主体性を持って働く」という哲学と、業務委託化が完全に一致している。② 報酬設計が元の待遇より**透明かつ有利**なため、移行への不安が少ない。③ 会社と個人が「対等な事業者」として関係を再構築することで、双方のモチベーションが上がった。④ フリーランス新法への対応含め、労務リスクの整理が先行して行われた。

[タニタ「日本活性化プロジェクト」note解説](https://note.com/newwork_newlife/n/nbad0acad10ce)

#### 🔥 ShapeFitへの応用可能性

> **業務委託トレーナー・コーチを「対等なパートナー」として設計する**

ShapeFitはすでに「正社員3名＋役員3名＋業務委託複数名」の混在体制。タニタ型の考え方を参考に、**業務委託メンバーが誇りを持って関われる制度設計**が鍵。

```
ShapeFit 業務委託設計の骨子（提案）
──────────────────────────────────
①「基本報酬」：セッション数や担当クラス数の固定保証
②「成果報酬」：担当クライアントの継続率・紹介数連動
③「コミュニティ参加権」：社内勉強会・大会への無償参加
④「Be→Doボーナス」：自分自身の行動変容記録を会社と共有し、
                      哲学体現者には報奨ポイント支給
⑤「ShapeFitパス」：自社ジム・オンラインコーチングの無料利用権
                    （業務委託含む全スタッフに付与）
──────────────────────────────────
```
