# なーすけさん壁打ち：山本さん提案の整理

## 1. 今日のゴール

今日の壁打ちで**3つのこと**を決める。

| 決めること | 内容 |
|-----------|------|
| ① 山本さんへの確認事項 | 判断するために何を聞くか |
| ② メンバー共有の方針 | いつ・誰に・どう伝えるか |
| ③ 優先順位 | カリキュラムローンチとの兼ね合い |

## 2. 現状の整理

### 山本さんからの提案

| 提案 | 概要 |
|------|------|
| インフルエンサー組織化 | 料理系・暮らし系の発信者を認定講師化し、教材を広める |
| リアル店舗 | 関節整体サロン（19店舗）と連携した新サービス |

### なーすけさんの現時点での考え

- やるなら**インフルエンサー組織化が先**
- 理由：ノーストレス減量をより広められるから
- 懸念：提案の**解像度がまだ荒い**

### なーすけさんが「譲れない」と言っていたこと

1. インフルエンサーとの関係性
2. Shape Fitの未来
3. ブランドを損なわないこと
4. 組織の熱量
5. 組織が崩れないこと

## 3. 今日話したいこと

### 話したいこと①：山本さんへの確認事項を決める

**なぜ話すか**
- なーすけさん自身「解像度が荒い」と感じている
- 判断するための情報が足りない
- **何を聞けば判断できるか**を整理したい

**確認候補リスト**

| カテゴリ | 確認したいこと | 確認する理由 |
|---------|---------------|-------------|
| 選定基準 | インフルエンサーは誰がどう選ぶのか？ | ブランドを守れるか判断するため |
| 役割範囲 | 認定講師はどこまで教えるのか？ | 品質管理ができるか判断するため |
| 撤退条件 | 合わなかった場合、やめられるのか？ | リスクを把握するため |
| 収益構造 | 山本さん側は何で儲けるのか？ | 利害が一致しているか確認するため |
| 優先順位 | 店舗事業を後回しにしても大丈夫か？ | 関係性への影響を確認するため |

**今日決めること**
- このリストの中で**特に重要なもの**はどれか？
- **追加で聞くべきこと**はあるか？

---

### 話したいこと②：メンバー共有の方針を決める

**なぜ話すか**
- なーすけさんは「懸念点を聞きたい」「違う角度からの視点が欲しい」と言っていた
- メンバーに**何をどう伝えるか**を決めておかないと、議論が発散するリスクがある

**決めるべきこと**

| 項目 | 選択肢 |
|------|--------|
| いつ共有するか？ | 山本さんヒアリング前 or 後 |
| 誰に共有するか？ | 全員 or コアメンバーのみ |
| 何を共有するか？ | 提案の概要だけ or なーすけさんの考えも含めて |
| 何を聞きたいか？ | 意見 or 懸念点 or 両方 |

**今日決めること**
- 上記の選択肢から**方針を決める**

---

### 話したいこと③：優先順位を決める

**なぜ話すか**
- カリキュラムローンチが進行中
- 山本さんの話に時間を使いすぎると、ローンチに影響する可能性がある
- **どちらを優先するか**を明確にしておきたい

**確認したいこと**

| 質問 | 意図 |
|------|------|
| カリキュラムローンチの目標時期は？ | 現状のスケジュールを確認 |
| 山本さんの話を「ローンチ後」にしたら何が失われる？ | 本当に今やる必要があるか確認 |
| 今のリソースで両方進められる？ | 現実的に可能か確認 |

**今日決めること**
- カリキュラムローンチと山本さんの話、**どちらが先か**
- 山本さんの話は**ローンチ後でもいいか**
