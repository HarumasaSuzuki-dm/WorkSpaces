## 調査依頼：東京近郊のアウトドア・グループ体験施設リサーチ

### 調査の目的
6名のチームで「非日常体験・内省・相互理解」を目的としたイベントを、2ヶ月に1回・平日を中心に開催したい。予算は1回あたり¥40,000（6名合計）。まず「日帰り」での実施可能性を確かめるため、具体的な施設候補を調査してほしい。

---

### 基本条件（必須）
- 参加人数：6名
- 開催形式：日帰り（宿泊なし）
- 想定日程：平日（月〜金）
- 予算：¥50,000以内（6名合計）。内訳は「施設使用料＋食材or食事代＋交通費」の合計
- 起点：東京都内（渋谷・新宿・池袋いずれかから出発）
- 移動時間：片道1時間30分以内（公共交通または車）

---

### 調べてほしい施設タイプ（3種類）

#### タイプA：手ぶらBBQ場
- BBQコンロ・炭・食材がレンタルまたはセットになっている施設
- 6名の少人数グループで予約可能なこと
- 例：千葉・埼玉・神奈川の河川敷・公園・アウトドア施設

#### タイプB：デイキャンプ場
- テントや焚き火台のレンタルがある（または持ち込みOKな）施設
- 手ぶらOKまたはレンタル充実のもの優先
- 例：奥多摩・高尾・秩父・千葉方面

#### タイプC：グループワーク＋食事ができる施設
- 屋内または半屋外で「対話・ワーク・食事」ができるスペース
- コワーキングスペース＋ランチ、貸し別荘のデイユース、合宿施設のデイプランなど
- 雨天でも開催できることが必須

---

### 各施設について調べてほしい項目

以下を施設ごとに表形式で整理してほしい。

| 項目 | 内容 |
|------|------|
| 施設名 | 正式名称 |
| 施設タイプ | A / B / C |
| 所在地・アクセス | 最寄り駅 or IC、東京中心部からの所要時間と交通手段 |
| 予約単位 | 何名から予約可能か、区画貸しか人数課金か |
| 施設使用料（6名） | 平日の費用概算 |
| 食材・食事費（6名） | 手ぶらプランの概算、または持ち込み可否 |
| 交通費（6名合計） | 電車・バス・レンタカー等の概算 |
| 合計費用目安 | 上記3項目の合計（¥40,000以内かどうかの判定） |
| 平日利用可否 | 〇/△/× |
| 雨天対応 | 屋内スペースの有無、タープ・屋根付きの有無 |
| 予約リードタイム | 何日前から予約可能か、当日予約の可否 |
| レンタル品 | BBQコンロ・テント・机・椅子など |
| URL・電話番号 | 公式サイトまたは予約ページ |
| 総合コメント | 「初回テスト開催に向いているか」を一言で |

---

### 優先して調べてほしい地域（候補エリア）
- 千葉県（市川・松戸・千葉市・館山方面）
- 埼玉県（所沢・飯能・長瀞方面）
- 神奈川県（相模原・横須賀・葉山方面）
- 東京西部（奥多摩・高尾・あきる野方面）
- ※都内（江東区・品川・大田区などの屋外BBQ施設も可）

---

### 出力形式
1. タイプA・B・Cそれぞれ3〜5施設ずつ（合計10〜15施設）
2. 各施設の上記表
3. 最後に「¥40,000以内で日帰り可能な最有力候補ベスト3」をまとめて推薦する（理由付き）

---

### 補足・判断基準
- 「手ぶらでOK・予約が簡単・少人数歓迎」の施設を優先する
- 初回はハードルを下げることが重要なので、「とにかく簡単に開催できる施設」を重視する
- 将来的に1泊ワーケーションや泊まりキャンプに発展させる可能性もあるため、泊まりプランもある施設だと加点
- 施設の写真・SNS発信映えは採用ブランディングの観点でもプラス評価
