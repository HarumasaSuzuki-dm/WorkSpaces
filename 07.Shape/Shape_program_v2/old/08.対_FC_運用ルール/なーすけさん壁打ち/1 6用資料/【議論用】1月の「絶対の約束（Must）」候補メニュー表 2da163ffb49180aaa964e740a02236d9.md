# Must候補 運用ガイド（FCオーナー向け）

**目的**：各Must候補について「何を・いつ・どうやるか」を具体的にイメージし、絞り込み議論の材料にする

---

## ① 24時間以内レスポンス

| 項目 | 内容 |
|------|------|
| **作業時間目安** | 週10〜15分（1日2〜3分 × 5日） |
| **使うツール** | Slack（またはチャットツール） |

**FCオーナーがやること**
1. 毎日1回、チャットを開く
2. 「全体連絡」タグ・名指しメッセージを確認
3. 24時間以内にスタンプ or 返信で反応

**記入例・サンプル**
```
本部：【全体連絡】1月の研修日程を確認してください
　↓
オーナー：✅（スタンプ）または「確認しました」
```

**例外時の対応**
- 体調不良・通信障害 → 復帰後に事務局へ一言連絡すればOK

## ② 週次会議100%参加

| 項目 | 内容 |
|------|------|
| **作業時間目安** | 週60分（会議）＋欠席時は追加30分（代替対応） |
| **使うツール** | Zoom（会議）、Googleドキュメント（議事録）、Slack（コメント提出） |

**FCオーナーがやること**
1. 毎週○曜○時の定例会議に参加
2. 欠席が事前に分かったら、前日までに事務局へ連絡
3. 欠席した場合は48時間以内に議事録を読み、質問・コメントを提出

**記入例・サンプル（代替対応）**
```
Slackで事務局へ：
「1/15の会議欠席しました。議事録確認済みです。
　質問：○○の件、詳細を教えてください」
```

**例外時の対応**
- 急な顧客対応・体調不良 → 事後でも代替対応を完了すればOK

## ③ セッションログ完全提出

| 項目 | 内容 |
|------|------|
| **作業時間目安** | 週15〜20分（確認・フォロー） |
| **使うツール** | ログフォーム（Googleフォーム）、管理シート（スプレッドシート） |

**FCオーナーがやること**
1. 週1回、管理シートで自店トレーナーの提出状況を確認
2. 未提出があればトレーナーに声かけ
3. 継続的な未提出があれば原因をヒアリング

**必須3項目（トレーナーが記入）**
| # | 項目 | 記入例 |
|---|------|--------|
| 1 | 顧客の言葉（引用） | 「最近、夜中にお菓子を食べちゃうんです」 |
| 2 | 弱み・本音 | 仕事のストレスで食べてしまう。自己嫌悪がある |
| 3 | 次回の一歩 | 夜10時以降はお菓子を見えない場所に移動する |

**例外時の対応**
- 時間切れ → 翌日午前中までに提出すればOK
- システム障害 → Slackで事務局に報告＋復旧後に提出

## ④ 未達の放置禁止

| 項目 | 内容 |
|------|------|
| **作業時間目安** | 週5〜10分（確認・声かけ） |
| **使うツール** | 振り返りシート（Googleフォーム） |

**FCオーナーがやること**
1. 週1回、振り返りシートの提出状況を確認
2. 提出があれば「出してくれてありがとう」と声かけ
3. 3回連続未達のトレーナーがいれば、一緒に対策を考える

**振り返りシート記入例（トレーナーが記入）**
| # | 項目 | 記入例 |
|---|------|--------|
| 1 | 何が起きた？ | お客様が終始うなずくだけで、自分の言葉が出なかった |
| 2 | なぜ本音が出なかった？（仮説） | 質問が抽象的すぎた。具体的な場面を聞けばよかった |
| 3 | 次回どうする？ | 「最近いちばん困った瞬間は？」と具体的に聞く |

**例外時の対応**
- 判断迷い → 迷ったら「未達」として提出してOK（出しすぎて怒られない）
- 時間不足 → 箇条書きでもOK、まず出すことが大事

## ⑤ 正解（ハウツー）販売禁止

| 項目 | 内容 |
|------|------|
| **作業時間目安** | 月30分（投稿前チェック＋月1回の見直し） |
| **使うツール** | Instagram / チラシ / LP、チェックリスト（後日配布） |

**FCオーナーがやること**
1. 投稿・販促物を作る前にNG表現リストを確認
2. 迷ったら投稿前に事務局へ相談
3. 月1回、過去投稿をセルフチェック

**NG表現 vs OK表現**
| NG（使わない） | OK（推奨） |
|----------------|------------|
| 「〇kg痩せた！」だけを強調 | 「なぜ続けられたか」のストーリー |
| 「この食事法で確実に痩せる」 | 「自分と向き合う時間ができた」という声 |
| ビフォーアフター写真だけ | 体重以外の変化（気持ち・習慣）を紹介 |
| 「〇〇するだけで簡単」 | 「一緒に考えながら進めます」 |

**例外時の対応**
- 過去投稿 → 1月中に見直しリストを作成（即削除は不要）
- 顧客から「数字を載せて」と要望 → ストーリーと一緒に載せる形で調整

## ⑥ 状態を測るNPS実施

| 項目 | 内容 |
|------|------|
| **作業時間目安** | 月15〜20分（月末に全顧客分の確認） |
| **使うツール** | NPSフォーム（Googleフォーム）、顧客管理シート |

**FCオーナーがやること**
1. 月末に顧客管理シートでNPS記録の有無を確認
2. 未記録の顧客がいればトレーナーにリマインド
3. スコアが低い顧客がいれば、トレーナーと対応を相談

**NPS質問例（トレーナーがセッション内で実施）**
| # | 質問 | 回答形式 |
|---|------|----------|
| 1 | 最近、自分で決めて行動できていますか？ | 10点満点 |
| 2 | ShapeFitを友人にすすめたいですか？ | 10点満点 |
| 3 | 自由コメント | 任意・自由記述 |

**例外時の対応**
- 顧客が回答拒否 → 「未実施（顧客意向）」と記録すればOK
- 新規顧客 → 入会後2回目以降のセッションで初回実施
- 休会中 → 対象外（復帰後に再開）

## ⑦ オーナー現場対話義務

| 項目 | 内容 |
|------|------|
| **作業時間目安** | 週10〜15分（対話5分＋記録5分） |
| **使うツール** | 週次報告フォーム（Googleフォーム or スプレッドシート） |

**FCオーナーがやること**
1. 週1回以上、トレーナーと5分以上の対話を実施
2. 対話後、当日中に週次報告フォームへ1行記録
3. 出張などで対面が難しい週はビデオ通話で代替（月2回まで）

**対話のポイント**
- 業務連絡だけでなく「最近どう？」「困っていることは？」を聞く
- 1対1でなくても、複数トレーナーとの会話でもOK

**記録例（週次報告フォーム）**
```
日付：1/15
対話相手：山田トレーナー、佐藤トレーナー
内容メモ：新規顧客の対応で悩んでいるとのこと。来週フォロー予定
```

**例外時の対応**
- 1週間以上不在 → 事前に事務局へ連絡、翌週に2回実施でカバー
- 体調不良 → ビデオ通話で代替、それも難しければ事務局へ報告
