# SFF × The Personal Gym（TPG）協業のご提案

**株式会社ShapeFit**

---

## はじめに：TPGさんへのメッセージ

パーソナルジムを運営されていると、こんな課題を感じることはありませんか？

> 「ダイエット目標を達成した会員様が、その後退会してしまう」
> 「"次の目標"を提案できれば継続率が上がるのに、その仕組みがない」
> 「大会出場に興味を持つ会員様がいるが、どう後押しすればいいか分からない」
> 「他ジムとの差別化が難しくなってきた」

私たちShape Fit Festival（SFF）は、まさにこの課題を一緒に解決できるパートナーだと考えています。

---

## 1. この提案で実現したいこと

### 双方にとってのゴール

| ShapeFit側 | TPG側 |
|------------|-------|
| SFFの選手数が増える | 会員様の継続率が上がる |
| 大会の価値・盛り上がりが高まる | 体験予約・入会が増える |
| 継続的なパートナーができる | 追加セッション売上が増える |
| | 「挑戦を応援するジム」として差別化できる |

### 基本方針

**「小さく始めて、うまくいったら拡大する」**

- いきなり大きな投資や工数をお願いしません
- まずは負荷の少ない施策からスタート
- 効果が見えたら、次のステップへ

---

## 2. なぜSFFなのか？

### SFFは「結果」ではなく「挑戦」を称える大会です

一般的なボディコンテストは「誰が一番仕上がっているか」を競います。
SFFは違います。

> **「挑戦する過程」「そこで何を感じたか」に価値を置く大会**

だからこそ、初めての方でも「出てよかった」と思える体験になります。

### TPGの会員様に紹介しやすい理由

| 一般的な大会 | SFF |
|--------------|-----|
| 結果がすべて | 挑戦と物語に価値がある |
| ガチ勢向け | 初挑戦でも歓迎 |
| 敷居が高い | 「やってみたい」が出発点 |
| 出場後の燃え尽き | 次の挑戦につながる設計 |

### SFFの実績

- 2024年：全6大会を黒字開催
- 参加者の約60%が大会初挑戦
- 「また出たい」と答えた参加者：90%以上
- 参加者の多くがジム継続率向上（トレーナーからのフィードバック）

```
【導入】─────────────────────────────────────────
  案A：相互特典（紹介）
```

## 4. 各プランの詳細

---

### 案A：相互特典で「行き来」を作る

**開始までの目安：即日〜1週間**
**TPG側の負荷：★☆☆☆☆（最小）**

#### 何をするか

TPGとSFFの間で、相互に特典を設けます。

**TPG → SFF方向**
- TPG会員様がSFFにエントリーする際、参加費を割引（例：3,000円OFF）
- または、優先エントリー枠を提供

**SFF → TPG方向**
- SFF参加者・来場者に、TPG体験の特典を案内（例：体験料2,000円OFF）
- 大会会場でのチラシ配布、QRコード設置

#### 告知方法（シンプルでOK）

- 店内POP
- 受付でのひとこと案内
- LINE / Instagram / メルマガ

#### TPG側のメリット

| メリット | 理由 |
|----------|------|
| 質の高い見込み客が増える | 大会に興味がある人＝本気度が高い |
| 広告費をかけずに集客 | 紹介ベースなのでCPA（顧客獲得単価）が低い |
| 工数がほぼかからない | 告知するだけで開始できる |

#### 期待効果（目安）

| 指標 | 目標値 |
|------|--------|
| TPG体験予約 | +3〜5件/月 |
| 入会 | +1〜2件/月 |
| SFF出場（TPG会員様） | +2〜3名/大会 |

#### 必要な準備

| 担当 | 内容 |
|------|------|
| ShapeFit | 告知文テンプレート、チラシデータ、特典コード発行 |
| TPG | 特典内容の決定、店内告知 |

### 案B：共同イベントで「出場の気持ち」を作る

**開始までの目安：2〜3週間**
**TPG側の負荷：★★☆☆☆**

#### 何をするか

TPG店舗（またはオンライン）で、60分程度のイベントを1回だけ実施します。

**イベント例：「SFFって何？初心者でも大丈夫？」説明会**

| 時間 | 内容 |
|------|------|
| 0〜30分 | SFFの紹介（雰囲気、参加者の声、初出場者の割合など） |
| 30〜45分 | 出場までの現実的な流れ（準備期間、費用、トレーニング内容） |
| 45〜60分 | Q&A、「やってみたい人」の個別相談 |

#### TPG側のメリット

| メリット | 理由 |
|----------|------|
| 参加者は関心が高い | 追加セッション・食事指導が売りやすい |
| 「挑戦を応援するジム」の印象が残る | 継続率アップ、口コミにつながる |
| イベント自体が集客コンテンツになる | SNS投稿のネタ、会員様への声かけのきっかけ |

#### 期待効果（目安）

| 指標 | 目標値 |
|------|--------|
| イベント参加者 | 15〜20名 |
| 「出場したい」と意思表示 | 5〜8名 |
| 実際の出場 | 3〜5名 |
| 追加セッション契約 | +2〜3名 |

#### 必要な準備

| 担当 | 内容 |
|------|------|
| ShapeFit | 説明会資料、申込フォーム、当日の運営（登壇） |
| TPG | 会場提供（店舗 or オンラインURL）、会員様への告知 |

### 案C：TPG代表決定戦 → SFF出場

**開始までの目安：4〜6週間**
**TPG側の負荷：★★★☆☆**

#### 何をするか

TPGの中で「代表」を決める小さなイベントを実施し、上位者にSFF出場特典を付与します。

**イベント例：「TPGチャレンジ」8週間プログラム**

| フェーズ | 内容 |
|----------|------|
| Week 1 | エントリー開始、目標設定シート提出 |
| Week 2-7 | 通常トレーニング＋週次チェックイン |
| Week 8 | 最終審査（写真・数値・変化・姿勢など） |

**審査基準（例）**
- 身体の変化（Before/After）
- 姿勢・表情の変化
- 取り組みの姿勢（トレーナー評価）

※「結果」だけでなく「過程」も評価する設計にすることで、SFFの思想と一致させます。

**上位者への特典（例）**
- SFFエントリー優先枠
- 参加費の一部サポート
- 「TPG代表」として大会でアナウンス

#### TPG側のメリット

| メリット | 理由 |
|----------|------|
| 会員様に「次の目標」ができる | 退会防止、継続率UP |
| 大会に向けた追加セッションが自然に増える | 売上UP |
| Before/Afterコンテンツが生まれる | SNS投稿、広告素材として活用可能 |
| 「代表」という称号で会員様のモチベーションUP | ジムへのロイヤルティ向上 |

#### 期待効果（目安）

| 指標 | 目標値 | 想定根拠 |
|------|--------|----------|
| エントリー | 10〜15名 | 会員100名の10%が興味を持つと想定 |
| 完走（8週間継続） | 8〜12名 | 80%が完走 |
| SFF出場 | 3〜5名 | 完走者の40%が出場 |
| 追加セッション売上 | +20〜30万円/8週間 | 出場者1名あたり週1追加×8週×5,000円 |
| 継続率改善 | +10〜15% | 目標がある会員の退会率は低い傾向 |

#### 必要な準備

| 担当 | 内容 |
|------|------|
| ShapeFit | ルール案、告知テンプレート、ポスターデザイン |
| TPG | 審査運営、会員様へのエントリー促進 |

### 案D：共同商品「SFF仕上げコース」

**開始までの目安：1〜2ヶ月**
**TPG側の負荷：★★★★☆**

#### 何をするか

「SFF出場を目指す人」向けに、短期集中コースを共同で企画・販売します。

**コース例：「SFF仕上げ12週間パッケージ」**

| 内容 | 詳細 |
|------|------|
| トレーニング | 週2回×12週間（計24回） |
| 食事指導 | LINEでの食事チェック、週次フィードバック |
| ポージング練習 | 大会前に2回のポージングレッスン（SFF側提供） |
| メンタルサポート | 不安解消のための個別相談（SFF側提供） |

**価格イメージ**
- 200,000〜300,000円（TPG通常コース＋αの価値を設計）
- 売上配分：TPG 80% / ShapeFit 20%（例）

#### TPG側のメリット

| メリット | 理由 |
|----------|------|
| 高単価パッケージで売上インパクト大 | 通常コースより高価格で販売可能 |
| 本気層が集まる | LTV（顧客生涯価値）が高い顧客が増える |
| 「公式コース」として差別化 | 他ジムにない独自サービスになる |
| 成功事例が広告素材になる | 「SFF出場者輩出」の実績をアピール可能 |

#### 期待効果（目安）

| 指標 | 目標値 |
|------|--------|
| コース申込 | 5〜10名/期 |
| 売上 | 100〜300万円/期 |
| SFF出場 | 申込者の80%以上 |

#### 必要な準備

| 担当 | 内容 |
|------|------|
| ShapeFit | ポージング指導、メンタルサポート、コース設計支援、LP作成 |
| TPG | トレーニング指導、食事指導、申込・決済管理 |
| 要協議 | 売上配分、責任分担、写真使用の同意取得 |

### 案E：協賛パートナー

**開始までの目安：大会の1〜2ヶ月前**
**TPG側の負荷：★★☆☆☆**

#### 何をするか

協賛金をいただき、「露出」だけでなく「成果」が見える協賛パッケージを提供します。

**協賛パッケージ例（ゴールドプラン）**

| 提供内容 | 詳細 |
|----------|------|
| 会場露出 | バナー掲出、アナウンスでの紹介 |
| 配信露出 | ライブ配信画面へのロゴ表示 |
| パンフレット | 全来場者配布パンフへの1ページ掲載 |
| 体験予約導線 | 会場内QRコードから予約フォームへ誘導 |
| 参加者リスト | 出場者へのDM送付許可（希望者のみ） |

**価格イメージ**
- 100,000〜300,000円/大会（プランにより変動）

#### TPG側のメリット

| メリット | 理由 |
|----------|------|
| 広告費として計上しやすい | 「何が得られるか」が明確 |
| 大会参加層にダイレクトにリーチ | 本気度の高い見込み客に訴求 |
| 体験予約への導線がある | 効果測定が可能 |

#### 期待効果（目安）

| 指標 | 目標値 |
|------|--------|
| QR経由の体験予約 | 5〜15件/大会 |
| 認知向上 | 来場者300〜500名にリーチ |

## 5. ShapeFitについて

### 会社概要

| 項目 | 内容 |
|------|------|
| 社名 | 株式会社ShapeFit |
| 事業 | フィットネス大会運営（SFF）、パーソナルジム（SFG）、アパレル、メディア |
| ミッション | 1人ひとりの「挑戦」と「物語」を共に |
| ビジョン | 1人ひとりが「自分の人生」を好きになる |

### 私たちが大切にしていること

> **「結果＝価値」ではない。挑戦とその物語に「価値」がある。**

SFFは、勝ち負けを決めるだけの大会ではありません。
参加者一人ひとりの「挑戦する姿」と「そこに至る物語」を称える場です。

だからこそ、初めての方でも安心して挑戦でき、
挑戦した後も「次もやりたい」と思える設計になっています。
