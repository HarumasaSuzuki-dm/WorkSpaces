# MYS-実験プロジェクト｜かおりさんへの協力依頼

---

## まず、お願いの背景

かおりさん、よろしくお願いします！

今回、なーすけさんのコーチングの「判断の仕方」を言語化するプロジェクトを動かしています。
なーすけさんが実験的にコーチングセッションを行い、その様子を動画で記録します。

かおりさんには、**その動画を見てフィードバックをまとめてもらう役割**をお願いしたいです。
稼働の目安は **週2時間程度**を想定しています。

---

## かおりさんの動き方（流れ）

```
① なーすけさんがセッション実施（1回あたり約60分）
      ↓
② 動画が格納される
      ↓
③ かおりさんが動画を視聴し、フィードバックをまとめる
      ↓
④ まとめた内容をなーすけさん・ぼんちゃんに共有する
```

---

## フィードバックでまとめてほしいこと

以下の観点で気づいたことをまとめてもらえると助かります。

- なーすけさんが何を聞いて、どう判断したか（判断の根拠が見えた瞬間）
- 参加者のどの発言・反応に対して、どんな返し方をしていたか
- 「これはなーすけさんならではだな」と感じたポイント

※ 評価・採点ではなく、 **「なーすけさんの判断の記述・記録」** が目的です。

---

## フィードバックのまとめ方

どちらでも構いません。やりやすい方法で。

**A. テキストでまとめる場合**
- 箇条書きでOK
- 時間軸（〇分頃）があると後で整理しやすいです

**B. 動画コメントでまとめる場合**
- 動画ツール上にコメントを残す形でもOK
- 【要確認：動画ツールのコメント機能が使えるか確認する】

---

## 共有の方法・タイミング

| 項目 | 内容 |
|------|------|
| 共有先 | 【要確認：Slackチャンネル名を記入する】 |
| 期限 | 動画共有後、**1週間以内** |
| フォーマット | 定型は設けません。気づきを書き出す感覚でOK |

**報告時のひな形（Slack投稿）**

```
【MYS実験FB｜〇〇さん・セッション①】

■ 気になった判断ポイント
- 〇分頃：〜という問いに対して、〜と返していた
- ...

■ なーすけさんらしいと感じた点
- ...

■ その他・気になったこと
- ...
```
