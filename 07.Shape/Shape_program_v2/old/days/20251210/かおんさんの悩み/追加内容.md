## Phase 2. 【勝機・未来】どう勝つのか？（構造化）

### Q11. 5年後の「勝利の景色」（タイムマシン）

* 
**社会の変化**: 業界全体が大きく変化している 。


* 
**競合他社の状況**: 大手フィットネス系などがSFを真似しようとするが、表面的な「ハウツー」は真似できても、その奥にあるバリューの体現（全員が信じて体現する難しさ）や、結果ではなくプロセスを大切にするという「怖いことに挑戦し続ける姿勢」までは再現できず、勝てない 。


* 
**賞賛される見出し**: 「結果にとらわれない考え方で、日本一に」 。


* 
**ドキュメンタリーのテーマ**: 「自分らしさを突き詰めた先にあるもの」や「らしさとは何か」を問われるような、内面とプロセスにフォーカスした内容が想定される 。



### Q12. 独自の「勝利の方程式」（Unfair Advantage）

* 
**真似できない武器**: ノウハウ以上に、全員がその価値観を信じ、同じ方向を向いて自分と他人に向き合い続ける「文化」や「空気感」 。


* 
**驚きのポイント**: 「ここなら何でも話せる」と思わせるフランクな空気感 。トレーナーが裏表のない「そのままの姿」で自己発信しており、顧客がいつの間にか自分をさらけ出してしまうような「言いやすさ」 。



### Q13. 解決される「社会的苦痛」

* 
**除去される痛み**: 「周りの正解に合わせなければならない」という苦しさ 。感情を無視して頭だけで考え、成果を出した後に残る「自分はこれで良かったのか」というモヤモヤやストレスを解消する 。


* 
**人生の好転**: 自分の人生を好きになり、今まで諦めていたことに一歩踏み出すのが怖くなくなる 。個人の行動が社会全体の好循環（お金の回りや給料の向上）を生み、それがまた個人に返ってくるというポジティブな連鎖が起きる 。



### Q14. 意図的な「非効率」

* 
**あえて時間をかけること**: 自己把握、他者理解、自分のことを考える時間 。コーチングの導入コスト 。


* 
**過剰なこだわり**: 効率性で「100%」に収めず、120%までこだわる姿勢 。


* 
**例**: 選手の声を控室に貼る。直接的な数値目標には見えづらくても、温かい空気感やパフォーマンス向上のための絶対的なこだわりとして継続する 。




* 
**言語化の時間**: 会社で決めた言葉を各メンバーが自分の言葉に落とし込むための対話時間を、組織力の源泉として大切にする 。



### Q15. 戦わない「土俵」

* 
**断る顧客**: 単なる「ハウツー（やり方）」や「結果だけ」を求める顧客。「それなら他社へ行ったほうがいい」と伝える 。


* 
**手を出さないもの**: 業界で一般的な「やり方だけ教えて独り立ちさせる」という表面的な考え方 。SFはもっと上流の、あり方やプロセスの変容を主戦場とする 。



### Q16. 推奨される「狂気」

* 
**称賛される行動**: 徹底的な「自己分析」 。コンフォートゾーンの外に高い目標を設定し、自分らしい「挑戦の物語」を歩む姿勢 。


* 
**盛り上がる瞬間**: 誰かが「ドキドキするような挑戦」を決めた時 。あるいは、自分の意思ややりがい、大切にしたいことを自分の言葉で伝えた時に最も盛り上がる 。



### Q17. 組織内の「ヒーロー」と「出世」

* 
**評価される人**: 自分をひたすら研究・内省し、言葉遣いや仕事の進め方でバリューを体現している人 。


* 
**英雄の条件**: 主語を「私たち（SF）」に置き、周囲から「この人がいるおかげでSFが加速している」と認められる、他者貢献意識の高い人 。


* 
**評価したくない人**: セッション数などの数字だけを盾にし、チーム全体の文化や空気感を考えられない個人主義的な行動 。



### Q18. 能力があっても「お断り」

* 
**採用したくない条件**: 理念やカルチャーを大切にする考え方に疑問を持つ人 。自己防衛に走り、率直な対話や議論ができない人 。


* 
**重視する人間性**: 「自己開示」ができ、言葉にできること 。過去の経歴において、何を信じて走ってきたかという「なぜやるのか」の軸に一貫性があること 。



### Q19. 許されない「凡庸」

* 
**平均的な仕事への対応**: 失敗はしないが提案もしない状態は、文化として「厳しく問い詰める」対象となる 。


* 
**組織のスタンス**: 「馴れ合い」のアットホームさではなく、同じ志を持って自分を極める「職人気質なプロ集団」 。物事に対して思考が深く、妥協を許さないストイックな文化を目指す 。



### Q20. 顧客体験の「ピーク」と「エンド」

* 
**ピークの体験**: 「自分が進んでいる」という確かな感覚を持てること 。


* 
**エンドの体験**: 考えていることが明確に言語化され、次にやるべきことがはっきりしている要因 。


* **具体的な施策**:
* セッション後の15分間はスマホをいじらず、自分と向き合う「内省の時間」を設ける 。


* 熱量が冷めないうちにすぐ行動（連絡など）を起こせる「即行動ルーム」のような仕掛け 。

## Phase 3. 【結晶化】言葉の力（仕上げ）

### Q21. 一言での「再現性」

* 
**一言での説明**: 「挑戦と物語を大切にする価値観をもとに事業をやっている」 。



### Q22. 記憶に残る「違和感」

* 
**アンチテーゼ要素**: 「結果主義ではない（結果を目指すがとらわれない）」 。「ノーストレス減量」 。



### Q23. 抽象名詞から「動詞」へ

* 
**顧客満足を動詞にする**: 「変容させる」「考え方を変える」「言語化する」「語る」「自己を確立させる」 。


* 
**挑戦を動詞にする**: 「目的に向かって走り出す」「一歩踏み出す」 。



### Q24. 覚悟を問う「踏み絵」

* 
**1億円の断捨離**: 理念に反する利益なら迷わず断る 。目先のPLよりも、組織の文化や資産価値をはるかに重要視している 。


* 
**社員の守り方**: 理念に従って行動した結果の失敗なら、会社が苦しくなっても社員を守る。やりたくないことで稼ぐより、理念を貫くことのほうがはるかに「成功」だと考えている 。



### Q25. 排除する「毒」

* 
**去っていく人**: 安定志向（現状維持が安心の人）や、コンフォートゾーンが現状に留まっている人 。


* 
**腹を立てる顧客**: 「お金を払うから何かしてくれ」という受動的な人（お客様は神様タイプ） 。SFは「自ら変わりたい」という内側からの欲求がある人をターゲットとする 。



### Q26〜Q27. オーナーシップと高揚感

* 
**生身の言葉**: 自分の腹の底から出た言葉であり、嘘や後ろめたさは一切ない 。


* 
**高揚感**: 理念を口にする時、ワクワクして自然と声が大きくなるほどの感覚がある 。



### Q28〜Q29. 現場への翻訳と耐用年数

* 
**判断基準の具体化**: 何が良い行動で何が悪い行動か、現場で判断できるレベルまで落とし込む必要がある 。


* 
**耐用年数**: 「理念に基づいて行動することが人生を好きにさせる」という本質は、100年後の人間にも通じる普遍的なものであると確信している 。



### Q30. 明日からの「変革」

* 
**やめること**: 主語を「私（個人）」にすること。「自分への矢印」の向け方。自己否定 。


* 
**新しく始めること**: 主語を「私たち（シェイプ）」に変える 。


* 
**仕組みの変更**: 会議を「発散」「収束」「決定」などの目的別に型化し、名称からやるべきことが明確になるように整理する 。コミュニケーションコストを抑えつつ、価値観を積み重ねる仕組みを構築する 。
