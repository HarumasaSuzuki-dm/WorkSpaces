# 📂 02_重要ルール・リスク管理

このセクションは、チャンネルの存続とブランド価値を守るための「守り」のルールです。
編集作業におけるクリエイティブな工夫の前に、**これらが守られていない納品物は承認されません。**

---

## 🚨 2-1. 【最重要】NGチェックリスト（リスク管理）

YouTube活動において、プライバシーの流出やコンプライアンス違反は致命的なリスクとなります。
「おしりちゃん（クライアント）が気づくだろう」ではなく、**編集者の段階でリスクをゼロにする**という意識でチェックを行ってください。

### 🛡 1. プライバシー保護（モザイク処理）
ロケ撮影や移動シーンでは、以下の情報の映り込みに細心の注意を払ってください。

* **住所の特定につながる情報:**
    * **電柱:** 住所表記プレート。
    * **特徴的な建物:** 自宅付近のマンション名、特徴的な外観の店。
    * **窓からの景色:** 特徴的なランドマークが見える場合。
    * **なーすけ宅の洗濯物:** 下着や特徴的な衣類が映り込むケースに注意。
* **個人情報:**
    * **配送伝票:** 宅配便の段ボールに貼られた住所・氏名。
    * **車のナンバープレート:** 自家用車、通行人の車すべて。
    * **スマホの画面:** 通知ポップアップ、LINEのトーク画面、待ち受け画面（**しおんさんの顔**が映っている場合など）。
    * **ゆめすけの顔:** 全面的にNG（必ず隠す）。

### ⚠️ 2. コンプライアンス・セーフティ
フィットネス動画という性質上、身体の露出が多くなりますが、YouTubeのポリシーおよびご本人の尊厳を守るためにチェックが必要です。

* **下着の映り込み:**
    * **下着までOK**としますが、トレーニング動作中（ベンチプレスやスクワットなど）に意図しない見え方になっていないか、リスク管理を徹底してください。
* **意図しない肌の露出:**
    * ご本人が意図していない過度な露出や、視聴者が不快に感じる可能性のあるアングル。
* **不適切な映り込み:**
    * 鏡や窓ガラスへの反射（撮影者や関係ない人物の映り込み）。

### ✅ 3. 納品前の自己チェックフロー
書き出しを行う前に、必ず映像を最初から最後まで通して確認し、以下の項目をクリアしているか確認してください。

* [ ] 自宅周辺の電柱・建物にモザイク漏れはないか？
* [ ] 宅配便の伝票などは隠れているか？
* [ ] トレーニング中の際どいアングルに配慮ができているか？
* [ ] モザイクの追従は外れていないか？（動きのあるシーン注意）

> **Point:**
> 「怪しいかな？」と思ったら、**迷わずモザイク**をかけてください。安全側に倒すのが鉄則です。

## 📝 2-2. 表記ルール・用語集

動画内のテロップにおける表記揺れ（バラつき）を防ぎ、ブランドとしての統一感を保つためのルールです。
VREWでの文字起こし修正時や、FCPXでのテロップ入力時に必ず参照してください。

### 🔡 1. 固有名詞・ブランド名の指定
企業名やスポンサー様のお名前は間違えると失礼にあたります。一字一句正確に記載してください。

| 対象 | 正しい表記 (OK) | 間違いやすい表記 (NG) |
| :--- | :--- | :--- |
| **チャンネル名** | **なーすけ Fitness** | なーすけフィットネス、なーすけFitness（半角スペースなしはNG） |
| **人物名** | **なーすけ** | ナースケ |
| **人物名** | **嫁すけ** | 嫁スケ |
| **人物名** | **ひーすけ** | ヒースケ |
| **人物名** | **寺島さん** | てらしまさん |
| **スポンサー** | **VITAS** | バイタス |
| **スポンサー** | **VEATM** | ピートム、Veatm |

### 🗣 2. 一般用語の統一ルール
読みやすさと柔らかさを出すために、ひらがな表記を推奨している言葉があります。

| 対象 | 正しい表記 (OK) | 間違いやすい表記 (NG) |
| :--- | :--- | :--- |
| **形容詞** | **すごい** | 凄い |
| **人数** | **1人ひとり** | 1人1人、一人一人 |
| **単位** | **g、kg** | グラム、キロ |
| **成分** | **PFC** | ピーエフシー |
| **掛け声** | **イキヤショウ！** | いきましょう |

### 🍽 3. フル食企画の「カロリー表記」ルール
フル食動画において、視聴者が**「自分も真似してみよう（再現性）」**と思えることがゴールです。
厳密な栄養計算よりも、「動画内の発言との整合性」と「わかりやすさ」を優先します。

#### 基本方針
* 計算上で商品が特定できない場合でも、**動画内でなーすけ様が発言した数値**に合わせて記載する。
* 誰が見ても明らかな間違い（例：映像は魚1切れなのに、テロップが1.2切れ分の数値になっている等）以外は、発言を優先してOK。
* あくまで「参考程度」の表記であることを前提とする。

#### グラム(g)表記するもの・しないもの
視聴者がスーパー等で購入して**再現しやすいものだけ**詳細に表記します。

* **詳細表記する（g数を入れる）:**
    * 米、肉（自炊用）、魚、卵、プロテイン粉末。
* **詳細表記しない（パッケージのままでOK）:**
    * 市販のパッケージ食品（お菓子、弁当など）。
    * 理由：視聴者がわざわざ「お菓子を○g測って食べる」ことはしないため。
