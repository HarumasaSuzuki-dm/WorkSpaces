# SFF RACI（完全版 v1.1）

> v1.1 変更方針
>
> * R / A / C / I の割当は **v1から変更なし**
> * 各行の「この行でやること」「運用メモ」に **判断停止を防ぐ追記のみ** を反映
> * 目的：
>
>   * 統括（かおんさん）の判断疲れを減らす
>   * 実行側の“どこまで任されているか”を明確にする
>   * RACIを“回せる道具”にする

---

## フェーズ全体像

| フェーズ         | 目的      | いつまでに     | 決める粒度  |
| ------------ | ------- | --------- | ------ |
| Phase1：設計確定  | 判断を軽くする | 今すぐ〜準備開始前 | 不可逆判断  |
| Phase2：実行準備  | 動かせる状態に | 大会3–4ヶ月前  | 実行RACI |
| Phase3：実行・当日 | 止めずに回す  | 大会直前〜当日   | 当日判断   |
| Phase4：事後対応  | 回収・改善   | 大会後       | 振り返り   |

## 補足（運用原則）

* RACIは**完全性より運用性**を優先する
* 迷ったら「運用メモ」に立ち返る
* 不安が出た行だけを次回修正対象とする
