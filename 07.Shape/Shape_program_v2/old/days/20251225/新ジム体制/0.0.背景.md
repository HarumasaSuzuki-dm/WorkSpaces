# プロジェクト「ShapeFit（SFF）」組織変革・運営モデル再構築 概要資料

## 1. プロジェクトの背景と目的
代表であるなーすけ氏の属人的なマネジメントが限界に達しており、組織のスケール（拡大）に向けて抜本的な構造改革が必要です。特に、初期から低ロイヤリティで参画しているフランチャイズ（FC）オーナーたちの「テイカー（もらうだけ）志向」が顕在化しており、ブランド価値を高める「共創」のフェーズに進めないことが最大の障壁となっています。
本プロジェクトでは、1月を「テスト期間」と位置づけ、なーすけ氏個人に依存しない「仕組みによる統治」への移行と、不適合なパートナーの選別・運営形態の変更を断行します 。

## 2. 現状（As-is）と理想（To-be）の詳細比較

| 項目 | As-is：現状の課題 | To-be：1月末の目標姿 |
| :--- | :--- | :--- |
| **オーナーのマインド** | 本部からの利益供与（GIVE）を当然視し、貢献意欲が低い | ブランド価値向上のための「共創パートナー」として自立する |
| **指揮系統** | なーすけ氏の発言しか聞かず、他の運営スタッフを軽視する | 事務局（スタッフ）が決めた「ルール」や「アジェンダ」を遵守する  |
| **会議の意義** | 損得勘定で参加を判断し、オフライン参加を渋る傾向 | 相互研鑽とブランド強化のための必須の場として機能する |
| **スケーラビリティ** | 代表が全店舗を直接管理しており、これ以上の拡大は不可能  | 代表は「承認者」に徹し、現場は仕組みで自律的に回る状態 |
| **運営モデル** | 契約上の縛りが弱く、マインドの低いオーナーを排除できない | ライセンス型など、本部が統制権を持つ多様な形態への転換 |

## 3. 本質的な潜在課題の分析
* **構造的欠陥**: 「低ロイヤリティ」が「安く看板を借りる権利」と誤解され、オーナーに経営者としての当事者意識が芽生えていない。
* **属人性の弊害**: なーすけ氏のカリスマ性に依存しすぎた結果、スタッフ（カオン氏、シオン氏等）の権威が確立されておらず、マネジメントの手が離れない 。
* **ミスマッチの放置**: 改善の余地が低いオーナー（直る見込みが薄い層）に対しても、具体的な撤退基準がないまま対話を続けてしまっている。

## 4. 論点およびサブ論点（MECE）

### 論点1：運営・契約モデルの最適化
* **形態の再定義**: 現行FCの継続可否判断、および「ライセンス形式」等への移行手段の特定。
* **経済条件の見直し**: 「共創」を前提としたロイヤリティ体系、またはサービス参画条件の再設計。
* **撤退シナリオの確立**: 1月末時点での「不適合オーナー」に対する契約解除、または直営化（買取り）の判断基準 。

### 論点2：組織統治（ガバナンス）の仕組み化
* **意思決定ラインの委譲**: 会議の進行、質疑応答、日常の指示をなーすけ氏から事務局スタッフへ完全移行する 。
* **判断の標準化**: 個人の裁量ではなく、SFF独自の「ルールブック」や「Q&A」に基づいた一貫性のある対応 。
* **モニタリング体制**: 各店舗のオーナーがルールを遵守しているかを可視化する「チェックリスト」の運用。

## 5. 1月始動：戦略的アクションプラン（1月6日〜31日）
1月6日からの週次ミーティング（15〜30分）を「踏み絵」の期間として活用します。

1. **新方針の「最後通告」**: 冒頭で「2026年はプロ組織化する。ついてこれない者は船を降りてもらう」という決意をなーすけ氏より表明。
2. **スタッフ主導の徹底**: なーすけ氏は会議で沈黙を守り、スタッフが司会進行・指示を行う「仕組み化」のテスト 。
3. **継続可否の判定（1月末）**: コミュニケーションが困難、または前進が見られない店舗を特定し、サービス継続の判断を下す。
