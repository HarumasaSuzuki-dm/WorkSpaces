# 📝 SFF会議・運営ガイド（保存版）

## 1. 会議のゴール

なーすけさんがいなくても、 **「決まったルール」** でテキパキとお店が良くなっていく状態を目指します。

* **事務局の立ち位置**：なーすけさんの代わりにルールを守らせ、判断を下す「リーダー」です。
* **参加者の心得**：教えてもらうのを待つ「お客さん」ではなく、一緒にブランドを育てる「プロのパートナー」として参加してもらいます。

---

## 2. 会議のタイムライン（60分）

| 時間 | 項目 | やること・チェック内容 |
| --- | --- | --- |
| **05分** | **ルールの確認** | 遅刻、資料の出し忘れがないか。あればその場で厳しく注意します。 |
| **10分** | **数字と宿題** | お店の数字（売上など）と、前回決めた宿題が終わっているか確認します。 |
| **20分** | **現場の状況** | 「ノーストレス減量」が正しく伝わっているか。現場を見ていない人の意見は流します。 |
| **20分** | **お悩み解決** | 困りごとに対し、事務局が「ルール」に沿って解決策を伝えます。 |
| **05分** | **次の約束** | 「誰が・いつまでに・何をやるか」を全員で確認して解散です。 |

## 3. 【重要】絶対に許さない「NG行動」

以下の行動が出たら、事務局はすぐに会議を止めて注意してください。

### ① 攻撃的なやり取り（いじめ・攻撃禁止）

* イヤミ、バカにする、大声で威圧する。
* 自分より立場が弱いスタッフをターゲットにする。
* 非を認めず、言い負かそうとする。

### ② ルール無視（当たり前のことをやる）

* 遅刻、直前キャンセル、連絡なし。
* アンケートや資料を出さない、返信をしない。
* 自分（オーナー）が現場に行かず、状況を知らないのに口だけ出す。

### ③ 「してくれ」ばかりの考え（テイカー禁止）

* 「〇〇のせいで進まない」と環境や人のせいにし、自分で解決しようとしない。
* 「意味がないから行かない」「作ってくれないからやらない」という態度の人。

## 4. 事務局が現場で使う「武器（返し技）」

会議を壊そうとする人が出たとき、こう言い返しましょう。

* **感情的に攻撃されたら**：
「感情的な言葉はやめてください。ブランドを良くするための建設的な話をしましょう」
* **他人のせいにし始めたら**：
「環境のせいにするのはやめましょう。今の状況で、あなたが良くするために『出せる価値（Give）』は何ですか？」
* **現場を知らずに口を出されたら**：
「現場を直接見ていない意見は、判断を狂わせるので採用できません」

## 5. 終わった後の「見極め評価」

会議のあと、事務局メンバーで以下の「3つの安心」ができているか確認しましょう。

1. **【人の安心】**：現場のトレーナーが前向きに動けているか？
2. **【物の安心】**：マニュアル通りに指導ができているか？
3. **【金の安心】**：売上が落ちない工夫ができているか？

> **メモ**：もし「ルールを守らない」「他人のせいにする」人がいたら、その人は「SFFの船から降りてもらう候補」として記録しておいてください。
