# 【SFF物語ノート】デザイン構成ラフ・指示書（決定版 Rev.2）

**■ 基本仕様**

* **サイズ:** B5（閉じた状態） / B4（見開き）
* **綴じ方:** 中綴じ（P12のみ袋とじ加工が入るため製本業者と要相談）
* **必須ギミック:** **しおり紐（スピン）を1本つける**（P6-7のすごろくページに挟むため）
* **デザインテーマ:** 「余白」「安心感」「絵本のような世界観」
* **フォント指定:**
* 本文：手書き風フォント、または丸ゴシック体（親しみやすさ重視）


* 見出し：明朝体ベースだが角が丸いもの、または手書き風


* **共通あしらい:**
* 
**おしりちゃん（マスコット）:** 随所に登場させ、ガイド役にする。


* 
**色味:** ペールトーン（淡い色）基調。

## **【P12】 エピローグ（袋とじ）**

**■ 構成イメージ**

```text
+--------------------------------------------------+
|  Epilogue (Fukuro-toji)                          |
|                                                  |
|  ========  [CAUTION]  =========                  |
|  ここから先は、全ての物語が終わるまで            |
|  開けないでください。                            |
|  ==============================                  |
|                                                  |
|  (以下、袋とじの内側の内容)                      |
|                                                  |
|  +--------------------------------------------+  |
|  | [手書きの手紙デザイン]                     |  |
|  |  挑戦したあなたへ                          |  |
|  |  (中略：なーすけさんからの熱いメッセージ)  |  |
|  |                      なーすけ              |  |
|  +--------------------------------------------+  |
|                                                  |
|        To be continued...                        |
|        Next Story is yours.                      |
|                                                  |
|        [ 次の夢を書く空白のスペース ]            |
+--------------------------------------------------+

```

* **デザイナーへの指示:**
* 
**重要仕様:** このページは**「袋とじ」**にします。


* 外側には「終わるまで開けないで」という注意書きのみを記載。
* 内側には、なーすけさんの直筆手紙（印刷）と、次の物語への招待状を配置してください。
