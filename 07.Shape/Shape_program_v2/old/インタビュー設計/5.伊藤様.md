# インタビュー設計書：香 (CHRO) 様

## 1. インタビューの目的 (Objectives)

1.  **「聖母型マネジメント」からの脱却**: メンバーのミスやタスクの隙間を「私がやればいい」と回収してしまう構造的・心理的要因を特定し、組織として自立させるための境界線（Line）を引く。
2.  **抽象的理念の「制度化」**: なーすけCEOの掲げる「挑戦」や「在り方」という抽象的な概念を、評価制度や採用基準といった**「具体的な人事システム」**にどう落とし込むか、その接続点を探る。
3.  **専門領域へのリソース集中**: 「雑務・調整」に忙殺されている現状 から、本来の強みである「コーチング・組織開発」 へ時間をシフトさせるための廃止・委譲タスクを合意する。

## 2. タイムラインと構成 (60min)

  * **00-05分**: アイスブレイク・趣旨説明（鈴木より：「香さんが『みんなのお母さん』役で疲弊するのを終わらせ、組織の『建築家』になるための時間です」）
  * **05-20分**: 【Being】役割定義と「守りたいもの」（CHROとしてのコア）
  * **20-35分**: 【Logic】「バッファ（尻拭い）」の構造解析（R\&R/Issues）
  * **35-50分**: 【Style】評価基準とコーチング・スタイル（Values/OS解析）
  * **50-60分**: クロージング（鈴木からの解決策提示）

## 3. 質問リスト詳細 (Interview Script)

### Part 1: 【Being】役割定義とミッション

**狙い:** 彼女が組織に対して抱いている「責任の正体」を解き明かし、CHROとして守るべき領域を再定義する。

| No. | 質問項目 (Question) | 深掘り・意図 (Intent & Deep Dive) |
| :--- | :--- | :--- |
| **Q1** | **「香さんの今の立ち位置を、人事や役員という言葉を使わずに『職業名』で例えるなら何ですか？」**<br>（例：お母さん、保健室の先生、神社の巫女、弁護士…） | **[自己認識の確認]**<br>・「ケアする人（母）」という認識か、「規律を作る人（法）」という認識か。<br>・現状は前者に偏りすぎている仮説を検証。 |
| **Q2** | **「この3ヶ月で、メンバー（紫音さんやスタッフ）の変化を見て、一番『私がいて良かった』と涙が出るほど嬉しかった瞬間はいつですか？」** | **[モチベーションの源泉]**<br>・人の成長（定性）に喜びを感じるか、組織の安定（定量）か。<br>・ここが評価制度設計の「コア（核）」になる。 |
| **Q3** | **「逆に、組織を見ていて『このままではShapeらしさが死んでしまう』と危機感を感じていることは何ですか？」** | **[文化の防衛ライン]**<br>・急拡大による「関係性の希薄化」や「ビジネス偏重」への懸念を引き出す。<br>・それを防ぐのが「制度」であると動機づけする。 |

### Part 2: 【Logic】「バッファ」の構造解析と解消

**狙い:** なぜ彼女にボールが集まるのか。**「組織図のバグ」**を特定し、鈴木が修正するための証拠を集める。

| No. | 質問項目 (Question) | 深掘り・意図 (Intent & Deep Dive) |
| :--- | :--- | :--- |
| **Q4** | **「正直、『これ、私じゃなくて神保さんや現場がやるべきだよね』と思いながら、つい手を出してしまっている業務（尻拭い）は何ですか？」** | **[業務の棚卸しと犠牲]**<br>・具体的なタスク（日程調整、クレーム対応、メンバーのメンタルケア）を列挙。<br>・それが「仕組みの欠如」によるものか確認。 |
| **Q5** | **「上記（Q4）のボールが落ちそうな時、なぜ『拾わない（失敗させる）』という選択ができないのですか？」**<br>（A: 顧客に迷惑がかかる、B: メンバーが可哀想、C: なーすけさんが怒る、D: 誰も気づいていない） | **[ボトルネックの真因]**<br>・Bの場合、それは「優しさ」ではなく「成長阻害」であるとリフレーミングが可能。<br>・CやDの場合、情報流通の仕組みに問題がある。 |
| **Q6** | **「もし、香さんが1ヶ月休暇を取ったら、組織の中で『誰と誰の仲』が悪くなる、あるいは連携が切れると思いますか？」** | **[隠れた人間関係の調整コスト]**<br>・組織図にはない「裏の調整ライン」を可視化する。<br>・その調整機能を「会議体」や「ルール」に置き換える。 |

### Part 3: 【Style】評価基準とコーチング・スタイル

**狙い:** 「なんとなく良い人」を評価するのではなく、**「Shapeとして評価すべき行動」**を言語化し、人事評価のプロトタイプを作る。

| No. | 質問項目 (Question) | 深掘り・意図 (Intent & Deep Dive) |
| :--- | :--- | :--- |
| **Q7** | **「今のShapeにおいて、売上を作っていなくても『この人は最高にShapeらしい（評価したい）』と思う行動は具体的にどんな行動ですか？」** | **[コア・バリューの具体化]**<br>・「挑戦して失敗した人」か「裏で支えた人」か。<br>・これを明文化したものが「評価シート」の項目になる。 |
| **Q8** | **「香さんがコーチングや面談をする際、相手の『どこ』を見て、その人の状態（良し悪し）を判断していますか？」** | **[不当な優位性 (Unfair Advantage)]**<br>・言葉の選び方？ 声のトーン？<br>・その「眼力（Insight）」を、組織の採用基準や配置基準に転用できるか検証。 |
| **Q9** | **「今後、法人研修 で香さんのリソースを『対外的なコーチング』に使うために、社内業務の『何を捨てる（諦める）』覚悟ができますか？」** | **[リソース配分の意思決定]**<br>・全てを守ることは不可能。<br>・「事務作業」を捨てるのか、「全メンバーとの1on1」を減らすのか、優先順位を問う。 |
| **Q10** | **「鈴木（私）に、組織の『冷徹な仕組みづくり（ルール化・評価制度）』を任せることに、どれくらい心理的な抵抗がありますか？」** | **[変化への受容度]**<br>・「管理されたくない」という文化的な拒否反応の有無を確認。<br>・「ルールは縛るものではなく、迷いを消すもの」という合意形成を行う。 |

## 4. インタビュアー（鈴木）へのメモ

  * **スタンスの取り方**:
      * 鈴木は**「組織の構造設計士」**として接してください。「香さんの『優しさ』が、今の組織の『弱さ』になっている可能性があります。その優しさを、属人的なケアではなく『制度』として実装しましょう」という、少し厳しいが愛のある提言が必要です。
  * **確認すべき「表情」**:
      * **Q5（なぜ拾うのか）**: ここで「私がやらないと回らないから…」と悲壮感が出る場合、彼女は**「自己犠牲による貢献」**に自身の存在意義を感じてしまっている危険性があります（共依存）。
          * **対策**: 「あなたがやることは、メンバーから『失敗して学ぶ機会』を奪っていることになります」と伝え、**「見守る勇気」**を持つことへの許可を出してください。
  * **深掘りのコツ**:
      * 「文化」や「空気感」という言葉が出たら、**「その空気が悪い時って、具体的にオフィスでどんな会話や行動が起きていますか？」**と、映像として見えるレベルまで落とし込んでください。それが「行動指針（Code of Conduct）」になります。
