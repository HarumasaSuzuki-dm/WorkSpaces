### 論点1：【財務・数値定義】現状の「正確な現在地」と「安全な投資余力」の確定

まず「何が現実か」を数字で固めるタスク群です。これがないと全ての計画が画餅になります。

* **1-1. 固定費（分母）の確定**
  * [ ] **地代家賃の集計**: 店舗および本社オフィスの賃貸借契約書を確認し、月額家賃（共益費込）をリスト化する。
  * [ ] **本社人件費の集計**: 現場（トレーナー・店舗スタッフ）を除く、本社機能（経営・バックオフィス）にかかる人件費（社会保険料込）を算出する。
  * [ ] **その他固定費の洗い出し**: 家賃・人件費以外で毎月必ず発生する固定費（システム利用料、顧問料等）がないか確認し、あれば分母に加えるか判断する。


* **1-2. 継続収益（分子）の現状把握**
  * [ ] **ジム売上（月額）の抽出**: 会員管理システムより、直近3ヶ月の月額会費の入金実績データを抽出する。
  * [ ] **コミュニティ会費の抽出**: オンラインサロン等のプラットフォームから、直近3ヶ月の会費入金実績データを抽出する。
  * [ ] **現状カバー率の算出**: `(ジム売上 + コミュ会費) ÷ (家賃 + 本社人件費)` を計算し、49.6%という現状値の精緻化を行う。


* **1-3. 投資余力（現預金）の実態解明**
  * [ ] **前受金の内訳分解**: 貸借対照表の「前受金（18,646千円）」を、「SFFチケット分」「ジム未消化分」「その他」に明細単位で分解する。
  * [ ] **役務提供スケジュールの可視化**: 分解した前受金が、いつの売上（役務提供）として消化される予定か、月別の消化予測表を作成する。
  * [ ] **実質的フリーキャッシュの算出**: 現預金残高から「前受金（返金リスク/消化義務あり）」「未払金」「納税予定額」を差し引き、投資に回せる上限額を算出する。


* **1-4. YouTube事業の貢献利益確定**
  * [ ] **YouTube関連コストの集計**: 外注費、機材費、出演費など、YouTube制作に直接紐づく経費をすべて洗い出す。
  * [ ] **社内内部委託費の整理**: YouTube制作に関わる社内リソースの工数を概算し、費用換算する（管理会計上の把握）。
  * [ ] **貢献利益の算出**: `YouTube売上 - (外部経費 + 内部コスト)` で、実際に他事業へ回せる利益額（目標3.5M/月）の現状値を確定する。

### 論点2：【運用・ガバナンス】「理念を守りながら数字を追う」会議体とKPI設計

数字が出た後に、それをどうモニタリングし判断するかという「仕組み」を作るタスク群です。

* **2-1. KPI定義と計測環境の整備**
  * [ ] **KPI定義書の作成**: 各指標（カバー率、教育機能回復度、予約枠充足率など）について、「計算式」「データ参照元」「更新担当者」「更新頻度」をドキュメント化する。
  * [ ] **ダッシュボード/管理表の作成**: 定義に基づき、毎週数値を入力・閲覧するためのスプレッドシート（またはツール）を作成する。
  * [ ] **異常値アラート基準の設定**: 各KPIについて「ここを下回ったら即対策会議」という撤退ライン・アラートラインを決める。


* **2-2. 会議体の設置とアジェンダ化**
  * [ ] **週次会議（30分）の設定**: カレンダーに予定を入れ、参加者を招待する。アジェンダは「指標の異常値チェック→打ち手決定」に固定する。
  * [ ] **月次会議（90分）の設定**: 投資判断と財務確認を行う会議を設定する。
  * [ ] **会議資料テンプレートの作成**: 毎回ゼロから作らなくて済むよう、KPI記入欄と議事録欄がセットになったフォーマットを用意する。

### 論点3：【教育・品質】「マニュアル100%」と「トレーナー増員」の実行準備

投資の最優先事項である「人」と「仕組み」に関する具体的準備です。

* **3-1. マニュアル化プロジェクトの始動**
  * [ ] **業務棚卸しと目次作成**: 接客、トレーニング指導、事務処理など、必要な業務を全項目リストアップし、マニュアルの「全体目次」を作る。
  * [ ] **現状の充足度チェック**: 目次に対し「既存マニュアルあり」「修正必要」「新規作成必要」のステータスを振る。
  * [ ] **作成担当と納期の割り当て**: 不足部分について、誰がいつまでに書くか（または動画を撮るか）を決定する。
  * [ ] **「ありたい姿(Be)」要素の組み込み**: 手順だけでなく「なぜやるか」「どういうスタンスで接するか」というVision要素をマニュアル冒頭に入れる規定を作る。


* **3-2. 採用・育成計画の策定**
  * [ ] **採用要件定義（ペルソナ）の作成**: スキルだけでなく、ShapeFitの理念（結果より状態）に共感できる人物像を言語化する。
  * [ ] **採用ロードマップの作成**: 3名→7名にするための「募集開始日」「面接期間」「入社日」「研修期間」のスケジュールを引く。
  * [ ] **研修カリキュラムの設計**: 入社後、どのマニュアルを使って何週間でデビューさせるかの標準スケジュールを作成する。

### 論点4：【投資実行】いつ、何に、いくら投資するかのルール化

お金を使う前の「ガードレール」を作るタスク群です。

* **4-1. 投資ルールの明文化**
  * [ ] **安全在庫月数の決定**: 「固定費の何ヶ月分を現預金として死守するか（例：6ヶ月分）」を決定し、金額換算する。
  * [ ] **投資優先順位の周知**: 「①教育 ②採用 ③拠点 ④コミュ」という順位を経営陣・マネージャー間で合意し、ドキュメントに残す。


* **4-2. 投資稟議プロセスの簡易設計**
  * [ ] **投資判断基準の策定**: 「教育マニュアルが進んでいないなら内装投資はしない」といった条件分岐（IF-THENルール）を書き出す。
  * [ ] **簡易稟議フォーマットの作成**: 投資案件が発生した際、「目的」「金額」「回収仮説（どのKPIに効くか）」を記載する1枚シートを作る。

### 論点5：【マーケティング・還流】単発を継続へ変換する導線設計

各事業をつなぐパイプを通すためのタスク群です。

* **5-1. YouTubeからの流入計測**
  * [ ] **アンケート項目の改修**: ジムの体験申込フォームに「YouTubeを見て」という選択肢だけでなく、具体的な「どの動画・企画を見て」かを聞く項目を追加する。
  * [ ] **視聴者限定オファーの検討**: 動画内で案内する「ジム体験」や「コミュニティ参加」への導線（URL/QR）が機能しているか確認する。


* **5-2. コミュニティ還流と定着**
  * [ ] **SFF後の案内フロー作成**: SFF終了直後の熱量の高い時期に、どのようにコミュニティへ誘導するか（メール、イベント等）の工程を作る。
  * [ ] **コミュニティKPIの計測開始**: 「新規流入数」「退会数」「アクティブ率」などのデータを月次で取れるよう、プラットフォームの管理画面や集計方法を確認する。


* **5-3. 社内言語化の実践**
  * [ ] **社内ワークショップの日程調整**: スタッフ自身が「自分のありたい姿」を言語化する時間を確保するため、業務時間を調整して日程を決める。
