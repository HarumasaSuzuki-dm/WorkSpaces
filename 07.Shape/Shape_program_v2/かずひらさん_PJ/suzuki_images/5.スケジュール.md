# 週次・月次の固定スケジュール（運用テンプレ）

### 週次（毎週）

* 月：未提出者リスト→復帰導線（テンプレ送付）
* 火〜金：マネージャー伴走（整理→次の一手→提出）
* 週末：チェックイン提出集計→フィードバック

### 月次（毎月）

* なーすけ講義（月1〜2回）
* KPIレビュー（提出率・継続率・離脱理由・クレーム/事故）
* 運用規定・テンプレ更新（小さく速く）
