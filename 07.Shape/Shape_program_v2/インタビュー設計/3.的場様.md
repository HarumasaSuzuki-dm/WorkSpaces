# インタビュー設計書：的場様 (CMO)

## 1. インタビューの目的 (Objectives)

1.  **「ブラックボックス」の特定**: マーケティング戦略を立てる上で、現在\*\*「欠落しているデータ（現場の数値、顧客の声、LTV等）」\*\*を洗い出し、ダッシュボード構築の要件にする。
2.  **承認プロセスの整流化**: クリエイティブや施策実行において、誰の確認待ちで時間が溶けているか（なーすけ様の感性チェック等）を特定し、権限委譲ラインを探る。
3.  **B2Bシフトへの準備**: 今後の「法人研修」展開 にあたり、現在のB2C（対個人）マーケティングのOSをどうアップグレードするか、リソースの懸念点を共有する。

## 2. タイムラインと構成 (60min)

  * **00-05分**: アイスブレイク・趣旨説明（鈴木様より：「的場さんの『感性』を最大化するために、裏側の『情報パイプライン』を私が整備します」）
  * **05-20分**: 【Being】役割定義と「勝ちパターン」（CMOとしてのコア）
  * **20-35分**: 【Logic】情報の非対称性と「欲しいデータ」（ガバナンス/連携）
  * **35-50分**: 【Style】仕事のスタイル・B2Bへの転換（スキル/OS解析）
  * **50-60分**: クロージング（鈴木様からの解決策提示）

## 3. 質問リスト詳細 (Interview Script)

### Part 1: 【Being】役割定義とミッション

**狙い:** 的場様が考える「マーケティングの定義」を確認し、単なる「集客屋」ではなく「市場価値の創造者」としての立ち位置を確立する。

| No. | 質問項目 (Question) | 深掘り・意図 (Intent & Deep Dive) |
| :--- | :--- | :--- |
| **Q1** | **「的場さんの今の役割を、マーケティング用語以外の『職業名』で例えるなら何ですか？」**<br>（例：翻訳家、ストーリーテラー、扇動者、狩人…） | **[自己認識の確認]**<br>・資料の「感情と数値を接続する」 という期待値に対し、ご自身がどちら（感情寄りor数値寄り）に軸足を置いているか確認。 |
| **Q2** | **「過去の施策の中で、一番『美しい（Shapeらしい）』成功をした事例はどれですか？また、それはなぜ成功したと分析していますか？」** | **[勝ちパターンの言語化]**<br>・数字（CPA/ROAS）が良かったからか、反響（定性的な熱量）が凄かったからか。<br>・Shapeにおける「Marketing Success」の定義を握る。 |
| **Q3** | **「逆に、やっていて一番『虚無感』を感じる、あるいは『これは私の仕事ではない』と感じる業務は何ですか？」** | **[リソースの無駄特定]**<br>・単純なSNS運用代行になっていないか。<br>・戦略策定以外の「作業」に埋没していないか確認。 |

### Part 2: 【Logic】情報の非対称性と連携不全

**狙い:** **鈴木様が構築する「情報連携システム」の要件**を引き出すパート。現場（SFF/ジム）とマーケティングの断絶を埋める。

| No. | 質問項目 (Question) | 深掘り・意図 (Intent & Deep Dive) |
| :--- | :--- | :--- |
| **Q4** | **「現在、施策を打つ上で『一番見えていないデータ（ブラックボックス）』は何ですか？」**<br>（例：入会後の継続率、退会理由、イベント参加者の熱量、LTV…） | **[情報欠損の特定]**<br>・ここが埋まればマーケティング精度が上がる。<br>・CRM/SFA導入（Salesforce等）の最重要要件となる。 |
| **Q5** | **「なーすけさんの『思いつき（Vision）』が降ってきた時、マーケティング責任者として一番困ることは何ですか？」** | **[ハレーションの実態]**<br>・「ターゲットの急な変更」か「スケジュールの短縮」か。<br>・その際、なーすけ様とどう交渉しているか（または言いなりか）。 |
| **Q6** | **「SFF（紫音さん）やジム（トレーナー）との連携で、もっと『こういう情報さえくれれば、もっと売れるのに』と思うことはありますか？」** | **[部門間連携の課題]**<br>・受動的な連携 から能動的な連携へ。<br>・「現場の『生の声』が届かない」等の課題を抽出。 |

### Part 3: 【Style】B2B展開とスキルセット

**狙い:** 今後の「法人研修」展開に向けた、マーケティングOSの切り替えと個人の強み確認。

| No. | 質問項目 (Question) | 深掘り・意図 (Intent & Deep Dive) |
| :--- | :--- | :--- |
| **Q7** | **「今後、法人向け（B2B）のマーケティング が始まるとしたら、今の体制で『何が足りない』と思いますか？」** | **[B2B対応力の検証]**<br>・リード獲得手法、ホワイトペーパー制作、決裁者へのアプローチ等のノウハウ有無。<br>・「感性（B2C）」と「論理（B2B）」の使い分けへの懸念。 |
| **Q8** | **「的場さんが、他のマーケターよりも圧倒的に『速く』あるいは『深く』できることは何ですか？」** | **[不当な優位性 (Unfair Advantage)]**<br>・コピーライティングの速さ？ 顧客心理の洞察？<br>・その強みを活かすために、誰（アシスタント等）が必要か。 |
| **Q9** | **「クリエイティブの承認フローにおいて、現状『ボトルネック（止まる原因）』になっているのは誰、あるいはどのプロセスですか？」** | **[意思決定スピードの阻害要因]**<br>・なーすけ様の「感覚チェック」で差し戻しが多い場合、事前に「判断基準（トンマナ）」を言語化する必要がある。 |
| **Q10** | **「もし明日、予算が無制限に使え、面倒な事務作業もゼロになったら、まずどんな手を打ちますか？」** | **[潜在的な戦略案]**<br>・リソース不足で諦めている「ホームラン級の施策」を引き出す。<br>・鈴木様のロードマップに組み込むネタにする。 |

## 4. インタビュアー（鈴木様）へのメモ

  * **スタンスの取り方**:
      * 鈴木様は\*\*「ロジカルな触媒」\*\*として振る舞ってください。「的場さんのクリエイティブな施策が、数字（PL）や現場のオペレーションと繋がっていないのはもったいない。私がその回路を繋ぎます」というアプローチが有効です。
  * **確認すべき「表情」**:
      * **Q4（見えていないデータ）**: ここで具体的なデータ名（LTV、CPA等）が即答される場合、的場様は「数字に強いマーケター」です。逆に「現場の空気感」等の定性情報が出る場合、「感性重視」です。
          * **数値重視の場合**: Dashboard構築を急ぐと伝えてください。
          * **感性重視の場合**: 「その感性を他者に説明するための『翻訳データ』を用意します」と伝えてください。
  * **深掘りのコツ**:
      * 「SFFとの連携がうまくいっていない」という話が出たら、\*\*「それは『定例会議』がないからですか？それとも『共通言語（KPI）』がないからですか？」\*\*と、構造的原因（会議体 or 基準）に切り分けてください。
