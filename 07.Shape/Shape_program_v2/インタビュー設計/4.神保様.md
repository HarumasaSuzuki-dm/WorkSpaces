# インタビュー設計書：神保様 (Ops/Admin)

## 1. インタビューの目的 (Objectives)

1.  **「受け身」からの脱却要因の特定**: 能力的にはもっと任せられるはずだが、なぜ現在「指示待ち」あるいは「タスク処理」に留まっているのか。その阻害要因（情報の欠落、遠慮、権限不足）を特定する。
2.  **B2B対応力のストレステスト**: 今後、法人研修（B2B）が始まった際、現在のバックオフィス（契約・請求・顧客対応）が耐えられるか、あるいはパンクするかを検証する。
3.  **「温度のある管理」の定義**: 「数値を扱いながらも温度を伝える」 という期待値に対し、具体的にどのような行動で貢献できるか、独自の強みを言語化する。

## 2. タイムラインと構成 (60min)

  * **00-05分**: アイスブレイク・趣旨説明（鈴木より：「リーダー層のパンクを防ぎ、神保さんがより『攻めのバックオフィス』になるための作戦会議です」）
  * **05-20分**: 【Being】役割定義と「架け橋」としてのプライド
  * **20-35分**: 【Logic】業務巻き取りとボトルネック（R\&R/Delegation）
  * **35-50分**: 【Style】B2B対応と仕事のスタイル（Skill/OS解析）
  * **50-60分**: クロージング（鈴木からの期待値調整）

## 3. 質問リスト詳細 (Interview Script)

### Part 1: 【Being】役割定義とミッション

**狙い:** 単なる「事務代行」ではなく、Shapeの文化を理解した「パートナー」としての自覚と、発揮したいバリューを確認する。

| No. | 質問項目 (Question) | 深掘り・意図 (Intent & Deep Dive) |
| :--- | :--- | :--- |
| **Q1** | **「神保さんの現在の役割を、事務や経理という言葉を使わずに『職業名』で例えるなら何ですか？」**<br>（例：守護神、翻訳者、潤滑油、外交官…） | **[自己認識の確認]**<br>・資料にある「実務＋共感の架け橋」 という期待値をどう解釈しているか。<br>・「守り」の意識が強いか、「攻め（組織改善）」の意識があるか。 |
| **Q2** | **「Shapeに関わっていて、事務的な処理が完了した時以上に『このチームにいて良かった』と熱くなった瞬間はいつですか？」** | **[モチベーションの源泉]**<br>・「温度を伝える」ことへの適性を確認。<br>・数字だけでなく、メンバーの感情やビジョンにどう共鳴しているか。 |
| **Q3** | **「逆に、現状のチームの動きを見ていて『もったいない』『もっとこうすれば楽なのに』と歯痒く感じていることはありますか？」** | **[潜在的な改善提案]**<br>・遠慮して言えていない「業務改善案」を引き出す。<br>・ここが「巻き取り」の第一候補となる。 |

### Part 2: 【Logic】業務巻き取りとボトルネック

**狙い:** 紫音様や香音様が抱えている業務 を、神保様側に移動させるための具体的条件（トリガー）を探る。

| No. | 質問項目 (Question) | 深掘り・意図 (Intent & Deep Dive) |
| :--- | :--- | :--- |
| **Q4** | **「正直、紫音さんや香音さんが忙しそうにしている作業の中で、『それ、権限さえくれれば明日から私がやりますよ』と思っているものはありますか？」** | **[委譲の受け皿確認]**<br>・具体的に（SFFのシフト調整、顧客対応、請求書発行など）挙げてもらう。<br>・「言われたらやります」か「奪い取りたい」か、姿勢を確認。 |
| **Q5** | **「もし上記（Q4）の業務を巻き取るとして、今『一番足りないもの』は何ですか？」**<br>（A: 具体的なマニュアル、B: なーすけさんの判断基準、C: アクセス権限、D: 失敗した時の免罪符） | **[阻害要因の特定]**<br>・Aなら鈴木が作成支援。<br>・Bなら判断基準の言語化。<br>・Dなら心理的安全性の確保。解決策を分岐させる問い。 |
| **Q6** | **「現在、外部パートナー（社外）という立場ゆえに、情報が入ってこなくて『作業が止まる』あるいは『手戻りになる』ことはどのくらい頻繁にありますか？」** | **[情報の非対称性]**<br>・コミュニケーションフローの欠陥を特定。<br>・Slackチャンネルの設計やMTG参加の必要性を判断する。 |

### Part 3: 【Style】B2B対応とスキルセット

**狙い:** 今後の法人研修（B2B）展開 に耐えうる事務基盤があるか、神保様のスキルセットを検証する。

| No. | 質問項目 (Question) | 深掘り・意図 (Intent & Deep Dive) |
| :--- | :--- | :--- |
| **Q7** | **「今後、法人顧客（B2B）が増えて請求書払いや契約書締結が急増した場合、今の事務フローで『どこがパンクする』と予想しますか？」** | **[スケーラビリティの検証]**<br>・「人力でなんとかなる」と思っているか、「システム化が必要」と見ているか。<br>・B2B紹介 の実務担当としてのリスク管理能力。 |
| **Q8** | **「神保さんが、他の事務担当者よりも圧倒的に『得意』で、やっていて『苦にならない』作業は何ですか？」** | **[不当な優位性 (Unfair Advantage)]**<br>・「カオスな領収書の整理」なのか「未入金者への優しい督促（温度ある対応）」なのか。<br>・Shapeにフィットする「人間味のあるバックオフィス」の要素を特定。 |
| **Q9** | **「鈴木（私）からの依頼や指示の出し方として、どういう形式が一番動きやすいですか？」**<br>（A: ゴールだけ示してあとはお任せ、B: 手順書通りに遂行、C: 背景と想いを共有してから着手） | **[マネジメントの取扱説明書]**<br>・「思考プロセスごと渡す」 ことができるタイプ（A/C）か、タスクワーカー（B）かを見極める。 |
| **Q10** | **「もし、香音さんたちの『右腕』としてもっと踏み込むために、一つだけ『武器（ツールや権限）』をもらえるとしたら、何をリクエストしますか？」** | **[次なる一手の要望]**<br>・具体的なツール（Salesforce権限など）が出るか、抽象的な要望が出るか。 |

## 4. インタビュアー（鈴木）へのメモ

  * **スタンスの取り方**:
      * 鈴木は**「交通整理係」**として接してください。「あなたがもっと活躍できるように、リーダーたちが握りしめているボールを、私がほぐしてパスします」という姿勢が、神保様の主体性を引き出します。
  * **確認すべき「表情」**:
      * **Q4（業務の巻き取り）**: ここで「食い気味に」提案が出てくる場合、神保様は「埋もれている即戦力」です。逆に「指示があれば…」と消極的な場合、**「委譲パッケージ（マニュアル）」** をかなり詳細に作り込む必要があります。
  * **深掘りのコツ**:
      * 「温度を伝える」という抽象的な表現が出たら、**「例えば、請求書を送るメールの一文で、どんな工夫をしていますか？」**と、具体的な行動レベルまで解像度を上げて聞いてください。それが業務マニュアルの「Tone & Manner」になります。
