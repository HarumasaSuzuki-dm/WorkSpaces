## 1. テスト報酬の金額について

クライアント様は1,000円〜2,000円の間で悩まれていますが、**「1,500円〜2,000円」**での設定を推奨します。

* **1,000円の場合：**
「フルテロップ」の作業が含まれるため、1,000円だと「時給換算で安すぎる」と判断され、優秀な層（実績のあるプロ）が応募を敬遠するリスクがあります 。

* **1,500円〜2,000円を推奨する理由：**
選考フローにある「3〜5分の完成動画」は、素材の確認から書き出しまで2〜3時間は要する作業です。この金額帯であれば「技術を正当に評価しようとしている」という誠実さが伝わり、質の高い編集者が集まりやすくなります 。


* **フィルター機能としての効果：**
「無償」ではなく「有償」であること自体が、冷やかしを防ぐ強力なフィルターとして機能するため、金額の差（500円程度）よりも「報酬がある」という事実が重要です 。

### 1. 採用選考コスト（単発）

募集から採用決定までにかかる費用のシミュレーションです。

| 項目                       | 内容                               | 概算金額                |
| -------------------------- | ---------------------------------- | ----------------------- |
| **応募数（予測）**         | フォロワーの0.05〜0.1%が応募と仮定 | 350人 〜 700人          |
| **一次選考（書類）**       | フォームによる自動絞り込み         | 0円（工数のみ）         |
| **二次選考（テスト報酬）** | 上位5名にAmazonギフト券で支払い    | 10,000円（2,000円×5名） |
| **面談・最終決定**         | 上位2名とオンライン面談            | 0円（工数のみ）         |
| **合計**                   |                                    | **約 10,000円**         |

* **ポイント：** 70万人規模だと応募が殺到するため、選考フローにある「フォームでの足切り」を厳格に行い、実際にテスト報酬を支払う対象を**5名程度**に限定することで、選考コストを1万円前後に抑えることが可能です。

### 2. 月間運用コスト（ランニング）

採用した1名に、週1回（月4本）のペースで編集を依頼した場合のシミュレーションです。
※報酬単価は「募集要項.md」の規定に基づきます。

#### パターンA：標準的なVlog（20分前後）の場合

* 単価：20,000円 / 本（15分〜30分未満）
* 月額：20,000円 × 4本 ＝ **80,000円 / 月**

#### パターンB：長尺・企画回（40分前後）の場合

* 単価：25,000円 / 本（30分〜45分未満）
* 月額：25,000円 × 4本 ＝ **100,000円 / 月**

## 2. 支払い方法（工数削減）について

クライアント様が提案された**「Amazonギフトカード（Eメールタイプ）」**は、現時点での**ベストな選択肢**です。

* **メリット：**
* 振込先情報のヒアリングや振込手数料、経理への振込依頼といった事務作業（工数）が一切発生しません 。


* メールアドレスさえ分かれば即時送付でき、履歴もAmazon上で一括管理できます。


* **鈴木からの補足提案：**
もしAmazon以外を検討されるなら「PayPay」や「Kyash」もありますが、ビジネス用途での受取拒否感が少なく、最も汎用性が高いのはやはりAmazonギフトカードです。

## 3. トライアル条件（第1条〜第6条）の提示タイミング

音声の中で「事前に提示する必要があるか」と確認があった点への回答案です 。

* **募集要項（一般公開）への記載：**
募集要項には現在の「報酬条件（減額なし）」や「期間（3ヶ月）」といった**概要のみ**を載せておけば十分です。
* **詳細（第1条〜第6条）の提示：**
音声で言及されていた細かい契約条項は、**「採用決定後（本契約前）」のタイミング**で、契約合意の最終確認として提示するのがスムーズです 。


* **理由：** 応募段階で長文の契約条項を見せすぎると、応募のハードル（心理的負荷）が高くなってしまうためです。
