# 法人向けパフォーマンス・マネジメント研修 概要書（仮）

## 1. サービスコンセプト

**「個人の『行動変容』を、組織の『文化変容』へ。」**

単に社員の運動不足を解消する福利厚生ではなく、科学的なアプローチ（GAP分析・コーチング）を用いて個人のパフォーマンスを最大化し、最終的に組織全体へ「自律的な健康管理の文化」を定着させることを目的とする。
特に、**認知科学コーチング（Cognitive Science Coaching）**の手法を取り入れ、単なる肉体改造ではなく、マインドセットの変革を促す。

## 2. ターゲットと目的

本プログラムは、受講対象者の役割に応じて**2つの目的（レイヤー）**を設定する。

| 対象層 | プログラムの目的（ゴール） |
| :--- | :--- |
| **① 経営者・役員層**<br>（スポンサー） | **「導入判断と説明力の獲得」**<br>自ら行動変容を体感し、そのロジック（なぜ変われたか）を理解することで、全社導入への**意思決定**と社内への**説明**ができる状態を目指す。<br>**目標:** 2〜3ヶ月のトライアルで効果を実感し、2026年3月頃の全社導入判断に繋げる。 |
| **② 一般社員・管理職**<br>（プレイヤー） | **「パフォーマンス最大化と自走」**<br>心身のコンディションを整え、業務における成果を最大化する。また、研修終了後も自律的に継続できる（自走する）状態を目指す。 |

## 3. 提供価値とアプローチ（Phase設計）

本研修は、「やって終わり」の一過性のイベントではなく、以下のフェーズを経て組織への定着を図る。

### Phase 1：自己変容（Self-Transformation） - トライアル期

*   **期間:** 2〜3ヶ月（経営者・リーダー層先行実施）
*   **概要:** 徹底した伴走による成功体験の創出。ボートレーニングや対面セッションを活用。
*   **内容:**
    *   現状の課題分析（GAP分析）と目標設定。
    *   バイタルデータや日報を活用した行動管理。
    *   **認知科学コーチング**に基づく対話セッション（週次/隔週）。
*   **成果:** 「気合い」ではなく「正しいアプローチ」によって身体と精神が変わることを実証する。

### Phase 2：組織展開（Organizational Deployment） - 導入期

*   **ターゲット:** 経営者による決裁後の全社/部署展開（2026年度〜想定）
*   **概要:** 成功体験の「種明かし」と「武器」の提供。
*   **内容:**
    *   **ロジックの言語化:** 「なぜ自分は変われたのか？（＝外部の強制力と仕組みがあったから）」を論理的に理解する振り返り。
    *   **意思決定の支援:** 精神論（「お前らも頑張れ」）ではなく、仕組み（「外部リソースを入れる」）として組織課題を解決するための判断軸を提供する。
*   **成果:** 経営者が「全社導入」の決裁を行い、社員に対してその必要性を自分の言葉で語れるようになる。

## 4. 運用体制のイメージ

属人性を排除しつつ、社内の専門リソースを最適配置する。

*   **リード（なーすけ CEO）:**
    *   経営者層へのPhase 1コーチング（ボートレーニング等のアクティビティ含む）。
    *   Phase 2における「導入の意思決定」のクロージング支援。
    *   全体のプログラム監修。
*   **認定スタッフ/トレーナー（香織様 等）:**
    *   **役割シフト:** 事務領域から、本来の強みである**「コーチング/メンタリング」**領域へリソースを重点配分する。
    *   一般社員向けのPhase 1実務（セッション、チャット対応）。
    *   運用マニュアルに基づく進行管理。
*   **IT/ツール:**
    *   動画教材（トレーニング方法、マインドセット講義）の活用。
    *   進捗管理ツールの導入により、対面工数を最適化。

## 5. 従来の研修との違い（差別化要素）

| 項目 | 一般的な企業フィットネス | **本プログラム（Shape for Biz）** |
| :--- | :--- | :--- |
| **ゴール** | 運動不足の解消・リフレッシュ | **「仕事の成果」が出る状態・自走** |
| **手法** | 一斉指導・イベント型 | **個別GAP分析・認知科学コーチング型** |
| **経営者の関与** | 導入して終わり（丸投げ） | **経営者自身がロジックを理解し、旗振り役になる** |
| **再現性** | 個人のモチベーション頼み | **「仕組み」として定着させる** |

## 6. 現状の進捗とネクストアクション

現在、PoC（概念実証）フェーズとして以下の活動が進行中。

*   **経営者アプローチ:** 3〜4名の経営者に対し、アポイントメントを取得済み。ボートレーニングやカフェでの対話を通じ、コーチング要素を含むトライアルを実施中。
*   **フィードバック収集:** 実際の対話から得られたニーズ（身体の悩みと経営課題のリンク）を収集し、プログラム詳細へ反映する。
*   **当面の目標:** トライアル実施中の経営者に対し、2026年3月を目処に「自社への導入」を意思決定してもらうための成果（身体的・精神的変化）を出すこと。
