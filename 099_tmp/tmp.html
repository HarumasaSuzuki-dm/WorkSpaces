FC店舗育成 ── フェーズ・ステップ遷移フロー

Mustの遵守状況に応じて、店舗ごとに異なるルートを進む全体像

フェーズA：Mustを守る（2〜3月）
なーすけさんが全店舗と週次30分の1on1で伴走。4つのMustの定着に集中する期間
フェーズA ─ 全店舗共通
4つのMustを「できている」状態にする
セッションログ完全提出 ／ 24時間以内レスポンス ／ NPS実施 ／ オーナー現場対話
毎週の1on1で「なぜできなかったか」を確認し、やり方を柔軟に調整する
担当：なーすけさん 頻度：週次1on1（30分） 記録：MUST運用シート
毎週判定
4つのMust
全て達成？
✓ 達成 ✗ 未達
守れている場合のルート
移行条件
4つのMust全てが「できている」状態が3週間連続で続いたらフェーズBへ
「できている」の定義：
・セッションログ → その週に1件以上提出
・24hレスポンス → 重要連絡に返事漏れなし（スタンプOK）
・NPS → その月の分が完了
・オーナー対話 → 記録が1on1前日までに記入済み
判定者：1on1担当者（今はなーすけさん → 4月以降ななこちゃんに移行）
フェーズB 目安：4〜5月
Mustをやって出てきた困りごとを一緒に解く
ゴール：オーナーが「うちの店にとってMustをやる意味」を自分の言葉で話せる状態
やること：
・現場の「困った」を具体的にヒアリング
・Mustの「なぜやるか」をオーナー自身が体感で理解
・ななこちゃんが1on1に同席開始
・オーナー同士の情報交換の場をお試し実施
店舗別テーマ例：本店＝フーテロちゃんの育成、柏＝離脱防止、大阪＝トレーナー差の見える化、上板橋＝Instagram運用
フェーズC 目安：6〜7月
「あなたの店舗、これからどうしたい？」を話す
ゴール：オーナーが自分たちで育成サイクルを回せる状態
やること：
・オーナー自身がやりたいこと・目指す姿を言語化
・セッション品質の基準をガイドにまとめる
・Shapeの考え方の伝達ステップを文書化
・Mustが安定している店舗から順にこのテーマへ移行
本部ビジョンをオーナーの声に重ねて共有（全体MTGで実施）
フェーズD 目安：8月〜
オーナーが自分で立てた目標を追う
ゴール：ななこちゃんが1on1主担当。なーすけさんは月1で全体確認のみ
やること：
・トレーナーがShapeの考え方を体で表現できるレベルに
・「お客様の挑戦の物語」を自分の言葉で語れる状態
・各店舗の育成の成果を振り返り、来年に活かす
1on1主担当：ななこちゃん ／ なーすけさん：月1全体確認
守れていない場合のルート
まず原因を3つに分ける（1on1で毎回メモ）
1 やり方が悪い：フォームが使いにくい、締切が非現実的 → 本部が仕組みを直す → ステップ3に進まない
2 手が足りない：トレーナー不足、シフトの問題 → 一緒に負荷を減らす → ステップ3に進まない
3 やる気・優先度の問題：仕組みも人も整っているのに後回し → 連続カウント開始 → ステップ3へ
※ やり方を直してから2週間たっても変わらなければ「やる気の問題」として判断し直す
ステップ1：事実の確認
「先週のログ見ると、ここが未提出ですね」
いつ：今週できていないMustが1つでもある時
何を：1on1でなぜできなかったか確認してメモに記録。責めない
担当：1on1の担当者
✓ 改善すればフェーズAの通常運用に戻る
ステップ2：サポート
原因に合わせて手を打つ
いつ：原因が「やり方」か「手が足りない」とわかった時
何を：やり方の調整・負担軽減の工夫をする
担当：1on1の担当者
✓ 改善すれば通常運用に戻る（連続カウントに入れない）
ステップ3：期限を決める
「いつまでにやるか」を具体的に決める
いつ：同じMustが3週続けてできていない時
何を：次の火曜日までの期限を設定 → チャットワークでも文面送付（記録）
担当：なーすけさん
運用ルール：
・伝え方：1on1で直接 ＋ チャットワークで記録
・やった連絡がない場合 → 木曜にリマインド → 金曜まで連絡なければ次の1on1で確認
・期限内にやれた場合 → 通常に戻す（連続カウントもリセット）
・期限内にやれなかった場合 → 翌週もう一度期限設定（2回目）→ 2回ともダメならステップ4へ
ステップ4：条件の見直し
FC契約の条件について話し合う
いつ：ステップ3を2回やっても改善しない時
何を：段階的に対応を検討
3段階で考える：
① まず：Mustの中身を変えられないか
② 次に：本部のサポート体制を変える（例：事務局がログ入力を一時代行）
③ 最後に：ロイヤリティ条件や契約継続について話す
最終判断：なーすけさん ／ 事務局は事実整理と選択肢の準備

オーナーへの開示タイミング

2月（テスト期間）：ステップ1〜2だけ伝える。「1on1で状況を一緒に確認して、困っていることがあればサポートします」
3月末 全体MTG：フロー全体を見せる。「テスト期間でわかったのは、原因のほとんどはやり方の問題でした。4月からはやり方を整えた上で見ていきます」
4月以降：個別に「今ステップ○です」とは言わない。ステップ3に入る時は「期限を決めましょう」と自然に伝わる形
判定のタイミング：毎週の1on1終了後、MUST運用シートを更新した時点で判定
記録方法：ホームシートに各店舗の今のフェーズ（A / B / C / D）を1行で管理
3週間の理由：2週間では習慣とは言えない。4週間では達成済み店舗のやる気が下がる。3週間がバランス最適
