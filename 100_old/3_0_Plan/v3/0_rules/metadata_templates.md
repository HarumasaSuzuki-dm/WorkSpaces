# Metadata Templates (Evidence-Ready)

## YouTube Shorts

### 1. タイトル（60文字以内｜数値＋行動＋効果）
- 例
  - Day9｜朝1分の呼吸で“起動”が楽に（研究より）
  - Day10｜寝る前5分“スマホ裏返し”で入眠がラク
  - Day15｜“3つだけ”ゼロタスクで罪悪感ゼロ就寝

### 2. 説明文（E→A→Rを1段落で）
SLEという背景を持つ僕が、研究の示唆（E）を“その場でできる行動”（A）に翻訳して、明日の自分を少し好きになる一歩（R）を共有します。  
※医療助言ではありません。効果には個人差があります。

▼Instagram（保存用テンプレ多め）
https://instagram.com/your_account

### 3. 固定コメント（出典＋A/Bフォーム）
【出典まとめ】  
1) {TITLE_1}（{AUTHORS_1}, {YEAR_1}）{LINK_1}  
2) {TITLE_2}（{AUTHORS_2}, {YEAR_2}）{LINK_2}  
※個人差あり。医療判断は主治医へ。  
【今日の一手】A: {OPT_A} / B: {OPT_B} → コメントで『A/B』だけでもOK

### 4. ハッシュタグ（文脈＋発見性＋短尺基礎）
