### ステップ1：動画の「目的（ゴール）」を明確にする

**「視聴者に、この動画を見終わった後、どうなってほしいのか？」**

まず自問すべき最も重要な質問はこれです。ストーリー構成は、この目的を達成するための手段でしかありません。

* **目的が「教える・スキルを習得させる」なら？**
    * 視聴者に具体的な手順を学んでほしい。
    * **→ 最適な候補:** `チュートリアル`, `ブレイクダウン`

* **目的が「説得する・納得させる」なら？**
    * 視聴者の悩みを解決し、あなたの提案を信じてほしい。
    * **→ 最適な候補:** `問題解決`, `ケーススタディ説明`

* **目的が「楽しませる・情報を整理して伝える」なら？**
    * 手軽に知識を得て、満足してほしい。
    * **→ 最適な候補:** `リスト`, `ニュースキャスター`

* **目的が「共感させる・ファンになってもらう」なら？**
    * あなたの人間性や価値観に触れて、親近感を持ってほしい。
    * **→ 最適な候補:** `個人的なストーリー`

**最初に「動画のゴール」を決めることで、選択肢は自然と2〜3個に絞られます。**

---

### ステップ2：「コンテンツの核」は何かを分析する

次に、あなたが動画で伝えたい**「最も面白い事実（前回のステップで発見したもの）」**の性質を分析します。コンテンツの性質が、最適な器（ストーリー構成）を教えてくれます。

* **核となる情報が「時系列のプロセス」なら？**
    * 例：「ある出来事が起こり、次に行動し、結果こうなった」
    * **→ 最適な候補:** `ケーススタディ説明`, `個人的なストーリー`, `チュートリアル`

* **核となる情報が「並列の知識・ポイント」なら？**
    * 例：「知っておくべきポイントが複数ある」「おすすめがいくつかある」
    * **→ 最適な候補:** `リスト`, `ブレイクダウン`

* **核となる情報が「因果関係」なら？**
    * 例：「〇〇という問題は、実は△△が原因で起こっている」
    * **→ 最適な候補:** `問題解決`, `ニュースキャスター`

**あなたの伝えたい「情報の形」と、ストーリー構成の「器の形」が一致していると、視聴者はストレスなく内容を理解できます。**

### ステップ3：あなたのジャンル（ニッチ）の成功法則を盗む

最後に、動画で最も強調されていた**「市場リサーチ」**を行います。あなたの頭の中だけで考えるのではなく、既に成功している動画から答えを見つけ出すのです。

1.  **YouTubeで自分の動画テーマを検索する。**
2.  **「フィルタ」機能で「視聴回数」順に並び替える。**
3.  **上位3〜5本の動画を「分析官」の目線で視聴する。**
    * 「この動画は、なぜこんなに再生されているんだろう？」
    * 「導入はどのフックパターンを使っているか？」
    * **「そして、全体は7つのうち、どのストーリー構成に沿って作られているか？」**

**例:**
あなたが「Excelの便利技」に関する動画を作りたいとします。検索して上位の動画を見ると、その多くが`リスト`形式（「知らないと損する便利技10選」）か、`問題解決`形式（「VLOOKUPでよくあるエラーを解決する方法」）で作られていることに気づくでしょう。

これは、そのジャンルの視聴者が**その形式で情報を受け取ることに慣れており、それを好んでいる**という強力な証拠です。最初は、この成功法則に従うのが最も確実な方法です。

### 【早見表】最適なストーリー構成の選び方

| 動画の目的・内容 | 最適なストーリー構成の候補 |
| :--- | :--- |
| **スキルや手順を教えたい** | `チュートリアル`, `ブレイクダウン` |
| **視聴者の悩みを解決したい** | `問題解決`, `ケーススタディ説明` |
| **複数の知識やコツを共有したい** | `リスト`, `ブレイクダウン` |
| **最新のニュースやトレンドを解説したい** | `ニュースキャスター`, `ブレイクダウン` |
| **商品やサービスを紹介・レビューしたい** | `ケーススタディ説明`, `リスト`, `問題解決` |
| **自分や誰かの体験談を伝えたい** | `個人的なストーリー`, `ケーススタディ説明` |
| **ファンになってもらい、信頼されたい** | `個人的なストーリー`, `問題解決` |

**結論として、最適な方法とは「目的を定め、内容を分析し、最後は市場の成功例に学ぶ」というプロセスそのものです。** これを繰り返すことで、あなたはどんなテーマでも自然に最適なストーリー構成を選べるようになります。
