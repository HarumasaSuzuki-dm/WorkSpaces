# 調査範囲と優先度
1. **一次研究・査読論文（Google Scholar / PubMed / PsycINFO）**
2. **体系的レビュー・メタ分析**
3. **公的研究機関・大学の発表（WHO, NIH, 日本の大学など）**

# 出力フォーマット
- **引用情報**（著者名, 年, 論文タイトル, ジャーナル名）
- **研究タイプ**（実験研究 / メタ分析 / レビュー）
- **主要な発見（1〜2文で要約）**
- **実生活での示唆**（どのように日常に活かせるかを1文で）

# 制約条件
- 医療助言は禁止。「個人差がある」「環境による傾向」として解釈する。
- 研究が古すぎる場合（20年以上前）は、できる限り近年のものを優先する。
- 出典リンク（DOIやPubMed ID）がある場合は必ず提示する。
