# **1. 修正後コアコンセプト**

> **「小さな科学で、自分を軽やかにアップデートする」**
>
> 疲労やモチベーション不足に悩む日常を、
> **“科学的マイクロアクション”×“等身大の挑戦”**で変えていく。
>
> まずは一緒に**“自分の目標”を探す旅**から始め、
> 見つかった目標に向かって**短期・中期・長期の物語**を紡いでいく。
>
> 長期ではフィジーク大会への挑戦を見据え、
> その過程をリアルタイムで共有する。

---

# **2. ストーリーアーク（段階的展開）**

| フェーズ              | 期間      | 主なテーマ          | 視聴者への体験                                         |
| ----------------- | ------- | -------------- | ----------------------------------------------- |
| **Phase 1：探索期**   | 0〜30日   | 「目標を見つける」      | 視聴者と一緒に「今の自分を知り、何を目指すか」を動画で探る。視聴者自身もコメントや保存で参加。 |
| **Phase 2：挑戦設計期** | 30〜60日  | 「100日チャレンジの設計」 | 見つかった中期目標を分解し、100日で達成できる形に落とし込む過程を共有。           |
| **Phase 3：挑戦期**   | 60〜160日 | 「100日チャレンジ実践」  | 進捗をリアルタイムで共有しながら一緒に挑戦。挫折や再起動も含めて物語化。            |
| **Phase 4：長期物語**  | 160日〜   | 「フィジーク大会」      | 長期目標としての大会に向け、ストーリーを積み上げ続ける。                    |

> Phase 1〜3は**中期的ゴール探索と達成**、Phase 4で**フィジークという長期ゴール**を自然に接続。

# **4. ファンが惹かれる理由（アップデート版）**

## ① **「自分ごと化できる目標探索」**

* 視聴者も「自分は何を目指したいか」を一緒に考える仕掛け。
* コメント参加型コンテンツで、共創感を強化。

💡心理効果：**自己決定理論** → 自分で決めた目標は行動維持につながる

---

## ② **「長期×短期の二重構造」**

* フィジーク大会という“遠い憧れ”と、
  100日でできる“身近なゴール”を両立。
* 視聴者は **夢物語の観客** としても、**仲間として実践**もできる。

---

## ③ **「挑戦のプロセスを可視化」**

* 目標がまだ曖昧な段階もコンテンツ化。

  * 例：「今日は目標を10分間書き出してみる」
* 完成された姿ではなく**“ぐちゃぐちゃな過程”**をリアルに見せることで、共感と保存を両立。

# **まとめ**

* 最初から100日チャレンジに固定せず、**「目標を探すプロセス」**をストーリー化。
* 中期（100日チャレンジ）と長期（大会）の**二層構造**で成長物語を描く。
* 視聴者が単なる観客ではなく**一緒に考え、一緒に進む仲間**として関われる設計にすることで、初期段階から熱量の高いファンを育てる。
